## 情境

HK 幼稚園面試季節（9-12 月），對 2-3 歲細路同家長嚟講殘酷得唔合理。

細路 K1 面試要對住陌生老師、做活動、答問題。家長口試 10-15 分鐘，校長 / 副校長坐你對面，問你嘅育兒理念、屋企習慣、點解揀呢間學校。

坊間中介開 8000-15000 蚊「面試 prep 班」，內容係：

- 細路死記答案：「我鍾意紅色，因為朱古力顏色」
- 家長背誦：「我哋重視小朋友嘅全人發展」
- 演練微笑、坐姿、握手

問題：校長一聽就知係夾過稿。問深一層就穿崩。**最重要嘅嘢——細路嘅真實性格、家長嘅真實教養方式——反而冇練到。**

呢篇教你用 Claude 做更有用嘅準備：細路練表達多過記答案、家長練講清楚自己嘅理念、做針對學校嘅資料搜集對應點演繹。

> 提提你：呢個唔保證入到。幼稚園收生有好多因素（兄姐、宗教、地區、運氣）係準備改變唔到。Claude 嘅角色係幫你個細路同你自己**真實咁呈現出嚟**，唔係扮另一個人。

## 跟住做

### 1. 列你嘅心水幼稚園 + 細路真實性格

開始之前寫低：

- **3-5 間心水幼稚園**，分夢想 / 實際 / 保底三層
- **細路真實檔案**：唔好寫「乖、聰明、聽話」呢類行貨。寫具體：「鍾意砌嘢，可以坐定半個鐘；陌生人面前會收埋；情緒大會喊但 5 分鐘平靜得返」
- **你屋企真實教養方式**：「我哋少睇 iPad，多去公園；返工時間長，主要週末親子；爸爸負責講故事時間」

呢啲原始材料之後喺 prompt 入面貼畀 Claude。**寫得越真實、越有具體細節，Claude 幫到你越多。**

### 2. PromptBlock — 揀啱問題類型畀細路練

細路 3 歲練「答問題」嘅目標 **唔係答得啱**，係練到佢面對陌生大人**夠膽出聲、表達意思**。

<PromptBlock label="完整 prompt — 細路面試表達力練習">
我細路 [年齡，例如：2 歲 10 個月] 仔／女，下星期面試 [幼稚園名]，K1。

佢嘅真實性格：

- [例：陌生人面前會收埋，要 5-10 分鐘先熱身得到]
- [例：鍾意講嘢，但會跳話題]
- [例：嬲嘅時候會僵住唔出聲]
- [例：對顏色 / 動物特別有興趣]

我想喺屋企同佢練，但唔想佢「死記答案」。我想培養佢嘅表達力、自信、自然反應。

請幫我整：

**Part 1：10 條適合佢年齡嘅開放式問題**

唔係「你叫咩名」呢類封閉式問題。要：

- 邀請佢講多兩句（「你最鍾意嗰隻動物係邊隻？點解？」）
- 對應佢個人興趣（顏色 / 動物 / 砌嘢）
- 包括情緒問題（「你開心嘅時候會點？」）—— 校長鍾意問
- 唔需要標準答案，鼓勵佢自己諗

**Part 2：5 個任務式情境**

幼稚園面試常考任務：執玩具、分顏色、跟指示、閱讀時間、自由玩。每個任務：

- 點喺屋企模擬
- 觀察重點（唔係佢做唔做到，係佢點樣處理）
- 如果佢拒絕，點反應（唔好硬迫，但教佢嘅應對方式）

**Part 3：練習嘅基本規則**

- 一日唔好練超過 [X 分鐘]
- 唔好罰、唔好對比兄姐 / 同學
- 練習應該玩得似遊戲，唔係操練
- 邊啲訊號顯示細路已經有壓力，要停

語氣：實際、有兒童發展觀念、唔好出怪獸家長嘅味道。
如果你覺得我細路嘅檔案唔適合 [幼稚園名]（例如重學術嘅學校但細路係愛表達型），老實講。
</PromptBlock>

關鍵：練完之後**畀細路自由發揮**。佢面試嗰日答「我鍾意紫色因為咁樣好靚」，比起夾過稿嘅「我鍾意紅色因為朱古力顏色」勁好多。**真實感係 3 歲細路無敵嘅優勢。**

### 3. 家長口試準備 — Claude 幫你講清楚自己嘅育兒理念

家長口試最論盡嘅答案：「我哋好重視小朋友嘅全人發展。」

校長聽過 500 次。

真正打動校長嘅，係**你有諗過、講得出具體例子、同你屋企做緊嘅嘢一致**嘅答案。

<PromptBlock label="家長口試講法準備">
我下星期參加 [幼稚園名] 嘅家長口試（10-15 分鐘）。我同我老婆 / 老公兩個都出席。

我哋嘅真實教養背景：

- 工作：[例：兩個都全職，朝 9 晚 7]
- 屋企日常：[例：婆婆平日主力照顧，週末我哋親子]
- 嗰啲我哋著緊嘅：[例：細路情緒調節、閱讀習慣、戶外時間]
- 嗰啲我哋唔太強嘅：[例：第二語言、有系統嘅學習]

校長預期會問：

1. 點解揀我哋呢間學校
2. 你哋嘅育兒理念
3. 小朋友遇到困難嘅時候你哋點處理
4. 屋企典型週末點過
5. 你期望小朋友 K3 畢業時係點

請幫我：

**Part 1：每條問題嘅講法框架**

唔係背稿，係框架：

- 主軸（你想帶出嘅 1 個核心訊息）
- 1-2 個具體嘅真實例子（例：「上個禮拜佢喺公園搶玩具，我哋點處理」唔係「我哋會引導佢」）
- 避免嘅陳腔濫調（例：「全人發展」、「全人教育」）

**Part 2：要避開嘅警號**

- 滿口潮語但講唔出實例
- 同另一半答得唔一致（要事先對好口徑）
- 過份吹噓 / 過份謙虛
- 講「我哋好谷佢，因為要準備小一」（呢間學校如果走開放路線會即扣分）

**Part 3：我同另一半要事先夾好嘅 3 個問題**

口試入面校長問 「邊個主力處理 [情境]」，兩個答唔同最尷尬。列我哋應該事前傾好嘅 3 個敏感話題。

語氣：實際、真實感，唔好幫我堆砌官腔。
</PromptBlock>

呢個準備嘅結果唔係背稿。係**你諗清楚自己屋企真實係點，講嗰陣自然有具體嘅細節帶出嚟**。

### 4. 面試完傾傾 — 同細路傾傾嘅框架

面試出嚟個細路通常累、可能喊、可能話「冇嘢呀」。家長心急想知「點呀？答得好唔好？」。直接問「老師問你乜呀？答咗咩？」基本攞唔到有用資料。

<PromptBlock label="面試完同細路傾傾嘅框架">
我細路啱啱面試完 [幼稚園名]，K1。佢 [情緒：累 / 喊咗 / 興奮 / 靜咗]。

我想：

- 知佢入面發生咗咩（佢嘅角度）
- 唔畀佢覺得自己被 「考核」
- 為下一間面試學到啲嘢

請畀我：

1. **唔好問嘅問題**（例：「你答啱咗冇？」「老師鍾唔鍾意你？」）
2. **應該問嘅開放式問題 5 條**，由細路角度，例如「老師個房有冇得意嘅嘢？」
3. **點處理佢嘅情緒**（如果佢沮喪或者覺得自己「衰咗」）
4. **觀察重點**：佢自然講出嚟嘅細節邊啲值得記低（之後準備第二間用）
5. 我自己焦慮嘅處理 —— 我點忍住唔問 「你答 X 問題嗰陣答咗咩」

語氣：以細路身心健康為先，唔係考評心態。
</PromptBlock>

呢一步一般家長唔做，但**對細路長遠嘅面試體驗同信心極關鍵**。一連幾間面試完都覺得安全，先有可能發揮到佢真正嘅水平。

## 變化例子

### 變化 1：K1 vs K2 vs 小一面試版

<PromptBlock label="變化 1 — 唔同年級嘅 prep">
我細路而家 [年齡]，面試 [K1 / K2 / 小一]。

唔同年級準備嘅重點應該唔同：

1. K1（2.5-3 歲）：應該著重邊 3 個能力？（提示：唔係學術）
2. K2（3.5-4 歲）：開始要求咩認知能力？
3. 小一（5.5-6 歲）：學術加重幾多？

請對應 [我細路年齡 + 目標年級]，列：

- 呢個年級面試評核者真正觀察緊邊 5 樣嘢
- 屋企可以做嘅準備同要避免嘅過度準備
- 邊啲 「補習班教嘅嘢」喺呢個年級反而扣分
</PromptBlock>

### 變化 2：國際學校（入學評估 + 家長文章）

<PromptBlock label="變化 2 — 國際學校入學">
我考慮 [國際學校名]，K1 收生，有：

- 細路入學評估（30 分鐘遊戲為本 + 觀察）
- 家長書面文章（300-500 字 English）
- 家長面談

我細路檔案：[填]
我屋企背景：[例：本地家庭 / 混血 / 外籍 / 回流]

請：

1. 國際學校同本地學校嘅評核標準點唔同（具體方面）
2. 家長文章 5 條典型題目同點演繹（要避開嘅陳腔濫調、想呈現嘅特質）
3. 細路遊戲為本評估點準備（提示：唔係操練 ABC）
4. 面談入面國際學校校長特別會問嘅問題類型
5. 如果我哋係本地家庭申請國際學校，點演繹 「合適」 嘅說法
</PromptBlock>

### 變化 3：SEN 細路嘅披露同演繹

<PromptBlock label="變化 3 — SEN 細路嘅準備">
我細路 [年齡]，有 [SEN 類型，例如：語言發展遲緩 / ASD 傾向 / ADHD 特徵 / 暫未確診但有疑慮]。

我哋有 [評估報告 / OT 報告 / 言語治療師筆記]。面試 [幼稚園名]。

我內心矛盾：

- 披露怕學校篩走
- 唔披露怕入到先發現唔夾，反而辛苦小朋友

請：

1. 唔同披露策略嘅取捨（完全披露 / 部分披露 / 觀望）
2. 點評估一間幼稚園係咪真係對 SEN 友善（唔係佢網站寫 「共融」就算）
3. 如果決定披露，點演繹先唔會聽落似推卸（著重小朋友嘅強項 + 需要嘅支援）
4. 面試嘅問題我可以反問校方，試佢哋對 SEN 嘅實際準備
5. 如果今間唔啱，可以諗嘅其他出路（特殊需要友善學校 / 主流 + 治療 / 推遲 K1）

語氣：以小朋友長期身心健康為先，唔係 「點樣入到主流名校」嘅心態。
</PromptBlock>

## 一個角度

幼稚園面試呢件事容易令家長變形。8000 蚊補習班、WhatsApp group 嘅 「攻略」、playgroup 入面嘅比較——所有誘因都係推你過度準備。

過度準備嘅代價唔係銀紙。係細路 3 歲已經內化咗 「我嘅價值由表現決定」呢個訊息。

> 校長見過 1000 個夾過稿嘅細路。佢真正想揾嘅，係一個喺呢個年齡**真實咁呈現自己**嘅細路，同一對**清楚知道自己想點教仔**嘅家長。

Claude 喺呢度嘅優勢：**唔係幫你講得更厲害，係幫你想清楚自己屋企真實係點、再講出嚟**。當你嘅答案有具體嘅真實細節、前後一致、配合你細路嘅真實性格——校長一聽就分得出。

最後提一句：入到邊間幼稚園唔決定你細路 15 年後嘅人生。**但你而家畀佢嘅準備體驗，決定佢點睇自己同被評核之間嘅關係。** 揀啱個準備方式，比揀啱間學校更長遠。

## 情境

你終於買咗 Final Fantasy 14 日服 sub，興奮咁開新角，jump 入新手 city。第一個 NPC 行埋嚟講：

> 「冒険者よ、ここは光の柱が空に伸びる地、グリダニア。エオルゼアの森に祝福を受けし者として、汝に最初の試練を授けよう。」

⋯⋯吓？

你開 Google Translate → 機械翻譯，意思 OK 但**完全冇 game lore context**——「光の柱」係咩？「エオルゼア」係邊？「最初の試練」係主線定 side quest？

或者你玩 Persona 5 Royal 嘅 JP 版，遇到一段 confidant dialogue：

> 「君のことを、もっと知りたいんだ。」

機翻：「我想更多了解你。」

但你唔知個角色當下嘅感情狀態、佢哋之前發生過咩事、應該答邊個 option 先 unlock 下個 social link tier。

呢個就係**JP-only 或者「中文 patch 仲未出」嘅 RPG 玩家**經典 friction：text wall 唔係翻譯問題，係 **context 缺失**問題。

呢篇用 Claude：

- ⏱ 5 秒翻譯一段 dialogue
- 🎯 連帶解 quest 背景、角色關係、揀對話 option 嘅 implication
- 📚 你 build 緊 lore knowledge，唔係 dump 答案

## 跟住做

### 1. 影低個 screen

開 Claude mobile app（或 desktop browser），新開一個 chat。

**iPhone**：撳 screenshot（電源 + 音量上）→ Photos 入面揀張相 → share to Claude → 加 prompt。

**Android**：類似 — Power + Volume Down → share to Claude。

**PC / Console**：用 Steam screenshot (F12) / PS5 share button → upload 去 Claude desktop。

⚠️ 直接打字輸入日文都得，但**影 screen 連帶 character portrait + UI context** 嘅效果好過好多——Claude 睇到邊個 NPC 講嘢、有冇 quest icon、揀緊邊個對話 option。

### 2. 用呢個 base prompt

<PromptBlock label="完整 prompt — JP RPG dialogue 翻譯 + lore context">
我而家玩緊 [遊戲名]（[platform / 版本，例：FF14 PC 日服 / Persona 5 Royal PS5 JP]）。

依家撞到呢段：[upload screenshot 或 paste 日文]

請整以下幾 part：

**Part 1：直譯（natural 中文）**
唔好機翻 — 用順口繁中或粵語俱可，保留說話者語氣（古風 / 少女 / 老人 / 武士等）。

**Part 2：Quest / scene 背景**
- 呢段對話發生喺邊個 chapter / area
- 講緊邊個事件 / 邊個 character relationship
- 我而家做緊嘅係 main story / side quest / event 定 character bond

**Part 3：關鍵詞解釋**
任何 game-specific 嘅 lore term（地名、種族、組織、技能、装備）— 用 1 句解
例：「エオルゼア = FF14 嘅大陸名，主舞台」

**Part 4：選擇 option 嘅 implication**
（如果係 dialogue choice scene）
- 每個 option 直譯
- 揀邊個會 unlock / lock 咩 (social link, affinity, route divergence)
- 如果係 RNG / temp reaction 唔影響 long-term，講明

**Part 5：⚠️ 預警**
- 我會唔會 lock 自己出某個 route / ending
- 有冇 missable item 喺呢個 scene

語氣：好似旁邊識嗰隻遊戲嘅朋友傾，唔係 wiki dump。
</PromptBlock>

### 3. Follow-up：問深入啲

收到 base response 之後，按需要 follow up：

> 「Part 2 你講『光の柱事件』係邊個 patch 嘅嘢？我宜家玩緊 2.0 base game 應該未發生？」

> 「我而家 affinity 同呢個 character 係 Rank 3，揀第 2 個 option 會 lock 我去 bad ending route 嗎？」

> 「Part 3 個『真名解放』skill 我學識咗未？定係 future content？」

Claude 對主流 JP RPG（FF14、Persona、Atlus 系列 SMT / P3-5、軌跡 / Trails、Tales of、Xenoblade、Monster Hunter）有相當深 lore knowledge——對於 niche / 新作可能要 google check，**所以遇到 lore 上嘅關鍵 turning point，建議 cross-check 一下中文 wiki**。

## 變化

### 變化 1：成個 cutscene 嘅完整摘要

如果你 record 咗一段 5-10 分鐘 cutscene（用 PS5 / Switch / OBS），可以 transcribe 之後貼落去：

<PromptBlock label="變化 1 — Cutscene 完整摘要 + 角色 motivation">
以下係 [遊戲名] 一段 cutscene 嘅日文 transcript（[幾耐]）：

[paste transcript]

請整：

**Scene summary（3-4 句）**
一個 episode-of-the-week 嘅 dramatized 描述

**角色 by 角色**
每個 character：
- 講咗咩
- emotional state（沮喪 / 怒 / 諷刺 / 真情流露 / 演技）
- 動機 motivation — 佢點解依家咁做

**轉折點 turning points**
邊一句 / 邊個 reaction 改變晒 scene 嘅方向

**未解嘅 thread**
依個 scene 提到但未解嘅 plot hook，可能會喺後面 chapter 收返

**Spoiler protection**
唔好 spoil 未來 chapter — 我宜家睇緊呢個 cutscene 已經係我嘅 spoiler frontier

語氣：劇本分析師 + 一個熟 lore 嘅 fan，唔係 wiki summary。
</PromptBlock>

### 變化 2：學日文 mode（一邊玩一邊學）

如果你係**有少少 N4-N3 日文基礎**嘅人，可以用 Claude 變語言學習 partner：

<PromptBlock label="變化 2 — 玩 game 同時學日文">
我學緊日文（[你嘅 level，例：N4]）。我玩緊 [遊戲名]。

呢段對話：[paste 日文 / screenshot]

請整：

**Part 1：原文 + 假名 furigana**
畀我每個漢字嘅假名讀音（例：冒険者[ぼうけんしゃ]）

**Part 2：逐句拆 grammar**
標出我 level 應該識嘅 pattern，同 1 個我未識嘅 pattern + 簡短解釋

**Part 3：3 個今日記嘅詞**
唔好太貪心 — 3 個就夠
例：祝福（しゅくふく）= blessing
每個給 1 個 example sentence

**Part 4：聽力 challenge（可選）**
如果呢段對話有 voice acting，你估有冇 contraction / colloquialism（だ→じゃ、ない→ねぇ 等）我應該留意

**Part 5：今日嘅自我練習**
畀我 1 條短句，用今日學嘅 grammar pattern + 1 個新詞，要我自己造嚟回應呢個 NPC

語氣：耐心 tutor，唔好 dump 一頁文法 table。
</PromptBlock>

### 變化 3：揀 build / equipment 顧問

JP-only RPG 嘅 mechanics text 通常都係日文：

<PromptBlock label="變化 3 — 裝備 / build / skill 揀對">
我玩緊 [遊戲名] 嘅 [class / 角色名]。

呢個装備 / skill 嘅描述：[screenshot / paste]

請：

**Part 1：直譯描述**
標出關鍵數字、條件、debuff

**Part 2：對我個 build 嘅實際 impact**
我現時嘅 build / role 係：[填，例：FF14 BLM caster，刷 raid 為主]
- 點解呢個對我有用 / 冇用
- vs 我宜家用緊嘅 [填裝備名]，邊個 expected DPS / mit 高
- 換唔換 worth it（畀個 quick yes / no + 1 句 reason）

**Part 3：升級 path**
- 呢件嘢喺邊度 farm / craft / 換
- 後面有冇 better tier，我跳過直接等好過？

**Part 4：⚠️ trap**
- 呢件嘢有冇「睇落好但其實 niche / 同 build 撞」嘅情況
- 有冇社群共識係 inferior / re-balance 過

語氣：副本老手朋友，畀 actionable 意見唔係 disclaimer。
</PromptBlock>

## 一個心態

JP-only RPG 對好多港人嚟講係**「想玩但驚浪費時間」**——你買咗 sub、買咗 console、買咗 game，但每次撞到一面 text wall 就好想 alt-tab。

Claude 唔係要取代你學日文（如果你想學，**變化 2** 嘅 mode 就係你嘅 daily reps）。Claude 嘅 value 係 **lower 個 friction threshold**——由「我要查 wiki + Google translate + 上日本攻略 site」嘅 5 分鐘 break，變成「影張相 → 5 秒返返個 quest」。

呢個 difference 對於每日只有 1-2 個 rdr session 嘅返工族嚟講，係**玩到完款唔玩**嘅分別。

最後一個 tip：**唔好 over-rely**。Boss 戰嘅 dialogue 你抽走全部 context 嚟睇翻譯，反而 lose 咗 voice acting + cinematography 嘅 emotional impact。Translate 嗰 D 真係卡住你進度嘅 text，畀自己慢慢沉浸喺氣氛入面。

GG WP，enjoy 你嘅遊戲。

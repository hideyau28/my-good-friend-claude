## 情境

你啱啱通關 Hollow Knight。100 個鐘以上，最後 boss 流晒淚。你想揾下一隻**類似氣氛**嘅遊戲——黏手戰鬥、神秘世界觀、氛圍式孤獨——但問題出咗：

- Steam「More Like This」？通常 push 你 Dead Cells、Blasphemous、Salt and Sanctuary。**全部都係 metroidvania，但氣氛唔同。**
- Reddit 推薦？top comment 講 Ori，第 2 個推 Celeste，第 3 個彈走講 Dark Souls。**冇人問你想要邊個維度似。**
- YouTube「Top 10 like Hollow Knight」？演算法只跟標籤「metroidvania」，**意境一致嘅遊戲就會走漏**（例：Outer Wilds 同 Hollow Knight 氣氛接近但機制完全唔同，永遠唔會撞到。）

問題本質：**「類似」唔係單一維度。** Hollow Knight 嘅「似」可以指：
- 探索式 metroidvania 機制
- 手繪美術 + 憂鬱 OST
- 環境敘事，唔靠過場片
- 孤獨氣氛，少 NPC 對話
- 死咗就由 checkpoint 重新刷嘅風險回報循環

要縮窄到啱你想要嘅維度，**先要拆清楚自己想要邊一面**。

呢篇用 Claude 5 分鐘揾到下一隻 pick——唔靠標籤、靠維度。

## 跟住做

### 1. 整理你嘅參考遊戲

開新 Claude chat。第一句直接：

> 我啱啱玩完 Hollow Knight，想揾下一隻氣氛接近嘅遊戲。

唔好就咁停喺度——加多兩句幫 Claude 縮窄：

> 之前我玩過 Outer Wilds、Celeste，都鍾意。但 Dark Souls 玩半個鐘就退出咗（覺得太刷）。

呢 3 個參考點（2 鍾意 + 1 唔合）已經畀到 Claude 嘅基準。

### 2. 用呢個 prompt

<PromptBlock label="完整 prompt — 由參考揾類遊戲">
我啱啱玩完 [遊戲名]，畀 [9/10] 分。

📚 **同系列比較**
我之前玩過嘅參考遊戲：
- ✓ 鍾意：[2-3 隻 + 點解]
- ✗ 唔合：[1-2 隻 + 退出嘅原因]

🎯 **我而家想要邊種氣氛（揀 1-2 個維度）**
- [ ] 機制 — 類似遊玩循環（platforming / souls-like / city-builder）
- [ ] 氣氛 — 氣氛接近，唔理機制
- [ ] 美術風格 — pixel art / 手繪 / low-poly / 寫實
- [ ] 故事節奏 — 緊湊敘事 / 開放世界探索 / 極少世界觀
- [ ] 遊玩時間預算 — 我得 [X] 個鐘，唔玩 100+ 鐘嘅 RPG

⚙️ **我嘅平台 / 預算**
- [PC / Switch / PS / Xbox]
- 預算 HK$[幾錢]
- 鍾唔鍾意 indie / AAA / 都得

請推薦：

**第一層：3 隻強力配對**（≥ 4 個維度啱）
每隻：
- 1 句簡介（氣氛 + 機制）
- 同我參考邊度似 / 邊度唔似
- 預計遊玩時間
- 港幣大約值（Steam HK / 港行）
- 風險 — 你會唔鍾意嘅 1 點

**第二層：3 隻冷門推薦**（2-3 個維度啱，但可能驚喜）
- 同第一層一樣格式
- 額外講：點解我可能會唔鍾意但都值得試

**第三層：1 隻「完全唔同但同氣氛」**
- 跨類型嘅驚喜推介
- 解釋點解你覺得我會中

⚠️ **避雷區**
3 隻表面類似但**我大機會退出**嘅，講原因。

語氣：好似識遊戲嘅朋友推介，唔係 IGN review 摘要。
</PromptBlock>

### 3. 跟進收窄

收到 9 隻推薦之後唔好即刻去 wishlist。跟進：

> 「第一層第 2 隻 [Tunic] 我細個玩過 Zelda，未玩過 souls-like，會唔會煩親我？」

> 「第二層 [Outer Wilds] 我嘅伴侶唔玩 game，可唔可以梳化雙人？」

> 「第三層 [Slay the Spire] 我已經砌咗 100 個鐘 Hearthstone，會唔會太似？」

呢類追問係 Claude 比演算法推薦嘅強項——你可以**追問 + 限定 + 反駁**。

## 變化

### 變化 1：「我得 30 分鐘抖氣，揾隻短局遊戲」

唔係每次都係買新 game，有時係要由 backlog 揀今晚玩邊隻。

<PromptBlock label="變化 1 — 由 backlog 揀今晚玩邊隻">
我 backlog（買咗未通關）：
- [遊戲 A — 玩咗幾耐 / 目前章節]
- [遊戲 B]
- [遊戲 C]
- [遊戲 D]

今晚我有 [30 / 60 / 90] 分鐘，[攰 / 精神好 / 想 chill / 想刺激]，想 [推進主線 / 探索 / 收集 / 純放鬆]。

請：

1. 推薦今晚玩邊隻
2. 由我目前狀態，建議：
   - 玩主線哪一段 / 抑或支線？
   - 1 個 30 分鐘嘅小目標（唔好推我刷 1 個鐘）
3. 如果我心情變咗，後備 pick 係邊隻

語氣：好似朋友幫你揀「今晚食乜」，唔係遊戲記者。
</PromptBlock>

### 變化 2：「我喺意一個極小眾嘅機制，揾類似」

有時你想要嘅唔係氣氛而係**一個好特別嘅機制**：

- Outer Wilds 嘅 22 分鐘時間循環
- Inscryption 嘅打破第四面牆
- Disco Elysium 嘅技能變角色內心對話
- Return of the Obra Dinn 嘅推理謎題

<PromptBlock label="變化 2 — 揾類似小眾機制嘅遊戲">
我特別鍾意 [遊戲名] 嘅 [某個機制，描述 1-2 句]。

例如：我鍾意 Outer Wilds 嘅「知識作為推進」——
你冇等級升級，你嘅角色點通關完全取決於你個玩家學識嘅世界觀。
我想要更多呢類設計。

請推薦 5 隻**核心機制接近**嘅遊戲，唔理類型。每隻：

- 個機制點重新演繹
- 「親兄弟設計」（設計哲學一脈相承）/「遠房親戚」（皮唔同骨同）
- 玩到呢個機制嘅解鎖條件（要刷幾耐先見到）
- 如果係冷門 / 1990 年代舊遊戲，講現代模擬器邊度玩到

語氣：遊戲設計講座 + 朋友推介。
</PromptBlock>

### 變化 3：「呢隻熱話 game，我啱唔啱玩」

新 game 推出之前你已經睇咗一堆預告 / 預覽，但唔知自己會唔會鍾意：

<PromptBlock label="變化 3 — 新作啱唔啱我">
我考慮緊買 [新 game 名，推出日期]。

我嘅遊戲基因：
- 過去 3 年最愛：[3 隻]
- 過去 3 年退出嘅：[3 隻 + 原因]
- 我唔鍾意嘅遊戲設計規律：[例如刷怪、抽卡、多人有毒、jump scare]
- 我嘅平台 / 預算

請整：

**配對分數 (1-10)** + 你嘅理由

**3 個綠燈**
呢隻 game 嘅元素你會鍾意

**2 個黃燈**
未必 dealbreaker 但會煩到你嘅嘢

**1 個紅燈**
最可能令你 60 個鐘後退出嘅風險

**等定唔等**
- 推出日買？
- 等 1 個月睇修補同評論？
- 等減價（通常 Steam 1 年內 25-40% off）？
- 跳過？

語氣：唔好做炒作銷售員，係要保護你個銀包同時間。
</PromptBlock>

## 一個思維框架

Steam 嘅「More Like This」係標籤為主。Reddit 推薦係人氣加權。YouTube「Top 10」係 SEO 為主。

呢 3 個來源都係用**集體統計**嚟揾「類似」，但你個人嘅「類似」係**維度具體**——你可能想要氣氛一致機制唔同，或者機制一致美術完全不同。

Claude 嘅獨特之處唔係資料庫大過 Steam，而係佢可以同你**逐個維度拆**。你輸入越精準（參考 + 想要邊面），輸出越啱。

最後提醒：**唔好用 Claude 揾完一張 9 隻嘅清單之後一隻一隻買晒**。揾完返去 Steam wishlist，等減價，1 個月後再揀其中 1 隻——畀自己一啲距離去消化推薦。

Wishlist 太長 = 真正想玩嘅信號會走失。

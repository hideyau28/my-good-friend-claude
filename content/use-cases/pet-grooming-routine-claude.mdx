## 情境

上星期六，你帶主子（你隻 5 歲長毛貓 / 狗）去寵物美容。

朝早 10 點放低。美容師同你傾咗一輪「有冇打結呀，要剪短啲嗎，室內貓啱唔啱剪獅子裝」。你簽咗張同意書。

晏晝 2 點接返。美容師個樣有啲唔自然：「主子比較不安。我哋盡量做晒，但打結太多有啲位剪得短咗 ⋯⋯ 佢有少少出聲但 OK 嘅 ⋯⋯」

你抱主子返屋企，佢一落地即刻衝入床底。

接落嚟 3 日：
- 主子唔出嚟食嘢
- 你裝吓床底，見到佢全身剪到一忽忽
- 第 2 日發現佢手仔有個小傷口（剪得太貼？）
- 第 3 日佢仲匿埋，但開始出嚟食少少嘢

你睇返張單：HK$680。你個心沉咗落去。

而家過咗 6 個月又到美容季節（夏天打結容易嚴重）。你冇信心 book 返同一間，但又唔知點揀。同時，你想減少將來美容嘅頻率——可唔可以喺屋企維持？

**呢篇教你用 Claude 拆解 3 樣嘢**：
1. 你可以喺屋企做嘅每週護理（減少美容頻率）
2. 揀新美容師嘅 5 條審查問題
3. 主子嘅 4 週減敏計劃（就算下次去美容，壓力都會細好多）

## 跟住做

### 1. 將主子嘅毛髮 + 行為資料話畀 Claude 知

開 Claude，先將你諗到嘅寫晒出嚟，唔好就咁叫 Claude 直接出計劃：

> 主子 5 歲，男仔，已絕育。長毛貓 (緬因貓混血？或者波斯混血 — 唔肯定品種)。
>
> 毛髮：有底毛 + 面毛，特別肚 + 大髀 + 腋下容易打結。全年都會甩毛，春秋兩季會大換毛。
>
> 行為：對梳毛嘅反應——頭 30 秒 OK，之後會扭來扭去 + 咬梳 + 走佬。極度抗拒剪甲，要 2 個人㩒住先剪到 1 隻爪。完全唔洗得白白（上次喺診所洗要打鎮靜劑先搞得掂）。
>
> 過去美容經驗：
> - 6 個月前美容師 A — 平 ($380) 但剪到一忽忽
> - 上星期美容師 B — 貴 ($680) 但主子嚇到匿咗 3 日
>
> 我每日作息：朝早 9 點 / 夜晚 7 點餵食。如果主子接受到，我可以喺早晚各抽 5-10 分鐘做護理。

呢類詳細資料，比起行貨「點樣喺屋企幫貓美容」嘅 prompt 出嚟嘅度身訂造計劃好太多。

### 2. 用呢個 prompt 整居家護理計劃

<PromptBlock label="完整 prompt — 居家護理每週計劃">
[貼上你上面嘅詳細資料]

請幫我整一個 **實際可行嘅居家護理計劃**——強調主子嘅接受程度，而唔係最理想嘅頻率。

我嘅目標：
- 減少專業美容頻率（由 6 個月 → 8-12 個月）
- 減少打結（特別係重災區）
- 主子接受到而唔會有陰影 — 循序漸進

請整 **每週時間表**：

📅 **每日（2-3 分鐘）**
- 邊樣最少功夫而主子應該每日接觸嘅？（例如：用闊齒梳喺重災區梳 5-10 下）

📅 **每週 2-3 次（5-10 分鐘）**
- 全身梳毛用咩工具 — 列出 2-3 種：
  - 針梳（清底毛）
  - 闊齒梳（解結）
  - 橡膠按摩手套（低壓力替代方案）
- 次序：由邊度開始（低焦慮位 — 背脊 / 面珠）→ 邊度尾（高焦慮位 — 肚 / 手腳）
- 主子嘅容忍信號：揈尾、縮耳、低吼 — 一出現 = 暫停 + 獎勵

📅 **每週 1 次（10-15 分鐘）**
- 檢查打結：重點檢查 5 個重災區（腋下 / 肚 / 耳背 / 尾部根部 / 腳後跟）— 用手指去搵出已經成形嘅結
- 微小打結：解結噴霧 + 手指慢慢解 — **絕對唔好用剪刀貼住皮剪**（居家護理最常見嘅受傷原因）
- 大結：唔好自己搞 — 記低，留畀專業美容師

📅 **每 2 週**
- 剪甲：主子嚴重抗拒 — 分階段減敏（參考本文步驟 4）
- 檢查耳仔：用棉花球 + 洗耳水（診所推薦牌子）抹外耳。**唔好用棉花棒插入耳道**

📅 **每月**
- 檢查體重（容易打結嘅貓容易因為體重變化影響毛髮狀態）
- 影相紀錄：影 5 個重災區特寫，每個月比較 — 你可以及早發現毛髮變化

🛒 **工具預算**（香港 2026 年一般價格）
- 針梳（FURminator / Trixie）: HK$120-180
- 闊齒梳: HK$60-100
- 橡膠按摩手套: HK$80
- 解結噴霧 (Earthbath / Burt's Bees): HK$120
- 寵物指甲鉗 (貓用鍘刀型): HK$80
- 零食袋（極吸引零食 — 凍乾雞肉）: $40 + 零食 $80

⏰ **長毛貓有美容焦慮嘅實際時間表**
- 唔好太有野心一開始就日日做 — 你會覺得攰 + 主子會將梳毛同你嘅焦慮聯繫埋一齊
- 由每週 2 次、每次 5 分鐘梳毛開始，4 個禮拜後再評估
- 進度標準：主子嘅容忍時間（唔走佬嘅情況下接受到）長咗 → 先升級

🚫 **唔好做**
- 喺屋企洗白白：長毛貓喺屋企洗白白失敗率極高 + 加深陰影（少數例外 — 用乾洗粉 / 濕紙巾局部清潔 OK）
- 用鉸剪貼皮剪結：受傷風險極高
- 強行㩒住主子完成護理：長遠只會加深焦慮
- 同社交媒體嘅寵物網紅比較 — 佢哋嘅寵物經過多年減敏訓練

語氣：貼地 + 唔好怪責。主人知道主子有焦慮唔等於主人失敗。
</PromptBlock>

Claude 出嘅計劃你印一份貼喺貓砂盆隔籬（你每日都會見到）。**頭 4 個禮拜集中喺堅持，唔係講求完美**——主子接受到你攞把梳出嚟，已經係進步。

### 3. 審查下一個美容師嘅 5 條問題

唔係所有美容師都適合容易焦慮嘅寵物。Claude 幫你出一份審查問題清單：

<PromptBlock label="完整 prompt — 審查美容師問題">
我要揀新美容師畀主子（[物種 + 品種 + 性格 + 過去美容陰影]）。

過去不愉快經驗：
- 美容師 B 上次令主子嚇到匿咗 3 日，剪到一忽忽，有小傷口
- 我而家極度謹慎去審查新美容師

請整 **5 條關鍵問題 + 點樣評估答案**：

❓ **問題 1：如果寵物喺美容過程明顯極度緊張，你會點處理？**
- 🟢 好答覆：「我哋會暫停 + 畀佢休息，部分情況我會建議分 2 次搞 / 或者轉介去診所打鎮靜劑」
- 🟡 尚可：「我哋會繼續但溫柔啲」
- 🔴 紅燈：「我哋有經驗處理，唔需要停」
- 額外追問：你哋有冇受過無恐懼美容 (fear-free grooming) 嘅訓練？

❓ **問題 2：你一般處理 1 隻貓 / 狗美容要幾耐？同其他寵物一齊定分開？**
- 🟢 好：每隻寵物 30-60 分鐘，**同一時間只處理 1 隻，喺獨立房間**
- 🟡 尚可：60-90 分鐘，但同冷靜嘅狗共用空間
- 🔴 紅燈：「一日做 100+ 隻，共用籠區，同時處理多隻」 = 流水作業模式，壓力極大

❓ **問題 3：你可唔可以中途 update 我，或者我留喺度睇？**
- 🟢 好：「主人可以留低或者我哋會影相 update，唔會影響美容質素」
- 🟡 尚可：「我哋傾向主人唔喺度（寵物表現會唔同）但會 WhatsApp update」
- 🔴 紅燈：「只准放低同接走，中途零溝通」 = 唔透明

❓ **問題 4：你哋有冇取消嘅個案？點處理？**
- 🟢 好：「如果主子過度緊張，我哋會提早停 + 只收部分費用。我哋唔會夾硬完成」 — 顯示有職業操守
- 🟡 尚可：「取消有取消費，但會解釋原因」
- 🔴 紅燈：「我哋一定會完成服務」 = 將完成工作建立喺寵物嘅痛苦之上

❓ **問題 5：可唔可以做試驗性質探訪？（10-20 分鐘短時間）**
- 🟢 好：「可以，我哋有減敏環節 $XXX，純粹熟習環境，唔會真剪」
- 🟡 尚可：「我哋可以試吓短時間梳毛睇吓啱唔啱」
- 🔴 紅燈：「我哋只做全套服務」 = 缺乏彈性

📋 **額外驗證（book 之前）**
- Google 評論 / FB 評論搜尋特定關鍵字：「焦慮」「驚」「領養」— 睇吓佢哋處理焦慮寵物嘅評價
- 踩場參觀一次但唔 book（搵藉口入去攞單張）— 觀察：
  - 寵物之間有冇分開？
  - 美容師狀態緊張定冷靜？
  - 氣味（尿味 / 恐懼費洛蒙 — 乾淨嘅地方唔應該有大陣味）
  - 同主人溝通方式（有耐性 / 敷衍）

🚩 **絕對調頭走嘅信號**
- 拒絕答你嘅問題
- 「我做咗 18 年從來冇寵物會緊張」 = 講大話或者觀察力差
- 價錢含糊 / 隱藏附加費
- 逼你盡快 book / 話只限今日優惠
- 拒絕你 book 之前參觀地方

語氣：賦予主人權力 — 主人唔係神經質，係寵物嘅代言人。美容師帶來嘅壓力反應 = 主子未來嘅美容體驗。
</PromptBlock>

5 條問題嘅答案質素比價錢更重要。**寧願畀多 30% 錢揀個冷靜嘅美容師，都唔好慳嗰 30% 搞到主子要半年先平復到**。

### 4. 主子嘅 4 週減敏計劃

家居減敏嘅目標唔係訓練主子「鍾意」美容——係將對美容嘅聯想由「恐慌」變成「中性 + 零食」。

<PromptBlock label="完整 prompt — 4 週美容減敏計劃">
主子對美容嚴重焦慮。我想喺屋企做 4 週減敏訓練，目標：下次去專業美容，主子壓力細好多。

主子目前嘅容忍底線：
- 被梳毛：30 秒 OK，之後就走佬
- 被掂手腳：極度抗拒
- 見到指甲鉗：即刻走佬
- 聽到美容相關聲音（電剪 / 風筒）：匿埋

請整 **Week 1-4 漸進計劃**（每日 2-3 分鐘）：

🗓 **Week 1 — 純粹存在**
**目標：令主子將美容工具同極吸引嘅零食聯繫埋一齊**
- Day 1-2：放把梳喺地下 5 米外，主子行埋去聞 = 獎零食
- Day 3-5：放把梳喺地下 1 米外，主子行埋去聞 = 獎零食
- Day 6-7：手揸住把梳，主子行埋嚟 = 獎零食（**唔係梳，淨係揸住**）
- 零食：凍乾雞肉 / 肉泥 / 任何主子最鍾意嘅
- 絕對唔梳。重複「工具 = 零食」嘅聯想

🗓 **Week 2 — 摸但唔用工具**
**目標：主子接受你用手摸重災區**
- Day 1-3：1 分鐘環節 — 摸主子頸 + 膊頭（低焦慮位），完成 → 獎零食
- Day 4-5：加入摸背脊
- Day 6-7：嘗試摸肚 1 秒（高焦慮位）→ 即刻獎零食
- 守則：主子一走佬 = 停。唔好追。等下次。

🗓 **Week 3 — 工具 + 最少接觸**
**目標：主子接受 1-2 下梳毛**
- Day 1-2：梳輕輕掂毛 1 下 → 獎零食
- Day 3-4：梳 3 下 → 獎零食
- Day 5-7：梳 5 下 → 獎零食。**頭幾下只梳低焦慮位（頸 / 面珠 / 膊頭）**
- 唔好搞重災區（肚 / 手腳 / 尾巴根部）

🗓 **Week 4 — 擴大範圍 + 加入其他工具**
**目標：主子接受完整環節 3-5 分鐘**
- Day 1-3：包埋背脊 + 腳外側
- Day 4-5：嘗試掂手腳 1 秒 → 獎零食
- Day 6-7：引入指甲鉗（唔剪 — 只放喺視線範圍 + 獎零食）
- 一直保持低期望：每次最多 5 分鐘

📈 **進度指標**（每週末檢討）
- Week 1 完：主子自願行埋工具度？Y / N
- Week 2 完：主子接受 30 秒觸摸而唔走佬？Y / N
- Week 3 完：主子接受梳 5 下？Y / N
- Week 4 完：主子喺 3 分鐘完整環節保持冷靜？Y / N

如果某個禮拜停滯不前（連續 7 日冇進步）：
- 絕對唔好夾硬推進去下一個禮拜
- 重複該週進度額外 1 個禮拜
- 縮短時間（2 分鐘 → 1 分鐘）
- 提高零食吸引力

🚨 **絕對停止信號**
- 完咗之後主子匿埋超過 1 個鐘：你推進得太快
- 主子開始驚你餵零食隻手：成個方法要推倒重來
- 主子抓傷自己 / 狂𢯎重災區：要去睇醫生，可能有潛在焦慮問題

🐾 **Bonus — 對聲音嘅反制約**
- Week 2 起：播電剪 YouTube 聲（低音量），主子行埋嚟 → 獎零食
- Week 4 起：嘗試開風筒極短時間（距離遠 + 細聲）

語氣：有耐性 + 貼地。4 個禮拜唔保證主子會「愛上美容」 — 但保證底線會有進步。長遠嚟講：持續護理 6-12 個月先會有大改善。
</PromptBlock>

呢個計劃嘅秘訣：**唔係訓練主子，係訓練你**。你嘅耐性 + 畀零食嘅時機 + 觀察主子身體語言嘅敏感度，全部都係技巧。主子嘅進步只係附帶結果。

## 變化例子

### 變化 1：短毛寵物（貓 / 狗）護理計劃

短毛寵物嘅護理同長毛唔同：

<PromptBlock label="變化 1 — 短毛寵物護理計劃">
我隻主子係短毛 [品種 + 物種]。佢冇打結問題，但 [甩毛嚴重 / 油脂多 / 皮屑多]。

請整短毛護理計劃：

🐾 **短毛護理重點**
唔係集中防打結，而係：
- 處理甩毛（家居衛生）
- 監察皮膚健康（短毛皮膚易見，容易發現問題）
- 剪甲 + 洗耳（同長毛一樣）
- 牙齒護理（特別重要 — 短毛品種容易有牙齒問題）

📅 **時間表**
- 每日：用橡膠按摩手套掃勻全身 3 分鐘（主要清廢毛）
- 每週 2 次：用豬鬃毛梳梳全身（刺激皮膚分泌油脂 + 分佈均勻）
- 每週 1 次：檢查指甲 / 耳仔
- 每月 1 次：檢查牙齒 / 刷牙（如果接受）
- 每 3 個月 1 次：沖涼（如有需要 — 短毛唔需要成日沖，過度沖涼反而會令皮膚乾燥）

🛒 **工具**
- 橡膠按摩手套（短毛主要工具）: HK$80
- 豬鬃毛梳: HK$100
- 寵物牙刷 + 專用牙膏: HK$120
- 寵物濕紙巾（局部清潔）: 每一包 HK$40

🔍 **短毛專屬注意位**
- 甩毛斑禿 / 紅腫 = 通常係敏感 / 壓力 / 寄生蟲 — 要睇醫生
- 季節性大量甩毛 vs 持續異常甩毛 — 後者要睇醫生
- 貓：短毛貓自己會搞掂 80% 護理 — 你嘅計劃只係輔助，唔係主要來源

語氣：精簡 — 短毛唔需要複雜計劃，過度護理反而有害。
</PromptBlock>

### 變化 2：狗隻美容 — 修剪造型 + 品種專屬剪法

狗隻美容比貓複雜（特定品種修剪風格 + 沖涼頻率）：

<PromptBlock label="變化 2 — 狗隻護理計劃 + 品種專屬修剪">
我隻主子狗 [品種 + 年齡 + 毛髮類型]。佢需要：
- 修剪頻率：[每 6 週 / 每 8 週 / 每 12 週]
- 沖涼頻率：[每 2 週 / 每月 / 視乎活動量]
- 每日散步 — 街上污糟積埋

請整：

📅 **時間表**
- 每日：行完街抹腳（清走街道灰塵 + 化學物質）
- 每週 2-3 次：全身梳毛
- 每週 1 次：沖涼（如果品種需要 — 貴婦狗 / 貴婦混種 / 部分爹利犬）或者 每 2-4 週（其他品種）
- 每月 1 次：清耳仔 + 刷牙
- 每 6-12 週：專業修剪

🦴 **品種專屬注意位**
你個品種嘅典型剪法 + 維持方法：
- 貴婦狗 / 貴婦混種: 毛髮不斷長，極易打結，需要每 6 週專業修剪
- 西施 / 摩天使: 絲質長毛，需要每日梳毛，眼周持續清潔
- 鬥牛犬 / 八哥: 皮膚摺位持續清潔（中午抹吓），少修剪需要
- 中型唐狗: 視乎毛髮類型，部分需要修剪，部分淨係梳毛就得

🛁 **喺屋企幫狗沖涼**
（貓唔建議，但狗通常 OK）
- 溫水 + 狗專用沖涼液（酸鹼值同人類唔同）
- 用棉花球塞住耳仔
- 徹底過水（殘留沖涼液 = 刺激皮膚）
- 完咗之後抹乾 + 低溫吹乾
- 全程獎勵零食

🚶 **行街相關護理**
- 香港街道熱 / 化學清潔劑：燙傷腳板底風險，夏天晏晝避免行熱石屎路
- 行山後檢查牛蜱：集中檢查腋下 / 耳仔 / 腳跟
- 長毛狗有泥：淨係沖腳，唔好全身沖（過度沖涼會令皮膚乾燥）

語氣：了解品種特性 — 唔同品種完全唔同護理現實。

### 變化 3：年老寵物護理 — 安全考慮

10 歲以上嘅寵物護理需要特別小心：

<PromptBlock label="變化 3 — 年老寵物（10 歲以上）安全護理">
我隻主子 [12 歲貓 / 14 歲狗]，最近 [行動慢 / 關節炎 / 腎病 / 皮膚問題]。我擔心美容對佢壓力太大，但又唔可以唔做（毛開始亂晒）。

請整針對年老寵物嘅方案：

⚕️ **護理前獸醫諮詢**
- 年老寵物護理前檢查：
  - 心臟負荷到鎮靜劑嗎（如需要）？
  - 關節炎嚴重程度 — 限制活動會引起痛楚？
  - 皮膚狀況 — 避開敏感部位
- 醫生可能建議：只做居家護理，唔再去專業美容

🏠 **居家護理調整**
- 表面：軟墊（瑜伽墊）— 硬地對關節炎有痛楚
- 時間：短（每次最多 2-3 分鐘），更頻密（每週 5-7 次）
- 姿勢：主子自己舒服嘅姿勢 — 唔強迫企起身
- 跳過：主子強烈抗拒嘅部位（唔好強迫 — 年老寵物嘅抗拒可能代表痛楚）

🛁 **沖涼替代方案**
- 年老寵物持續濕身 = 凍親風險 + 皮膚問題
- 用乾洗粉 / 濕紙巾局部清潔
- 如果必須沖涼（意外整污糟）：局部沖洗 + 即刻抹乾 + 保暖
- 避免風筒嘅聲 + 熱 — 淨係用毛巾抹乾

✂️ **處理打結**
- 年老寵物打結通常因為減少自我清潔
- 細小打結用解結噴霧 + 手指
- 大結: **絕對唔好自己貼皮剪** — 老人家皮薄 + 容易撕裂。搵上門美容師 / 診所處理
- 嚴重污糟打結 = 考慮剷獅子裝 (全身剷短) 一次，然後容易啲打理

⚠️ **紅燈信號**
- 主子喺護理期間出聲 = 痛，停手 + 睇醫生
- 喺護理期間發現新腫塊 = 睇醫生
- 皮膚情況惡化 = 睇醫生
- 主子護理後唔肯食嘢 = 壓力爆煲

📋 **臨終階段考慮**
如果主子喺安寧護理階段（見 [pet-end-of-life-grief-claude]），護理優先次序要改變：
- 目標唔再係靚 — 係舒服 + 衛生
- 接受毛髮唔靚作為換取唔打擾主子嘅代價
- 唔小心整污糟嘅局部清潔為主，全套護理唔重要

語氣：尊重年老寵物 — 護理唔係主人貪靚，係為主子舒服。步入老年，主子生活方式改變，護理方式都要跟住變。
</PromptBlock>

## 一個思維框架

護理喺主人嘅概念入面通常被視為「主子嘅事」——主子鍾唔鍾意，主子反應大唔大。

但其實，護理嘅最終效果 80% 取決於 **主人嘅技巧 + 耐性 + 堅持**。美容師一個月見主子 1 次 1 個鐘，你一個月見主子 720 個鐘。**你嘅居家護理所佔嘅比重，遠超任何一次單獨嘅美容體驗**。

呢個思維嘅轉變：居家護理唔係為咗慳錢，係做主人嘅基本技能。同你學煮飯 / 學洗衫分顏色一樣，**係長期同寵物共同生活嘅基本能力**。

最後提醒：

- ✅ **進度標準唔係「主子鍾意護理」**——係「主子接受到而冇陰影」。要求唔高，絕對做得到。
- ⚠️ **唔好比較你嘅主子同社交媒體嗰啲完美護理寵物**。佢哋花咗 2-3 年時間做減敏訓練。
- 🆘 **嚴重護理焦慮超過 6 個月居家訓練後仲冇進步** = 考慮獸醫開嘅焦慮藥（短期 + 持續行為訓練）+ 專門無恐懼美容師。主子嘅焦慮唔係性格缺陷 — 係醫療問題值得正視。

下次去美容，主子唔會完全冇焦慮，但焦慮底線高過上次——已經算係成功。

## 情境

你拎到 offer letter + 僱傭合約。HR 話：

> 「簽完 send 返畀我，下星期一報到。」

合約 8-15 頁，律師體中英夾雜。你快速睇過薪金、花紅、假期，覺得 OK，準備簽。但你冇細讀：

- **Non-compete clause（競業禁止條款）**——可能限制你離職後 6 個月不可入同行
- **IP assignment（知識產權轉讓）**——你週末寫嘅 side project 可能屬於公司
- **Garden leave（冷河期）**——離職時公司可以出糧畀你但唔畀你做嘢 / 唔畀去新公司
- **Bonus clawback（花紅退還）**——拎咗花紅但離職 12 個月內觸發嘅退還條件
- **Confidentiality scope（保密範圍）**——廣度可能涵蓋你成個職業生涯已知嘅行業知識

呢啲條款**唔影響你頭三個月嘅工作**，但**深刻影響你未來 10 年嘅事業彈性**。

呢篇用 Claude 做 30 分鐘嘅策略性檢視：搵出唔尋常 / 對打工仔不利嘅條款、預警可能失去嘅未來選擇空間。

> 利益申報：呢個唔係法律建議。Claude 嘅輸出係幫你**識得問問題**，唔取代律師。涉及高層職位（C-level、高級合夥人）或者你唔肯定，畀 $2-3K 搵僱傭法律師審半個鐘回報永遠係正面嘅。

## 跟住做

### 1. 上傳合約（連附件都要齊）

成份合約包括：
- 主要僱傭協議
- 薪酬附件（底薪、花紅結構）
- Restrictive covenants（限制性條款）附件（non-compete、non-solicit）
- IP / 保密協議附件
- 任何參考嘅政策文件（如「跟公司員工手冊」）

如果 HR 只 send 你主要協議，**主動要求睇埋附件**——通常呢度埋伏嘅陷阱最多。

### 2. 用呢個 prompt

<PromptBlock label="完整 prompt — 僱傭合約策略性檢視">
我會簽呢份 HK 僱傭合約（新工）。請幫我檢視。我嘅背景：

- 職位 + 級別：[填，例：Senior Marketing Manager / Director / 等]
- 行業：[填]
- 年薪：[底薪 + 花紅結構，要具體]
- 我而家 / 上一份工係邊間：[填，相關性：可能影響 non-compete 嘅範圍]
- 我嘅長遠彈性考量：[例子：未來 3 年想入同行 startup / 未來想自己創業 / 想保留 side project 自由]

請整以下 5 個部分：

**Part 1：薪酬細節核對表**

- 底薪：金額 + 幾時檢討
- 花紅：佔底薪百分比、是否保證、計算邏輯、派發時間
- 13 個月糧 / 滿約酬金：有冇、計算方法
- 股票 / 期權：歸屬時間表 (vesting schedule)、懸崖期 (cliff)、加速歸屬條款 (accelerator clauses)
- 迎新獎金 (Sign-on bonus)：金額 + 退還條件 (clawback condition)
- 福利：MPF 補貼、保險、年假、病假

如果某項冇提及，標「未列明」。

**Part 2：⚠️ Restrictive covenants（最重要嘅一段）**

逐個檢視：

- **Non-compete**：期限、地理範圍、行業範圍、是否有補償。問：條款可執行嗎？（HK 法律下 non-compete 必須範圍 + 期限合理 + 有補償先可執行）
- **Non-solicit**：只涵蓋客戶 / 只涵蓋員工 / 兩者皆是？期限？
- **Non-deal**：有冇？呢個比 non-solicit 更廣，禁止你同前公司客戶做生意，甚至對方主動接觸你都唔得
- **Garden leave**：公司可以強制？最長幾耐？呢段時間係咪出全份糧?

每項：原文引用 + 是否標準 + 對我未來嘅具體影響 + 應該商議邊度。

**Part 3：⚠️ IP / Confidentiality**

- **Invention assignment**：公司擁有嘅 IP 範圍有幾廣？是否涵蓋工餘時間嘅創作？是否涵蓋喺簽約前已經有嘅發明（pre-existing IP 豁免條款）?
- **Confidentiality 期限**：永久 vs 有期限? 範圍係只包商業秘密定係涵蓋更廣嘅機密資料?
- **Side project / 兼職**：有冇明文禁止? 有冇豁免機制?

**Part 4：⚠️ 解僱條款 (Termination clauses)**

- **通知期 (Notice period)**：你畀通知同公司畀通知對等嗎？
- **有合理解僱理由 (Cause) vs 無合理解僱理由 (without cause)**：兩個觸發點嘅後果
- **解僱時花紅 / 股票嘅處理**：唔同觸發點有唔同結果？
- **遣散費 / 代通知金 (payment in lieu)**：用 PILON 替代通知期嘅權利屬於邊個？

**Part 5：我應該同 HR / 老闆商議嘅 5 個要求**

由以上分析抽出 5 個對我長遠彈性最關鍵嘅嘢，我簽約前要反駁嘅。畀埋：
- 點樣表達（避免聽落唔信任 / 麻煩）
- 預想 HR 嘅可能回應
- 我嘅退讓底線（如果爭取唔到全部要求，可以有咩替代方案）

語氣：一針見血、具體、可採取行動。唔出「簽約前要小心」呢類行貨廢話。
</PromptBlock>

### 3. 核對兩三個關鍵條款嘅原文

呢類合約嘅 5% AI 錯誤可以等於幾年嘅事業限制。至少核對：
- **Non-compete 期限同範圍嘅原文** —— 一個字差別好大
- **IP assignment 嘅範圍** —— 有冇「conceived or developed during employment」嘅字眼？呢個會涵蓋工餘時間
- **解僱時花紅 / 股票處理** —— 已歸屬 (vested) vs 未歸屬 (unvested) 點處理

### 4. 商議（HR 比你想像中更有彈性）

大部分 HK 打工仔從未商議過 non-compete / IP 條款，因為**唔知有得傾**。

事實係 HR 通常已準備兩個版本：全面限制版同合理版。一個有準備嘅求職者主動提出反駁，**60-70% 嘅情況公司會調整**（因為 HR 知唔合理嘅條款上到法庭反而不可執行）。

**你嘅唯一籌碼就係簽約前**。簽約後想重新商議 = 基本上唔可能。

## 變化例子

### 變化 1：高級管理層 / C-suite 合約

<PromptBlock label="變化 1 — 高級職位深入分析">
我嘅職位係 [VP / Director / C-suite]，合約更複雜（有 LTIP、change-of-control provision 等）。

請特別深入分析：
1. **控制權變更觸發點 (Change-of-control trigger)**：如果公司被收購、IPO、合併，我嘅股票歸屬 / 花紅 / 遣散費點處理
2. **績效股份 / LTIP**：基於目標定基於時間？有冇上限？
3. **彌償 / D&O 保險**：個人責任保護範圍
4. **推定解僱觸發點 (Constructive dismissal triggers)**：如果公司單方面降職 / 調職 / 大改職責範圍，我嘅權利
5. **超越僱傭關係嘅保密責任**：舉報人豁免、合作義務

呢類高級合約，律師審視幾乎係必須。Claude 嘅輸出係幫你準備律師會議。
</PromptBlock>

### 變化 2：你係 freelance / contractor，唔係 full-time

<PromptBlock label="變化 2 — Freelance / contractor 合約">
我簽嘅係承辦商協議 (contractor agreement)（freelance / consulting / part-time），唔係全職僱傭。

請特別集中分析：
1. **獨立承辦商身分界定**：條款確保你唔會被視為僱員（避免未來勞資審裁處反向索償）
2. **IP assignment**：你交付成果嘅擁有權？你原有 IP 嘅保留權？
3. **付款條款**：預先付款 / 按階段 / 30 日找數？延遲找數嘅追討方法
4. **終止合約**：你畀通知同客戶畀通知對等？
5. **彌償 (Indemnity)**：客戶被第三方控告嘅話，邊個負責
6. **承辦商嘅 Non-compete**：通常較窄範圍應該夠
</PromptBlock>

### 變化 3：你已簽咗，而家事後想了解清楚

<PromptBlock label="變化 3 — 簽完之後檢視">
我已簽咗（[幾耐之前]），先睇返條款。請：

1. 用上面相同 5 個部分整理
2. 標示我而家簽咗所面對嘅劣勢
3. 列出我幾時可以重新商議（例：年度檢討、達到績效指標、加薪商議）
4. 具體指出：呢份合約嘅邊啲條款我下次轉工 / 下次檢討應該提出反駁

最後問：基於呢份合約，我可以做啲乜避免未來後悔（例：記錄我工餘創作 IP 嘅時間戳、避免簽署額外嘅限制性文件）
</PromptBlock>

## 一個思維轉變

HK 打工仔商議合約嘅最大迷思：「公司唔會讓步，我反駁就會無咗個 offer」。

事實係，**HR 嘅預設做法係零商議**——但係零商議唔等於「預算冇彈性」。大部分 HR 拎住標準合約範本，唔係特登設陷阱跣你。**一個有具體要求嘅求職者，HR 通常會提交上去畀法務部 / 請人嘅經理認真考慮**。

你最壞結果：HR 話「呢條冇得改」。你最好結果：拎走幾條限制條款。

**所以提出反駁嘅期望值永遠係正面嘅**。

Claude 嘅籌碼就係令你由「唔知有得傾」變「知有得傾 + 點樣表達」。30 分鐘嘅檢視功夫，可以保護你未來 10 年嘅事業選擇空間。

## 情境

你開緊間小店，客戶用 IG DM、WhatsApp、有時甚至 email 落單。 每個星期你要花一個鐘做嘅嘢：

- 揾返呢個星期啲訂單訊息
- 抄入 Google Sheet（客戶名、產品、銀碼、地址）
- 計返呢個星期幾單，總共幾錢

呢一個鐘係**重覆嘢**——同一套步驟、同一個 sheet 格式、淨係內容唔同。Chat 唔啱做呢類嘢（你要逐單貼）。**Cowork 嘅賣點就係呢度：set 一次，佢自動每星期執行**。

> 重要 caveat：Cowork 仲係 research preview，要 Pro/Max plan。佢有 agentic 能力會直接操作你部機嘅檔案同連接服務，**先開一個 dedicated 工作 folder 畀佢**，唔好畀佢掂你嘅敏感檔案。

## 跟住做

### 1. 入 Claude Desktop，開個新嘅 Cowork session

桌面版 Claude（唔係 web） → 撳「Cowork」mode。佢會問你個 task 係乜。

### 2. 一次過畀佢成個流程嘅 prompt

呢個 prompt 比較長，但好 critical——你要寫清楚**每星期要做乜、邊度攞 input、出乜 output**。Claude Cowork 會 plan 同 execute，你睇住佢做。

<PromptBlock label="完整 prompt — set 一次，每星期自動跑">
我開一間香港小店，賣手作飾物。我想你每個星期日早上 10 點自動做以下嘅嘢：

1. 連去我嘅 Gmail，揾返過去 7 日（星期一到星期日）入面所有提到「訂單」、「order」、「想要」、「reserve」嘅電郵
2. 同樣噉睇 IG DM（如果連咗 Meta Business Suite）
3. 抽出以下資料：
   - 客戶名（or IG handle）
   - 產品（盡量 match 我嘅 product 名單，喺 product-list.txt）
   - 數量
   - 銀碼（如果有講；冇就標「未報價」）
   - 地址 / 自取（如果有講）
   - 訊息時間
4. 將呢啲資料 append 落我嘅 Google Sheet「2026 訂單」嘅 "Weekly Log" sheet
5. 計返呢個星期：
   - 總共幾單
   - 總銀碼
   - 邊個 product 最多人要
6. 將呢個總結用 email 寄返畀我（subject 寫「第 X 期訂單總結」）

注意：
- 如果某單 info 唔齊，標黃色，喺 Notes column 寫缺啲乜
- 唔好自動回覆任何訊息——只係抽 data
- 如果有疑問，等我回覆先繼續

set 好之後 confirm 我，再 schedule 落每星期日 10am 自動跑。
</PromptBlock>

### 3. 同 Cowork 行一次 dry run

第一次跑唔好直接 schedule。叫佢「dry run 一次，showw 我 output，唔好真係 append 落 Google Sheet」。 你睇下：

- 佢有冇識認你啲 product 名
- 抽 data 嘅準確度
- 個 summary 嘅格式啱唔啱你

執完 prompt 之後，先 enable scheduled task。Cowork 會用 scheduled task 嘅 feature 幫你每週跑。

## 點解 Cowork 同 Chat 唔同

呢個情境用 Chat 唔係做唔到，但你要：
- 每星期主動開 Chat 一次
- 逐單 paste 訊息畀佢
- 自己抄入 Sheet
- 自己計總數

**Chat 係你問一次佢答一次。Cowork 係你 set 一次，佢做成年**。 對「重覆但唔複雜」嘅任務，Cowork 嗰幾粒鐘 set up 嘅成本，幾個星期就回本。

但 caveat 重提：呢個 mode 比較新，唔好叫佢做 critical / 唔可以錯嘅嘢（例如自動發 email 畀客戶、自動 charge 錢）。**佢做 read-only data extraction 同 reporting 最 sure**，做 write actions 要小心 + 一定要 dry run。

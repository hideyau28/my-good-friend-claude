## 情境

你計劃緊 8 月家庭 trip 去大阪 5 日 4 夜。

家庭組成：

- 你（38，marketing 經理）
- 伴侶（40，會計師）
- 8 歲女（升小三，鍾意 Disney + Universal）
- 5 歲仔（K2，專注力最多 30 分鐘，爆煲觸發包括「行得耐」/「肚餓」/「熱」）

你嘅 trip 偏好：

- 食日本嘢 ramen / sushi / yakiniku
- 行心齋橋 + 道頓堀 + 美食街
- 1 日 USJ
- 1 日京都 day trip
- 奈良鹿如果可以加埋

你伴侶嘅偏好：

- 避免過度趕
- 細路爆煲 = 全家 ruin day
- 預每日 1-2 個鐘休息時間
- 現金流唔好超預算

8 歲女嘅偏好：

- USJ 全日（Harry Potter + Minions + Hello Kitty）
- 買 K-pop / 玩具
- 食雪糕 / 士多啤梨甜品

5 歲仔嘅偏好：

- 坐火車（最鍾意新幹線）
- 公園跑 + 玩水
- 食 udon / 薯條 / 飯團
- 唔肯食生嘢
- 行路 30 分鐘為頂

你試自己列 5 日行程：

- Day 1：抵步 + 心齋橋
- Day 2：USJ
- Day 3：京都 + 奈良
- Day 4：大阪城 + 大阪城 area
- Day 5：心齋橋 + 返港

你望住個 list 半個鐘，發現問題：

- **Day 2 USJ**：8 歲女 + 你想玩晒 11 個 ride，5 歲仔玩唔到 7 個。佢點頂得住全日？
- **Day 3 京都 + 奈良**：1 日 day-trip 對 5 歲仔太進取。坐新幹線 + 京都古蹟 + 奈良鹿 = 8+ 個鐘戶外 = 爆煲風險 100%
- **每日食飯**：你想試 ramen / sushi / yakiniku；細路淨係食 udon / 飯團。你哋分開食 vs 揾兩者都得嘅餐廳？
- **Energy 管理**：5 歲仔 11am 準時飢餓爆煲；2pm 後午睡 window；4pm 後第二陣風但易爆煲

呢個家庭 trip 計劃嘅死結：**4 個旅客 × 4 個唔同需要 = 行程 90% 嘅衝突張力**。

你預設邊個照顧？

- 8 歲女 + 大人想 = 5 歲仔受苦
- 5 歲仔做定海神針 = 8 歲女悶 → 自己爆煲
- 對半分 = 全日嘈交

呢篇用 Claude **將全家偏好 + 限制寫晒，出平衡 5 日行程**——每日有大人亮點 + 細路定海神針 + 餐 + 休息 buffer，減少爆煲機會。

📋 **科技 / 連線**
- 1 個 pocket wifi（覆蓋 4 個裝置）
- 或每部電話有 eSIM 日本 plan
- 酒店名 + 地址用日文（taxi 溝通）
- 大使館電話（失蹤時用）

🆘 **緊急 kit**
- HK 保險聯絡 + 保單號碼
- 大阪家庭友善醫院清單（trip 前 research）
- 海外卡備份
- 兩個細路 1 套額外衫喺手提（行李延誤防護）

💼 **手提 vs 寄艙分裝策略**
- 寄艙：大量衫 + 手信
- 手提：1 套細路衫 + 藥物 + 1 日小食 + 貴重物品
- 萬一行李延誤你頂 24 小時無壓力

語氣：dad-mode 實用，唔過度過裝。
</PromptBlock>

### 4. 用呢個 prompt 出爆煲應變 + 夜晚大人時刻計劃

呢步唔顯眼但家庭 trip 成功關鍵：

<PromptBlock label="完整 prompt — 爆煲應變 + 大人時刻計劃">
[貼上你嘅家庭資料]

請整兩個計劃：

🆘 **爆煲應變劇本**

**5 歲仔型爆煲：**
- 觸發 A：餓 → 反應：即時小食（唔好談判）
- 觸發 B：攰 → 反應：BB 車 + 30 分鐘內返酒店
- 觸發 C：熱 → 反應：入最近冷氣空間 + 降溫噴霧
- 觸發 D：悶 / 等 → 反應：分心玩具 + 用未來雪糕做利誘
- 觸發 E：過度刺激 → 反應：安靜空間 + 30 分鐘睇 screen OK

**8 歲女型爆煲：**
- 觸發 A：妒忌細仔注意力 → 反應：每日專屬 1 對 1 1 個鐘（parent 輪流）
- 觸發 B：悶活動 → 反應：分組，畀佢吸引嘢做
- 觸發 C：餓 → 反應：同樣小食協議
- 觸發 D：熱 / 攰 → 反應：同 5 歲仔同樣
- 觸發 E：被剝奪 shopping → 反應：預先分配每日手信 HK$ 預算

**Parent 型爆煲**（唔好忽視）：
- 觸發 A：細路同時鬧脾氣 → 反應：輪流，唔羞愧離開 5 分鐘
- 觸發 B：行程滑坡 → 反應：放棄可選活動，唔推
- 觸發 C：預算焦慮 → 反應：Day 1 預先 commit 預算，中途唔重新計算
- 觸發 D：parent vs parent 不和 → 反應：埋單 5 分鐘，決定，唔好喺細路面前嘈

🍷 **大人時刻計劃（5 日家庭 trip 必需）**

**點解重要**：5 日高警戒 parenting 冇大人時刻 = 燒乾 = 第二日更差 parent。

**每日 30-60 分鐘大人時刻選擇：**

選擇 A（大部分日）：**瞓覺後溜出去**
- 細路 7:30-8pm 瞓覺
- 8-9pm：一個 parent 留低（如有酒店職員 / 1 個監察 app 用）
- 另一個 parent 單獨心齋橋行 + 1 杯飲
- Parent 每日輪流
- 9pm 前返酒店

選擇 B（Day 4 夜晚）：**聯合大人晚餐**
- 細路早食（5:30pm）家庭餐廳
- 一個 parent 7pm 帶細路返酒店
- 另一個 parent 安排 babysitter / 酒店兒童節目 OR 移細路早瞓 + 監察
- 大人聯合晚餐 7:30-9pm 行得到地方

選擇 C：**酒店內靜時間**
- 細路瞓緊 8pm 後
- 坐酒店露台 / 酒店酒吧
- 1 杯飲 + 同伴侶傾偈
- 30 分鐘鬆口氣

選擇 D：**休息日合作單獨時間**
- 一個 parent 帶細路「泳池日」
- 另一個 parent 單獨下午 shopping
- 第二日對調如想

🎯 **大人時刻基本規則**
- 提前排程，唔好「let's see if energy permits」（精力唔會 permit，硬推）
- 唔好比較你 trip 同單人 / 情侶旅客（唔同遊戲）
- 唔好 spend 大人時刻講細路後勤（傾夢想、新聞、八卦、其他任何嘢）

📷 **Trip 前家庭對齊時刻**
- Trip 前：家庭會議對齊期望
- 個個（包括 5 歲仔）名 1 樣「必做」嘢佢期待
- 承認每個人嘅揀
- 每日 30 秒 check-in：「你今日最鍾意嘅時刻？」
- 呢樣建立家庭回憶，唔淨係相

語氣：將 parent 壓力正常化 + 容許大人時間。
</PromptBlock>

呢個應變 + 大人時刻計劃比行程本身更關鍵。**Trip 成功 = 90% 精力管理 + 10% 觀光**。

## 變化例子

### 變化 1：單身 / 情侶 trip（冇細路）

冇細路限制嘅 trip 完全唔同：

<PromptBlock label="變化 1 — 單身 / 情侶 trip 行程">
我哋情侶去 [目的地] [N 日]，冇細路 / 老人家限制。

請整單身 / 情侶最優化行程：

🎯 **冇細路 trip 嘅自由**
- 較早開始時間 OK（冇瞓覺時間表）
- 較遲夜晚 OK（冇就寢時間）
- Skip 家庭餐廳
- 冒險試小眾本地位
- 戶外長行可接受
- 遲晚餐 OK

📐 **每日結構（vs 家庭模式）**
- 早上：進取觀光開始
- 午餐：自由 — 隨意街頭食 / 小眾餐廳
- 下午：繼續或休息由你心情
- 夜晚：遲晚餐 + 酒吧 / club / 夜市
- 夜：遲返可接受

✅ **適合情侶嘅活動類型**
- 烹飪班 / 文化體驗
- 行山 / 戶外探險
- 博物館 + 美術館（容許長觀賞）
- 早上本地市場
- 深夜居酒屋觀察本地生活

💑 **情侶專屬考慮**
- 1 日各自「自己揀」（畀伴侶完整 1 日主導權）
- 1 餐「揮霍」體驗（情侶回憶錨點）
- 1 個「本地浸沒」（民宿 / 班 / 義工）
- 避免：衝過每個 check-list 項目（休息 = 旅遊質素）

📊 **預算分配唔同**
- 較高 % 喺體驗（烹飪班 / 私人團）
- 較高 % 喺食（值得情侶投資）
- 較低 % 喺交通（少行李拖）

🚫 **陷阱**
- 情侶但每日 100% 一齊 → 過度飽和
- 排程半日各自單獨係健康

語氣：放縱 + 自主。
</PromptBlock>

### 變化 2：老人家 trip（行得較慢 + 醫療考慮）

帶父母 / 公公婆婆出 trip 完全唔同動態：

<PromptBlock label="變化 2 — 老人家父母 trip 規劃">
我帶 [我父母 65 + 70 / 公公婆婆 75+] 出 trip [目的地] [N 日]。佢哋 [行路速度 + 醫療狀況 + 飲食]。

請整老人家父母最優化行程：

🏥 **老人家專屬出發前準備**
- 家庭醫生 medical clearance
- 全面旅遊保險連老人保障
- 藥物 1.5 倍劑量（買額外）
- 處方醫生信（手提）
- 病歷總結用英文（緊急海外醫院如用）
- 國際 SOS 會籍考慮

⏰ **節奏調整**
- 行路速度：年輕人嘅 50-60%
- 每日 1 個主要活動（最多 2 個）
- 休息排程：每 90 分鐘
- 避免樓梯多嘅地點
- 升降機 / 自動梯偏好

🚶 **行動規劃**
- 酒店：升降機 + 可達廁所
- 交通：避免 8 個鐘車程，偏好火車 + taxi
- 活動鄰近：聚埋附近減少車程
- 行路鞋關鍵（新鞋唔好出 trip 試）
- 隨身助行器 / 拐杖（如適用）

🍽 **飲食考慮**
- 糖尿 / 高血壓專屬 menu
- 落單時要求少鹽 / 少油
- 熟悉食物錨點每日 1-2 餐（飯 + 簡單熟菜）
- 避免生海鮮 / 街邊未經驗證
- 腸胃不適反應 kit
- 茶偏好（年長 parent 鍾意一到熱茶）

🌡 **氣候考慮**
- 避免極熱 / 極凍目的地
- 80°F 對好多年長人唔舒服
- 浸溫泉 / 溫泉要注意心血管狀況
- 高原目的地（西藏 / 高海拔秘魯）有心血管風險

💊 **每日例行維持**
- 朝藥同屋企同一時間
- 排便習慣維持（1-2 日中斷正常但 3+ 日要警覺醫療）
- 瞓覺時間表：早瞓早起
- 每日行（15-30 分鐘）維持

🎯 **活動配對**
- 鍾意嘅老人活動：
  - 古蹟 / 神廟（坐 + 欣賞 OK）
  - 花園行
  - 遊船河（坐住 + 望）
  - 烹飪班（較輕版）
  - 溫泉（連醫療 OK）
  - 茶道
- 唔啱：
  - 高刺激（主題公園）
  - 進取行山
  - 長途機冇休息日
  - 冒險食物

🆘 **緊急規劃**
- 酒店職員知老人特別需要
- 最近醫院 trip 前認清
- 本地語緊急用語保存喺電話
- 家人緊急聯絡 24/7
- 大使館 / 領事館聯絡保存

🤝 **跨代時刻**
- 每日排程 1 個「故事時間」（老人講你 / 你細路嘅過去）
- 同 parent 喺有意義位置影相
- 帶 parent 揀 1 個手信畀孫（參與任務）
- 唔好理所當然 parent - 呢個 trip 可能有限

語氣：尊重 + 照顧但唔屈尊。

## 一個思維框架

5 日家庭 trip 嘅成功指標唔係「我哋打卡晒所有景點」——係 **5 日後返港，全屋企人有美好回憶 + 想再 trip together**。

呢個指標嘅含義：家庭 trip 規劃嘅敵人唔係 missed 景點，係 **累積爆煲侵蝕家庭感情**。Day 3 全屋企人熱 + 攰 + 餓嘈交嘅 90 分鐘，比 missed 京都嘅失望影響更深。

呢個思維嘅轉變：家庭行程唔係「最優化景點數量」——係 **同時滿足 4 個唔同需要嘅 window**。每日有 1 個大人亮點 + 1 個細路定海神針 + 兩者都鍾意嘅餐 + 休息 = 成功。

Claude 嘅角色喺家庭 trip：你單獨估計多人偏好嘅能力有限。Claude 幫你有系統咁將全家偏好 + 限制寫晒，再出一個深思熟慮嘅行程。**唔係完美行程 — 係尊重每個家庭成員嘅深思熟慮起點**。

最後提醒：

- ✅ **行程唔需要 100% 跟**。家庭 trip 本質彈性，預 30% buffer 畀細路心情 / 天氣 / 突發。
- ⚠️ **每日 30 分鐘大人時刻唔係 optional**。Skip 一日你 OK，連續 5 日略過等於燒乾。
- 🎯 **累積回憶 > 每日效率**。8 歲女 + 5 歲仔 5 年後唔記得行程細節，但記得「我哋一齊去大阪嗰個 8 月，5 歲仔喺奈良餵鹿笑到趴喺地」。為嗰個時刻設計。

一路平安，屋企人。

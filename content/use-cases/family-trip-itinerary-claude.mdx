## 情境

你計劃緊 8 月家庭 trip 去大阪 5 日 4 夜。

家庭組成：

- 你（38，市場推廣經理）
- 伴侶（40，會計師）
- 8 歲女（升小三，鍾意 Disney + Universal）
- 5 歲仔（K2，專注力最多 30 分鐘，爆煲觸發包括「行得耐」/「肚餓」/「熱」）

你嘅 trip 偏好：

- 食日本嘢 ramen / sushi / yakiniku
- 行心齋橋 + 道頓堀 + 美食街
- 1 日 USJ
- 1 日京都 day trip
- 奈良鹿如果可以加埋

你伴侶嘅偏好：

- 避免過度趕
- 細路爆煲 = 全家毀咗成日
- 預每日 1-2 個鐘休息時間
- 現金流唔好超預算

8 歲女嘅偏好：

- USJ 全日（Harry Potter + Minions + Hello Kitty）
- 買 K-pop / 玩具
- 食雪糕 / 士多啤梨甜品

5 歲仔嘅偏好：

- 坐火車（最鍾意新幹線）
- 公園跑 + 玩水
- 食 udon / 薯條 / 飯團
- 唔肯食生嘢
- 行路 30 分鐘為頂

你試自己列 5 日行程：

- Day 1：抵步 + 心齋橋
- Day 2：USJ
- Day 3：京都 + 奈良
- Day 4：大阪城 + 大阪城 area
- Day 5：心齋橋 + 返港

你望住個 list 半個鐘，發現問題：

- **Day 2 USJ**：8 歲女 + 你想玩晒 11 個 ride，5 歲仔玩唔到 7 個。佢點頂得住全日？
- **Day 3 京都 + 奈良**：一日來回對 5 歲仔太趕。坐新幹線 + 京都古蹟 + 奈良鹿 = 8 個鐘以上戶外 = 爆煲風險 100%
- **每日食飯**：你想試 ramen / sushi / yakiniku；細路淨係食 udon / 飯團。你哋分開食，定係揾兩者都得嘅餐廳？
- **精力管理**：5 歲仔 11am 準時肚餓爆煲；2pm 後午睡時段；4pm 後返到啲精神但好易爆煲

呢個家庭 trip 計劃嘅死結：**4 個旅客 × 4 個唔同需要 = 行程 90% 嘅衝突張力**。

你預設邊個照顧？

- 8 歲女 + 大人想 = 5 歲仔受苦
- 5 歲仔做定海神針 = 8 歲女悶 → 自己爆煲
- 對半分 = 全日嘈交

呢篇用 Claude **將全家偏好 + 限制寫晒，出平衡 5 日行程**——每日有大人亮點 + 細路定海神針 + 餐 + 休息時間，減少爆煲機會。

🎈 **保持理智嘅工具（畀大人）**
- 耳塞（細仔喺酒店興奮到瞓唔着時用）
- 1 罐紅酒 / 飲品喺機場買（細路瞓咗之後嘆）
- 有聲書 / podcast，喺交通工具上隔走細仔嘅嘈音
- 旅行枕

📋 **科技 / 連線**
- 1 個 pocket wifi（覆蓋 4 個裝置）
- 或每部電話有 eSIM 日本 plan
- 酒店名 + 地址用日文（taxi 溝通）
- 大使館電話（失蹤時用）

🆘 **緊急 kit**
- HK 保險聯絡 + 保單號碼
- 大阪家庭友善醫院清單（trip 前搵定）
- 海外卡備份
- 兩個細路 1 套額外衫喺手提（行李延誤防護）

💼 **手提 vs 寄艙分裝策略**
- 寄艙：大量衫 + 手信
- 手提：1 套細路衫 + 藥物 + 1 日小食 + 貴重物品
- 萬一行李延誤，你都頂得住 24 小時冇壓力

語氣：老竇模式咁實用，唔好帶太多嘢。
</PromptBlock>

### 4. 用呢個 prompt 出爆煲應變 + 夜晚大人時刻計劃

呢步唔顯眼但家庭 trip 成功關鍵：

<PromptBlock label="完整 prompt — 爆煲應變 + 大人時刻計劃">
[貼上你嘅家庭資料]

請整兩個計劃：

🆘 **爆煲應變劇本**

**5 歲仔型爆煲：**
- 觸發 A：餓 → 反應：即時小食（唔好談判）
- 觸發 B：攰 → 反應：BB 車 + 30 分鐘內返酒店
- 觸發 C：熱 → 反應：入最近冷氣空間 + 降溫噴霧
- 觸發 D：悶 / 等 → 反應：用玩具分散注意力 + 用陣間有雪糕做利誘
- 觸發 E：過度刺激 → 反應：揾個安靜空間 + 畀佢睇 30 分鐘螢幕

**8 歲女型爆煲：**
- 觸發 A：妒忌大人錫晒細仔 → 反應：每日有 1 個鐘專屬一對一時間（大人輪流）
- 觸發 B：活動悶 → 反應：分組，畀佢做啲吸引佢嘅嘢
- 觸發 C：餓 → 反應：同樣即刻畀小食
- 觸發 D：熱 / 攰 → 反應：同 5 歲仔做法一樣
- 觸發 E：唔畀 shopping → 反應：預先分配每日手信 HK$ 預算

**大人型爆煲**（唔好忽視）：
- 觸發 A：細路同時鬧脾氣 → 反應：輪流，離開 5 分鐘唔使內疚
- 觸發 B：行程開始崩 → 反應：放棄可選活動，唔好硬推
- 觸發 C：預算焦慮 → 反應：Day 1 預先定好預算，中途唔好再計過
- 觸發 D：兩個大人意見不合 → 反應：埋位傾 5 分鐘，做個決定，唔好喺細路面前嘈

🍷 **大人時刻計劃（5 日家庭 trip 必需）**

**點解重要**：5 日高度戒備咁湊仔，冇大人時刻 = 燒乾 = 第二日更加做唔到個好家長。

**每日 30-60 分鐘大人時刻選擇：**

選擇 A（大部分日子）：**細路瞓咗就溜出去**
- 細路 7:30-8pm 瞓覺
- 8-9pm：一個大人留低（如有酒店職員 / 用個監察 app）
- 另一個大人自己去心齋橋行 + 飲一杯
- 兩個大人每日輪流
- 9pm 前返酒店

選擇 B（Day 4 夜晚）：**夫婦一齊食晚餐**
- 細路早食（5:30pm）家庭餐廳
- 一個大人 7pm 帶細路返酒店
- 另一個大人安排 babysitter / 酒店兒童節目，或者放細路早瞓再睇住佢哋
- 兩公婆一齊食晚餐 7:30-9pm，揀個行得到嘅地方

選擇 C：**酒店內靜時間**
- 細路 8pm 後瞓緊
- 坐酒店露台 / 酒店酒吧
- 飲一杯 + 同伴侶傾偈
- 30 分鐘鬆口氣

選擇 D：**休息日各自抖一抖**
- 一個大人帶細路過「泳池日」
- 另一個大人自己去下午 shopping
- 第二日想對調都得

🎯 **大人時刻基本規則**
- 提前排好，唔好諗住「睇下有冇精力先」（永遠都唔會有精力，要硬推）
- 唔好攞你嘅 trip 同單人 / 情侶旅客比較（玩法唔同）
- 唔好喺大人時刻講細路嘅後勤嘢（傾下夢想、新聞、八卦、咩都得）

📷 **Trip 前家庭對齊時刻**
- Trip 前：家庭會議對齊期望
- 個個（包括 5 歲仔）名 1 樣「必做」嘢佢期待
- 承認每個人嘅揀
- 每日 30 秒傾兩句：「你今日最鍾意嘅時刻係咩？」
- 呢樣建立家庭回憶，唔淨係影相

語氣：將家長壓力正常化 + 容許大人時間。
</PromptBlock>

呢個應變 + 大人時刻計劃，比行程本身更關鍵。**Trip 成功 = 90% 精力管理 + 10% 觀光**。

## 變化例子

### 變化 1：單身 / 情侶 trip（冇細路）

冇細路限制嘅 trip 完全唔同：

<PromptBlock label="變化 1 — 單身 / 情侶 trip 行程">
我哋情侶去 [目的地] [N 日]，冇細路 / 老人家限制。

請整單身 / 情侶最優化行程：

🎯 **冇細路 trip 嘅自由**
- 早啲開始都得（冇瞓覺時間表）
- 夜啲都得（冇就寢時間）
- 唔使去家庭餐廳
- 大膽啲試啲小眾本地位
- 戶外長行可接受
- 遲食晚餐 OK

📐 **每日結構（同家庭模式比較）**
- 早上：一早就開始密集觀光
- 午餐：自由 — 隨意食街頭嘢 / 小眾餐廳
- 下午：繼續定休息，睇你心情
- 夜晚：遲食晚餐 + 酒吧 / club / 夜市
- 夜：夜啲返都得

✅ **適合情侶嘅活動類型**
- 烹飪班 / 文化體驗
- 行山 / 戶外探險
- 博物館 + 美術館（容許長觀賞）
- 早上本地市場
- 深夜居酒屋觀察本地生活

💑 **情侶專屬考慮**
- 一日各自「自己揀」（畀伴侶完整一日話事權）
- 一餐「奢侈」體驗（情侶嘅回憶錨點）
- 一個「本地浸沉體驗」（民宿 / 學堂 / 義工）
- 避免：衝晒清單上每個項目（識得休息 = 旅遊質素）

📊 **預算分配唔同**
- 較高 % 放喺體驗（烹飪班 / 私人團）
- 較高 % 放喺食（情侶值得投資）
- 較低 % 放喺交通（行李少，唔使拖）

🚫 **陷阱**
- 情侶但每日 100% 黐埋一齊 → 太密太焗
- 排一個半日各自抖一抖係健康嘅

語氣：放縱 + 自主。
</PromptBlock>

### 變化 2：老人家 trip（行得較慢 + 醫療考慮）

帶父母 / 公公婆婆出 trip 完全唔同動態：

<PromptBlock label="變化 2 — 老人家父母 trip 規劃">
我帶 [我父母 65 + 70 / 公公婆婆 75+] 出 trip [目的地] [N 日]。佢哋 [行路速度 + 醫療狀況 + 飲食]。

請整老人家父母最優化行程：

🏥 **老人家專屬出發前準備**
- 家庭醫生開出嘅適航證明
- 全面旅遊保險連老人保障
- 藥物帶 1.5 倍劑量（多買啲）
- 醫生處方信（隨身手提）
- 病歷總結用英文寫（緊急時喺海外醫院用）
- 考慮加入國際 SOS 會籍

⏰ **節奏調整**
- 行路速度：年輕人嘅 50-60%
- 每日 1 個主要活動（最多 2 個）
- 休息安排：每 90 分鐘一次
- 避免樓梯多嘅地點
- 盡量揀有升降機 / 扶手電梯嘅

🚶 **行動規劃**
- 酒店：有升降機 + 廁所易去
- 交通：避免 8 個鐘車程，盡量揀火車 + taxi
- 活動就近安排：聚埋喺附近，減少車程
- 行路鞋好緊要（新鞋唔好留到出 trip 先試）
- 隨身助行器 / 拐杖（如適用）

🍽 **飲食考慮**
- 糖尿 / 高血壓專屬 menu
- 落單時要求少鹽 / 少油
- 每日 1-2 餐食佢哋熟悉嘅嘢（飯 + 簡單熟菜）做定心丸
- 避免生海鮮 / 街邊冇保證嘅嘢
- 帶定腸胃不適嘅藥
- 茶嘅喜好（年長嘅父母一到就鍾意有杯熱茶）

🌡 **氣候考慮**
- 避免極熱 / 極凍目的地
- 80°F 對好多年長人嚟講都唔舒服
- 浸溫泉要注意心血管狀況
- 高原目的地（西藏 / 高海拔秘魯）有心血管風險

💊 **維持每日例行**
- 朝早食藥同喺屋企同一時間
- 維持排便習慣（亂咗 1-2 日正常，但超過 3 日就要留意身體）
- 瞓覺時間表：早瞓早起
- 維持每日行下（15-30 分鐘）

🎯 **活動配對**
- 鍾意嘅老人活動：
  - 古蹟 / 神廟（坐 + 欣賞 OK）
  - 花園行
  - 遊船河（坐住 + 望）
  - 烹飪班（較輕版）
  - 溫泉（連醫療 OK）
  - 茶道
- 唔啱：
  - 高刺激（主題公園）
  - 辛苦嘅行山
  - 長途機後冇休息日
  - 試新奇食物

🆘 **緊急規劃**
- 畀酒店職員知道老人嘅特別需要
- 出 trip 前認清最近嘅醫院
- 將當地語言嘅緊急用語存喺電話
- 24/7 家人緊急聯絡
- 存定大使館 / 領事館聯絡

🤝 **跨代時刻**
- 每日排 1 個「故事時間」（老人家講你 / 你細路嘅過去）
- 同父母喺有意義嘅地方影相
- 帶父母揀 1 件手信畀孫（畀佢哋參與）
- 唔好當父母係必然 - 呢個 trip 可能係難得

語氣：尊重 + 照顧但唔好擺出高高在上嘅姿態。

- 夜晚飲品 / 觀星 / 深夜傾偈

## 一個思維框架

5 日家庭 trip 嘅成功指標唔係「我哋打卡晒所有景點」——係 **5 日後返港，全屋企人有美好回憶 + 想再一齊去 trip**。

呢個指標嘅含義：家庭 trip 規劃嘅敵人唔係去漏咗景點，係 **累積嘅爆煲侵蝕緊家庭感情**。Day 3 全屋企人又熱又攰又餓嘈交嘅 90 分鐘，比起去漏咗京都嗰種失望，影響更深。

呢個思維嘅轉變：家庭行程唔係「最優化景點數量」——係 **同時滿足 4 個唔同需要嘅時機**。每日有 1 個大人亮點 + 1 個細路定海神針 + 兩者都鍾意嘅餐 + 休息 = 成功。

Claude 喺家庭 trip 嘅角色：你一個人去估計咁多人嘅偏好，能力始終有限。Claude 幫你有系統咁將全家偏好 + 限制寫晒，再出一個諗得周到嘅行程。**唔係話佢係完美行程 — 而係一個尊重每個家庭成員、諗得周到嘅起點**。

最後提醒：

- ✅ **行程唔需要 100% 跟足**。家庭 trip 本質就係要彈性，預 30% 緩衝畀細路心情 / 天氣 / 突發。
- ⚠️ **每日 30 分鐘大人時刻唔係可有可無**。略過一日冇問題，連續 5 日略過就等於燒乾。
- 🎯 **累積回憶 > 每日效率**。8 歲女同 5 歲仔 5 年後唔會記得行程細節，但會記得「我哋一齊去大阪嗰個 8 月，5 歲仔喺奈良餵鹿笑到趴喺地」。為嗰個時刻設計。

一路平安，屋企人。

## 情境

你主子（隻 8 歲短毛貓）平時食 80g Royal Canin Indoor，過去 5 年都係咁。

過去 4 日：

- **Day 1**：碗仲剩 30g。你以為佢前一晚零食食得多。
- **Day 2**：剩 50g。你聞吓隻碗——食物正常，無變壞。
- **Day 3**：30g 都唔郁。你開罐 churu，佢秒食。
- **Day 4**：朝早搵到一灘黃色嘔吐物，無毛球，半流質。佢匿喺床下底。

你開始驚。Google 搜「貓唔食 4 日」⋯⋯95% 結果係「即刻送獸醫，可能腎衰」，5% 結果係「貓正常一日唔食」。兩極到完全幫唔到你。

呢類唔上唔落嘅情況最折磨——可能係：

- **醫療問題**：牙痛 / 早期 CKD / 胰臟炎 / 毛球 / 感染
- **行為問題**：扭計王 / 轉季胃口差 / 食物突然改配方 / 壓力（新傢俬／搬屋／家中新人）
- **環境問題**：碗位變咗 / 旁邊有新氣味 / 食物擺太耐氧化
- **主人問題**：你上一星期餵多咗，而家佢自己調節返

如果係醫療 → 即刻去獸醫；如果係行為 → 7 日轉糧，唔好硬轉；如果係環境 → 換碗位 / 換食物時間試吓；如果係主人 → 冷靜，觀察。

呢篇用 Claude **10 分鐘**先分辨原因，如果需要再設計一個 7 日轉糧計劃，少啲「硬轉糧搞到主子嘔」嘅情況。

⚠️ Claude 唔係取代獸醫。如果有危險病徵（嘔血 / 完全唔飲水 / 企唔起身 / 連續 48 小時零卡路里），**直接去獸醫**，呢篇計劃唔適用。

## 跟住做

### 1. 寫低基準 + 觀察

打開 Claude Chat。將你最原始嘅觀察照寫低：

> 主子（8 歲 male neutered cat, 4kg, indoor only, FIV-negative, 上次體檢 6 個月前 normal）。
> 
> 平日食量：每日 80g Royal Canin Indoor Adult + 1 罐 Hill's i/d 濕糧（傍晚）。每日 1.5 盤水。屎正常，一條條實淨 1 次 / 日，尿正常 2-3 次 / 日。
> 
> 過去 4 日：
> - Day -3：剩 30g 乾糧。濕糧食一半。
> - Day -2：剩 50g。濕糧食一半。
> - Day -1：30g 唔郁。churu 秒食。
> - Today：朝早嘔黃色半流質一灘。匿床底。
> 
> 體重：上週末量 4kg，今朝 3.85kg。
> 
> 屋企變化：上週末買咗新地毯。其他作息一樣。

### 2. 用呢個 prompt — 先分辨原因

<PromptBlock label="完整 prompt — 挑食 / 食量異常拆解">
我主子（[填 species / age / weight / sex / neutered status / indoor or outdoor / 慢性病]）食量出現以下變化：

[貼上你嘅 timeline + 平日觀察筆記]

我希望你幫我有系統咁拆解原因：

**🚨 Part 1：危險警號（呢 5 樣有任何一樣 = 即去獸醫，唔好跟呢個計劃）**
- [ ] 連續 48 小時零卡路里（貓尤其危險 — 脂肪肝 hepatic lipidosis）
- [ ] 嘔吐含血 / 黑色（咖啡渣樣）
- [ ] 完全唔飲水 24 小時
- [ ] 黃疸（眼白 / 牙肉 / 耳內發黃）
- [ ] 企唔起身 / 跳唔到

如果以上有任何一樣，停手，直接去獸醫，唔好用呢個計劃。

**🩺 Part 2：醫療可能性（3-5 個最有機會嘅原因）**
基於我嘅描述，呢啲係獸醫通常會評估嘅原因（按可能性排）：
- 病名 — 點樣對得上我主子嘅情況 + 1 個喺屋企觀察到嘅跡象去確認 / 排除
- ...

**🧠 Part 3：行為 / 環境可能性**
非醫療嘅合理原因：
- 行為（扭計 / 食物改配方 / 轉季胃口差）
- 環境（新傢俬／搬屋／陌生人）
- 主人引致（餵太多 / 零食過量 / 時間表變動）

**🎯 Part 4：48-72 小時在家觀察計劃**
（前提：危險警號都冇）
- 觀察邊 3 個跡象 + 點量化
- 加 / 唔加邊種引誘嘢食（溫水濕糧 / 魚湯 / churu）
- 飲水量點記錄
- 屎尿次數記錄

**🚦 Part 5：判斷流程**
- 如果觀察 24h：[條件] → 行動
- 如果觀察 48h：[條件] → 行動
- 如果觀察 72h：[條件] → 去獸醫
- 如果惡化：→ 即去獸醫

語氣：似一個有同理心嘅獸醫護士兼貓主人，唔好淨係怪我餵太多講說教，幫我整理觀察。
</PromptBlock>

### 3. 如果係行為 / 環境問題 → 7 日轉糧計劃

如果 Part 2 醫療風險低，而你想主動試**轉糧 / 加水分**（獸醫推薦 / 你自己決定）：

<PromptBlock label="完整 prompt — 7 日漸進轉糧計劃">
我主子（[species / age / weight / current health]）需要由：

- **目前糧**：[brand / line / dry or wet]
- **目標糧**：[brand / line / dry or wet / 點解轉，例：vet 推薦 prescription kidney diet / 我想升級質素 / current 糧改配方出事]

請整 **7 日漸進轉糧 plan**：

**📅 每日時間表**
Day 1-2: 75% 舊 + 25% 新
Day 3-4: 50% 舊 + 50% 新
Day 5-6: 25% 舊 + 75% 新
Day 7+: 100% 新

每日：
- 朝早份量 (g / kcal)
- 夜晚份量
- 兩溝點準備（加暖水？溝勻？）
- 點量度食量

**🎯 成功指標（觀察邊 3 樣）**
- 食量（返唔返到平日嘅 80%+）
- 屎狀態（轉糧期間正常會略爛 1-2 日）
- 精神 / 行為（肯唔肯玩）

**🚦 停止 / 減慢指標**
- 如果 Day 2 嘔 → 停，食返 100% 舊糧，等 3 日再試慢啲（每次加 10%）
- 如果 Day 3 完全唔食 → 打去搵獸醫
- 如果屎連續 3 日水狀 → 停新糧

**💡 應付挑食小貼士**
- 叮暖食物到體溫（38-39°C）— 唔好叮太耐
- 溝少少濕糧 / churu / 吞拿魚水引誘
- ⚠️ 避免：針筒強行灌食（會傷食道）/ 餓到佢屈服（貓真係會餓死自己）

**🍴 長遠計劃**
轉糧成功之後再觀察 4 星期：
- 體重穩唔穩定
- 屎質素
- 毛髮狀態（貓 4 星期可見）
- 精神

**🐱 vs 🐕 物種提醒**
（如果主子係貓 → 強調脂肪肝風險、唔可以斷食）
（如果主子係狗 → 強調突發腸胃不適風險、可加益生菌補充品）

語氣：似一個有耐性嘅貓 / 狗主人朋友。
</PromptBlock>

### 4. 7 日每日報到

每日食完餐之後用 30 秒打兩句：

> 「Day 3。今朝食咗 75% 份量，夜晚食 50%。屎正常，少少軟。精神 OK，主動跳上梳化。」

Claude 評估 → 「症狀喺預期之內，繼續按計劃進行，聽日留意大便」

如果出問題 → 啟動停止 / 減慢機制。

## 變化

### 變化 1：處方 / 醫療糧嘅嚴格遵從

如果獸醫處方咗醫療糧（KD 保腎 / HD 保心 / DM 糖尿 / IBD 等）：

<PromptBlock label="變化 1 — 處方糧嚴格遵從 + 安全零食清單">
我主子（[慢性病]）開始食 [處方糧名]。

獸醫指示：
- 嚴格飲食 — 唔可以溝其他食物
- 每日份量: [g / kcal]
- 期限: [試驗期 / 終生]

問題：
- 主子唔肯食呢隻處方糧（味唔啱）
- 屋企人成日想偷偷餵零食
- 我自己心痛佢食得少

請幫我整：

**Part 1：執行現實核對**
- 呢個處方糧嘅原理點解一定要嚴格執行（用我隻主子嘅病嚟解，唔好行貨）
- 偷偷餵零食嘅實際醫療風險（可以的話量化）

**Part 2：點樣令唔願食嘅主子食處方糧**
- 推薦轉糧速度（普通要 7 日，處方糧可能要 14-21 日）
- 溫度 / 改變質感技巧
- 同一個處方系列嘅濕糧、乾糧、肉泥逐樣試
- 如果完全拒絕 → 同獸醫傾其他牌子（同原理但唔同味）

**Part 3：屋企人共識**
我嘅伴侶 / 阿媽 / 工人姐姐成日偷偷餵主子嘢食（覺得餓親主子好可憐）。請幫我：
- 寫一段簡短說明解釋處方糧重要性，畀我畀屋企人睇
- 3 條可接受界線（例如：每日最多 1 粒處方零食，指定牌子）
- ⚠️ 絕對唔可以畀主子食嘅名單（取決於病種）

**Part 4：試驗期評估**
獸醫通常 6-8 星期會跟進評估。請整：
- 期間我每日記錄邊 5 樣（胃口 / 精神 / 體重 / 屎尿 / 針對病徵嘅指標）
- 試驗期完之後，帶去見獸醫嘅數據總結點準備

語氣：要有同理心 — 我知強迫主子食佢唔鍾意嘅嘢會心痛，但醫療上係必要。
</PromptBlock>

### 變化 2：自家煮鮮食 diet planning

你想幫主子半自煮 / 全自煮：

<PromptBlock label="變化 2 — 自家鮮食可行性 + 食譜框架">
我考慮自家煮主子嘅飯，由：

- [每日幾多杯 / 罐商業糧]
- 轉去：[半自煮 / 全自煮]

我嘅動機：[控制質素 / 主子過敏 / 驚商業糧回收 / 興趣 / 成本]

我煮飯嘅能力：[每日 / 每週煮定一批 / 每月]

請做現實評估：

**Part 1：老實可行性評估**
- 貓同狗嘅自煮複雜度差異（貓對牛磺酸 / 精氨酸極度敏感，狗寬容啲）
- 你嘅投入程度配唔配自煮（每日新鮮 vs 每週煮定一批嘅儲存風險）

**Part 2：必須補充嘅微量營養素**
- 唔加就一定會缺乏嘅（貓嘅牛磺酸 / 狗嘅維他命 D / 鈣磷比例）
- 推薦嘅綜合維他命 / 單一營養補充品（籠統講種類，唔推銷牌子）
- 邊度買（香港寵物店 / 海外網購）

**Part 3：每週基本食譜框架**
- 蛋白質來源比例（貓 60-70% / 狗 40-50%）
- 碳水可接受範圍
- 油脂 + 必需脂肪酸
- 蔬菜 / 纖維（貓少 / 狗就無問題）

**⚠️ Part 4：絕對唔得**
- 對貓 / 狗致命嘅食材（朱古力、洋蔥、提子、木糖醇、貓嘅生骨頭等）
- 儲存規則（雪櫃下格 3 日 / 冰格 1 個月）
- 烹調溫度（肉類一定要全熟，避免沙門氏菌 / 寄生蟲）

**Part 5：6 星期試驗計劃**
- 起步先小量（25% 鮮食 + 75% 商業糧）轉 2 星期
- 觀察 5 個指標（屎 / 體重 / 毛髮 / 精神 / 食慾）
- 6 星期見獸醫嗰陣應該驗咩血

**🚨 Part 6：幾時要停**
- 任何營養缺乏跡象（貓心臟問題 / 狗骨骼問題）→ 即刻轉返商業糧
- 如果你煮到透支 → 千祈唔好求其，要轉返商業糧（求其煮比純商業糧更危險）

語氣：撐你嘅動機但保護主子嘅健康，唔好盲推鮮食亦唔好潑冷水。
</PromptBlock>

### 變化 3：多寵物混食問題（貓狗互偷嘢食）

你屋企有貓有狗（或者兩隻貓各有需求），佢哋成日偷食對方啲糧：

<PromptBlock label="變化 3 — 多寵物餵食安排">
我屋企：
- 主子（貓）— 食 [brand / 特殊需求 e.g. 處方腎臟糧]
- Coco（狗）— 食 [brand / 特殊需求 e.g. 體重管理糧]

問題：
- 互相偷食（貓食狗糧 / 狗食貓糧）
- 雙方需求唔同，但餵食時間 / 地點難分開

請整：

**Part 1：互食嘅健康風險**
- 貓食狗糧 → 長期牛磺酸 / 精氨酸不足，心臟風險
- 狗食貓糧 → 蛋白質過高 → 腎臟負擔 + 肥胖
- 針對你哋情況：主子腎病 + Coco 減肥 = 錯配風險極高

**Part 2：物理分隔策略**
- 飯碗高度 / 位置（貓可以跳，喺高處餵貓 / 狗喺地下）
- 錯開時間餵（餵完一隻全部食完先放第二隻出嚟）
- 晶片感應餵食器（香港寵物店有冇現貨，推薦種類、大約預算）
- 分房（唔同房餵 + 閂門）

**Part 3：每隻日常時間表（總時間表）**
畀我 1 個 7am-10pm 時間表，列明：
- 邊個時間餵邊隻
- 收碗時間
- 邊隻可以長擺長食，邊隻要控制份量
- 邊個時段要睇住

**Part 4：寵物保姆分隔指引**
精簡一段，保姆入嚟睇得明個分隔指引（呼應 issue 051）

**Part 5：⚠️ 常見錯誤**
- 兩隻都長擺長食 = 包保互食
- 太信「主子識自己唔食 Coco 啲嘢」（寵物唔識嘅）
- 用高度分隔但狗學識爬上去

語氣：貼地咁講多寵物家庭嘅現實考量，唔好理想化。
</PromptBlock>

## 一個心態

寵物嘅胃口係**最敏感嘅早期訊號**——比體重、活動量、屎尿都更早睇到變化。一隻貓突然唔食，可以喺 48-72 小時急轉直下變成脂肪肝；一隻狗突然嘔足幾日，可能係胃扭轉（外科急症）。

但同時，**寵物挑食都係日常現實**——換牌子改配方、轉季胃口變、屋企新傢俬有壓力——好多時觀察 48 小時就會回復正常。

Claude 嘅角色唔係斷症，係**幫你分辨「值得驚」定「耐心觀察」**。危險病徵 → 獸醫。冇危險病徵 → 有系統觀察 + 漸進介入。

最後提一提：

- ✅ 任何嘔吐含血 / 黃疸 / 企唔起身 / 48 小時零卡路里 = 直接送急症獸醫
- ✅ 轉糧永遠要漸進（最少 7 日，處方糧 14-21 日）
- ❌ 餓到主子屈服先肯食新糧 — 貓真係會餓死自己
- ❌ 針筒強行灌食唔識吞嘅貓 — 會傷食道
- 🐾 主子挑食可能只係扭計王 — 但你嘅冷靜，永遠大過你嘅慌亂

主子食得開心，你食得安樂。

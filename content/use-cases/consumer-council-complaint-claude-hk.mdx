## 情境

你預繳咗 $28,000 嘅美容療程，做咗 3 次，第 4 次門市話「療程師辭咗職，等通知」。等咗兩個月，門市冇人接電話。

或者：你訂咗婚紗 package $58,000，sales 落單時口頭話「包當日 6 小時 outdoor + 50 張精修」，到簽合約嗰刻被加錢加到 $72,000，仲話「精修 20 張，多影另計」。

又或者：你喺旅行社訂咗一個夏天嘅 package tour $24,000，出發前 3 個禮拜旅行社突然 email「業務調整，全數退款處理中」——之後就冇人覆。

呢類爭議嘅特點：**金額大到肉痛，但細到搵律師唔抵**。十居其九嘅人放棄，自認倒霉。

但香港其實有一條升級階梯：

- **消委會** —— 調解，免費，唔判決但有優勢
- **海關** —— 商品說明條例 (TDO) 刑事檢控，針對虛假商品說明 / 誤導性遺漏
- **小額錢債審裁處** —— 申索額 $75,000 以下，無需律師
- **區域法院** —— 申索額 $75,000 至 $3,000,000

呢篇用 Claude 35 分鐘做兩件事：拆清楚件事屬於邊一條路，然後寫返封企得起身嘅正式投訴。

> 利益申報：呢個唔取代律師。涉及金額過 $75,000、涉及人身傷害、或者商戶有刑事詐騙嫌疑（拐帶款項潛逃），直接搵律師 + 報警。Claude 嘅輸出係幫你**喺正確渠道講清楚件事**，攞返錢嘅機會視乎商戶仲有冇償債能力——預繳商戶執笠之後嘅追討成功率老老實實好低。

## 跟住做

### 1. 齊嗮證據先（10 分鐘）

未開 prompt 之前，手頭整理：

- **合約 / 服務確認書**（紙本 / email / WhatsApp 截圖都算）—— 揾出口頭承諾同書面條款之間嘅落差
- **付款紀錄**（信用卡月結單、銀行轉賬截圖、收據）—— 證明你已經畀錢
- **服務交付紀錄**（已做咗幾多次、剩低幾多次、未交付項目）
- **商戶宣傳材料**（網站截圖、IG post、單張、sales 展示嘅簡報）—— 證明宣傳同實際嘅落差
- **與商戶嘅溝通紀錄**（WhatsApp、email、通話錄音如有）—— 證明你已嘗試合理解決
- **同類個案**（消委會公開個案、新聞報導、討論區帖文）—— 證明唔係單一事件

冇齊唔緊要，照 prompt，Claude 會話你知缺咗邊樣會影響邊一條路。

### 2. 用呢個 prompt 拆法律角度

<PromptBlock label="完整 prompt — 拆爭議性質 + 揀升級路徑">
我有一單消費爭議，想搞清楚屬於邊個法律角度，應該行邊條升級路徑。

爭議概況：

- 商戶類型：[例：美容院 / 健身中心 / 旅行社 / 婚紗店 / 補習社 / 網購平台]
- 商戶名稱 + 地點：[填]
- 我預繳 / 已付金額：[例：HK$28,000]
- 服務 / 商品性質：[例：30 次美容療程 package，已做 3 次，剩 27 次未交付]
- 爭議發生經過（時間線，事實）：[3-5 句講清楚乜事，唔加形容詞]
- 商戶現狀：[例：門市仍開但拒接電話 / 已執笠 / 老闆失蹤 / 仍正常營業但拒絕退款]
- 我手上嘅證據：[列點：合約、付款紀錄、WhatsApp 等]
- Sales 落單時嘅口頭承諾（如有）：[例：「保證 6 個月見效」、「療程師全程同一位」]
- 我想要嘅結果：[退款 / 餘下服務轉移 / 履行原合約 / 唔做只想攞返錢]

請幫我做以下分析：

**1. 法律性質定性**

呢單爭議最可能涉及邊類條文：
- 《商品說明條例》(Cap. 362) —— 虛假商品說明、誤導性遺漏、具威嚇性營業行為、不當地接受付款（接受付款但無意 / 無能力交付）
- 《不合情理合約條例》(Cap. 458) —— 合約條款明顯不公平
- 《管制免責條款條例》(Cap. 71) —— 商戶試圖用合約條款免除合理責任
- 普通法下嘅「違反合約」(breach of contract)
- 「失實陳述」(misrepresentation) —— sales 口頭承諾同實際唔符

請逐項評估適用性，並指出最強嘅一兩條。

**2. 升級路徑建議**

請排序邊個渠道最值得行，附成本 / 時間 / 預期結果：

(a) **消費者委員會調解** —— 免費、約 2-3 個月、無判決權但對仍營運嘅商戶有聲譽上嘅籌碼
(b) **海關 商品說明條例組** 28922222 —— 免費、刑事檢控、適用於有虛假陳述 / 具威嚇性營業手法嘅個案
(c) **小額錢債審裁處** —— 申索額 ≤ $75,000、申索費約 $120-$1,045、無需律師、判決有強制執行力
(d) **區域法院 / 高等法院** —— 申索額大、需律師、成本高
(e) **報警** —— 如有刑事詐騙嫌疑（拐款潛逃、偽造合約）

針對我個情況，邊個係主路徑、邊個並行行、邊個唔抵行？

**3. 攞返錢嘅實際機會**

老實評估：
- 如果商戶仍營運：成功追討嘅百分比
- 如果商戶已執笠 / 進入清盤：成功追討嘅百分比
- 如果商戶轉名 / 老闆開新公司繼續經營：點追查

唔好過度樂觀。如果機會低，請直接講。

**4. 缺漏嘅證據**

我手上嘅證據有冇漏洞？要再補咩先入紙投訴會更紮實？

**5. 時間敏感度**

呢類個案有冇追溯時限（例：違反合約 6 年、商品說明條例檢控時限）？我而家入紙緊唔緊張？

請用平實語氣，唔需要免責聲明連篇，但對機會要老實。
</PromptBlock>

呢個 prompt 嘅核心唔係叫 Claude 扮律師，係幫你**將模糊嘅「我畀人呃」翻譯成法律分類**——咁先有得對應渠道。

### 3. 寫正式投訴：消委會表格 + 海關表格（同步）

拆完路徑之後，分別寫兩封：

**消委會投訴**（如 Claude 建議行主路徑）：

- 上 consumer.org.hk 開網上個案，或者 fax / 郵寄
- 內容要包：商戶完整資料（名 / 地址 / BR no.）、爭議時間線、付款金額、要求嘅補救、已採取嘅行動、附件清單
- 語氣：清晰、依時序、唔渲染情緒。消委會職員每日睇幾十單，越有條理越快被人跟進

**海關商品說明條例投訴**（如有虛假陳述 / 預繳商戶懷疑無意交付）：

- 熱線 2545-6182，或網上 customs.gov.hk「投訴及表揚」表格
- 重點：引用邊一條條文（s.7 虛假商品說明 / s.13E 不當地接受付款 / s.13F 具威嚇性營業手法）
- 附 sales 口頭承諾同實際交付嘅對照、付款紀錄、商戶宣傳材料

兩條路並行行冇衝突。消委會走調解、海關走刑事——前者幫你拎錢、後者幫商戶留紀錄（甚至檢控）。

> Pro tip：消委會嘅個案編號，喺後續任何書面通訊都引用返。對方收到「來自消委會跟進嘅查詢」嘅心理份量，比一般客戶投訴高一截。

### 4. 14 / 30 / 90 日跟進節奏 + 小額錢債成本效益

寄信 / 開個案之後，定三個檢查點：

- **Day 14** —— 消委會應該已確認收到。如未，主動跟進拎個案編號。同步檢查商戶有冇回應你嘅追討信
- **Day 30** —— 消委會通常會嘗試調解一兩次。如商戶零回應，呢個階段開始考慮升級到小額錢債
- **Day 90** —— 如仍未解決，做最後成本效益評估：

**小額錢債審裁處成本效益清單**：

| 項目 | 預計 cost / 時間 |
|---|---|
| 申索費（$10,000-$50,000 段） | 約 $370 |
| 申索費（$50,000-$75,000 段） | 約 $1,045 |
| 申請文件起草時間 | 8-12 小時（用 Claude 起草約 3 小時） |
| 首次聆訊輪候 | 約 6-10 星期 |
| 整個程序預計 | 4-8 個月 |
| 勝訴後執行（商戶冇錢 / 走數） | 額外金錢同心力 |

簡單原則：申索金額 ≥ $15,000 + 商戶仍有償債能力 = 抵告。 < $15,000 + 商戶執笠 = 心痛但放手嘅機會成本更低。

## 變化例子

### 變化 1：預繳式服務（健身 / 美容 / 補習班）倒閉

<PromptBlock label="變化 1 — 預繳商戶倒閉個案">
我嘅情況屬於「預繳式服務商戶倒閉 / 突然停業」：

- 行業：[健身會所 / 美容院 / 補習社 / 興趣班 / 月費共享空間]
- 預繳金額：[填]
- 已使用比例：[例：30 堂剩 22 堂未上 / 12 個月會籍剩 7 個月]
- 商戶現狀：[已貼結業告示 / 進入清盤 / 老闆失蹤 / 轉名再經營]
- 付款方式：[一次過畀現金 / 信用卡分期 / 簽分期供款計劃]

請特別處理以下：

**1. 信用卡 chargeback 可能性**

如果我用信用卡畀錢，仲未過 chargeback 時限（一般係交易日起計 120 日，但商戶倒閉「服務無法交付」嘅 chargeback 有部分發卡行接受更長時限），我要點啟動 chargeback 權利？我應該邊個時間點聯絡發卡行？要準備咩文件？

**2. 簽咗分期供款計劃 / 私人貸款**

如果我簽咗分期供款計劃（其實係透過商戶介紹嘅財務公司借錢），商戶倒閉，我仲要供？呢個係預繳商戶嘅常見陷阱。請分析 Cap. 622 公司條例 / 放債人條例 下嘅角度，同建議我應該即刻點做（通知財務公司 / 停止供款 / 法律咨詢）。

**3. 集體訴訟 / 集體投訴**

通常呢類個案會有幾百個受害者，請建議：
- 點加入現有嘅投訴群組（消委會通常會合併處理同一商戶嘅多宗投訴）
- 自己入紙同加入集體投訴嘅取捨
- 媒體曝光（東張 / 蘋果消費版）對追討嘅實際效果

**4. 清盤 (winding-up) 程序**

如果商戶正式進入清盤，請解釋：
- 我作為無抵押債權人嘅排位
- 點向清盤人提交債權證明
- 實際追討成功率（行業通常係每蚊只追回幾仙）

最後一句老實話：呢類個案追回 $0 嘅機會大不大？
</PromptBlock>

### 變化 2：婚禮 / 旅遊服務突然取消

<PromptBlock label="變化 2 — 婚禮 / 旅遊服務取消">
我嘅情況屬於「重大人生大事服務突然取消」：

- 服務類型：[婚禮 package / 婚紗攝影 / 婚宴酒席 / 旅行團 / 機票酒店套票 / 郵輪 booking]
- 合約金額：[填]
- 服務日期：[填] vs 通知取消日期：[填]
- 商戶提供嘅理由：[業務調整 / 不可抗力 / 員工問題 / 話全數退款但拖數]
- 我已花咗嘅連帶開支：[例：已寄結婚請柬、已訂酒店、親友已買機票飛來港、已請假]

請特別處理：

**1. 條文角度**

- 違反合約 (breach of contract) 嘅金額計算：除咗已付款，仲可唔可以申索相應損失（已花咗嘅連帶開支）？引用相關案例原理（唔需要真實案例名，講原理就得）
- 不可抗力條款 (force majeure) 嘅適用範圍 —— 商戶通常會搬出嚟用，但呢條條款適用範圍其實受限
- 如涉及旅行社：旅遊業議會 (TIC) 嘅「旅遊業賠償基金」適用條件

**2. 時間敏感 — 補救行動**

距離原定服務日期仲有時間嘅話，可以即時搵替代服務。請建議：
- 點記錄替代服務嘅成本差額（用作日後申索嘅金額基礎）
- 應否減低損失（法律上有減損義務）

**3. 媒體 + 監管雙線**

呢類個案媒體曝光嘅籌碼大（「新人婚禮兩日前被取消」係好題材）。請建議：
- 幾時接觸傳媒、幾時先按住唔出
- 對應監管機構：婚紗業界冇統一監管機構，但旅遊業有 TIC + 商務及經濟發展局；婚宴酒店有食物環境衞生署牌照角度（極端情況）

**4. 信件語氣**

呢類個案商戶通常已知衰咗，會好快拋一個壓到好低嘅和解方案。請建議信件中點留位畀對方提案，但同時錨定一個合理嘅申索金額（已付款 + 證實嘅連帶開支），唔好開頭就喊打喊殺，亦唔好天真接受 30% 退款就算。
</PromptBlock>

### 變化 3：網購（Taobao / iHerb / Amazon 寄港）爭議

<PromptBlock label="變化 3 — 跨境網購爭議">
我嘅情況屬於「跨境網購爭議」：

- 平台：[Taobao / Tmall / 京東 / iHerb / Amazon / eBay / SHEIN / 其他]
- 訂單金額：[填]
- 爭議類型：[揀]
  - 收貨同 listing 描述不符（假貨 / 縮水 / 壞 / 過期）
  - 完全未收貨（運送途中遺失 / 商家未發 / 物流卡關）
  - 商家拒絕退貨退款
  - 平台爭議系統判我輸但我認為唔合理
- 已嘗試嘅步驟：[平台爭議、聯絡商家、申請退款等]

請特別處理：

**1. 司法管轄現實檢查**

跨境網購爭議嘅老實答案：香港消委會 / 海關對境外商戶冇執法權。請直接講清楚：
- 對 Taobao / Tmall：可投訴上「12315」（內地消費者投訴熱線）+ 阿里巴巴內部仲裁機制
- 對 Amazon / iHerb：用 A-to-Z Guarantee / 平台爭議系統
- 香港法院判決對境外被告嘅執行難處

呢條路嘅核心唔係告，係善用平台嘅爭議系統。

**2. 信用卡 chargeback 做預設**

跨境網購爭議嘅最高勝率路徑，通常係信用卡 chargeback。請建議：
- 幾時直接跳過平台爭議去 chargeback（如平台拖你超過 30 日無進展）
- 點寫 chargeback 嘅理由代碼（merchandise not as described / goods not received / services not rendered）
- 要準備嘅證據包（listing 截圖、收貨相同 listing 對比、物流紀錄、與商家溝通紀錄）

**3. 平台投訴信寫法**

平台爭議系統有字數上限（通常 500-2000 字），請示範一個英文版 + 中文版範本，包含：
- 一句總結
- 分點列出嘅時間線
- 具體要求嘅補救（refund / replacement / partial refund）
- 附件證據清單參照

**4. 唔好做嘅嘢**

- 唔好情緒化（平台演算法同審核人員都會調低你嘅排序）
- 唔好同時開多個爭議（會被系統標記成濫用）
- 收到貨之後唔好即刻喺平台度確認收貨（確認之後爭議時限會縮短）
</PromptBlock>

## 一個老實嘅現實檢查

香港消費者維權嘅難處唔係「冇渠道」，係「渠道多但分散，每條路嘅回報都唔高」。 消委會冇判決權、海關刑事檢控門檻高、小額錢債告完商戶冇錢 = 一紙空判、跨境網購基本上唔屬本地司法管轄。

> 行內人嘅一句話：「真正攞返錢嘅個案，七成靠文件嘅完整度，兩成靠肯堅持嘅跟進節奏，一成靠運氣（商戶仲有錢 + 仍想保聲譽）。律師意見值錢，但對小金額爭議嘅邊際效益其實有限。」

Claude 嘅優勢喺呢度有兩個：(1) 幫你 30 分鐘做完原本要 2 個鐘揾資料 + 寫信嘅功夫，等你成本低到可以同時行三條路；(2) 幫你寫一封讀起嚟好似你做足功課嘅投訴，畀對家覺得「呢個客唔好搞，畀返佢算」。

35 分鐘嘅功夫，唔保證攞返錢，但保證你由「畀人呃完冇晒辦法」變成「該行嘅渠道都行過、該入紙嘅個案都入過」。後者就算最終追回 $0，至少瞓得着。**重點唔係贏，係企得起身行完條階梯**。

> 呢篇唔避諱講死。如果你而家係父母剛走咗嘅頭 48 小時，跳到「跟住做」第 1 步——睇清楚眼前要即刻處理嘅嘢。如果你係預先安排，由「變化 1」入。Claude 唔取代殯儀顧問或律師，但佢幫你撐住個流程框架，等你喺哀傷入面唔會俾人加碼推銷。

## 情境

可能你係呢兩個情境之一。

**情境 A — 父母剛走咗**：

凌晨 4 點，醫院 call 你。你阿爸 / 阿媽喺老人科病房走咗。你趕到醫院，護士遞一張表畀你簽。醫生開咗 **死因醫學證明書（Form 18）**，叫你 24 小時內去**生死註冊處**換**死亡證**。你問下一步點？護士好同情你，但講得好簡短：「殯儀館會聯絡你，揀邊間？」

你冇心理準備。你只係知道——

- 屋企人喺 WhatsApp group 開始問「幾時 set 安息禮拜？」
- 殯儀館中介喺醫院門口已經派咭片
- 你阿姨話「世界殯儀館好啲」，你舅父話「紅磡殯儀館傳統啲」
- 你唔知棺木幾錢、化妝幾錢、紙紮幾錢
- 你唔知火葬定土葬、邊度有位、要等幾耐

**呢 48 小時嘅每一個決定都涉及錢同情感**。殯儀業界唔係邪惡，但**資訊極度唔透明**——你冇個底價參考，就好易俾人推銷一個 $80k 嘅套餐，當中可能有一半你根本唔需要。

**情境 B — 你想預先安排自己嘅身後事**：

你 55 歲，剛幫朋友處理完佢阿爸喪事，見到佢同兄弟姊妹喺殯儀館門口為咗「燒邊種紙紮」嗌交。你諗：我唔想下一代搞成咁。

兩個情境，Claude 都可以幫你 **搭起個流程框架**。佢唔取代殯儀顧問，但佢令你**有個底去傾**。

## 跟住做

### 1. 頭 48 小時 / 7 日 / 30 日 任務 checklist

最焦慮嘅嘢係「我係咪漏咗啲嘢」。先用 Claude 出一份分時段 checklist，印出嚟，逐項剔。

<PromptBlock label="完整 prompt — 香港身後事 48 小時 / 7 日 / 30 日 checklist">
我[父親 / 母親 / 配偶]今日（[日期]）喺[醫院 / 屋企 / 院舍]過身。我需要一份**香港 2026 身後事 task checklist**，分三個時段：

⏰ **首 48 小時**
- 死因醫學證明書（Form 18）邊個醫生簽：醫院 / 簽發機構
- 屋企走嘅情況：打 999 定家庭醫生？要唔要驗屍？
- 接收死者個人物品：身份證、銀行卡、藥物、眼鏡（影低件件登記）
- 通知至親：點排先後（配偶 → 子女 → 兄弟姊妹 → 摯友）
- 殯儀館接體：可以等，唔好慌住揀；醫院殮房通常免費 keep 48-72 小時
- 揀**生死註冊處**地點（[港九新界 4 個辦事處]）

📋 **首 7 日**
- 死亡證正本（換 Form 18 → 死亡登記證明書）—— 我需要帶咩文件
- 火葬／土葬決定：食環署火葬場 vs 私營（如華永會、教會墓地）
- 殯儀館報價（步驟 2 詳細處理）—— 至少報 3 間
- 訂場：殯儀廳一晚 vs 兩晚、出殯日期（避開公眾假期 + 預留時間畀親友飛返嚟）
- 通知：公司 / 學校 / 銀行 / 政府部門（強積金、長者津貼、公屋）—— 邊啲要書面、邊啲打去就得
- 訂訃聞（如要登報）
- 同宗教領袖確認：神父 / 法師 / 牧師 / 無宗教司儀揀邊個

📅 **首 30 日**
- 火化後：骨灰位申請（食環署輪候期 4-5 年 vs 私營即得）
- 「頭七」如有做傳統儀式（佛教 / 道教 case）
- 死亡證核證副本（銀行 / 保險各要一張，建議申請 10 張）
- 啟動遺產處理（步驟 4 詳細）：揾律師 vs 自己搞小額遺產
- 收回死者公屋 / 私樓（如獨居）
- 取消：八達通自動增值、SIM card、Netflix、銀行戶口（要遺產承辦書先做）
- 100 日 / 對年（傳統家庭）安排

🛑 **唔好做（常見陷阱）**
- 唔好喺醫院門口即場簽殯儀館合約 —— 中介有佣金
- 唔好預付全數（30-50% 訂金合理，餘額完工後先俾）
- 唔好淨係同一間殯儀館講「我趕」—— 你越急佢越唔讓步
- 唔好喺哀傷最激烈嗰陣揀 $50k+ 棺木 —— 你會後悔

語氣：實際、莊重、清楚講時間線。我而家個腦轉唔到，需要一步一步嚟。
</PromptBlock>

Claude 出嗰份 checklist 你印出嚟，貼喺廚房。每剔一項，你冇遺漏嘅安全感就多一分。

### 2. 殯儀館報價問題 + 費用透明

呢一步係 **最易俾人搵笨** 嘅環節。香港私營殯儀館報價結構超複雜——一個「基本套餐」可以由 $25k 跳去 $120k，視乎你加幾多項目。

<PromptBlock label="完整 prompt — 香港殯儀館報價逐項拆解 + 比較表">
我要為[父親 / 母親]安排 [佛教 / 基督教 / 無宗教 / 簡約] 喪禮。我想 Claude 幫我整一份**殯儀館報價拆解框架**，等我打去 3 間問價嗰陣，知道應該問乜、應該對比乜。

🪦 **香港殯儀報價項目（典型拆解，2026 大概數）**

請列出**每一條項目** + 香港市場典型價格範圍 + 「易俾人加碼推銷嘅版本」：

1. **基本服務費**（接體 / 化妝 / 入殮 / 出殯協調）—— 典型價格範圍
2. **棺木 / 骨灰盅**
   - 火葬用簡約紙棺：[價格範圍]
   - 中價實木：[價格範圍]

3. **殯儀廳場地租金**（一晚 vs 兩晚）
   - 大廳 vs 細廳
   - 紅磡 / 北角 / 鑽石山 / 新界 各區典型價格範圍
4. **化妝 + 整容服務**（基本 vs 嚴重外傷情況）
5. **遺體運送**（醫院 → 殯儀館 → 火葬場每一程）
6. **儀式人員**
   - 法師誦經（佛教 / 道教）
   - 神父 / 牧師（基督 / 天主教）
   - 無宗教司儀
7. **紙紮 / 祭品**（傳統家庭，可以由 $2k 到 $30k+）—— 哪啲必要、哪啲純情感

10. **訃聞登報**（明報 / 東方 / SCMP 各自收費）
11. **回禮**（毛巾 + 糖 + 利是封）—— 數量計法
12. **火化爐租用**（食環署 vs 私營）
13. **骨灰位**（食環署輪候 vs 私營即時）—— 申請流程

🔍 **打去殯儀館要問嘅問題清單**
- 報價有冇**書面逐項報價**？（要白紙黑字，唔好淨係聽口頭）
- 邊幾條係「必須」、邊幾條係「可有可無」？（迫佢分清楚）
- 訂金幾多 %？退款條款？
- 有冇額外**手續費 / 服務附加費**？（部分殯儀館有 10-15% 行政費收喺最後）
- 出殯當日**人手配備**（幾多個工作人員照應），有冇加班費？
- 如果出殯延期（火葬場排唔到），點計？

💰 **三檔預算參考**（2026，只供定錨用）
- 簡約火葬：HK$30-50k（基本服務 + 簡約棺 + 1 晚廳 + 食環署火化）
- 中位數傳統：HK$60-100k（中價棺 + 2 晚廳 + 法師 + 適度紙紮 + 私營骨灰位）
- 大型傳統：HK$150k+（高檔棺 + 大廳 + 全套紙紮 + 私營骨灰塔位）

⚠️ **加碼推銷紅旗**
- 「呢個係你父母最起碼應得嘅 standard」—— 情感勒索
- 「呢個加購而家訂有折」—— 製造假緊迫感
- 「其他客都揀呢個套餐」—— 拉你跟大隊
- 唔肯畀逐項報價，淨係報一個總價

請整成**一個對比表範本**（殯儀館 A / B / C 三欄），我打完三個電話可以即刻填入對比。

語氣：識行情、唔評判家庭點花錢、但保護我唔俾人擺布。
</PromptBlock>

打三間電話。**唔好同一日打**——分開兩日，你嘅情緒狀態唔同，你會聽到唔同嘅嘢。最平嗰間唔一定揀，但**最貴嗰間應該畀到你最具體嘅逐項拆解**——如果佢含糊其辭，掉頭走。

### 3. 儀式風格 — 文化 / 死者心願 / 親友期望三角

呢個係 **家庭衝突嘅高發點**。你阿媽生前話「簡單啲就好」，但阿姨話「你阿爸地下會嬲」、舅父話「冇法師唔得」。Claude 唔解決家庭政治，但幫你**搭起平衡三方嘅框架**。

<PromptBlock label="完整 prompt — 儀式風格決策框架">
我[父親 / 母親]嘅信仰 / 文化背景：[佛教 / 基督教 / 天主教 / 道教 / 無宗教 / 混合]。

死者生前講過嘅心願（如有）：[簡單 / 傳統 / 海葬 / 環保 / 唔肯講過]
我自己傾向：[簡約 / 傳統 / 環保 / 唔知]
親友期望：[長輩想全套傳統 / 兄弟姊妹分歧 / 屋企人一致]

請幫我整**四種香港主流儀式風格嘅比較**：

⛩️ **傳統佛教 / 道教**
- 流程：頭七、做七、出殯、火化、骨灰安奉
- 時長：通常 7 日內出殯，全程儀式 3-5 個鐘
- 法師費 + 誦經費典型價格範圍
- 服飾要求（孝服 / 麻衣）
- 適合：長輩仍在生堅持、死者生前傳統、屋企人需要儀式畀到安頓嘅感覺

✝️ **基督教 / 天主教安息禮拜**
- 流程：守夜 + 安息禮拜 + 出殯 + 火化 / 土葬
- 時長：可以快過傳統，5-7 日內完成
- 牧師 / 神父費（部分教會免費，奉獻為主）
- 適合：死者有教會生活、屋企人多基督徒、需要教友支援

🕊️ **無宗教告別會**
- 流程：自由設計，分享環節 + 音樂 + 投影片 + 致辭
- 時長：1.5-3 個鐘
- 司儀費 + 影音設備
- 適合：死者無宗教、家庭多元信仰、想做得個人化
- 場地：殯儀館廳，或餐廳 / 社區會堂（部分私營殯儀館肯承辦）

🌿 **簡約 / 環保 / 綠色殯葬**
- 香港選項：海上撒灰（食環署免費安排）/ 紀念花園撒灰 / 環保紙棺
- 流程：可以無正式儀式，純家庭聚會
- 成本最低：HK$15-30k 完成
- 適合：死者明確要求、家庭意見一致、無強烈宗教 / 傳統壓力

🎯 **決策框架**
請幫我答以下，得出我嘅傾向方向：

1. **死者心願嘅份量**（你哋傾過嗎？寫過嗎？）—— 寫低咗嘅心願，份量最重
2. **長輩期望嘅強烈程度**（拒絕妥協 vs 不滿但接受）
3. **屋企人共識**（一致 / 分裂）
4. **預算限制**（簡約 $30k vs 傳統 $80k vs 大型 $150k）
5. **海外親友**（要唔要直播 / 延後儀式等佢哋返港）

⚖️ **如果有衝突**
- 三方衝突常見模式：死者心願 vs 長輩期望
- 調解建議：核心儀式跟長輩（保面子），個人化嘅悼念跟死者心願（一段投影片 / 一首歌 / 一段朗讀）
- 「兩階段」做法：殯儀館內傳統儀式 + 屋企 / 餐廳輕鬆悼念聚會排後一日

請幫我列出**三條跟進問題**，我應該問屋企其他成員，等決策更有根據。

語氣：尊重所有信仰、唔貶低任何風格、認可家庭政治係現實。
</PromptBlock>

Claude 出嘅框架你睇咗，**唔好即刻喺 family WhatsApp group 公佈**。先一對一同最近你嗰個兄弟姊妹傾，你哋兩個夾好咗，再一致對外同長輩傾。

### 4. 喪禮之後嘅手續 — 遺產 / 銀行 / 遺產稅

火化完之後，你以為完——其實手續部分先開始。香港遺產稅 2006 年已廢除，但**遺產承辦書（Grant of Probate / Letters of Administration）**仍然係取出死者銀行戶口、變賣物業嘅前提。

<PromptBlock label="完整 prompt — 香港遺產處理 + 政府部門通知 checklist">
我[父親 / 母親]剛火化完。死者資產：[銀行戶口 / 物業 / 強積金 / 保險 / 股票 / 公司股權 — 列出大概]。死者有冇遺囑：[有 — 律師處 / 屋企 / 無]。

請幫我整**香港遺產處理 30-90 日 checklist**：

📜 **有遺囑 vs 無遺囑**
- 有遺囑 → 申請 **Grant of Probate**（遺囑認證）
- 無遺囑 → 申請 **Letters of Administration**（遺產管理書）
- 小額遺產（HK$150k 以下）→ 可以走 **小額遺產簡易程序**，免律師

🏦 **銀行戶口**
- 死者單名戶口：凍結，要遺產承辦書先解凍
- 聯名戶口（視乎 and/or 寫法）：在生嗰位通常可以提取
- 信用卡：通知銀行取消，部分卡有身故保障（債務豁免）
- 自動轉賬：全部列出，逐個取消（電費 / 水費 / 電訊）

🏘️ **物業**
- 單名 → 入遺產，要承辦書先轉名
- 聯名（joint tenancy）→ 自動轉去在生嗰位，但仍要去土地註冊處備案
- 公屋 → 通知房署，由在生戶主繼承戶籍

💼 **強積金 / 退休金**
- 強積金：填妥強積金身故索償，有指定受益人就直接發放
- 政府公務員退休金：通知部門

🛡️ **保險**
- 人壽保險：受益人直接索償，唔入遺產
- 醫療 / 危疾未索償過嘅，最後一次索償醫療費

📋 **政府 / 公共部門通知清單**
- 入境處（身份證註銷）
- 房署 / 公屋
- 食衛局（長者醫療券餘額）
- 社署（長者生活津貼 / 綜援）
- 稅局（個人入息評稅最後一年）
- 選舉登記處
- 強積金管理局

💸 **遺產稅現況**
- 香港 2006 年起廢除遺產稅
- 但 **死者去世前 7 年內送出嘅大額饋贈** 仍可能涉及法律爭議（如有家庭糾紛）
- 跨境資產（中國內地 / 海外）—— 當地稅制另行適用

🧑‍⚖️ **律師 vs 自己搞**
- 小額遺產 + 無爭議 → 可以自己搞（高等法院 Probate Registry 有自助指引）
- 物業 / 多名繼承人 / 跨境 → 揾律師（典型收費 HK$15-40k）
- 紅旗：兄弟姊妹有人提出異議 → 一定揾律師

⏰ **時間線**
- Grant of Probate / LA：典型 3-9 個月
- 期間銀行戶口凍結 —— **預先喺自己戶口留起喪葬費 + 6 個月家用**
- 物業出售（如要）：承辦書出咗先可以放盤

請整成**一張 30-60-90 日時間表**，每段我應該完成乜。

語氣：實事求是、唔煽情，但提醒呢個階段嘅疲勞係真嘅 ——「事都做完，個哀傷先正式浮上嚟」。
</PromptBlock>

呢一份印兩份。一份俾你，一份俾最信得過嗰個兄弟姊妹。**呢啲手續唔應該由一個人孭晒**。

## 變化

### 變化 1：預先安排自己嘅身後事（55-70 歲讀者）

你健康，但你睇住朋友處理父母喪事時嘅混亂——你想下一代少受罪。

<PromptBlock label="變化 1 — 預先安排自己嘅身後事 + 寫畀至親嘅信">
我[年紀]，健康，但想預先安排自己嘅身後事，等屋企人之後唔使估。

請幫我起草**3 份文件**：

📝 **A. 寫畀至親嘅心願信**
框架包括：
- 我想要嘅儀式風格（傳統 / 無宗教 / 簡約 / 環保）+ 點解
- 火葬 vs 土葬 + 骨灰處理（撒海 / 撒灰花園 / 安奉骨灰位 / 帶返鄉下）
- 預算上限提示（「唔好俾人加碼推銷，[$X] 之內掂晒就 OK」）
- 我想播嘅音樂 / 想讀嘅文字 / 想請嘅人
- 我**唔想**嘅嘢（例：唔想登報、唔想搞紙紮、唔想喊得太慘）
- 一段「畀下一代嘅諒解同許可」—— 由得佢哋用自己嘅方式哀傷

🏛️ **B. 預設醫療指示（Living will）**
- 香港 2025 起預設醫療指示法律框架更清晰
- 列明：拒絕嘅延命治療（CPR / 插喉 / 維生機）
- 適用條件：末期 / 持續植物人 / 不可逆昏迷
- 邊度簽：兩個成年見證人（其中一個唔可以係受益人）
- 副本俾：家庭醫生 / 信任嘅至親 / 律師

📋 **C. 實用資料表**
- 銀行 / 保險 / 強積金戶口清單（**唔寫密碼**，寫機構 + 戶口號碼）
- 律師 / 會計師聯絡
- 遺囑放邊（律師處 / 銀行保險箱 / 屋企夾萬）
- 數碼身後事：email / Facebook / iCloud 點處理（部分平台有身故聯絡人設定）
- 寵物安排（如有）—— 邊個接手

語氣：唔忌諱、實際、平和。預先安排唔係陰沉，係留畀仔女嘅最後一份禮物。
</PromptBlock>

寫完 3 份，印出嚟，放夾萬。同你最信得過嗰個至親講「夾萬入面有嘢」，**唔需要而家睇**。

### 變化 2：海外亡親 — 運遺體返港 vs 海外火化

父母移咗民去[加拿大 / 英國 / 澳洲 / 中國內地]，喺嗰邊走咗。

<PromptBlock label="變化 2 — 海外亡親處理安排">
我[父親 / 母親]喺[國家 / 城市]過身。我喺香港，需要決定：

✈️ **選項 A：運遺體返港**
- 海外殯儀館防腐處理（香港火葬場要求）
- 國際運送費：典型 [USD 範圍]，視距離 + 航空公司
- 文件：當地死亡證 + 公證 + 中文翻譯
- 香港入境：食環署遺體入境批准
- 時間線：通常 2-4 週
- 適合：屋企人堅持喺香港安葬、有家族墓地 / 骨灰位

🔥 **選項 B：海外火化 + 骨灰返港**
- 當地火化（成本通常較平）
- 骨灰運返：可隨身（國際航空大部分接受，要齊文件）
- 香港入境骨灰：相對簡單
- 時間線：1-2 週
- 適合：屋企人接受海外火化、想減成本同運送負擔

🌍 **選項 C：海外安葬，喺香港辦悼念會**
- 死者葬喺移民地（部分人希望留喺嗰度）
- 香港親友辦悼念會（無遺體）
- 成本最低，但部分屋企人會覺得「冇個了結」
- 適合：死者明確希望留喺移民地、香港屋企人接受

📋 **決策問題**
- 死者生前有冇講過心願？
- 香港屋企人能唔能去當地出席？
- 預算容忍度？
- 香港有冇家族墓地 / 骨灰位留緊？
- 跨境法律：當地遺產 vs 香港遺產，要分開處理

語氣：實際、認可跨境哀傷有種獨特嘅沉重 ——「你冇喺現場」嘅內疚好常見，但唔合理。
</PromptBlock>

### 變化 3：環保土葬 / 海上撒灰（HK options）

死者生前講過「唔想喺骨灰塔等 4 年」/「想回歸自然」。

<PromptBlock label="變化 3 — 香港環保身後事選項">
死者心願：環保 / 自然 / 簡約。請列香港 2026 環保身後事選項：

🌊 **海上撒灰**
- 食環署免費安排（每月固定船次，由南丫島 / 西貢出發）
- 流程：火化 → 申請 → 上船 → 撒灰
- 時間線：火化後 1-2 個月排到期
- 親友出席：船有限額，通常 6-8 人

🌳 **紀念花園撒灰**
- 食環署管理 13 個紀念花園（鑽石山 / 葵涌 / 富山等）
- 免費 / 低收費
- 撒灰後種紀念樹（部分花園）—— 唔可以標明個別位置
- 適合：想有「specific 拜祭點」但唔要骨灰位嘅家庭

♻️ **環保紙棺 / 竹棺**
- 香港部分殯儀館有供應
- 成本：$3-8k（vs 木棺 $15-50k+）
- 火化後完全降解
- 外觀上樸素啲 ——部分長輩起初會抗拒，但喪禮上會見到嘅人少

🍃 **無儀式，純家庭聚會**
- 火化後屋企人小型聚會，無正式儀式
- 成本只係火化費 + 簡單棺木 = HK$15-25k
- 適合：死者明確心願 + 家庭意見一致

⚖️ **何時呢條路會撞牆**
- 長輩堅持「先人要有風光大葬」
- 親戚之後話「你阿爸地下嬲你」
- 妥協做法：正式儀式簡化 + 撒灰嘅儀式反而做隆重啲

🏛️ **香港法律提醒**
- 私人土地撒灰：業權人同意可以
- 公園 / 沙灘 / 山徑：技術上唔合法（但好少執法）
- 海上撒灰：要走食環署正式途徑，唔好私自出海

語氣：認可環保選擇唔係「唔孝順」—— 死者心願應該優先。
</PromptBlock>

## 一個心態

殯儀業界唔係邪惡。佢哋每日見死亡，部分前線甚至比家屬更冷靜。但呢個行業嘅利益誘因，同你哀傷時嘅脆弱**剛好相反**——你越急、越唔識、越情緒，佢哋越容易賣到貴嘢。

**Claude 喺呢個流程嘅角色唔係取代殯儀顧問或律師**。佢嘅角色係：

- 喺你個腦轉唔到嘅 48 小時，**撐住份 checklist 框架**，等你唔會漏文件
- 喺你打殯儀館電話之前，**畀你打底嘅基本知識**，等你問得到對嘅問題
- 喺家庭衝突嘅時候，**搭起平衡三方利益嘅框架**，等你唔係靠情緒嗌交
- 喺手續嘅疲勞期，**排好政府部門 + 銀行 + 律師嘅次序**，等你睇得到終點

> 莊重，唔需要等於昂貴。簡約，唔等於唔孝順。
>
> 死者真正帶走嘅，係佢同你共同經歷嘅嗰啲時間。棺木幾錢、紙紮幾大件，30 年後冇人會記得。記得嘅，係你哋互相講過嘅最後幾句說話、你頭七嘅嗰一頓飯、你撒灰嗰日嘅天氣。
>
> 用 Claude 撐住個框架，等你嘅哀傷有空間。

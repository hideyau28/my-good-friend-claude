> ⚠️ 重要聲明：Claude 唔取代 GP / oncologist。呢篇所有框架係教你**問啱問題**、揀啱 screening package，唔係臨床建議。任何 screening / 跟進 / 治療決定 = 同你嘅醫生討論。

## 情境

你 40 歲。WhatsApp 開始收到體檢 plan 推銷：

- **「夫婦尊尚體檢套餐 $28,800」** —— 80 個項目、AI body scan、全身 MRI
- **公司年度體檢券** $3,000 喺中銀 / Bupa，揀邊間？
- **保險經紀**：「呢個 plan 包大腸鏡 + mammogram，要趁未供完……」
- **家族大伯 55 歲確診大腸癌**——你應該幾時開始照？

每個項目 marketing 都 sell「early detection saves lives」。但：

- **乳房 MRI** 對 average risk 女性可能 over-screen，false positive 高
- **PSA**（前列腺）50 歲前 routine screen 國際 guideline 唔推薦
- **「全身 MRI」「AI cancer scan」** 喺 USPSTF / NHS / CDC 入面：**冇任何一個 guideline 推薦做 average-risk screening**
- **大腸鏡** 公立要排 2 年，私營 $8,000-$15,000——但 50 歲開始照（高 risk 45 歲）係真係 evidence-based

呢篇用 Claude **30 分鐘**對應你 risk profile 整：

1. Personalized screening list（基於 USPSTF / NHS / HK Department of Health guideline）
2. 私營 package 逐個 item 拆 evidence
3. 5-year cadence + 公私混合策略
4. 萬一 abnormal 嘅 mental + financial prep

## 跟住做

### 1. 列家族病史 + 個人 risk factor + 年齡

冇呢啲輸入資料，Claude 出嘅 screening list 同 marketing brochure 冇分別。寫低（Apple Notes / 紙）：

- **年齡 + 性別 + 種族**（south Asian / 華人對某啲癌 risk 唔同）
- **家族病史**（盡量具體）：
  - 邊個親人（父母 / 兄弟姊妹 = first-degree）
  - 咩癌
  - 確診年齡（50 歲前確診 = 高 risk signal）
- **個人 risk factors**：
  - 吸煙史（pack-years）
  - 飲酒
  - BMI / 體重
  - 已知 condition（HPV、乙肝、Helicobacter pylori、糖尿）
- **生育 / 荷爾蒙史**（女性）：初經年齡、第一胎年齡、HRT、口服避孕藥年期
- **過去 screening 結果**：上次 mammogram / pap smear / 大腸鏡日期 + 結果
- **暴露**：石棉、輻射、工作環境
- **已知基因 marker**（如有做過 BRCA 等）

⚠️ 規矩：
- 用 international guideline 嚟做 reference（USPSTF 2023+ / NHS / CDC）
- 唔好基於私營 marketing claim

- 任何 borderline 都 flag「同 GP 討論」

Claude 出 list 之後讀一次：對應到你風險嘅 1-2 個重點通常浮現（例如：母親 48 歲乳癌 → mammogram 30-35 歲開始、加 MRI、考慮 BRCA test）。

### 3. 私營 package 逐個 item 拆解

收到報價單之後，唔好淨係比 total price。用呢個拆：

<PromptBlock label="完整 prompt — 私營體檢 package 拆解">
我收到 [私營體檢中心 / 保險公司] 報價，套餐 $[X]，包括 [N] 個項目。

📋 **完整項目清單**
[逐個 paste：項目名、檢查方法、價錢如果有 line item]

🎯 **請逐個 item 出**

| Item | Evidence-based？ | 我風險點解需要 / 唔需要 | 公立可唔可以做 | 私營報價 vs 單買 |
|------|----------------|------------------------|--------------|----------------|

特別 flag：

**Over-tested（我唔需要但 package 有）**
- 例：全身 MRI、tumor marker bundle、AI body scan、unnecessary 重複 ultrasound
- 風險：false positive → 過度跟進 → 焦慮 + 額外 cost
- 建議：搵 package 唔包呢啲 / 講明唔做

**Under-tested（我需要但 package 冇）**
- 例：50 歲後缺大腸鏡 / 高 risk 女性缺 BRCA 諮詢 / 吸煙者缺 low-dose CT 肺
- 建議：分開私營買 / 公立排

**啱 evidence + 我風險嘅核心 item**
- 應該保留嘅
- 對比公立可唔可以做（cost saving）

📊 **重整：Bare-minimum vs Marketing-loaded**

請出 2 個版本：
1. **Bare-minimum evidence-based**（基於我風險）—— 大約 cost
2. **Marketing-loaded**（package 全買）—— 大約 cost
3. **差價**買到咩額外 value（如果有）

⚠️ 唔好幫我做最終決定。呢個係比較表，唔係建議書。
</PromptBlock>

⏰ **典型發現**：HK 私營 $15,000 package 入面，evidence-based 對你 profile 真係有 value 通常 $4,000-$6,000。其餘係 over-screen / marketing weight。

### 4. 5-year cadence + 萬一 abnormal 嘅 mental prep

<PromptBlock label="完整 prompt — 5-year cadence + abnormal follow-up plan">
基於我 [年齡 + risk profile] 已選定嘅 screening list，請整 5-year cadence。

📅 **5 年計劃表**

| 年份 | 我嘅年齡 | Screening | 公立 / 私營 | Cost 估計 |
|------|---------|-----------|-----------|----------|

⏰ **Cadence 邏輯**
- 大腸鏡（average risk 10 年 / high risk 5 年）
- Mammogram（2 年 / annual depend on risk）
- Pap + HPV（5 年 if HPV co-test negative）
- Low-dose CT 肺（吸煙者 annual）
- HBV / liver function（已感染者）

🏥 **公私混合策略**
- 公立 free / 資助：大腸 FIT、婦女健康中心 mammogram + pap
- 公立輪候長但可預約：大腸鏡（如 FIT 陽性）
- 私營較合理：annual GP + blood work、特定 high-risk 跟進
- **唔好做嘅**：私營重複公立啱啱做完嘅檢查（hospital authority 紀錄 6 個月內）

🧠 **Abnormal result 嘅 mental prep**

請預先框架我心理準備：

**Stage 1：收到 abnormal 通知**
- Abnormal screening result ≠ 確診
- 大部分 abnormal → 跟進 → 無事 / benign
- 第一步：搵 GP 安排正式跟進，**唔好上網自己研究**
- 唔好喺等跟進嗰幾日做大決定（辭職 / 賣樓 / 嗌交）

**Stage 2：跟進檢查（biopsy / imaging）**
- 通常 1-4 週內安排
- 私營快 / 公立排
- 第二意見：唔係不信任，係 standard practice for 重大診斷

**Stage 3：如真係確診**
- 唔好淨係睇 Google 5 年存活率（嗰個 stat 包晒全部 stage / 過時數據）
- HK 公立 oncology 治療 international standard
- 私營可考慮但 cost 大（$500K-$2M+ 一個療程唔出奇）
- 醫療保險 review：呢個係攞 [insurance-policy-review-claude] 嘅時機
- 工作 / 收入 / 家人安排：分階段，唔好一日內決定

💰 **Financial buffer**
- Abnormal follow-up 平均 cost：$3,000-$15,000（imaging + biopsy + 諮詢）
- 應急基金內預留 = 唔需要因 cost 拖延跟進
- 醫療 / 危疾保險 review 一次（最後一次 update：[填]）

⚠️ **最重要嘅 framing**

Screening 嘅目的 ≠ 「保我無事」。Screening 嘅目的：

- 提升早期發現率（早期治癒率顯著高過晚期）
- 接受 false positive 嘅 cost（焦慮 + 跟進）係 trade-off
- 唔做 screening ≠ 健康。係延後資訊。

語氣：實用、唔 alarmist、唔 dismissive。
</PromptBlock>

每 2-3 年 re-run 一次（新家族史浮現 / 年齡跳階段 / guideline update）。

## 變化例子

### 變化 1：男性 40 歲（前列腺 / 大腸 / 心血管）

<PromptBlock label="變化 1 — 男性 40 歲 screening">
我男性 [年齡]，無顯著家族癌史。請出 evidence-based screening list。

🎯 **男性 40-50 重點**

**大腸（最高 yield）**
- 50 歲開始 FIT 每 2 年（HK 政府資助 50-75）
- 大腸鏡 50 歲首次（high risk 45）
- 兄弟 / 父確診 < 60 歲：早 10 年開始

**前列腺（PSA — 證據 nuanced）**
- USPSTF：50-69 歲 individual decision（唔係 routine）
- < 50 歲 average risk：唔推薦 routine PSA
- 黑人 / 家族史：45 歲考慮
- 同 GP 討論 over-diagnosis trade-off

**心血管（唔係 cancer 但 40+ 男性最大殺手）**
- Lipid panel、BP、HbA1c（糖尿）每 1-3 年
- ASCVD risk calculator（10-year risk）

**肝（HK 男性高 HBV 流行）**
- 乙肝 status 確認
- 如帶菌：肝功能 + α-fetoprotein + ultrasound 每 6 個月

**肺（吸煙者）**
- Low-dose CT annual：50-80 歲、20+ pack-years、戒煙 < 15 年
- 無吸煙史：唔推薦

**唔需要 routine**
- 全身 MRI、tumor marker panel、CA 19-9、AI body scan

請按我 profile 排優先次序。
</PromptBlock>

### 變化 2：女性 35-50（乳房 / 子宮 / HPV）

<PromptBlock label="變化 2 — 女性 35-50 screening">
我女性 [年齡]，[家族史 / 生育史 / HPV 狀態]。

🎯 **女性 35-50 重點**

**乳房**
- Average risk：40 歲開始 mammogram 每 1-2 年（USPSTF 2024 update）
- 一級親屬乳癌 / 卵巢癌：30 歲開始、加 MRI、考慮 BRCA 諮詢
- 緊乳房（dense breast）：補 ultrasound

**子宮頸（最 evidence-based）**
- HPV co-test 5 年一次（25-65）
- HPV 已知陽性：頻密啲
- HK 婦女健康中心 / Family Planning Association 資助

**HPV vaccine 補打**
- 9-valent vaccine 至 45 歲都有 benefit（如未完成）
- HK 私營 $4,000-$6,000 全程
- 過 45 同 GP 討論 individual benefit

**卵巢**
- Average risk：CA-125 + ultrasound **唔推薦 routine**（false positive 高）
- BRCA 陽性 / 強家族史：salpingo-oophorectomy 諮詢

**甲狀腺（HK 私營經常 sell）**
- USPSTF：無病徵 average risk **唔推薦 routine ultrasound**
- 結節常見、極多 over-diagnosis

**心血管 + 骨**
- 40+ 開始 lipid + BP
- Stop period 前後：骨密度 baseline 考慮

請對應我風險出個人化版本。
</PromptBlock>

### 變化 3：高 risk 家族（parent 50 歲前確診 cancer）

<PromptBlock label="變化 3 — 高 risk 家族策略">

🎯 **第一步：唔好恐慌，但要正視**

50 歲前確診 first-degree = 提早 + 加密 screening signal。可能但唔一定基因。

**Part 1：基因諮詢轉介問題**

我應唔應該轉介基因諮詢？呢啲癌種強 genetic：
- 乳癌 / 卵巢癌（BRCA1/2、PALB2）
- 大腸癌（Lynch syndrome）
- 胰臟 / 前列腺早發（BRCA）
- 黑色素瘤
- 子宮內膜 + 大腸（Lynch）

HK 公立：港大 / 中大臨床遺傳科。私營：腫瘤科轉介。

**Part 2：Screening cadence 提早 + 加密**

- 大腸鏡：親人確診年齡 - 10 年（最遲 40）
- Mammogram：30-35 歲開始
- 加 modality（MRI、ultrasound 補 mammogram）

**Part 3：心理 + 家庭**

- Survivor's anxiety 真實
- 兄弟姊妹 / 子女同樣 risk 嘅溝通
- 家族基因資訊嘅 sharing ethic

**Part 4：保險 timing**

⚠️ 重要：基因 test 後**買保險變難**（pre-existing disclosure 義務）。考慮：
- 危疾 / 醫療保險先 secure
- 之後做基因 test
- 同保險經紀坦白討論 timing

⚠️ 規矩：
- 唔好基於 Claude output 自己決定要唔要 BRCA test
- 基因諮詢 = 必經一步（唔係淨係驗血）
</PromptBlock>

## 一個心態把關

40 歲後 screening 決策嘅核心錯誤：**將 screening 等同「保險」**——你買 30K package 心理上覺得「我有買 = 我冇事」。

真實 picture：

- Screening 揀啱 = 提升早期發現率（顯著降低 mortality）
- Screening 揀錯 = false positive 焦慮 + 重複跟進 + over-diagnosis 治療冇必要嘅 lesion
- **唔做 evidence-based screening ≠ 健康**。係延後資訊嘅 cost。

Claude 喺呢度嘅角色：

- 幫你**對應 international guideline + 個人 risk profile** 揀啱
- 拆私營 marketing 嘅 over-test / under-test
- 預設 abnormal result 嘅 mental + financial buffer

⚠️ Claude 唔可以做嘅：

- 替代 GP / 家庭醫生嘅個人化臨床判斷
- 解讀你嘅 imaging / pathology report
- 預測你嘅 risk percentage
- 取代基因諮詢專業

Screening 唔係一次性消費。係 5-10 年嘅 cadence。今年揀啱啲，下個 5 年身體有啲乜嘢，你會有 baseline 可以對比——而呢個 baseline 就係早期發現嘅根基。

⚕️ **任何擔憂、任何 abnormal、任何家族 cluster——同你嘅 GP 講。Claude 係準備夥伴，唔係診症室。**

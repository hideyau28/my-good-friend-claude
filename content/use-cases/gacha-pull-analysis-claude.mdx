## 情境

你打開原神，見到 banner：「夜蘭復刻！限時 21 日。」

腦入面立即跳出嗰句：

> 「夜蘭一年先復刻一次⋯⋯今次唔抽下次唔知幾時⋯⋯而且我已經抽過好多五星⋯⋯抽多 30 連應該有保底啦⋯⋯」

呢段內心戲，係咪好熟悉？呢個就係 gacha 嘅設計：用 **稀缺（限時）+ FOMO（一年一次）+ 沉沒成本（之前抽過嗰啲）+ 樂觀偏誤（應該有保底）** 4 個槓桿同時推你撳「+10」掣。

**唔係話唔可以抽——而係**：「我每月真係划得到幾多錢去抽」、「我而家嘅 pity 喺邊」、「機率上 30 連真係夠？」呢 3 條問題，你抽之前有冇答到？

呢篇用 Claude 喺**你撳「確認」之前嗰 60 秒**，幫你由「感覺」抽變理性決定。

⚠️ 呢篇唔係叫你戒，係幫你**知道自己抽緊乜**。知道自己抽緊乜嘅玩家，通常都健康啲。

## 跟住做

### 1. 列出你嘅 baseline（一次過設定，之後重用）

開新 Claude chat，第一次先建立 baseline：

<PromptBlock label="完整 prompt — 建立你嘅 gacha baseline">
我玩 [遊戲名，例：原神 / 星穹鐵道 / 藍檔]。

📊 **基本 spec**
- 我嘅 server / region：[asia / global]
- 玩咗：[幾耐]
- 帳號 stats：[level / pity 數 / 5★ 角 / 5★ 武器]

💰 **過去 3 個月課金記錄**
（睇 Apple / Google Play 個訂閱同 IAP 記錄）
- 月份 1：$X
- 月份 2：$Y
- 月份 3：$Z
- 合共：$XXX HK

🎯 **我嘅合理 budget**
我屋企／生活負擔：[簡單描述]
我覺得每月最多花 [$Y] 喺呢隻遊戲，先唔影響必要開支

🧮 **數學 baseline**
- 呢隻 game 嘅 pity 系統（如：原神 90 個保 5★、180 硬保 50/50）
- 1 個 5★ 角色嘅預期成本（畀我 in-game currency + 折算 HKD）
- 1 個「想抽嘅指定 5★」預期成本（包括要連 weapon banner 一齊抽嘅情況）

📂 **儲起呢個 baseline**
我之後每次想抽 banner 之前會 paste 返呢段 baseline，請即時對返我嘅 budget + pity 同呢個 banner，畀「抽 / 唔抽 / 抽幾多」嘅建議。

語氣：朋友 + 數學補習老師，唔好淨係叫我戒，係幫我抽得清楚明白。
</PromptBlock>

⚠️ **關鍵**：呢個 baseline 你只係設定一次，儲喺一個 Claude project 或者你個 notes app 入面，下次用直接 paste。

### 2. Banner 出現時，做個 check

新 banner 出嘅時候，唔好直接撳「+10」。先 paste：

> 「呢個 banner 係：[角色名] + [武器名]。我而家 pity 係 [數字]，距離保 50/50 仲差 [數字]。原神今期我已經花咗 $[數字]，仲有 [幾日] 落架。請按我 baseline，答我：抽 / 唔抽 / 抽幾多。」

Claude 應該畀到類似：

```
🎯 Quick answer: 抽 10 連觀察 50/50，唔好 commit 90 抽。

📊 Reasoning:
- 你 baseline budget $300/月，今期已花 $180
- 你呢個月 pity = 53，距 90 保底差 37
- 10 連 expected ≈ $32 USD ≈ HK$250 (剛剛超出你今月剩餘 $120)
- 50/50 機率：6%（10 連 hit 5★）× 50%（指定）= 3%

⚠️ Trap warning:
- 你今期 spend 已超出 monthly avg 40%
- Pity 53 仲遠，今期 commit 90 = 過 $1,500 HK
- 角色 power 對你而家 abyss progress 並非必需
```

呢個唔係叫你唔抽，係**抽之前同抽之後，認清現實一次**。

### 3. 抽完後更新課金記錄

抽完，無論中咗定輸咗 50/50，更新返你個記錄：

> 「我啱啱抽咗 10 連，中咗 4★。新 pity = 63。今期 total spend 更新做 $XXX。請更新我嘅 baseline。」

呢個閉環令你下次對數對得更準。

## 變化

### 變化 1：3 個月戒課檢視

每季尾揾一晚，paste 過去 3 個月所有課金記錄：

<PromptBlock label="變化 1 — 季度 gacha 自我審視">
以下係我過去 3 個月喺 [遊戲] 嘅課金記錄：

[月份 1]
- $X1: [purpose, e.g. monthly pass]
- $X2: [purpose, e.g. dual banner pull]

[月份 2]
⋯（續上）

請分析：

**第 1 部分：課金模式**
- 月平均
- 課得最勁嗰個月 + 原因（係咪 anniversary / 出新角）
- 「衝動 cluster」（同一日多次 IAP 嘅規律，通常 = 情緒上頭抽）

**第 2 部分：ROI（自己定義）**
畀我估算：
- 過去 3 個月我抽咗幾多個 5★
- 平均每個 5★ 用咗幾多 HKD
- 對比官方預期成本（畀個對比，話我知係好彩 / 平均 / 黑仔）

**第 3 部分：對生活嘅影響**
- 一季 total spend 等於我嘅 [其他開支，例：食午餐 / 月票 / netflix] 嘅幾多倍
- 如果呢筆錢去咗 [其他用途，例：ETF、旅行基金]，長遠會點

**第 4 部分：觸發規律**
我邊類 banner / 邊類情境最易抽過頭？
（限時復刻？對戰力有幫助？我已經有嗰個 series 嘅新角？）

**第 5 部分：未來 3 個月建議**
- 推薦 baseline budget
- 建議幾條硬規矩（例：唔抽 weapon banner、唔抽 50/50）
- 1 個具體行為改變，下個月試行

語氣：當我係朋友 + 理財顧問。唔好淨係講「玩 game 唔好花錢」。
</PromptBlock>

### 變化 2：新 banner 抽唔抽嘅集體決定

如果你有公會 / Discord 朋友都玩同款，可以做集體理性 check：

<PromptBlock label="變化 2 — 集體 banner discussion 客觀化">
我哋 [N 個] friends 都玩 [遊戲]，今期 banner 係 [角色]。

我哋每人 baseline：
- 玩家 A：pity X，月 budget $X，account 主力 [DPS / support / shield]
- 玩家 B：⋯
- 玩家 C：⋯

請畀**個人化建議**：

每個玩家：
- 抽 / 唔抽 / 睇情況（50/50 贏咗先續）
- 理由（1-2 句）
- 風險程度（低 / 中 / 高）

**全組共識**
- 角色定位 vs 我哋而家嘅 team comp
- 邊個玩家抽會抵過邊個
- 如果有人花 $1000 抽同一個角色，全組「抽完後悔」程度會點

語氣：唔好做 hype man，做我哋嘅清醒第二意見。
</PromptBlock>

### 變化 3：新手玩家評估要唔要入坑

如果你正考慮玩一隻新嘅 gacha 手遊：

<PromptBlock label="變化 3 — 評估係咪要踩呢個 gacha 坑">
我正考慮玩 [新遊戲名]。

我嘅 gacha 履歷：
- 之前玩過嘅 gacha：[列出]
- 平均月花：$X
- 棄過嘅 gacha + 原因：[列出]

我嘅生活狀態：
- 返工強度
- 屋企／伴侶嘅態度
- 我而家其他玩開嘅 hobbies

請評：

**契合度評分（1-10）**
+ 理由

**呢隻 game 嘅課金狼度評級**
（佛系 / 中等 / 油田 / 抽老婆地獄）

**未來 1 年預計花費**
如果我同之前嘅 gacha 規律一樣，預計花幾多

**3 個吸引位**
我會玩得開心嘅元素

**3 個危險位**
我好大機會課過頭 / 玩到攰嘅規律

**建議**
免費試玩一個月先再入坑？跳過？定淨係玩 monthly pass tier？

語氣：唔做 marketer 角色，做個想你玩得開心但又護住你個荷包嘅人。
</PromptBlock>

## 一個心態

Gacha 嘅設計循環係 **先撩起情緒，數學擺第二**。Banner 出嘅嗰一刻，game 已經贏咗——你由平常狀態變咗「FOMO 狀態」，所有決定都帶住情緒稅。

Claude 嘅角色唔係反 gacha，係**幫你塞一個 60 秒嘅冷靜時間**，喺撳「+10」之前同自己對返一次：

- 我今期 budget 仲剩幾多？
- 我而家 pity 嘅預期成本，啱唔啱我 budget？
- 我抽呢個角，係因為我真係想要佢，定係因為**「限時」兩個字**？

呢個 30 秒嘅停頓，長遠會幫你慳返四五位數港幣。

最後：**如果你發覺自己每次 paste baseline 入 Claude 之後依然抽到爆 budget——咁問題就唔係資訊夠唔夠，而係情緒管理**。呢個時候要搵嘅唔係 AI，而係朋友 / 伴侶 / 心理輔導，認真。

GG，理性抽，玩得開心。

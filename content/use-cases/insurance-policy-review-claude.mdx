## 情境

代理 send 你份保單 PDF：

> 「簽完 send 返畀我，下星期生效。」

你快速掃過：保費、保額、續保條款 —— 你覺得無問題。但條款細節入面：

- **不保項目** —— 邊啲病症 / 治療唔保障
- **已存在病症定義** —— 你以為包嘅慢性病，可能落入不保項目
- **等候期** —— 簽完幾耐先會生效
- **賠償上限** —— 每個病症 / 每年嘅隱藏上限
- **「意外」/「危疾」嘅定義** —— 行業標準同一般人嘅理解差好遠

呢類條款細節 90% 嘅買家唔會讀（律師體中文 8 頁）。但呢 8 頁決定你索償嗰陣「係咪真係有保障」。

呢篇用 Claude 做策略性檢視。

> ⚠️ 利益申報：呢個唔係財務 / 保險建議。涉及大保額或複雜病症，搵獨立保險經紀。

## 跟住做

### 1. Upload 整份保單 + 利益說明書

兩份都要：

- **保單文件**（實際合約）
- **利益說明書**（代理 show 畀你睇嘅銷售文件）

兩份內容應該一致，但偶爾會有出入。Claude 會幫你交叉核對。

### 2. 用呢個 prompt

<PromptBlock label="完整 prompt — 保單 30 分鐘策略性檢視">
我會簽呢份保單（[人壽 / 醫療 / 危疾]）。請幫我做檢視。我嘅背景：

- 保單類型：[填]
- 我嘅年齡 + 健康狀況：[填，特別提及任何 pre-existing condition]
- 月 / 年保費：[填]
- 保額：[填]
- 我嘅首要考慮：[揀 — 想保障重病、想保護家人、想有現金價值、純粹保障醫療開支]

請整 5 個部分：

**Part 1：核心保障總結**
- 主要保障 + 保額
- 受保人 + 受益人
- 保費時間表（平準 / 遞增 / 一次過）
- 保單年期

**Part 2：⚠️ 不保項目（最重要）**
逐個不保項目檢視：
- 條款原文
- 解釋簡單廣東話點理解
- 我嘅特定風險（基於我健康 + 生活習慣）
- 如何應對（例：簽之前申報、買附加保障補救）

特別留意：
- Pre-existing condition 嘅定義廣度
- 自殺條款（人壽通常 2 年等候期）
- 「Acts of war」 / 傳染病大流行不保項目
- 高風險活動不保項目（潛水、攀石、賽車）

**Part 3：等候期 + 賠償上限**
- 每個保障項目嘅等候期
- 每個病症上限
- 每年上限
- 終身上限

**Part 4：續保 / 終止**
- 我終止 vs 保險公司終止嘅條件
- 續保權利屬於我（保證）定保險公司（酌情決定）
- 保費將來嘅調整機制

**Part 5：5 條我應該問代理 / 經紀嘅問題**
基於以上資料，列出針對性問題。
</PromptBlock>

### 3. 逐個核對不保項目

Claude 出嘅不保項目總結 95% 啱，但每條條款嘅原文你都要睇一次。保險合約一字之差影響可以好大：「any pre-existing」 vs 「disclosed pre-existing」。

### 4. 帶住針對性問題清單搵代理

代理見到有準備嘅客，**通常會老實申報**。你呢 30 分鐘準備嘅回報：可能免卻你索償嗰陣被拒絕嘅麻煩。

## 變化例子

### 變化 1：你已簽約，事後想了解清楚

<PromptBlock label="變化 1 — 已簽嘅事後檢視">
我已簽咗保險 [幾耐]。請：
1. 用上面同一個結構檢視
2. 標示我而家被困住嘅劣勢
3. 列我可以做嘅行動：升級附加保障 / 買補充保單 / 將來嘅檢視時機
4. 列我未來索償嗰陣最容易撞到嘅爭議風險
</PromptBlock>

### 變化 2：對比兩個 plan（同公司唔同級別）

<PromptBlock label="變化 2 — 兩個 plan 對比">
我考慮 2 個 plan（同保險公司，唔同級別）：[Plan A] vs [Plan B]。
請：
1. 並排保障對比
2. 保費差異 vs 保障差異嘅性價比
3. 我嘅具體情況，邊個啱
4. 「將來升級」vs「即刻簽高一級」嘅取捨
</PromptBlock>

### 變化 3：VHIS（自願醫保）vs 公司醫療保險

<PromptBlock label="變化 3 — VHIS vs 公司醫療">
我公司已包公司醫療保險（policy attached）。我考慮買額外 VHIS。

請：
1. 對比公司 plan vs VHIS 嘅保障範圍
2. 搵出缺口：邊啲病症 / 治療公司唔包但 VHIS 包
3. 計吓 VHIS 嘅扣稅優惠（每人最高 $8K 扣減）
4. 建議：買 VHIS 嘅理據充份嗎？
</PromptBlock>

## 一個思維轉變

保險業嘅資訊不對等比按揭仲嚴重——你索償嗰陣先發現「冇保障」。

Claude 嘅用處：簽之前先搵出不保項目，畀你做知情決定。**30 分鐘嘅準備，可能避免未來 $X,XXX 嘅自費開支**。

最後提醒：呢個過程唔可以取代經紀 / 律師。大保額 plan 最後要搵專業人士確認。

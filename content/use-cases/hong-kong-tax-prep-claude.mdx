## 情境

每年 5 月，稅務局 (IRD) 寄個人入息稅報表（俗稱「綠色表」）落你信箱。

你打開：

- 你年薪、花紅 (bonus)、股票歸屬 (stock vesting)
- 強積金扣除、自願性 MPF 供款
- 進修開支、慈善捐款、供養長者父母
- 居所貸款利息 / 強積金、自願醫保

正常人睇到呢度就頭暈。IRD 份表又密麻麻、又用緊 1989 年嘅排版。你唯有希望唔好填錯：

- **填漏扣稅額 (deduction)**——典型嘅香港中產平均漏報 $5-10K 價值嘅扣稅額
- **填錯金額**——花紅唔記得計埋股票歸屬嘅現金價值
- **漏報供養老人 / 兄弟姊妹**——最常被遺忘嘅扣稅額
- **唔識點樣申索進修費 / 居所貸款利息**——條款細節太多

呢篇教你用 Claude 一次過讀晒你嘅糧單 (payslip)、IR56B、扣稅單據，**交叉核對 + 標示出你唔記得咗嘅扣稅額**，準備好一份可以直接填表嘅數據。

> 利益申報：呢個唔係稅務建議。Claude 嘅輸出係幫你**識得整理 + 識得問**，唔可以取代稅務會計師。涉及複雜情況（信託、海外收入、做生意），請聘請會計師。

## 跟住做

### 1. 收齊文件

最少要有：

- **僱主畀你嘅 IR56B 或 employer's return**（每年 4-5 月由 HR 畀你）
- **過去 12 個月嘅糧單**（4 月底前嘅糧單 + 5 月初嗰張）
- **MPF 帳戶結單**——主動供款嘅紀錄
- **進修費收據**（如有；只認可指定課程）
- **慈善捐款收據**（累積總額多過 $100）
- **自願醫保 / 強積金自願性供款嘅結單**
- **居所貸款利息結單**（如有）

文件齊就上傳落 Claude。

### 2. 用呢個 prompt

<PromptBlock label="完整 prompt — 報稅準備 + 扣稅額審視">
我準備填香港個人入息稅報表（IRD BIR60 表）。我嘅文件已上傳。我嘅背景：

- 課稅年度：[填，例：2025/26]
- 婚姻狀況：[單身 / 已婚 - 個別評稅 / 已婚 - 合併評稅]
- 配偶有冇自己嘅收入：[填]
- 撫養家屬：[列：配偶、子女（年齡）、父母（年齡及是否同住）、兄弟姊妹（如撫養）]
- 全年符合香港稅務居民身份：[Yes / No - 如果部分時間符合，請詳細說明海外日數]

請幫我整以下 5 個部分：

**Part 1：收入確認**

由 IR56B + 12 個月糧單交叉核對：

- **總應評稅入息**：底薪 + 花紅 + 佣金 + 其他（要加埋，亦核對 IR56B 個總數）
- **股票 / 期權嘅現金價值**：歸屬 (vest) 嗰陣嘅市值（IR56B 通常有紀錄，但要驗證）
- **迎新 / 留任花紅嘅申報**：應該包埋喺 IR56B
- **任何「實物利益」**：僱主提供嘅房屋、汽車、教育津貼等（如有，IR56B 會分開列出）

如果總數同 IR56B 或者我嘅糧單總數唔夾，標示出嚟 + 估計邊度出錯。

**Part 2：強制性扣除確認**

- **MPF 強制性供款**：每年最高 HKD 18,000。確認 IR56B 個數字啱
- **其他強制性供款**：例如某啲公務員供款
- **受僱工作支出**：通常係 0，除非有特定嘅受僱開支

**Part 3：⚠️ 自願性扣除（最易漏嘅一段）**

詳細檢視每個潛在嘅扣稅額：

1. **MPF 可扣稅自願性供款（TVC）**：每年最高 HKD 60,000，扣稅
2. **VHIS（自願醫保）保費**：每名受保人每年最高 HKD 8,000，扣稅 — 包埋屋企人嘅 plan
3. **居所貸款利息**：每年最高 HKD 100,000，最長 20 年
4. **認可進修開支**：認可課程，每年最高 HKD 100,000（每位納稅人）
5. **慈善捐款**：受認可嘅慈善機構，累積多過 HKD 100/年先可以扣（最高為入息嘅 35%）
6. **供養父母 / 祖父母 / 外祖父母**：每位長者免稅額
   - 55-59 歲：$25K
   - 60 歲或以上：$50K
   - 同住額外加：$50K
7. **供養兄弟姊妹**：每位免稅額 $15K（如未婚、不能自給）
8. **供養子女**：每個小朋友 $130K（出世嗰年加 $130K）
9. **單親免稅額**：$132K（如符合）
10. **長者住宿照顧開支**：每位長者最高 $100K

逐項列出：
- 我嘅文件有冇證據
- 我潛在可扣但忘記咗嘅（例如阿媽上年夠 60 歲但你未更新）
- 具體金額 + 申索填落 BIR60 第幾行
- 注意位 / 常見陷阱

**Part 4：稅款估算**

基於 Part 1 - 3，畀我：

- 應課稅入息（扣除後）
- 應繳稅款（用累進稅率或標準稅率，揀較低者）
- 實際稅率
- 對比上年指標（如果有上年數據）

⚠️ 呢個估算用嚟做初步核對，唔可以取代 IRD 官方嘅計算。

**Part 5：行動清單 + 5 條問會計師嘅問題**

- 我要跟進收集嘅缺失文件（例如 Part 3 入面忘記咗嘅扣稅額）
- 5 條如果唔肯定，應該搵稅務會計師問嘅尖銳問題
- 提示下年應該保留嘅文件紀錄
</PromptBlock>

### 3. 交叉核對 Part 1 嘅數字

Claude 對收入嘅交叉核對通常 99% 啱，但每年都有 1 個個案花紅 / 佣金計法唔同。最少要驗證：

- IR56B 嘅「入息」總額
- 你自己計嘅糧單總額（12 個月底薪 + 浮動薪酬）
- 如果相差 5% 以上，搵 HR 確認

### 4. 由 Part 3 列出缺失文件，去跟進

呢個係成個流程回報最高嘅部分。大部分香港打工仔每年都漏報扣稅額：

- 父母 60 歲生日嗰年唔記得更新
- 進修費收據收埋咗但搵唔到對應嘅欄位
- 一年入面每月定期捐款合計多過 $1K，但唔記得填

收齊缺失文件，填表時多扣 $5-10K 係好常見嘅事。

### 5. 填完表，再用 Claude 檢查一次

填完表（紙質 / 報稅易）之前截圖，再用 Claude 驗證：

<PromptBlock label="最後表格檢查 prompt">
我填完 IRD BIR60。截圖 / 掃描喺下面（連附頁）。

請：
1. 對比我之前分享嘅文件 + 你之前嘅分析
2. 標示任何數字同上次分析唔同嘅地方
3. 標示任何表格入面看似矛盾嘅地方（例：配偶有收入但你 tick 咗「沒有」）
4. 我有冇漏填某頁 / 某部分

呢個係最後安全網。交畀 IRD 之前最後一查。
</PromptBlock>

## 變化例子

### 變化 1：你係自由工作者 / 自僱人士

<PromptBlock label="變化 1 — 自僱報稅">
我係自由工作者 / 自僱人士。需要填 BIR60 嘅「利得稅」部分（同僱員嘅「薪俸稅」唔同）。

請：
1. 計算我嘅營業額（由發票數據得出）
2. 列出我可以扣減嘅業務開支（租金、設備、專業費用、見客交通費等）
3. 計出應評稅利潤
4. 強調我需要保留 7 年嘅文件紀錄
5. 標示：自僱人士有冇額外扣稅額（例如：自己嘅 MPF 自願性供款）
6. 如果我預計下年營業額大幅改變，提示暫繳稅嘅影響
</PromptBlock>

### 變化 2：你有海外收入 / 海外資產

<PromptBlock label="變化 2 — 海外收入處理">
我有部分年度嘅海外收入（例：[年 / 月] 喺 [國家]，剩低喺香港）。
或者
我有海外資產（出租物業、海外股息）。

請：
1. **明確警告**：呢類情況強烈建議搵稅務會計師，因為避免雙重課稅協定 + 司法管轄區問題好複雜
2. 但準備工夫我可以做嘅：整理文件、計算每項收入來源、認清 IRD 要求申報嘅嘢
3. 標示：香港嘅「地域來源原則」對我個案嘅意義
4. 整一份「我會問會計師嘅問題清單」（5-7 條）

呢個情況唔好嘗試自己搞報稅。準備好資料之後直接請專家。
</PromptBlock>

### 變化 3：你係小生意老闆 / 公司創辦人

<PromptBlock label="變化 3 — 公司創辦人報稅">
我有自己嘅有限公司（香港註冊）。我嘅個人稅務涉及：
- 從我公司收嘅薪金
- 從我公司收嘅股息
- 任何董事貸款
- 任何股份增值

請：
1. 強調：個人稅務 + 公司稅務分開處理。呢個 prompt 只涵蓋個人方面
2. 個人報稅角度：從香港公司收嘅股息 = 唔扣個人薪俸稅（因為已經喺公司層面扣咗利得稅）
3. 董事貸款嘅影響
4. 嚟自我自己公司嘅股份薪酬：稅務處理
5. 預警 5 個常見嘅老闆報稅陷阱

⚠️ 呢類情況強烈建議搵執業會計師 (CPA)。我畀呢啲資料係為咗準備用，唔可以取代專業建議。
</PromptBlock>

## 一個思維框架

報稅嘅心理負擔唔在於「畀錢」——係 **「我可能填漏 + 麻煩稅務局寫信嚟問」嘅焦慮**。

Claude 嘅槓桿在於 **將「我覺得我可能會忘記」嘅疑慮，變成「我已經交叉核對過、有標註、知道有咩取捨」嘅自信**。

呢個轉化值得花 45 分鐘嘅功夫。

最後提醒：**呢個 prompt 嘅輸出唔係稅務建議**。稅務局嘅最終決定永遠係以稅務局為準。你嘅責任係盡最大誠意去申報。如果你唔肯定，畀 $1-2K 搵會計師永遠係值得嘅投資——保你免被查數、保你晚晚有覺好瞓。

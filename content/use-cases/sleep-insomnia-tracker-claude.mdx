> 急性失眠（3 個月以下）Claude 可以幫追蹤。慢性失眠（超過 3 個月）**必須睇 GP** — 慢性失眠通常有潛在原因（焦慮 / 抑鬱 / 睡眠窒息症 / 藥物）。自我追蹤唔取代醫療診斷。

## 情境

而家係 1:14am。

你 11:30pm 上床，「今晚要早瞓」決心強烈。讀 30 頁書，瞓低 set 鬧鐘 7am。

11:55pm：仍然清醒。Sigh。

12:15am：個腦跳到「明日 9am meeting 要準備」。

12:30am：你 check 電話（just to silence rate）。而家你已經清醒。電話 scroll 20 分鐘。

12:50am：電話放低，再嘗試瞓。個腦想「呢樣我諗緊嘅 — 等我快快諗清楚」。

1:14am：你發現已經 1 個鐘以上上床後，瞓覺 window 縮緊。對唔瞓得到嘅焦慮（meta 焦慮）開始主導。「If 我而家唔瞓，明日 destroyed」嘅循環思維。

3:20am：終於瞓著。

7:00am：鬧鐘。4 個鐘睡眠。

7:30am：腦霧。咖啡。唔知原因。

呢個循環喺過去 3 個禮拜重複。你試過：

- 早瞓（11pm）：仍然 1:30am 先瞓著
- 唔喺床上用電話（Day 5）：Day 1-3 較好，Day 4-7 一樣
- 安眠藥 OTC（褪黑激素 3mg）：Day 1-2 work，Day 3-7 一樣
- 「就唔好諗」：唔 work（思考停頓嘅死結）

呢個急性失眠嘅根本原因：

- **嫌疑兇手太多**（咖啡因時機 / 酒精 / 屏幕 / 焦慮 / 運動時機 / 食 / 室溫 / 鬧鐘 / 伴侶打鼻鼾 / 壓力 / 荷爾蒙）
- **冇有系統嘅數據** — 你嘅睡眠記憶 24 小時後扭曲
- **「就咁瞓」嘅邏輯唔 work** — 焦慮 + 腦過度激活 = 矛盾性失眠
- **隨機嘗試唔隔離變量** — 你改咗 3 樣同時，唔知邊樣 work

呢篇用 Claude **14 日睡眠紀錄**，有系統咁排查 4 大兇手，識別你個人觸發規律。

## 跟住做

### 1. 理解 4 大兇手 + 自我評估

開始記錄前，先建立心理模型邊 4 樣最可能係兇手：

<PromptBlock label="完整 prompt — 睡眠兇手概覽 + 我嘅評估">
我急性失眠 3 個禮拜。想有系統咁診斷自己觸發之前先記錄。

請解釋 4 大兇手 + 我自我評估邊個風險最高：

🍵 **兇手 1：咖啡因**
- 半衰期 5-6 個鐘（你 2pm 飲咖啡 = 8pm 仍然 50% 喺身體入面）
- 影響：延遲瞓著 + 減低深度睡眠質素
- 自我評估：
  - 我每日杯數：[_]
  - 最後一杯時機：[_]
  - 飲：[咖啡 / 茶 / 能量飲品 / 朱古力（係，有咖啡因）]
- 高風險如：超過 3 杯/日 OR 最後一杯喺 12pm 之後
- 快速測試：1 週限朝早只飲（11am 前 cut off）

🍷 **兇手 2：酒精**
- 錯誤思維：「酒精幫瞓覺」
- 現實：酒精加速瞓著 BUT 嚴重降低睡眠質素（破壞 REM、半夜醒）
- 自我評估：
  - 每週飲：[_]
  - 夜晚飲頻繁度：[_]
- 高風險如：超過 2 杯/週 OR 每晚都飲
- 快速測試：1 週零酒精 6pm 後

📱 **兇手 3：屏幕光 + 刺激**
- 藍光延遲褪黑激素
- 心理刺激（Instagram / 新聞 / 工作 email）高皮質醇
- 自我評估：
  - 上床 30 分鐘電話使用：[_]
  - 類型：[scroll IG / 回工作 / 睇片 / 讀]
  - 熄燈後仲用電話？
- 高風險如：上床 30 分鐘有電話（任何類型）
- 快速測試：1 週電話喺睡房外 10pm 起

🧠 **兇手 4：焦慮 / 大腦過度激活**
- 「焦慮思考」觸發皮質醇釋放
- 個腦將未解問題視為威脅 → 激活
- 自我評估：
  - 上床後隨機思緒入侵：[_]
  - 對唔瞓得到嘅預期焦慮：[_]
  - 日間壓力水平 1-10：[_]
- 高風險如：上床後持續活躍思考，日間壓力超過 7
- 快速測試：瞓前倒思緒日記（下一個 prompt）

🌡 **值得提嘅（影響細啲但要 check）**：
- 室溫：最理想 16-19°C（HK 夏天有挑戰）
- 鬧鐘光（數字鐘強光擾人）
- 伴侶打鼻鼾
- 運動時機（高強度運動瞓覺前 3 個鐘內降低睡眠質素）
- 遲食重餐
- 荷爾蒙（女性月經週期）

📊 **我嘅初步診斷自我評估（1-5 量表）**：
- 咖啡因風險：__/5
- 酒精風險：__/5
- 屏幕風險：__/5
- 焦慮風險：__/5

請對我嘅評估，建議 14 日排除次序（最高風險先）：
- Day 1-7：集中邊個兇手
- Day 8-14：集中邊個兇手

語氣：有系統，無判斷。
</PromptBlock>

Claude 出你具體優先次序。如咖啡因 + 焦慮高風險 → Day 1-7 咖啡因排除測試，Day 8-14 焦慮協議測試。

請整每日睡眠紀錄範本，每晚瞓覺 + 每朝起身 5 分鐘填。

- 今日運動（強度 + 時機）：___
- 戶外日光暴露：___ 分鐘
- 今日最大壓力事件（1-10）：___

📋 **每週檢視 prompt**

7 日紀錄後，貼入 Claude：
「請檢視過去 7 日紀錄。識別：
- 邊一晚瞓得最好？邊一晚最差？
- 好嘅晚之間共同規律？
- 差嘅晚之間共同規律？
- 主要兇手對應我嘅風險側寫？

語氣：詳細但快速填（最多 5 分鐘）。
</PromptBlock>

⏰ **可持續記錄嘅貼士**：
- Set 電話鬧鐘 9:55pm 每日提醒
- 喺 notes app 留住範本
- 朝早填之前唔好 check email（避免工作皮質醇污染）
- 14 日堅持 — 唔好 Day 5 退出因為「煩，仲未有進步」

### 3. 瞓前倒思緒儀式（焦慮兇手）

如焦慮高風險，呢步必須：

<PromptBlock label="完整 prompt — 瞓前倒思緒儀式">
我焦慮係失眠高風險。我想要一個 5 分鐘瞓前日記儀式。

🌃 **設定（做一次）**
- 紙本日記放床邊（NOT 電話 — 電話引刺激）
- 1 支筆攞住
- 1 個具體時間：上床前 30 分鐘

📝 **每晚 5 分鐘結構**：

**第 1 分鐘：倒思緒（1 分鐘 raw）**
- 寫低任何喺心入面嘅嘢
- 連住寫，唔審查
- 5 分鐘前你諗緊咩？擔憂清單？隨機？

**第 2 分鐘：「明日清單」（1 分鐘）**
- 明日必做 top 3 項目
- 預先寫喺紙 = 個腦釋放「我必須記得」嘅負擔
- 「If 我唔瞓，我仲知明日優先次序」

**第 3 分鐘：「擔憂清單」（1 分鐘）**
- 對於每個當前心入面嘅擔憂：
  - 可行動？下一行動：___
  - 唔可行動 / 未來事件？「擔憂停車場 — 星期三重看」
- 將擔憂外化 — 個腦理解「呢個已記低，我可以放低直到星期三」

**第 4 分鐘：1 個感激（1 分鐘）**
- 唔好行貨「I'm grateful for health」
- 具體：「今日 [具體時刻] feel 好因為 [具體原因]」
- 唔係自助 cliche — 係神經學重置去副交感神經

**第 5 分鐘：身體掃描（1 分鐘）**
- 瞓低
- 由腳趾 → 頭，每個身體部位 1 秒
- 留意緊張位，唔評判
- 喺每個位置呼出

💤 **然後瞓**
- 唔好對瞓著速度有壓力
- 如上床 20 分鐘仍清醒：起身，輕讀 10 分鐘（NOT 電話），再返床
- 唔好望住個鐘 — 將佢轉開
- 5am 完全清醒？起身 — 好過攰夜晚 + 推下一個上床時間早啲，唔好留床上恐懼

📋 **同儀式互補嘅工具**
- 上床 30 分鐘前：1 杯洋甘菊茶（溫和）
- 室溫 18°C
- 黑窗簾（光線入侵打亂 REM）
- 白噪音機如街道嘈
- 鎂甘氨酸 200-400mg 床前 1 個鐘（如已服藥諮詢 GP）

🚨 **唔好**
- 唔好用 OTC 安眠藥持續超過 7 日（依賴風險）
- 唔好用酒精「幫」（嚴重降低睡眠質素）
- 唔好喺床上躺超過 20 分鐘試瞓 — 破壞睡眠 - 床嘅連繫

語氣：平靜、具體、可持續儀式。
</PromptBlock>

呢個儀式嘅關鍵領悟：**個腦嘅「我必須諗清楚呢樣嘢」屬於認知激活**。將佢外化上紙 = 個腦釋放「處理」命令 = 副交感神經激活 = 啟用睡眠。

### 4. Day 14 檢視 + 升級決定

14 日後做有系統嘅檢視：

<PromptBlock label="完整 prompt — Day 14 睡眠檢視 + 升級">
我完成 14 日睡眠紀錄。請有系統咁檢視。

📊 **貼數據總結：**
[14 日每日關鍵 bullet：
- 上床 / 瞓著 / 起身 / 時長
- 睡眠質素 1-10
- 咖啡因時機
- 酒精 Y/N
- 屏幕 ≥ 9:30pm Y/N
- 日間焦慮 1-10
- 顯著]

📈 **分析：**

**1. 睡眠時長趨勢**
- 14 日平均
- 趨勢（改善 / 停滯 / 下滑）？
- 變異（一致或大幅波動）？

**2. 兇手相關性**
- 邊一晚最好 = 共同規律？
- 邊一晚最差 = 共同規律？
- 具體因素顯著相關於差睡眠？
- 唔永遠直覺（例：運動日之夜諷刺地較難瞓）

**3. 對排除協議嘅遵從度**
- 我嘅排除測試（咖啡因 / 酒精 / 電話）執行完整？
- 失誤？顯著失誤對應睡眠影響？

**4. 介入有效性**
- 邊個介入 worked？
- 邊個介入顯示無效？
- 邊個我哋未測試？

📋 **判決**

**選擇 A：識別咗明確觸發 + 有改善**
- 持續進行協議
- 重測 7 日確認
- 6 個月後評估可持續性

**選擇 B：有明顯改善但仍 5-6/10 質素**
- 加 1 個新變量測試下 7 日
- 具體：邊一個風險變量未隔離？
- 持續紀錄

**選擇 C：冇明確改善 / 持續**
- 升級：GP 探訪
- 可能潛在問題（焦慮症 / 抑鬱 / 睡眠窒息症 / 慢性痛）
- 唔好繼續自我協議如 3 個月累積冇進步

🚨 **升級門檻（停自我追蹤 + GP）：**
- 持續失眠超過 3 個月
- 日間功能受損（工作 / 心情 / 意外）
- 睡眠少於 5 個鐘定期
- 半夜醒定期 3-4am 無法返
- 懷疑睡眠窒息症（大聲鼻鼾 + 朝醒筋疲力盡儘管 7 個鐘）
- 自我服藥酒精 / OTC 安眠藥持續

📞 **GP 探訪準備**
- 帶睡眠紀錄（14 日數據）
- 具體投訴：「過去 3 個月持續失眠影響日常功能。自我追蹤排除測試 4 個兇手，冇明確可隔離觸發。想探索潛在原因。」
- GP 可能轉介到：
  - 睡眠診所做多導睡眠圖
  - 精神科評估焦慮 / 抑鬱
  - 慢性痛 / 內科（如具體投訴）

語氣：數據導向判決，適當升級。
</PromptBlock>

## 變化例子

### 變化 1：輪班工作 / 通宵時間表失眠

夜班 / 三班制時間表打亂：

<PromptBlock label="變化 1 — 輪班工作失眠適應">
我 [款待業 / 護士 / 24 小時運作 / 輪班工作] 持續下移睡眠時間表。

📐 **輪班工作生理時鐘挑戰**
- 身體天然演化為日活躍
- 強迫夜活躍 = 生理時鐘錯位
- 慢性輪班工作者有較高心血管 / 心理健康風險
- 但現實：唔係所有工作允許標準時間

✅ **策略（喺限制內）**

🛏 **日睡眠最優化**
- 全黑窗簾（必要，唔係 optional）
- 白噪音遮蓋日間噪音
- 室溫進取地涼（16-17°C）
- 電話鎖出房
- 家人 / 同住者知道「4pm 前唔好打擾」

🍵 **咖啡因策略**
- 班頭：full mug
- 班中：小額外
- 班結束前 4 個鐘：cut off（咁你瞓上床）

🌞 **陽光時機**
- 班結束 → 屋企：戴太陽眼鏡（擋朝早光，保留褪黑激素）
- 屋企 → 床：dim 燈 30 分鐘斜坡
- 起身：5 分鐘戶外陽光（循環重置）

📅 **休息日策略**
- 唔好完全翻轉時間表
- 「錨點睡眠」中 4-5 個鐘一致跨所有日
- 例：永遠瞓 2pm-9pm 最少，即使休息日

🍽 **餐時機**
- 重餐喺「晚餐」（夜班工作者班中）
- 細小食班中
- 唔好食 2 個鐘前瞓覺，不論鐘時間

💪 **健康影響監察**
- 每 6 個月 GP check
- 心血管指標
- 心理健康 check
- 維他命 D（陽光剝奪常見）

🔚 **長遠考慮**
- 職業軌跡：輪班工作年可持續？
- 健康成本真實 — 唔忽視
- 商討彈性如有選擇

語氣：現實，唔係所有問題輪班工作持續時可解決。
</PromptBlock>

### 變化 2：懷疑睡眠窒息症

如打鼻鼾 + 日間筋疲力盡儘管 7+ 個鐘：

<PromptBlock label="變化 2 — 睡眠窒息症懷疑 + 醫療路徑">
我留意到：
- 持續大聲打鼻鼾（伴侶 / 家人報告）
- 朝醒筋疲力盡儘管 7-8 個鐘「睡眠」
- 半夜抽氣 / 哽咽感
- 朝頭痛
- 日間困頻繁

請解釋 + 醫療路徑：

🫁 **睡眠窒息症係咩**
- 睡眠期間重複氣道阻塞
- 身體短暫醒嚟重啟呼吸（你唔記得）
- 嚴重個案：每晚數百次微醒
- 睡眠數量 OK 但質素嚴重降低

⚠️ **風險因素**
- 男性，40+
- 過重 / 肥胖（但都有瘦人）
- 厚頸圍
- 家族病史
- 吸煙
- 酒精（惡化）
- 鼻竇 / 過敏阻塞

🏥 **點解診斷關鍵**
- 未治：心血管疾病 2-3 倍風險
- 中風風險提升
- 認知受損
- 工作場所意外 / 車禍風險（疲駕）
- 生活質素大影響

📋 **自我篩查（GP 之前）**
1. Print STOP-BANG 問卷（網上免費）
2. 睡眠伴侶觀察：數呼吸暫停 / 抽氣
3. 自己瞓覺錄影 / 錄音（電話錄音喺床邊）
4. 記日間病徵

🩺 **GP 探訪稿**
「我懷疑睡眠窒息症。過去個月：持續大聲打鼻鼾（有人見證）、抽氣醒、朝頭痛、日間筋疲力盡儘管 8 個鐘睡眠。STOP-BANG 分數：[X]。想轉介去睡眠診所做多導睡眠圖。」

🏥 **HK 睡眠診所選擇**
- 公營：GP 轉介去 QMH / 瑪嘉烈睡眠中心（輪候 6-12 個月）
- 私家：養和 / 港島東醫院睡眠中心（較快但 $$）
- 在家睡眠測試（HSAT）：高性價比初步篩查

🛌 **如診斷可能治療**
- CPAP 機（黃金標準，對 OSA 改變生命）
- 下顎前移裝置（輕度個案）
- 減重（如適用）
- 姿勢治療
- 手術（罕，嚴重個案）

💰 **保險覆蓋**
- 大部分 HK 保險：診斷測試 + CPAP 部分覆蓋
- 通常需要預先批准
- 睡眠診所幫你 navigate

🚨 **唔忽視**
- 「只係打鼻鼾」嘅心態危險
- 窒息症 ≠ 心臟弱 — 生理學問題
- 治療顯著改善生活質素
- 未治嘅心血管後果累積

語氣：迫切但唔慌張。

### 變化 3：旅行 / 時差協議

跨時區 trip 後失眠：

<PromptBlock label="變化 3 — 時差恢復 + 旅行睡眠">
我 trip 返自 [目的地 X 時區差]。時差影響睡眠 [N] 日。

請整恢復協議：

✈️ **時差生理學**
- 身體「內部時鐘」（生理時鐘）同新時區錯位
- 經驗規則：每 1 小時時差復原 1 日
- 向東（例：HK→US 東岸）難過向西（US→HK）
- 跨越褪黑激素、皮質醇、體溫循環

📅 **行前準備（如預期 trip）**
- 出發前 3-4 日：每日將上床時間移 1 小時向目的地
- 「預先適應」減少時差影響
- 進取地補水

🛫 **機上**
- 保持補水（長途 3L 水）
- 限酒精 + 咖啡因
- 每 90 分鐘郁（循環 + DVT 預防）
- 喺目的地夜時瞓覺
- 喺目的地日時清醒

🌅 **Day 1-3 抵步調整**
- 早朝早陽光 30 分鐘（同步信號）
- 強迫保持清醒到目的地上床時間（唔好午睡超過 30 分鐘）
- 早早餐 / 朝早 routine 目的地時間
- 戶外運動（合併太陽 + 動）
- 褪黑激素 0.3-1mg 瞓前 1 個鐘目的地（NOT 5mg 藥房預設）
- Day 1-3 避免酒精

📋 **Day 4-7 鞏固**
- 睡眠應該正常化
- 持續問題過 Day 7：通常睡眠衛生基線 check
- 持續超過 2 週 = 諮詢 GP

💊 **輔助（少用）**
- 褪黑激素（時機 > 劑量）
- 偶用 zolpidem（只處方，NOT 慣性）
- 鎂為放鬆
- 洋甘菊 / 纈草茶

🚫 **唔好**
- 酒精做「安眠藥」（更差時差）
- 「補睡眠」週末後 — 推唔重置週期
- 咖啡因中午後 Day 1-5
- 重餐瞓前

📊 **追蹤恢復**
- 每日瞓著、起身、時長
- 主觀精力
- 期望：完整恢復 by Day [小時時區差數]

語氣：耐心，時差真實唔係軟弱。
</PromptBlock>

## 一個思維框架

失眠嘅死結真相：**「我必須瞓」嘅念越強，越難瞓**。

呢個死結嘅機制：個腦將「無法瞓著」當作威脅。威脅 = 交感神經激活 = 皮質醇釋放 = 激活。激活同睡眠相反。

呢個思維嘅轉變：14 日睡眠紀錄嘅更深目的唔係「最優化睡眠數字」——係 **釋放你對睡眠嘅過度控制**。
- 睡眠唔係你「成就」嘅嘢，係你「允許」嘅嘢
- 你個腦過度關注睡眠 = 無法瞓
- 將追蹤外化畀 Claude / 紀錄 = 個腦釋放控制命令

Claude 喺呢度嘅角色：你獨自追蹤 14 日心理負擔高，容易放棄。Claude 容器 = 可持續、結構化、無評判。你倒數據，Claude 反映規律。

最後提醒：

- ✅ **3 個月係關鍵時間表**。3 個月以下 = 急性，自我追蹤可能足夠。3 個月+ = 慢性，GP 探訪強制。
- ⚠️ **唔好「補番」失去嘅睡眠**。睡眠唔係銀行戶口。睡眠週期重置比睡眠時長累積更重要。
- 🆘 **如果你而家躺床上 2am 讀緊呢篇**：起身、調暗燈、讀 15 分鐘、真攰時返。唔好喺床上強迫瞓。

Sleep well。或者唔 — 都 OK。14 日後睇規律。

## 情境

而家係 12:14am。

你個鐘頭前開咗窗，因為新界 5 月晚上 28 度悶到死，你想等空氣對流啲先瞓。窗台寬 30cm — 你心諗主子（你隻 4 歲短毛貓）平時都唔會跳出去，5 樓嘅鐵窗有條鐵枝佢應該過唔到。

你瞓著咗 15 分鐘，醒返。

主子唔喺佢平時瞓喺你腳邊個位。你叫佢，冇反應。你開燈，行勻成屋——梳化底、書架後、廁所、廚房櫃。冇。

你企喺廚房，望住個開咗嘅窗，個鐵枝其實有條 12cm 隙位你之前冇留意——啱啱夠隻貓過。

你個胃沉落去。

**你嘅腦袋而家完全當咗機。** 你想喊。你打「lost cat hong kong」入 Google，搜出 500 條結果，每條 link 你都打開唔到。你揾你伴侶，佢瞓緊覺，你 30 秒之內又閂咗個 message 唔想嘈醒佢。

呢個 30 分鐘極重要。**頭 48 小時係搵返走失寵物嘅黃金時間。** 你需要：

1. 一份睇得明嘅走失寵物傳單（實體版印出嚟 + 電子版 share）
2. 一個有系統嘅社區通知計劃（邊度貼 / 邊個 group post / 邊個部門 call）
3. 一個搜索策略（實地搜索範圍 + 時間段）

Claude 唔識搵返你主子，但 Claude 識喺你個腦袋當機嘅 30 分鐘，**幫你將驚慌整理成可以行動嘅步驟**。

呢篇就係教你點用。

## 跟住做

### 1. 先停低驚慌 — 頭 30 分鐘做呢 4 樣嘢（唔需要 Claude）

開 Claude 之前，先做呢啲，因為貓走失頭 1 小時佢仲可能喺度：

- **🔦 樓下 + 同層走廊搜一次**：手電（唔好太光）+ 細聲叫主子個名 + 帶包佢平時鍾意嘅零食（churu / 凍乾雞肉）。貓走失通常匿喺**最近嘅黑暗匿藏位**——後巷、單車停泊位、停車場底、樓梯間。
- **🚪 同層鄰居 — 留字條唔好夜半敲門**：寫張手寫字條塞入信箱：「主子走失 [時間 + 描述 + 你電話] — 如見到請 WhatsApp」。等朝早再跟進。
- **🪟 開返扇窗 + 喺窗外放隻貓砂盆 + 佢嘅毛巾**：貓用嗅覺定位，佢自己嘅貓砂氣味 + 你嘅氣味可以引佢返。
- **📸 揾返主子最近 5-6 張不同角度嘅相**：正面、側面、全身、特徵（白手手 / 斑紋 / 黑點）。傳單 + post 都會用。

呢 4 樣做完，**你個驚慌程度應該由 100 跌到 70**——你採取咗行動。

而家坐低，打開 Claude，開始整傳單。

<PromptBlock label="完整 prompt — 走失寵物傳單（貓版）">
我隻貓主子喺 [日期 + 時間 + 地址 area] 走失。請幫我整一份好用嘅走失貓傳單，內容包括實體傳單文字 + 一個 80 字嘅電子 post 版本。

🐾 **主子嘅 details**：
- 名：主子
- 品種 / 毛色：[短毛 / 長毛 / 混種]
- 顏色：[tabby / 黑 / 白 / 橙 / 雙色 / 三色 + 詳細描述]
- 性別 + 絕育：[male/female + 已絕育 Y/N]
- 大約 size：[幾 kg / 細 / 中 / 大]
- 年齡：[歲]
- 特徵（**最重要**）：
  - [眼色 — 黃 / 綠 / 藍 / 異色瞳]
  - [頸帶 Y/N — 顏色 + 名牌有冇電話]

📍 **走失 location**：

- 你電話：[]

- 如 call 唔到，second contact：[]

請整：

**A. 實體傳單文字（A4 印出嚟貼街）**
- 大字大標題："走失！LOST CAT"（雙語）
- 1 張大相位 + 2 張細相位（不同角度）
- 主子嘅描述（bullet points，最多 5 條 — 太多 detail 反而冇人睇）
- 走失 location + 時間
- 聯絡 + 酬金
- 結尾 1 句情感呼籲（短，唔好過度戲劇化）

**B. 電子 post（WhatsApp / FB / IG 用）**
80 字中英混合香港講法。Emoji 適度。結尾呼籲「請 share + 如見到請 WhatsApp [number]」。

語氣：urgent but factual。**唔好誇張情緒**（路人睇傳單 5 秒就走，靠描述具體唔係靠情感）。
</PromptBlock>

Claude 出嘅傳單你會見到佢有清晰嘅層次 — 大標題 / 相 / 5 條 bullet 描述 / 聯絡。你複製入 Canva（免費 template「Lost Pet Flyer A4」就有），擺 5 張相，印 30 份。

**A4 印 30 份係最低消費**。後面要貼。

### 3. 香港社區通知計劃 — 邊度貼 / 邊個 group post

傳單印好，下一步係落地執行。Claude 幫你整通知 checklist 按重要性排。

<PromptBlock label="完整 prompt — HK 走失寵物社區通知計劃">
我隻貓主子喺 [地址 + 區] 走失。我已經印咗 30 份 A4 傳單 + 整咗 80 字電子 post。

我要一份**香港走失貓社區通知行動計劃**，按重要性排（最高 impact 排頭）：

請涵蓋：

🏪 **實體張貼地點（24 小時內貼晒）**
- 同棟大廈：lift lobby / 信箱房 / 物管 office 告示板 — 聯絡人：物管 office（提供建議講法點同保安講）
- 同屋苑 / 附近屋苑：同樣
- 半徑 500m 範圍：
  - 寵物店（PetPet Park / Whiskers N Paws / 小型獨立 pet shop）
  - 獸醫診所內部告示板
  - 7-Eleven / 便利店（部分接受社區通告）
  - 商場 customer service desk
  - 街市附近燈柱（**注意：唔好亂貼私人物業，可能被人撕。集中喺公共社區告示板**）
- 走失方向估計嘅熱點：公園 / 後巷 / 停車場社區告示板

📱 **香港走失寵物 FB / 網上群組（按重要性排）**
請列出香港最 active 嘅走失寵物社區群組（包括 cat-specific + 一般走失寵物），加埋每個 group 嘅 posting 規矩提示（例如：部分 group 要 admin approve / 部分要先讀 pinned rules）。我知道 examples 包括：
- 香港愛護動物協會 (SPCA) 失寵尋寵
- HK Lost & Found Pets
- 區域性 group（Tai Po / Sha Tin / 新界東 等跟我住嘅區）

🏛 **政府 + NGO 部門 call list**
- 漁護署（AFCD）— 晶片資料庫報失流程
- SPCA — lost & found service 點 register
- 任何 24 小時動物醫院 — 通常會 keep 走失寵物紀錄（路人撿到會送嚟）
- KCRC / MTR / Bus depot 失物認領 — 如果走失點靠近 station

📧 **個人 network 動員**
- 你嘅獸醫 — call / email，請診所通知佢 client base
- 街坊群組（屋苑 WhatsApp / Telegram channel）
- 你嘅清潔姐姐 / 保安 / 信差 / 早晨買報紙嘅檔主 — 呢類人**最常見到走失貓**（佢哋喺街上時間最多）

⏰ **48 小時時間表**
- Hour 0-2：實地搜索 + 樓下屋苑貼傳單
- Hour 2-6：網上 post 全部出晒
- Hour 6-24：第 2 輪實地搜索（黃昏時段 — 貓通常 dawn/dusk 活躍）
- Hour 24-48：擴大範圍到 1km，重貼脫落傳單
- Day 3+：每日 routine 搜索 + 升級到誘捕籠如冇目擊

請每個地點加備註：要唔要事先請准、大概 cost（多數 free）、估計 effectiveness。
</PromptBlock>

Claude 會出一份 ~15-20 個可以做嘅 items list，你可以一個一個剔走。

**最易唔記得嘅一樣**：你嘅獸醫。獸醫診所嘅 staff 收目擊報告嘅頻率好高，路人撿到貓會送去獸醫核實晶片。Day 1 朝早 9am 就打你獸醫電話：「我隻主子走失，麻煩 broadcast 畀你診所嘅 client」。

### 4. 用 Claude 整個每日搜索記錄 + 升級條件

走失貓唔係單一事件 — 係多日搜索過程。你需要一個記錄 + 點先升級到下一步。

<PromptBlock label="完整 prompt — 每日搜索記錄 + 升級計劃">
我隻主子走失咗 [幾日 / 幾小時]。我要一個**每日搜索記錄範本 + 升級條件**。

請整：

📋 **每日搜索記錄（你每日填一次）**
- 日期 + 時間段
- 搜索地點（具體 — 後巷名 / 樓層 / 公園）
- 用咗咩引誘（零食 / 玩具聲 / 你把聲）
- 觀察：見過任何目擊 / 路人 report / 留低嘅食物有冇被食
- 鄰居 / 保安 feedback（邊個報過見到 / 邊個 area / 何時）
- 下次搜索點調整：[根據今日觀察改變搜索地點 / 時間 / 方法]

🚨 **升級條件 — 出現呢啲就升級**

如果 24 小時冇目擊：
- 重貼脫落傳單（雨水 / 風 / 人撕）
- 擴大範圍到 800m
- Post 多 2-3 個區域性 FB group
- 聯絡多 1 間區內獸醫診所

如果 48 小時冇目擊：
- 誘捕籠（向 SPCA / cat rescue volunteer 借）
- 攝錄機（行山 camera / Wyze cam）放後巷 / 餵流浪貓嘅 spot
- 提高酬金（如果一開始細）
- 請寵物偵探（香港有少數，例如「香港尋寵協會」）

如果 1 週冇目擊：
- 心理重整 — 走失貓 1-3 週後返自己地頭嘅 case 唔少
- 持續每週 check 獸醫診所 + 漁護署 pound 紀錄
- 保持傳單喺某啲高人流地點（**唔好全部撤** — 8 週後仲有人報目擊嘅 case 存在）

🛑 **情緒節奏提醒**
- 唔好 Day 1 通宵搜 — 你 Day 2-7 仲要有體力 + 判斷力
- Day 3-7 係情緒低點，預先 prepare
- 同伴侶 / 家人溝通唔好獨自承擔
- 如果情緒崩潰 = SPCA bereavement helpline / 信任嘅朋友傾偈

請語氣：calm + structured，唔好過度安慰（虛假希望比真實期望更傷）。
</PromptBlock>

Claude 整咗呢份記錄，你貼喺雪櫃。每日填一次。**唔係每次搜索都填**——一日結束填一次，做一日總結反思。

呢個記錄嘅隱藏價值：當你第 5 日情緒低，你回頭睇 Day 1-4 你做咗咩，會發覺你做咗有意義嘅嘢，唔係無助。

## 變化

### 變化 1：純室內貓走失（vs 室外）

純室內貓走失同室外貓走失嘅搜索策略完全唔同：

<PromptBlock label="變化 1 — 純室內貓走失搜索調整">
我隻主子係純室內貓，從來冇行過街。佢 [點走失 — 窗 / 門開咗 / 搬屋 / 修理工人入屋]。

請調整搜索策略：

🏠 **室內貓走失行為**
- 室內貓走失通常**唔會行遠**（80% 喺 500m 範圍內搵返）
- 佢哋會匿喺**最近嘅黑暗 + 安靜匿藏位**——熟悉大廈嘅後巷 / 停車場 / 樓梯間 / 雪櫃機房
- 佢哋極驚，**唔會回應你叫名**（會匿得更實）
- 反而會喺 night 2am-5am 慢慢探索

🔦 **調整搜索方法**
- 搜索時間：dawn 5-6am + dusk 7-9pm（貓活躍時段）
- 搜索方法：**唔好叫名** — 貓太驚唔會回應。改用：
  - 帶把佢平時鍾意嘅零食個袋（搖出聲）
  - 帶個熟悉嘅玩具（逗貓棒 / 鈴鐺）
  - 影音 record 佢平時嘅 meow 聲 play out（佢可能回應同類聲）
- 搜索範圍：頭 24 小時 focus 50m 內，慢慢擴大
- 留嘢喺大廈外：佢用過嘅貓砂（嗅覺 marker）+ 你著過嘅 t-shirt + 食物盒（記得 cover 防其他動物）

📷 **野生動物 cam / 攝錄機**
便宜 setup（HK$200-400）：
- TP-Link Tapo C100 室內 camera 放後巷對準食物盒
- 24-48 小時可以確認佢有冇出嚟食
- 如見到，落誘捕籠時機到

⏰ **唔好太快升級**
室內貓第 1-3 日通常太驚唔會出嚟食。**Day 2-4 先開始有目擊係正常**。唔好 Day 1 就放誘捕籠（貓太驚 trigger 個籠反而嚇得更實）。

語氣：理性，提醒室內貓主人唔好放棄太快——80%+ 室內貓 1-2 週內喺 50m 內搵返。
</PromptBlock>

### 變化 2：狗走失（不同於貓嘅搜索）

狗走失嘅搜索方式同貓完全唔同：

<PromptBlock label="變化 2 — 香港走失狗搜索計劃">
我隻狗 [名 + 品種 + size + age] 喺 [地點 + 時間] 走失。

請整 **走失狗**專屬計劃（同貓唔同 because 狗會 social + 行得遠）：

🐕 **狗走失行為**
- 狗會行得遠（10km+ 唔出奇，尤其驚嘅狗）
- 狗會跟氣味 / 跟陌生人 / 跟其他狗
- 狗 visible — 路人見到報告率比貓高
- 狗驚會定格唔郁，唔好直接撲過去捉 — 會嚇佢逃跑

🔍 **調整搜索**
- 搜索範圍：頭 24 小時 1-2km，48 小時擴大到 5km
- 行 walking trail / 公園（狗會本能揾草地 / 樹）
- 聯絡附近所有獸醫診所 + 漁護署 pound（**狗強制晶片 — 路人撿到必須 scan**）
- Post 區內 FB group + 行山 group（行山客係主要目擊來源）
- 巴士 / 小巴司機 — 沿途見到狗 alert（部分 route 願意 share message）

🏛 **重點升級 — 漁護署**
香港狗強制晶片 + 牌照。路人撿到送漁護署嘅機會率高。
- 漁護署動物管理中心 hotline：1823
- 24 小時內 register 走失狗報告
- 持續每日 check（佢哋 keep 流浪狗 14 日，過期可能人道毀滅 — 緊迫）

📷 **目擊點點捉（最關鍵）**
- 如果有目擊，**唔好衝過去**——狗驚會逃跑得更遠
- 帶高嗅覺食物（cooked chicken / 香腸）
- 蹲低，唔好直接 eye contact
- 用佢熟悉嘅聲調 calmly 叫佢
- 慢慢 approach，等佢自己過嚟

🚨 **誘捕籠 & 升級**
如 48 小時冇捉到：
- 喺目擊熱點設餵食點 + camera
- 聯絡香港愛護動物協會 / 保護遺棄動物協會 借誘捕籠
- 請寵物偵探（專長追狗）

語氣：urgent — 狗喺街上 risk 更高（車禍 + 漁護署 pound 時間限制）。
</PromptBlock>

### 變化 3：失而復得之後 — 防止再走失 + 後遺處理

主子返咗——首先恭喜你。但走失過嘅貓 / 狗通常會有行為後遺：

<PromptBlock label="變化 3 — 失而復得善後計劃">
我隻主子走失咗 [幾日] 之後返咗。佢宜家：
- 體重：[輕咗 / 同走前差不多]
- 行為：[匿、唔肯食、過度黐人、聽到聲就驚]
- 受傷：[Y/N — 邊度 / 程度]

請整 **失而復得照顧 + 防止再走失** 計劃：

🏥 **24-48 小時內睇獸醫**
即使主子表面 OK：
- Check 脫水 + 體重 baseline
- Scan 晶片確認仍 functional
- Check 隱性傷口 / 寄生蟲（戶外幾日易有 fleas / ticks）
- 大便樣本（戶外食 strange thing 可能影響腸胃）

🍽 **慢慢重新餵**
唔好一返就大餐：
- 頭 24 小時：1/2 份量多餐（4-5 次）
- Day 2-3：慢慢加返正常份量
- 水：encourage drinking，唔好強迫
- 觀察嘔吐 / 大便變化，異常即睇獸醫

🧘 **行為後遺**
走失過嘅貓常見變化：
- 對引起走失嘅事件變敏感（窗 / 開門 / 維修工人 / 大聲 — 邊樣 trigger 主子走失？要特別 address）
- 過度黐人 1-2 週（正常 — 畀安全感）
- 突然匿埋：正常 phase，唔好強拉佢出
- 睡眠 pattern 亂咗（白天瞓 / 夜晚活躍）— 通常 2-4 週重整

🚪 **防止再走失 — 實地審查**
請整審查 checklist：
- 窗：所有窗安裝鐵窗網（防貓 mesh — 唔係普通鐵枝，要 6mm grid max）
- 門：玄關設雙門系統 / 寵物閘（外人入屋時主子唔即衝出）
- 露台：完全圍封（**香港主子常見走失地點**）
- 晶片：主子有冇晶片？冇就趁睇獸醫即做（HK$300 一次 lifetime）
- 名牌頸帶：刻咗電話號碼（**雖然貓唔鍾意，但重要**）

🆘 **以防萬一 kit**
喺屋企 keep 一個走失寵物 kit：
- 最新相（每年更新）
- 主子嘅外貌描述一頁（背得熟，急時可以即報）
- 晶片號碼寫低
- 寵物店 + 獸醫 + 漁護署電話寫低
- 5 張後備傳單（萬一）

語氣：reassuring 但實際。主子已經返咗，但創傷真實，照顧者亦需要消化自己嘅焦慮。

## 一個心態

走失寵物嘅頭 30 分鐘，你嘅腦袋完全當咗機。

呢個當機狀態唔係你軟弱——係生物本能。當你感知到一個重要依附對象失去咗，你嘅皮質醇 spike，前額葉（負責 planning）短暫 shutdown，你變成只能 react 嘅生物。

**呢個心態嘅關鍵：你個腦當機嗰陣，需要一個外部結構接管。**

呢個外部結構唔一定要係 Claude — 可以係一張提前寫好嘅字條、可以係一個信任嘅朋友 call 你 walk through、可以係一個你 bookmark 咗嘅 SPCA emergency guide。

Claude 嘅價值喺呢度：你慌嗰陣個腦諗唔到順序行動計劃，但你仲打得到「主子走失，幫我」入 Claude。佢可以喺 30 秒內 畀返你個結構。

**最後三樣提醒**：

- ✅ **走失頭 48 小時黃金時間 — 但 8 週後仲搵返嘅 case 唔少**。唔好早放棄。
- ⚠️ **晶片唔係保證搵返**——但係搵返嘅最強指標。冇晶片嘅寵物走失 return rate ≈10%，有晶片 ≈40%。趁宜家就做。
- 🆘 **唔好獨自承擔**。SPCA bereavement helpline (2802 0501) 唔淨係處理寵物死亡——亦提供走失寵物主人嘅情緒支援。打去傾偈唔等於放棄。

希望你主子平安返到。

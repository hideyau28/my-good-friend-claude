## 情境

而家係 12:14am。

你個鐘頭前開咗窗，因為新界 5 月晚上 28 度悶到死，你想等空氣對流啲先瞓。窗台寛 30cm — 你心諗主子（你隻 4 歲 short-hair cat）平時都唔會跳出去，5 樓嘅鐵窗有條鐵枝佢應該唔過得到。

你瞓著咗 15 分鐘，醒返。

主子唔係佢平時瞓喺你腳邊個位。你叫佢，冇反應。你開燈，行嗮成屋——梳化底、書架後、廁所、廚房櫃。冇。

你企喺廚房，望住個開咗嘅窗，個鐵枝其實有條 12cm 隙位你之前冇留意——剛好夠隻 cat 過。

你嘅胃沉落去。

**你嘅腦袋而家完全 offline。** 你想喊。你 type「lost cat hong kong」入 Google，搜出 500 條結果，每條 link 你都打開唔到。你揾你 partner，佢瞓緊覺，你 30 秒之內又 close 個 message 唔想嘈醒佢。

呢個 30 分鐘極重要。**頭 48 小時係搵返走失 pet 嘅黃金時間。** 你需要：

1. 一份睇得明嘅 lost pet flyer（實體版印出嚟 + digital 版 share）
2. 一個有系統嘅社區通知 plan（邊度貼 / 邊個 group post / 邊個部門 call）
3. 一個搜索策略（實地搜索 radius + 時間段）

Claude 唔識搵返你主子，但 Claude 識喺你個腦袋 offline 嘅 30 分鐘，**幫你將 panic 整理成可以行動嘅 steps**。

呢篇就係教你點用。

## 跟住做

### 1. Stop panic — 頭 30 分鐘做呢 4 樣嘢（唔需要 Claude）

開 Claude 之前，先做呢啲，因為 cat 走失頭 1 小時佢仲可能喺度：

- **🔦 樓下 + 同層走廊搜一次**：手電（唔好太光）+ 細聲叫主子個名 + 帶包佢平時鐘意嘅 treat（churu / freeze-dried 雞肉）。Cat 走失通常匿喺**最近嘅黑暗匿藏位**——後巷、單車停泊位、停車場底、樓梯間。
- **🚪 同層鄰居 — 留 note 唔好夜半敲門**：寫張手寫紙條塞入信箱：「主子走失 [時間 + 描述 + 你電話] — 如見到請 WhatsApp」。等朝早再 follow-up。
- **🪟 開返扇窗 + 喺窗外放隻 litter box + 佢嘅毛巾**：cat 用嗅覺定位，佢自己嘅 litter smell + 你嘅 scent 可以引佢返。
- **📸 揾返主子最近 5-6 張不同角度嘅 photo**：face front、side profile、full body、特徵（白手手 / 斑紋 / 黑點）。Flyer + post 都會用。

呢 4 樣做完，**你個 panic level 應該由 100 跌到 70**——你採取咗行動。

而家坐低，打開 Claude，開始整 flyer。

**A. 實體 flyer text（A4 print 出嚟貼街）**

**B. Digital post（WhatsApp / FB / IG 用）**

**A4 print 30 份係 minimum**。後面要貼。

- 街坊群組（屋苑 WhatsApp / Telegram channel）
- 你嘅 cleaner / 保安 / 信差 / 早晨買報紙嘅檔主 — 呢類人**最常見到走失 cat**（佢哋街上時間最多）

**最易唔記得嘅一樣**：你嘅 vet。Vet clinic 嘅 staff 收 sighting report 嘅頻率好高，路人撿到貓會送去 vet 確認 chip。Day 1 朝早 9am 就打你 vet phone：「我隻主子走失，麻煩 broadcast 畀你 clinic 嘅 client」。

### 4. 用 Claude 整個每日搜索 log + 升級條件

走失 cat 唔係單一 event — 係多日 search process。你需要一個 log + 點先升級到下一步。

<PromptBlock label="完整 prompt — Daily search log + 升級 plan">
我隻主子走失咗 [幾日 / 幾小時]。我要一個**每日搜索 log template + 升級條件**。

請整：

📋 **每日搜索 log（你每日填一次）**
- 日期 + 時間段
- 搜索地點（具體 — 後巷名 / 樓層 / 公園）
- 用咗咩引誘（treat / 玩具聲 / 你 voice）
- 觀察：見過任何 sighting / 路人 report / 留低嘅食物 trace 有冇被食
- 鄰居 / 保安 feedback（邊個 reported 見過 / 邊個 area / 何時）
- 下次搜索點調整：[根據今日觀察改變搜索地點 / 時間 / 方法]

🚨 **升級條件 — 出現呢啲就升級**

如果 24 小時冇 sighting：
- 重貼脫落 flyer（雨水 / 風 / 人撕）
- 擴大 radius 到 800m
- Post 多 2-3 個區域性 FB group
- Contact 多 1 間 area vet clinic

如果 48 小時冇 sighting：
- Humane trap（向 SPCA / cat rescue volunteer 借）
- Camera trap（行山 camera / Wyze cam）放後巷 / 餵 stray cat 嘅 spot
- 提高酬金（如果一開始細）
- 請寵物偵探（HK 有少數，例如「香港尋寵協會」）

如果 1 週冇 sighting：
- 心理 reset — 走失 cat 1-3 週後返自己地頭嘅 case 唔少
- 持續每週 check vet clinic + AFCD pound log
- 保持 flyer 喺某啲 high-traffic 地點（**唔好全部撤** — 8 週後仲有人 report sighting 嘅 case 存在）

🛑 **情緒節奏 reminder**
- 唔好 Day 1 通宵搜 — 你 Day 2-7 仲要有體力 + judgment
- Day 3-7 係情緒低點，預先 prepare
- 同 partner / 家人 communicate 唔好獨自承擔
- 如果情緒崩潰 = SPCA bereavement helpline / 信任嘅朋友傾偈

請語氣：calm + structured，唔好過度 reassuring（虛假希望比真實期望更傷）。
</PromptBlock>

Claude 整咗呢份 log，你貼喺 fridge。每日填一次。**唔係每次搜索都填**——一日結束填一次，做一日總結反思。

呢個 log 嘅隱藏價值：當你第 5 日情緒低，你回頭睇 Day 1-4 你做咗咩，會發覺你做咗有意義嘅嘢，唔係無助。

## 變化

### 變化 1：Indoor-only cat 走失（vs outdoor）

Indoor-only cat 走失同 outdoor cat 走失嘅搜索策略完全唔同：

<PromptBlock label="變化 1 — Indoor-only cat 走失搜索調整">
我隻主子係 indoor-only cat，從來冇行過街。佢 [點走失 — 窗 / 門開咗 / 搬屋 / 修理工人入屋]。

請調整搜索策略：

🏠 **Indoor cat 走失行為**
- Indoor cat 走失通常**唔會行遠**（80% 找回喺 500m radius 內）
- 佢哋會匿喺**最近嘅黑暗 + 安靜匿藏位**——熟悉 building 嘅後巷 / 停車場 / 樓梯間 / 雪櫃機房
- 佢哋極驚，**唔會 respond 你叫名**（會匿得更實）
- 反而會喺 night 2am-5am 慢慢探索

🔦 **調整搜索方法**
- 搜索時間：dawn 5-6am + dusk 7-9pm（cat active 時段）
- 搜索方法：**唔好叫名** — cat 太驚唔會 respond。改用：
  - 帶把佢平時鐘意嘅 treat 個袋（搖出聲）
  - 帶個熟悉嘅 toy（fishing wand / 鈴鐺）
  - 影音 record 佢平時嘅 meow 聲 play out（佢可能 respond 同類聲）
- 搜索 radius：頭 24 小時 focus 50m 內，慢慢擴大
- 留嘢喺 building 外：佢嘅 used litter（嗅覺 marker）+ 你穿過嘅 t-shirt + 食物盒（記得 cover 防其他動物）

📷 **Wildlife cam / camera trap**
便宜 setup（HK$200-400）：
- TP-Link Tapo C100 indoor camera 放後巷對準食物盒
- 24-48 小時可以確認佢有冇出嚟食
- 如見到，trap 時機到

⏰ **唔好太快升級**
Indoor cat 第 1-3 日通常太驚唔會出嚟食。**Day 2-4 才開始 sighting 係正常**。唔好 Day 1 就放 humane trap（cat 太驚 trigger trap 反而嚇得更實）。

語氣：理性，提醒 indoor cat owner 唔好放棄太快——80%+ indoor cat 1-2 週內喺 50m 內 found。
</PromptBlock>

### 變化 2：狗走失（不同於 cat 嘅搜索）

狗走失嘅搜索方式同 cat 完全唔同：

<PromptBlock label="變化 2 — Lost dog HK 搜索 plan">
我隻狗 [名 + breed + size + age] 喺 [地點 + 時間] 走失。

請整 **lost dog**-specific plan（同 cat 唔同 because 狗會 social + 行得遠）：

🐕 **狗走失行為**
- 狗會行得遠（10km+ 唔出奇，尤其驚嘅 dog）
- 狗會 follow scent trail / 跟陌生人 / 跟其他狗
- 狗 visible — 路人見到報告率比 cat 高
- 狗驚會 freeze，唔好直接撲過去 catch — 會嚇佢逃跑

🔍 **調整搜索**
- 搜索 radius：頭 24 小時 1-2km，48 小時擴大到 5km
- 走 walking trail / 公園（狗會本能揾草地 / 樹）
- Contact 附近所有 vet clinic + AFCD pound（**狗強制 microchip — 路人撿到必須 scan**）
- Post 區內 FB group + 行山 group（行山客係主要 sighting source）
- 巴士 / 小巴司機 — 沿途見到狗 alert（部分 route 願意 share message）

🏛 **重點升級 — AFCD**
HK 狗強制 microchip + license。路人撿到送 AFCD 嘅機會率高。
- AFCD 動物管理中心 hotline：1823
- 24 小時內 register lost dog report
- 持續每日 check（佢哋 keep stray dog 14 日，過期可能 euthanize — 緊迫）

📷 **Sighting 點 catch（最關鍵）**
- 如果有 sighting，**唔好衝過去**——狗驚會逃跑得更遠
- 帶高嗅覺 food（cooked chicken / sausage）
- 蹲低，唔好直接 eye contact
- 用佢熟悉嘅聲調 calmly 叫佢
- 慢慢 approach，等佢自己過嚟

🚨 **Trap & 升級**
如 48 小時冇 catch：
- 喺 sighting hotspot 設 feeding station + camera
- Contact 香港愛護動物協會 / 保護遺棄動物協會 借 humane trap
- 請寵物偵探（專長追狗）

語氣：urgent — 狗喺街上 risk higher（車禍 + AFCD pound 時間限制）。
</PromptBlock>

### 變化 3：失而復得之後 — 防止再走失 + 後遺處理

主子返咗——首先恭喜你。但走失過嘅 cat / dog 通常會有行為後遺：

<PromptBlock label="變化 3 — 失而復得 aftercare plan">
我隻主子走失咗 [幾日] 之後返咗。佢宜家：
- 體重：[輕咗 / 同走前差不多]
- 行為：[匿、唔肯食、過度黏人、jumpy at sounds]
- 受傷：[Y/N — 邊度 / 程度]

請整 **失而復得 care + 防止再走失** plan：

🏥 **24-48 小時內 vet visit**
即使主子表面 OK：
- Check 脫水 + 體重 baseline
- Scan microchip 確認仍 functional
- Check 隱性 wound / parasite（戶外幾日易有 fleas / ticks）
- Stool sample（戶外食 strange thing 可能影響腸胃）

🍽 **慢慢 refeed**
唔好一返就大餐：
- 頭 24 小時：1/2 portion 多餐（4-5 次）
- Day 2-3：慢慢加返正常 portion
- 水：encourage drinking，唔好強迫
- 觀察嘔吐 / 屎變化，異常即 vet

🧘 **行為後遺**
走失過嘅 cat 常見變化：
- 對引起走失嘅事件變敏感（窗 / 開門 / 維修工人 / 大聲 — 邊樣 trigger 主子走失？要 specifically address）
- 過度黏人 1-2 週（正常 — 畀安全感）
- 突然 hide：正常 phase，唔好強拉佢出
- 睡眠 pattern 亂咗（白天 sleep / night active）— 通常 2-4 週 reset

🚪 **防止再走失 — 實地審查**
請整 audit checklist：
- 窗：所有 windows install 鐵窗網（cat-proof mesh — 唔係普通鐵枝，要 6mm grid max）
- 門：玄關設 double door system / pet gate（外人入屋時主子唔即衝出）
- Balcony：full enclosure（**HK 主子常見走失地點**）
- Microchip：主子有冇 chip？冇就趁 vet visit 即做（HK$300 一次 lifetime）
- ID collar + tag：phone number engraved（**雖然 cat 唔鐘意，但重要**）

🆘 **Just-in-case kit**
喺屋企 keep 一個 lost pet kit：
- 最新 photo（每年更新）
- 主子嘅外貌描述一頁（背得熟，急時可以即報）
- Microchip number 寫低
- 寵物店 + vet + AFCD phone 寫低
- 5 張 spare flyer（萬一）

語氣：reassuring 但實際。主子已經返咗，但 trauma 真實，照顧者亦需要消化自己嘅焦慮。
</PromptBlock>

## 一個心態

走失寵物嘅頭 30 分鐘，你嘅腦袋完全 offline。

呢個 offline 狀態唔係你軟弱——係生物本能。當你 perceive 一個重要依附對象失去咗，你嘅 cortisol spike，prefrontal cortex（負責 planning）短暫 shutdown，你變成只能 react 嘅生物。

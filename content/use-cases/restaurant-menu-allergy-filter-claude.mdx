## 情境

你有食物限制：

- **Vegan / 素食**：避免肉、蛋、奶、蜜糖
- **麩質敏感 / 麩質不耐**：避免麥、米麵、醬油酵素問題
- **海鮮敏感**：避免蝦、蟹、貝，有時連魚都唔得
- **無酒精**：宗教 / 健康 / 懷孕
- **低鈉**：高血壓 / 心臟病要小心
- **清真 / 猶太餐**

你同朋友 / 家人出去食飯。一入餐廳，侍應遞上個 menu——20 頁中英對照，菜名大多數係「金粟豆芽炒河」呢類，冇列明成份。

你慣常嘅做法：

- **問侍應**：花 5 分鐘逐個菜詳細形容，但侍應有時自己都唔肯定
- **食「安全」嗰款**（白飯 + 淨味菜）：可能就咁錯過咗成間餐廳嘅招牌菜
- **博一博落單**：一中招撞到觸發食物，餐又浪費咗，身體又唔舒服

呢篇教你用 Claude 手機 app：**影張 menu 相，30 秒就篩選出你食得安全嘅菜式**。

## 跟住做

### 1. 開 Claude.ai 手機 app

iOS / Android 都得。第一次用之前登入一次，之後再用就唔使再設定。

### 2. 影 menu

成個 menu 拍清啲（影多張相都得，Claude 支援多張圖）。

⚠️ 貼士：
- **避免反光** —— 個 menu 唔好過光
- **覆蓋整頁** —— 唔好剪到 menu 邊
- **如果你只關心某頁**，淨係影嗰頁

### 3. 用呢個 prompt

<PromptBlock label="完整 prompt — 食物篩選">
影咗呢個餐廳嘅 menu。我嘅飲食限制：

- [列你嘅限制，例：vegan、gluten-free、避免海鮮]

請：

**Part 1：✓ 安全菜式**
列出所有我可以食嘅菜：
- 菜名 + 英文 / 中文（用 menu 顯示嘅）
- 點解安全（明確原因）
- 1-2 個「可能擔心位」我應該問侍應確認嘅地方（醬料、煮食油同其他菜共用等）

**Part 2：⚠️ 可能可以嘅菜式**
邊啲菜「可能安全但唔肯定」需要問侍應：
- 菜名
- 具體問題我應該問（例：「呢個炒河用咩油？同蝦嗰個鑊共用㗎？」）

**Part 3：✗ 避開**
邊啲菜明顯唔安全：
- 菜名
- 觸發成份
- 唔需要詳細解釋，短一句

**Part 4：推薦**
基於安全 + 可能類別，推薦：
- **單人落單**：3 個主菜我可以食 + 均衡
- **分享落單**（如果同朋友 / 家人食）：仲有邊啲「對我安全 + 對佢哋受歡迎」嘅菜

語氣：直接、實用。餐廳場景下你冇時間讀長文。

**重要：** menu 入面有時用簡寫或者不尋常嘅菜名。如果唔肯定，老實標「需要同侍應確認」，唔好估。
</PromptBlock>

### 4. 直接同侍應確認可能類別

帶住 Claude 嘅輸出，侍應面前有效率咁問：

> 「呢個炒河同呢個炒飯，請問用嘅油會唔會同其他菜共用？蝦炒飯嗰個鑊同呢碟係咪同一個？」

具體問題好過「我食唔食得？」好多——有具體問題,侍應先至容易答到你。

### 5. 食完更新你自己嘅飲食紀錄

如果食完有反應（少少腸胃不適、麻煩），記低：
- 邊間餐廳
- 邊個菜
- 用咗「安全」/「可能」/「避開」嘅判斷

呢類小小嘅紀錄累積落嚟，**你對自己嘅觸發規律、邊間餐廳信得過**就會越嚟越清楚。

## 變化例子

### 變化 1：陪小朋友（揀飲擇食 + 敏感）

<PromptBlock label="變化 1 — 小朋友友善篩選">
影咗呢個餐廳嘅 menu。我陪 [年齡] 小朋友食飯。

小朋友限制：
- 食物敏感：[列]
- 揀飲擇食偏好：[列，例：唔食蘑菇、唔食辣]
- 鍾意：[列]

請：
1. 篩選安全 + 小朋友可能會食嘅菜（兩個條件）
2. 識別「安全但悶」vs「安全 + 吸引」
3. 推薦分享份量我點落單（小朋友食啲，我食啲）
4. ⚠️ 提示：菜式即使安全，但份量大 / 辣度唔啱小朋友
</PromptBlock>

### 變化 2：日本 / 韓國 / 東南亞嘅 menu（外語）

<PromptBlock label="變化 2 — 外文 menu">
我喺 [日本 / 韓國 / 越南 / 等] 旅行。影咗本地 menu（[語言]）。

我嘅限制：[列]

請：
1. **翻譯**菜名做英文 + 中文
2. 用同樣「安全 / 可能 / 避開」框架
3. 加：文化備註 — 邊啲隱藏成份喺呢個菜系常出但 menu 唔特別提（例：日本 dashi 通常含鰹魚、韓國醬料常含蝦醬）
4. 具體「問侍應」用語用嗰種語言（我手機顯示畀侍應）
</PromptBlock>

### 變化 3：街市 / 街邊小食檔（冇 menu，只有口頭落單）

<PromptBlock label="變化 3 — 街市快速判斷">
我喺街邊小食檔 / 街市。Menu 係腦海清單 / 小黑板 / 一兩款選擇。

我影咗檔口外觀 / 小黑板 / 已準備好嘅食物。我嘅限制：[列]

請：
1. 識別可能嘅食物 / 烹飪方式
2. 「可以問」vs「避開」嘅快速判斷
3. 具體問題我可以用廣東話問檔主（簡短）
4. ⚠️ 提示：街頭食物信任永遠基於檔主嘅老實 + 交叉污染風險一定有
</PromptBlock>

## 一個現實檢視

呢個 prompt 嘅可靠程度唔係 100%：

- **Menu 描述模糊** —— Claude 同你一樣難判斷
- **隱藏成份** —— 醬料 / 高湯 / 煮食油常含觸發物
- **交叉污染** —— 即使菜嘅成份安全，共用嘅鑊 / 廚具都可能觸發

呢個流程嘅**實際價值**係**「先篩走明顯要避開嘅，再幫你縮窄要問侍應嘅具體問題」**。佢唔係要取代你自己應有嘅小心。

如果係嚴重過敏（例如花生過敏會引致過敏性休克、乳糜瀉等），**呢個 prompt 只係一個起點**。一定要親口同侍應 / 廚房確認，有需要就記得帶 EpiPen。

如果係冇咁嚴重嘅偏好（vegan、低鈉、戒酒精），呢個 prompt 就真係幫你簡化成個食飯過程，**令你可以安心享受餐飯，唔使落單落得心驚膽戰**。

最後提醒：呢個用法最妙嘅地方，唔係取代你「自己睇個 menu」，而係**令你有信心同侍應問具體問題**。由「我食唔食得？」嗰種模糊焦慮，變成「呢三個菜，個鑊有冇同人共用？啲油可唔可以換？」嘅具體對話。

呢個信心上嘅轉變，對於日日都要面對食物限制嘅人嚟講，係一個細微但實實在在嘅生活質素提升。

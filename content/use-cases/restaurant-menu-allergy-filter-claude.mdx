## 情境

你有食物限制：

- **Vegan / vegetarian**：避免肉、蛋、奶、蜜糖
- **Gluten intolerance / coeliac**：避免麥、米麵、醬油 enzyme issue
- **海鮮敏感**：避免蝦、蟹、貝、有時連魚都唔得
- **無酒精**：宗教 / health / pregnancy
- **低 sodium**：高血壓 / 心臟病要 watch
- **Halal / Kosher**

你同朋友 / 家人去食飯。 進餐廳，個 menu 來了——20 頁中文 + 英文，dish 名以「金粟豆芽炒河」呢類 描述 為主，無 ingredient 詳列。

你嘅 default option：

- **問侍應**：5 分鐘 detail 描述每個 dish，侍應有時都唔肯定
- **食 「safe」 default**（白飯 + plain 蔬菜）：可能 miss 整個餐廳嘅 highlight
- **冒險 order**：撞到 trigger 食物，浪費餐 + 身體不舒服

呢篇用 Claude 手機 chat：**影個 menu 一張相，30 秒 filter 出你食得安全嘅 dish**。

## 跟住做

### 1. 開 Claude.ai 手機 app

iOS / Android 都得。 第一次 use 之前 sign in，等再次用就唔需要 setup。

### 2. 拍 menu

整個 menu 拍清啲（多張相 OK，Claude 支援 multi-image）。

⚠️ Tips：
- **避免 reflection** —— 個 menu 唔好過光
- **覆蓋整 page** —— 唔好 cut menu 邊
- **如果你只 interested 某 page**，淨係拍 嗰 page

### 3. 用呢個 prompt

<PromptBlock label="完整 prompt — 食物 filter">
影咗呢個餐廳嘅 menu。 我嘅 dietary restriction：

- [列你嘅 restriction，例：vegan、gluten-free、避免海鮮]

請：

**Part 1：✓ Safe dishes**
List 所有我可以食嘅 dishes：
- Dish name + 英文 / 中文（用 menu 顯示嘅）
- 點解 safe（明確 reason）
- 1-2 個 「possible concern」 我應該 ask 侍應 confirm 嘅地方（醬料、cooking oil shared with 其他 dish 等）

**Part 2：⚠️ Maybe dishes**
邊啲 dish 「likely safe but uncertain」 需要 ask 侍應：
- Dish name
- Specific question 我 should ask（例：「呢個炒河 用咩 oil? Share with 蝦 嗰個 wok 嗎?」）

**Part 3：✗ Avoid**
邊啲 dish 明顯 NOT safe：
- Dish name
- Trigger ingredient
- 唔需要 elaborate，short 一句

**Part 4：Recommendation**
基於 Safe + Maybe，推薦：
- **Solo order**：3 個 main dish 我可以食 + balanced
- **Sharing order**（如果 dining with 朋友 / 家人）：仲有邊啲 「safe for me + popular for them」 嘅 dish

語氣：直接、實用。 餐廳場景下你冇時間 read long output。

**重要：** menu 入面有時用 abbreviation 或者 unusual dish name。 如果 unsure，老實標 「需 confirm with server」，唔好估。
</PromptBlock>

### 4. 直接同侍應 confirm Maybe list

帶住 Claude 嘅 output，侍應前面 efficient ask：

> 「呢個炒河同呢個炒飯，請問用嘅 oil 同其他 dish share 嗎? 蝦炒飯 嗰個 wok 同呢碟 share 嗎?」

具體 question 比 「我食唔食得?」 強好多——侍應有 specific question 容易答。

### 5. 食完 update 你 own dietary log

如果食完出反應（少少 GI distress、麻煩），記低：
- 邊間餐廳
- 邊個 dish
- 用咗 「Safe」 / 「Maybe」 / 「Avoid」 嘅 prompt judgement

呢類 micro-feedback 累積落，**你嘅 trigger pattern 同 餐廳 trust scoring** 越來越 sharp。

## 變化例子

### 變化 1：陪小朋友（picky eater + 敏感）

<PromptBlock label="變化 1 — 小朋友 friendly filter">
影咗呢個餐廳嘅 menu。 我陪 [年齡] 小朋友食飯。

小朋友 restriction：
- 食物敏感：[列]
- Picky preference：[列，例：唔食 mushroom、唔食 spicy]
- 喜歡：[列]

請：
1. Filter safe + 小朋友 likely eat 嘅 dish（兩個 criteria）
2. 識別 「safe but boring」 vs 「safe + appealing」
3. 推薦 sharing portion 我點 order（小朋友食啲、我食啲）
4. ⚠️ Flag：菜式 even safe，但 portion size 大／spicy degree 唔啱小朋友
</PromptBlock>

### 變化 2：日本 / 韓國 / 東南亞 嘅 menu（外語）

<PromptBlock label="變化 2 — 外文 menu">
我喺 [日本 / 韓國 / 越南 / 等] 旅行。 影咗本地 menu（[語言]）。

我嘅 restriction：[列]

請：
1. **Translate** dish names 落英 + 中文
2. 用 same 「Safe / Maybe / Avoid」 framework
3. 加：cultural note — 邊啲 hidden ingredient 喺呢個 cuisine 常出但 menu 唔特別 mention（例：日本 dashi 通常 contain 鰹魚、韓國 醬料常 contain shrimp paste）
4. Specific 「ask the server」 phrase 用嗰種語言（我手機 show 畀侍應）
</PromptBlock>

### 變化 3：街市 / 街邊小食檔（無 menu，只有 verbal order）

<PromptBlock label="變化 3 — 街市 quick check">
我喺街邊小食檔 / 街市。 menu 係 mental list / 小黑板 / 一兩款選擇。

我影咗 stall 嘅外觀 / 小黑板 / 已準備好嘅 food。 我嘅 restriction：[列]

請：
1. Identify likely 食物 / 烹飪方式
2. 「Safe to ask for」 vs 「Avoid」 嘅 quick judge
3. Specific question 我可以用廣東話問 stall owner（簡短）
4. ⚠️ Caveat：street food trust 永遠基於 stall owner 嘅 honesty + cross-contamination risk 一定有
</PromptBlock>

## 一個 reality check

呢個 prompt 嘅 reliability 唔係 100%：

- **Menu 描述 vague** —— Claude 同你一樣難 judge
- **Hidden ingredient** —— 醬料 / 高湯 / cooking oil 常 contain trigger
- **Cross-contamination** —— 即使 dish ingredient safe，shared wok / utensil 可能 trigger

呢個 process 嘅 **realistic value** 係 **「先 filter 走 obvious avoid，再 narrow down 你 ask server 嘅 specific question」**。 唔係取代你 due diligence。

對於 severe allergy（peanut anaphylaxis、coeliac等），**呢個 prompt 只係 starting point**。 一定要 verbally confirm 同 server / kitchen，必要 carry EpiPen。

對於 less severe preference（vegan、low sodium、避免酒精），呢個 prompt 真係 streamline 食飯經驗，**令你 enjoy meal 而唔係 panic order**。

最終 reminder：呢個 use case 嘅 magic 唔係取代 「自己讀 menu」，係**令你 confident 同侍應 ask 具體 question**。 由 「我食唔食得?」 嘅 vague anxiety， 變 「呢三個 dish，share wok 嗎? Oil substitute 嗎?」 嘅 specific dialogue。

呢個 confidence shift，對於 daily 撞食物 restriction 嘅人，係 small but real 嘅 quality-of-life upgrade。

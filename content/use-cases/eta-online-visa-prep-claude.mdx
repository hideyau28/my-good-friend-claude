## 情境

你 6 月計劃 3 個月小休 — UK 兩星期，跟住飛美國探朋友 10 日，最後韓國 5 日返香港。

3 個地方而家全部都要網上預批：

- UK ETA（2025 年全面強制）— £16，48 小時批
- US ESTA — US$21，72 小時內通常批
- Korea K-ETA — ₩10,000，72 小時批

你打開 UK 官網 `gov.uk/guidance/apply-for-an-electronic-travel-authorisation-eta`。

表面好簡單 — 11 條問題，全部 yes/no 或者下拉選單。

但係：

> "Have you ever been refused entry to the UK or any other country?"

你 2019 年去美國轉機嗰次，邊個官員問你 "intent of stay" 你答得唔好，被帶去 secondary inspection 問 45 分鐘先放你走。**呢個算唔算 refused entry？**

> "Have you ever been arrested, charged, or convicted of any offence in any country?"

你大學時試過遊行被警察查身份證影低，冇起訴 — 但**有冇紀錄？**

> "Do you have any other nationality or citizenship?"

你父母同時有 BNO + HKSAR — **填 BNO 定 HKSAR？兩個都填？**

你喺呢三條問題上面停咗 20 分鐘。Google 嘅答案矛盾。Reddit 嘅人各有各講。

呢度個關鍵 — **ETA 拒簽之後通常 7 日內唔可以重新申請同一個系統**。UK ETA 拒簽 = 改申請 Standard Visitor Visa（£127 + 3 週處理 + 面試）。ESTA 拒簽 = 改申請 B1/B2 visa（US$185 + 領事館面試 + 數月排期）。

而拒簽嘅理由唔會講畀你聽。你只會收一封「your application was not successful」嘅 email。

呢個網上 ETA 系統嘅陷阱：

- **表面簡單 → 你以為 5 分鐘搞掂**（其實每條問題都有陷阱）
- **問題用法律術語**（"refused entry"、"convicted"、"intent to remain"）但你用日常理解作答
- **錯一條 = 永久紀錄**（將來所有國家簽證問題都要披露）
- **無法重試**（同一 ETA 系統拒咗，等於關咗呢條路）

呢篇用 Claude **25 分鐘預先篩查** — 用 HKSAR / BNO 視角，揾你背景嘅高危位、教你每欄點填、安排多個目的地嘅 timing 次序，連拒簽之後嘅後備路都齊。

唔取代移民律師。如果你背景有任何刑事紀錄、過去簽證被拒、雙重國籍複雜情況，**先諮詢律師再申請**。Claude 嘅角色係幫你睇清楚邊度應該諮詢、邊度可以自己搞。

## 跟住做

### 1. 列你背景 + 旅遊計劃

開 Claude 之前，自己離線寫一份背景概覽。**呢份嘢唔好上雲端 / 分享出去** — 純粹自己睇住跟 Claude 對答用。

包括：

> **護照狀態**
> - HKSAR 護照（號碼、發放日期、到期日）
> - BNO 護照（如有，到期日、過去 5 年用咗幾多次）
> - 其他護照 / OCI / PR 卡（如有）
>
> **過去 5 年出入境**
> - 每次去過邊度（日期 + 國家）
> - 有冇被 secondary inspection / 問話超過 15 分鐘
> - 有冇申請過簽證（批 / 拒 / 撤回）
> - 有冇喺任何國家逾期停留（即使 1 日）
>
> **法律 / 警方紀錄**
> - 有冇被捕（即使冇起訴）
> - 有冇被警察查問 / 帶返警署
> - 有冇法庭傳召（即使民事）
> - 有冇政治參與被紀錄（遊行、捐款、組織）
>
> **僱主 + 財務**
> - 當前公司 + 任職幾耐
> - 月薪 + 存款大概範圍
> - 過去 12 個月有冇換工
>
> **今次行程**
> - 邊 3 個（或更多）目的地
> - 每個停幾耐
> - 入境次序
> - 探親 / 朋友 / 公幹 / 純旅遊嘅組合
> - 同行者

呢份概覽寫得越具體，Claude 嘅篩查越準。模糊輸入 = 行貨輸出。

### 2. 用呢個 prompt 揾高危欄位 + 各欄合適寫法

<PromptBlock label="完整 prompt — ETA / ESTA 申請預先篩查 + 逐欄解讀">
[貼上你嘅背景概覽]

我 HKSAR 護照（+ BNO 如有），準備同步申請：[UK ETA / US ESTA / Korea K-ETA / Japan eVisa / AUS ETA — 列出邊幾個]。

請逐個系統預先篩查我嘅背景，揾出 **高危欄位** + 提供 **合適寫法**。

🚨 **第 1 部分：跨系統高危問題識別**

請逐個列我背景入面對應以下類別嘅欄位，每個解釋風險 + 建議處理：

**類別 A：過去被拒入境 / 簽證被拒**
- 「被拒入境」嘅法律定義（vs 你以為嘅「被問話多」/「secondary inspection」）
- 香港申請人常見誤解：轉機被問話 ≠ refused entry
- 我嘅情況點答（具體）

**類別 B：被捕 / 刑事紀錄**
- 「被捕」vs「被警察查問」vs「拘留」嘅法律區分
- 香港警方查身份證 / 留資料 ≠ arrested
- Caution / fixed penalty notice 算唔算
- 我嘅情況點答

**類別 C：雙重國籍 / 多重公民身份**
- HKSAR 護照持有人嘅複雜性：
  - 同時持有 BNO → 申報定唔申報？
  - 中國國籍嘅技術定位
  - OCI / PR 卡（澳洲、加拿大、英國 settled status 等）算唔算「另一國籍」
- UK ETA 對 BNO 嘅特殊處理（**BNO 持有人唔需要 UK ETA**，但係要小心表格邊度宣告）
- 我嘅情況點答

**類別 D：居留意圖 / 雙重目的**
- 「Will you work / study / engage in business during your visit?」嘅範圍
- 幫香港僱主 remote work 算唔算喺目的地工作？
- 同目的地嘅朋友合作搞 side project 嘅敏感度
- 我嘅情況點答

**類別 E：過去簽證逾期**
- 1 日逾期 vs 1 個月逾期嘅披露要求差異
- 自願離境 vs 被遣送嘅紀錄
- 我嘅情況點答

🎯 **第 2 部分：逐個目的地嘅表格重點欄位**

**🇬🇧 UK ETA（如適用）**
- Employment 填法：「Employed」/「Self-employed」/「Student」 — 我嘅情況揀邊個
- Home address：香港住址中譯英嘅標準格式（line 1 / 2 / postcode 處理）
- Intended stay 日期 + 地點：寫具體 city 定 region
- 「Have you ever been involved in...」嘅長 list — 我點逐條核對
- BNO 持有人特別注意：如果你揀 BNO 入境，**唔需要 ETA**（呢個系統唔適用）

**🇺🇸 US ESTA（如適用）**
- Eligibility questions 嘅 9 條 yes/no — 我每條嘅答案 + 理由
- 「Communicable disease」嘅範圍（HIV 2010 後已刪除，但其他仍適用）
- 「Engaged in espionage / sabotage / terrorism」— 政治活動嘅邊界
- Employer / contact info：香港公司英文名要對應 HR 系統正式名
- Address while in US：朋友屋企地址寫法（zip code 必須）

**🇰🇷 K-ETA（如適用）**
- Occupation 下拉選單嘅 70+ 選項 — 邊個最符我嘅實際職位
- 韓國境內 address：朋友屋企 / Airbnb / 酒店嘅填法
- 「Have you been refused entry to Korea or any country?」— 類別 A 答案重用

**🇯🇵 Japan eVisa（如適用）**
- HKSAR 護照持有人現時仍免簽 90 日 — eVisa 主要係特定情況（長期商務、結婚簽證等）
- 確認你今次行程真係需要 eVisa（而唔係單純免簽入境）
- 如真係需要：Sponsor / Inviting party 嘅欄位點填

**🇦🇺 AUS ETA（如適用）**
- Subclass 601（ETA）vs 651（eVisitor，BNO 適用）vs 600（Visitor visa）嘅揀法
- 「Convicted of any offence」嘅澳洲特別嚴格定義
- Character declaration 嘅完整列表

⏰ **第 3 部分：同步申請嘅 timing 策略**

我計劃 [出發日期]。請排申請次序：

- **邊個系統先申請**（基於處理時間 + 拒簽後後備路嘅複雜度）
- **每個之間隔幾耐**（避免一個拒簽影響下一個申報）
- **如果系統 A 拒咗，系統 B 仲未申請，要點調整寫法**（披露義務由邊一刻觸發）

建議排序例子：
1. 處理時間最長 + 拒簽後最難補救嘅先申請（通常 US ESTA — 拒簽後要做 B1/B2 面試）
2. 中等難度跟住
3. 最快 + 最寬鬆嘅最後（通常 K-ETA）

🚨 **第 4 部分：拒簽之後嘅後備路**

對每個系統，列：

- 拒簽之後可唔可以即時重試同一系統（**通常唔可以 — 至少等 1 個申請週期**）
- 後備簽證類別（標準訪客簽證 / B1-B2 / Standard Visitor）嘅申請成本 + 時間
- 拒簽對其他國家將來簽證嘅披露義務（**呢點關鍵 — 一次拒簽 5-10 年要 declare**）
- 領事館上訴 / waiver path（部分國家有 humanitarian / business waiver）

🛑 **第 5 部分：你睇我嘅情況，需唔需要搵律師**

請老實判斷我嘅背景風險級別：
- **低**：直接申請，跟你建議寫法
- **中**：申請之前再睇 1 次官方 guidance + 喺灰色地帶揀保守答案
- **高**：先諮詢移民律師（香港顧問費 HK$2,000-5,000 一次 — 比拒簽後嘅成本平好多）

語氣：法律精確，唔過度安撫；香港持照人視角；避免行貨「consult a professional」式免責聲明，要具體解釋我嘅情況屬咩級別。
</PromptBlock>

Claude 出嘅篩查你印出嚟，逐條行一次先開官網表格。**唔好表格開咗一半先諗** — 網上 ETA 系統通常 30 分鐘 timeout，半途卡喺敏感問題會慌住亂答。

### 3. 同時申請多個目的地嘅 timing

3 個 ETA 同步申請睇似簡單，但披露義務嘅 timing 好微妙：

<PromptBlock label="完整 prompt — 多目的地 ETA timing 策略">
我要同時申請 [UK ETA + US ESTA + Korea K-ETA]，出發 [日期]。

請整 **timing 決策樹**：

📅 **理想時序**

- T-60 日：開始準備背景概覽
- T-45 日：申請第一個（處理時間最長 / 風險最高 — 通常 US ESTA）
- T-40 日：收到系統 A 結果之後，先申請系統 B
- T-35 日：收到 B 結果之後，申請系統 C
- T-7 日：所有 ETA 應該已批，印出 + 存電子副本

⚠️ **點解唔同步遞交 3 個**

- 如果 A 拒咗，B + C 嘅「Have you been refused a visa / ETA for any country?」答案由「No」變「Yes」
- 已遞交嘅 B + C 你冇得改 — 等於變相虛假陳述
- 一個拒簽連鎖 3 個拒簽嘅風險

📋 **披露觸發時機**

「Have you been refused...」嘅披露義務由 **收到拒簽通知嗰一刻** 觸發。所以：

- A 遞交 + 等緊 → B 申報「No refused」OK（截至遞交時刻屬實）
- A 拒簽通知收到 → B / C 仲未遞交，**必須改答「Yes」+ 解釋**
- A 拒簽通知收到 → B / C 已遞交但未批，**主動聯絡領事館修正**（罕但要做）

🎯 **建議申請次序（香港視角）**

1. **US ESTA 先**（拒簽後後備路最複雜，處理 72 小時通常）
2. **UK ETA 跟住**（拒簽後 Standard Visitor 仲有出路，48 小時處理）
3. **K-ETA 最後**（最寬鬆，72 小時）

如果你係 BNO 持有人去 UK：

- 唔好申請 UK ETA — BNO 入境唔需要
- 但 ESTA 嘅「other nationality」要申報 BNO
- K-ETA 嘅「other nationality」同樣申報

🚨 **特殊狀況**

- 如出發前少過 30 日：壓縮時序，每個系統之間至少隔 24 小時
- 如出發前少過 14 日：直接付加急費（US ESTA 唔接受加錢，UK / K-ETA 有 priority option）
- 如出發前少過 7 日：放棄 ETA，改申請落地簽（如有）或者改機票

語氣：實際 + 顧及風險，唔好過度簡化。
</PromptBlock>

呢個次序嘅核心：**每個 ETA 之間留呼吸位**。你以為平行遞交慳時間，但係拒簽連鎖嘅成本遠大過慢幾日。

### 4. 拒簽應變（waiver / 領事館面試路）

最壞情況準備：

<PromptBlock label="完整 prompt — ETA 拒簽之後嘅應變">
假設我 [其中一個] ETA 拒咗。請畀我 **應變決策樹**：

📨 **收到拒簽嗰刻**

**唔好做**：
- ❌ 即時重試同一系統（通常鎖 7-90 日）
- ❌ 改答案重新遞交（呢個係詐騙，永久封禁）
- ❌ 用其他護照申請同一系統（雙重國籍唔申報 = 詐騙）
- ❌ 取消機票同酒店（拒簽唔等於行程死，仲有後備）

**要做**：
- ✅ 印 / screenshot 拒簽 email + 任何 reference number
- ✅ 列你後備路嘅死線（出發前要批咗嘅 cutoff）
- ✅ 評估係咪值得繼續去（簽證費 + 律師費 vs 行程價值）

🛂 **每個系統嘅後備路**

**US ESTA 拒簽 → B1/B2 visa**
- 費用：US$185 + DS-160 填寫
- 處理時間：香港領事館預約等 2-8 週
- 必須親身面試
- 拒咗 ESTA 後申請 B1/B2 成功率較低（officer 會睇拒簽紀錄）
- 建議：搵移民律師寫 cover letter 解釋 ESTA 拒簽原因

**UK ETA 拒簽 → Standard Visitor Visa**
- 費用：£127（短期）/ £400+（多次 2-10 年）
- 處理時間：3-15 個工作日
- 通常文件審核，部分情況要面試
- 拒咗 ETA 之後申請 Standard Visitor 較常見，但要完整解釋
- 建議：自己草擬 cover letter + 搵律師 review

**K-ETA 拒簽 → 韓國 C-3 短期訪問簽證**
- 費用：~HK$300
- 處理時間：3-5 個工作日
- 香港韓國領事館申請
- 通常文件 OK 就批
- 建議：可以自己搞，唔需要律師

**AUS ETA 拒簽 → Visitor visa subclass 600**
- 費用：AU$190
- 處理時間：可以快至 1 個工作日，部分情況數週
- 全網上，少數情況要面試
- 建議：自己搞 OK

**Japan eVisa 拒簽 → 領事館 visa**
- 視乎邊種 eVisa 拒咗（商務 / 結婚 / 親屬）
- 通常領事館申請後備路較寬鬆
- 香港日本總領事館預約

💰 **拒簽嘅長期成本**

未來 5-10 年所有國家簽證表格都要 declare 呢次拒簽。具體影響：

- 申請第 4 個國家簽證嘅表格「Have you been refused...」答「Yes」
- 通常需要附 1 頁解釋信
- 部分資深領事館 officer 會額外問話
- **唔等於將來全部拒簽**，但需要額外功夫

🆘 **幾時必須搵律師**

- ESTA 拒簽嘅同時你想申請 US visa（律師費 HK$3,000-8,000 — 值得）
- UK ETA 拒簽而你過去有 UK 學生 / 工作簽證歷史
- 任何 ETA 拒簽加刑事紀錄 / 過去簽證逾期
- ETA 拒簽影響到工作（公幹行程要批）

⏰ **拒簽嗰刻嘅行程修改決定**

- 機票取消保險 check（部分覆蓋簽證被拒）
- 改路線跳過拒簽嘅目的地可行性
- 同僱主 / 朋友重新安排行程嘅成本
- 直接放棄個目的地，下年再計劃

語氣：實際 + 唔慌張，畀我清楚下一個 24 小時要做咩。
</PromptBlock>

呢個後備 prompt 你拒簽之前都應該行一次。**有 plan B 喺手嘅人，申請嗰陣少啲焦慮，反而表現更好**。

## 變化例子

### 變化 1：UK ETA + BNO 嘅相互關係

BNO 持有人去 UK 嘅特殊狀況：

<PromptBlock label="變化 1 — BNO + UK ETA 決策樹">
我同時持有 HKSAR 護照 + BNO 護照。今次去 UK 旅遊 2 星期。請釐清：

🇬🇧 **BNO 入境 UK 嘅身份**

- BNO 護照持有人入境 UK：**唔需要 ETA**（免簽 6 個月訪問）
- 用 BNO 入境 = 用 BNO 過關 = 走 UK / EEA passport channel
- 用 HKSAR 入境 = 需要 ETA（£16，48 小時批）

🤔 **應該用邊本護照入境？**

**用 BNO 嘅好處**：
- 慳 ETA 費 + 申請
- 慳所有 ETA 申報問題（refused entry / criminal / nationality 等）
- 走 e-gate 快過 visitor channel
- 將來 BNO visa 路徑嘅入境紀錄保留

**用 BNO 嘅壞處**：
- 如果你計劃將來去美國，US ESTA 嘅「other nationality」一定要申報 BNO（即使入境用 HKSAR）
- BNO 入境 UK 6 個月限制嚴格執行
- 如果你之後想轉去 BN(O) visa 5 年路徑，呢次入境嘅 stamp / 紀錄唔可以混亂

**建議**：
- 純旅遊少過 2 星期 → 用 BNO 入境，唔使申請 ETA
- 任何工作 / 學習成分 → 搵移民律師釐清

📋 **ETA 表格上嘅國籍申報**

如果你決定用 HKSAR 入境（申請 UK ETA）：
- "Nationality"：HKSAR（Chinese）
- "Do you hold any other nationality?"：**Yes，British National (Overseas)**
- 唔申報 = 虛假陳述（即使 BNO 喺 UK 法律下身份模糊）

🇺🇸 **同步申請 US ESTA 嘅影響**

ESTA 嘅「Do you hold or have you ever held any other nationality / passport?」：
- **必須申報 BNO**（即使從未用 BNO 入境美國）
- BNO passport 號碼 + 國家「United Kingdom」填入 additional nationality field
- 唔影響 ESTA 批核（BNO 屬於 Visa Waiver Program 兼容）

🇰🇷 **K-ETA 嘅 BNO 申報**

K-ETA 表格上「Other nationality」一欄：
- 申報 BNO + UK 為發證國家
- 韓國對 BNO 持有人同 HKSAR 持有人免簽待遇相同（90 日）

🚨 **唔申報 BNO 嘅長期風險**

如果你選擇唔申報 BNO（例：用 HKSAR 申請 ESTA 但隱瞞 BNO）：
- 任何將來 US visa 面試，officer database check 會 surface BNO 紀錄
- 「Why did you not declare in ESTA?」= 永久封禁風險
- 唔值得為咗慳填一行嘢冒呢個風險

✅ **建議統一規則**

- 所有 ETA / 簽證表格嘅國籍一欄：申報你**所有持有嘅有效旅行證件**
- 入境用邊本：另一回事，揀對你最有利
- 兩者一致性：盡量行程期間用同一本入同一國，避免 stamp 混亂

語氣：法律精確 + 實際考慮一齊。
</PromptBlock>

### 變化 2：ESTA 拒簽之後改用 B1/B2

ESTA 嘅拒簽處理係 ETA 系統入面最複雜嘅：

<PromptBlock label="變化 2 — ESTA 拒簽之後嘅 B1/B2 路線">
我 ESTA 申請拒咗。行程喺 [日期]。請整 B1/B2 visa 救援計劃：

🇺🇸 **ESTA 拒簽之後嘅實況**

- ESTA 唔可以上訴（呢個唔係 visa，係預先授權）
- 唔可以即時重新申請 ESTA（要等背景改變，例如新工作 / 新護照）
- **唯一後備：申請 B1/B2 visa**
- 即使你之前 ESTA 批過，今次拒咗 = 連鎖將來 ESTA 都會拒

📋 **B1/B2 visa 流程**

**第 1 步：DS-160 表格**
- 網上填，~90 分鐘
- 比 ESTA 詳細好多（過去 5 年僱主、過去國家、social media handle）
- 遞交後生成 barcode page，要印

**第 2 步：付費 + 預約**
- US$185 MRV fee
- 香港美國領事館預約面試
- 旺季排期 4-8 週（暑假 / 聖誕前）
- 淡季 1-3 週

**第 3 步：面試準備**
- 帶文件：DS-160 confirmation、appointment letter、護照、相片、ESTA 拒簽 email、工作證明、財務證明、行程
- 對「Why was your ESTA rejected?」要有 1 句答（你都唔知原因，但你要展示你誠實 + 已準備好）
- 對「同香港嘅強連繫」要有具體 evidence：工作合約、物業、家庭

**第 4 步：面試當日**
- 通常 3-5 分鐘對話
- Officer 一個決定批 / 拒
- 批：護照留低 1-5 個工作日領取
- 拒：212(b) 信解釋大致原因，可以重新申請但要 address concerns

🚨 **ESTA 拒簽後 B1/B2 嘅特殊考慮**

Officer 一定會睇 ESTA 拒簽紀錄。你要主動處理：

- 唔好假裝唔知（「I don't know why」係差答）
- 唔好過度解釋（「Maybe because of...」串講猜想）
- 中性答：「I was not given a specific reason. I am here to demonstrate my eligibility under standard visa requirements.」

📅 **行程 timing 評估**

- 標準行程 6 月出發
- 4 月 ESTA 拒簽收到 → 即日開始 B1/B2
- 4 月底完成 DS-160 + 付款
- 5 月初預約 slot（旺季可能 6 月）
- 5 月中面試
- 5 月底 / 6 月初 visa 批

如果 timing 太緊：
- 搵律師寫加急申請（emergency travel）
- 改行程押後 2-3 個月
- 放棄美國部分，行程繼續 UK + Korea

💰 **總成本**

- DS-160 + MRV fee：US$185
- 律師費（ESTA-拒簽情況建議搵）：HK$3,000-8,000
- 機票改期：HK$500-3,000
- 酒店改期：視乎 booking policy
- 時間成本：3-6 週

vs 原本 ESTA US$21 + 5 分鐘 — **20 倍嘅成本**。

🆘 **幾時放棄美國**

- 律師評估你 B1/B2 成功率低過 50%
- 行程入面美國部分嘅唯一價值係探朋友（可以視訊 / 朋友嚟香港）
- 律師費 + 簽證費 + 時間成本 > 行程入面美國部分嘅價值

語氣：清醒 + 步驟具體，唔做安慰式言論。
</PromptBlock>

### 變化 3：多國組合（Schengen + ETA combo）

歐洲行程加美國 / UK / 韓國嘅混合情況：

<PromptBlock label="變化 3 — Schengen + ETA mix-and-match">
我 9 月 6 星期行程：法國 + 意大利（Schengen）→ UK 1 星期 → 紐約 1 星期。請整 visa / ETA combo 計劃：

🌍 **Combo 概覽**

- 法國 + 意大利：**Schengen visa**（短期訪問，C visa）
- UK：**ETA**（£16，48 小時）— 注意 UK 已脫歐，唔係 Schengen
- US：**ESTA**（US$21，72 小時）

⏰ **申請次序 + timing**

**T-90 日：開始 Schengen**
- Schengen 處理時間 15-45 工作日
- 香港 VFS 預約可能要等 2-4 週
- 文件最重 — bank statement、僱主信、保險、行程、酒店、機票
- 詳細流程睇我哋之前篇 [Schengen 簽證文章](/use-cases/visa-application-claude)

**T-45 日：Schengen 批咗之後申請 ESTA**
- 點解 Schengen 先？因為 ESTA 表格嘅「Have you been issued a visa by any country in the past 5 years?」要申報 Schengen
- Schengen approved → ESTA 填得有底氣
- ESTA 72 小時批

**T-30 日：UK ETA**
- UK ETA 同 ESTA 都係短 ETA，順序可以倒轉
- UK ETA 48 小時批
- 兩個都申報你已有 Schengen + 即將出發嘅行程

📋 **三個系統嘅申報一致性**

最容易出錯：3 份表格嘅以下欄位必須完全一致：

- 護照號碼 + 發放日期 + 到期日
- 全名拼寫（對應護照中英對照）
- 香港住址（line 1 / 2 / postcode 寫法統一）
- 僱主英文全名（對應 HR 系統，唔好簡寫）
- 過去 5 年旅遊紀錄
- 入境目的地嘅日期 + 地址（朋友屋企 / 酒店）

建議：開 1 個 Google Doc / Notes 記低「參考值」，3 份表格都 copy-paste 同一啲字。

🛂 **入境次序對申報嘅影響**

如果你計劃：香港 → 巴黎 → 倫敦 → 紐約 → 香港

- 入境 Schengen：法國海關，用 Schengen visa
- 出 Schengen 入 UK：英國海關，用 UK ETA
- 出 UK 入美國：美國海關，用 ESTA
- 每次過關 officer 會睇前一個國家 stamp

UK + 美國嘅 officer 通常會問：「How long in Europe before here?」— 預一個 1 句答（對應你表格入面寫嘅）。

⚠️ **三系統都有申報義務**

如果 Schengen 拒簽 → ESTA + UK ETA 表格上嘅「visa refused」答「Yes」
如果 ESTA 喺 UK ETA 批之前拒簽 → 你已遞交嘅 UK ETA 可能受影響（雖然 UK 唔特別問美國 ETA refusal）
如果 UK ETA 拒簽 → 通常唔影響 ESTA / Schengen（互相獨立系統）

🎯 **如其中一個拒簽嘅行程救援**

- Schengen 拒簽：放棄歐洲段，剩低 UK + 美國（2 星期 + 1 星期）
- ESTA 拒簽：B1/B2 後備或者跳過美國，多花時間喺 UK + Schengen
- UK ETA 拒簽：Standard Visitor visa 或者跳過 UK（直接 Schengen → 美國）

每個拒簽之前心入面預先諗定一個救援計劃，唔好拒簽嗰刻先諗。

💰 **總成本估算**

- Schengen visa：~HK$1,500-2,500
- UK ETA：£16 = ~HK$160
- US ESTA：US$21 = ~HK$165
- 旅遊保險：HK$1,000-2,000
- 文件費 + VFS 預約：HK$500-1,000
- **總簽證 / ETA 成本：~HK$3,500-6,000**

唔大，但要預留時間。

語氣：複雜情況，要序列清晰。
</PromptBlock>

## 一個思維框架

網上 ETA 系統嘅 user experience 設計目標係**篩走風險個案，唔係幫你 navigate**。

表格上面每條問題嘅寫法故意保留法律模糊性 — 留 officer 後續解讀空間。對你嚟講，呢個模糊就係風險來源：你用日常意義理解問題，officer 用法律 / 政策意義裁決你嘅答案。

> 「Have you ever been refused entry?」喺你嘅腦：「我曾經被推返嚟？」
>
> 喺 officer 嘅腦：「呢人有冇 record of inadmissibility？」

呢兩個唔同問題。**Claude 嘅角色喺呢度 — 幫你由日常理解反推返 officer 嘅 mental model，再校準你嘅寫法**。

呢個思維轉變：網上 ETA 唔係「填表格」嘅行政 task — 係**法律自我披露練習**。每 click 遞交一次，你就喺 5-10 個國家嘅移民資料庫留低永久紀錄。25 分鐘預先篩查嘅成本，對比一次拒簽嘅 5-10 年 follow-up 成本，比例非常划算。

最後提醒：

- ✅ **預先篩查比拒簽後應變平好多**。Claude 25 分鐘 vs 律師 HK$5,000 + 4 星期，揀邊樣明顯。
- ⚠️ **Claude 唔取代移民律師**。如果你有刑事紀錄、過去簽證被拒、雙重國籍複雜、過去逾期 — 律師費係必要開支，唔係可有可無。
- 🛂 **一致性比聰明答案更重要**。3 份表格之間嘅欄位唔對應，officer 即時 flag。寧願老實 + 一致，唔好聰明 + 矛盾。

ETA 系統設計嚟篩你，但你有 Claude 幫你睇清楚每條問題嘅真正含義。25 分鐘預先篩查，行程出發前心安理得。一路平安。

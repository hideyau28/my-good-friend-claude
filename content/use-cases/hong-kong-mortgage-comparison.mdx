## 情境

你要決定買樓 / 轉按，揾咗三間銀行 ——滙豐、恒生、中銀——攞咗份 product term sheet。三份睇落差唔多：「最優惠利率減 X」、「現金回贈 Y」、「最高貸款額 Z%」。

但細心啲睇：
- 一份話「H + 1.3%、上限 P - 1.5%」
- 一份話「H + 1.25% with rate floor at P - 1.55%」
- 一份話「prime rate 跟隨同業」

**正常人根本睇唔出邊間實際抵啲、邊間有伏**。你又冇時間上 banker forum 逐間格。

呢篇教你點用 Claude 做一次有結構嘅比較，**唔係齋做摘要，係幫你揾返真正嘅取捨喺邊**。

> 重要提示：呢個流程出嘅結果係幫你**睇明產品**，唔係取代專業顧問。最終點揀仲係要你自己拍板（或者問按揭轉介公司 / 財務顧問）。Claude 出嘅嘢唔等於財務意見。

## 跟住做

### 1. 攞齊三份 PDF（或者 product page）

去銀行官網下載個 product term sheet / fact sheet。唔係宣傳網頁，係**真正嘅 T&C 文件**。如果得個網頁攞唔到 PDF，cmd+P 存做 PDF 都得。

### 2. 一次過 upload 三份畀 Claude

Claude.ai 個對話框可以一次過收多個 PDF。上載完之後，用以下嘅 prompt。

<PromptBlock label="完整 prompt — 比較三個按揭產品">
我會喺呢三個按揭產品入面揀一個。請幫我做以下嘅比較：

**Part 1：左右對照表**

請整一個表，欄係三間銀行，每行包：
- 實際利率（明寫 H+X% 或 P-X%，連 cap / floor）
- 罰息期 ——幾耐、提早還款罰幾多
- 現金回贈 ——金額 + 要鎖定幾耐
- 按揭保險計劃的要求 ——邊個 tier 要、保費點計
- 最高貸款成數
- 申請費 / 律師費 ——邊個包、邊個自己畀
- 浮息／定息 ——邊間點配搭、揀邊個

如果某間銀行某一行冇明確列出，標「未列明，需詢問」。

**Part 2：實際數字例子**

假設我借 $5M、供 30 年。請計：
- 每間銀行頭一年嘅月供
- 罰息期內如果我提早還清，會蝕幾多
- 5 年總利息（假設最優惠利率維持不變）

**Part 3：每間嘅伏 / 隱藏取捨（最重要嘅一段）**

每間銀行請列 1–2 點「宣傳網頁唔會講，但 fine print 嗰度寫住嘅缺點」。例如：
- 「現金回贈嗰 $X 要鎖定按揭 3 年，提早走要還返」
- 「H + 1.25% 表面最低，但下限利率比另外兩間高 0.1%，長遠反而貴」
- 「申請費豁免但律師費要自己畀，計落實際 $XX,XXX」

如果某間冇明顯嘅伏，老實講「未見明顯陷阱」，唔好硬作。

**Part 4：我下一步應該問銀行 / 轉介公司嘅問題（3 條）**

由你嘅分析出發，畀我 3 條尖銳、有針對性嘅問題。
</PromptBlock>

### 3. Cross-check 兩、三個關鍵數字

Claude 出嘅結果通常 95% 啱，但**喺金融文件度，5% 嘅錯誤可能就係幾萬蚊嘅分別**。

至少對返以下幾個：
- 每間銀行嘅實際利率
- 每間嘅罰息期幾耐
- 任何標咗「未列明」嘅嘢——返去 PDF 揾返，可能你睇漏咗

### 4. 拎個對照表 + Part 3 + Part 4 去問 banker

帶住一張清晰嘅對照表同 3 條具體問題去見 banker，**你會由「畀人 sell」變返「主動嗰個」**。銀行職員見到你準備充足，多數會直接講重點，唔再 sell 啲唔相干嘅優惠。

## 變化例子

### 變化 1：比較 3 張信用卡（cashback / mile / privilege）

<PromptBlock label="變化 1 — 信用卡比較">
我而家有 3 張信用卡可以揀，每張嘅 T&C 我上載咗。請：

1. 整表比較：年費（豁免條件、首年 vs 第二年）、現金回贈率、里數賺取率、全年上限、迎新優惠
2. 模擬：假設我每月碌 $20,000（5,000 食肆、5,000 超市、5,000 交通、5,000 網購），邊張回贈最多？計埋年費之後嘅淨值
3. 隱藏陷阱：每張卡 fine print 入面有冇缺點（例：上限、特定類別不適用、迎新優惠要最低簽賬等等）
4. 以我嘅生活習慣應該揀邊張？簡短推薦 + 理由

唔使行貨嘅信用卡意見，淨係專注呢 3 張。
</PromptBlock>

### 變化 2：比較 2 個人壽保險 plan

<PromptBlock label="變化 2 — 人壽保險比較">
我喺度諗緊兩個人壽保險計劃（保單文件已上載）。請：

1. 左右對照表：保額、保費（按年）、保障年期、退保價值（5 年 / 10 年 / 20 年）、附加保障（危疾、意外等）
2. 每個計劃嘅「最啱邊類人」：邊類人最適合買呢個
3. 每個計劃嘅「警號」：保單入面任何不尋常嘅條款（例：不保事項清單異常長、聲稱保證但其實唔保證嘅現金價值、有隱藏收費等）
4. 5 條我應該問代理嘅關鍵問題

請務必根據文件內容做分析，唔好用行貨知識亂講。如果文件冇講某個細節，標「未列明」。
</PromptBlock>

### 變化 3：比較 2 個個人貸款

<PromptBlock label="變化 3 — 個人貸款比較">
我考慮緊 2 個個人貸款。請：

1. 比較 APR、實際年利率（如果有手續費等）、還款期、提早還款罰款
2. 假設我借 $200K 還 36 個月，每月供款 + 總利息
3. 每個嘅取捨
4. 唔係財務意見，但純粹計數嘅角度，邊個平啲？
</PromptBlock>

## 兩個 caveat

**第一：呢個流程唔係用嚟取代專業顧問**

Claude 識讀文件，但唔識你嘅完整財務狀況、未來計劃、風險承受能力。按揭轉介公司 / 財務顧問嘅價值唔係幫你睇 fine print（呢樣 Claude 做到），係幫你諗長遠啱唔啱。**用 Claude 做準備，留個專業顧問幫你拍最後嗰板**。

**第二：銀行嘅 T&C 隨時會改**

你上載嘅 PDF 係今日嘅版本。簽之前一定要問 banker：「呢份報價幾時到期？啲優惠條件會唔會簽完之後變？」

不過喺準備上花 30 分鐘用 Claude 做一次有結構嘅比較，比起懵下懵下、唔知自己係魚定係餌咁入銀行傾，整個層次都唔同。

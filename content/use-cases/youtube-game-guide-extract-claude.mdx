## 情境

你卡關喺 Elden Ring 嘅 Malenia。或者 Hollow Knight 嘅 Path of Pain。或者 Persona 5 嘅 Velvet Room boss。

你打開 YouTube，搵到一條「Malenia EASY KILL GUIDE — No Mimic Tear Required」。15 分鐘長。

你開始睇——3 分鐘背景介紹，2 分鐘裝備配搭，5 分鐘第一階段講解，5 分鐘第二階段。

打 boss 中途，你只想知**「而家應該避去邊個方向」**——但你要返去段片度拉時間軸搵返嗰個時機。死完又再睇，再死又再睇，半個鐘淨係玩到兩場。

問題在於：**影片係線性媒體，但你需要嘅係隨機存取嘅參考資料。**

呢篇教你用 Claude 花 **10 分鐘**，將 15 分鐘嘅片變成**1 頁打機筆記**——按階段排好時機、攻擊模式、應對方法。印出嚟或者用電話開住，唔使再喺段片度拉來拉去。

## 跟住做

### 1. 攞 YouTube 逐字稿

YouTube 大部分片都有自動生成嘅逐字稿（包括非英文）。攞法：

**桌面瀏覽器**：
1. 開條片
2. 撳 `...` (更多) → `Show transcript`
3. 將成段複製（包含時間戳記）

**iPhone / Android**：
- 用 Apple Notes 嘅捷徑「Get YouTube Transcript」（一搜尋 shortcuts gallery 就搵到）
- 或者用瀏覽器外掛「YouTube Transcript & Summary」
- 或者去 `youtubetranscript.com`，貼上網址

你會收到 ~500-1500 字嘅逐字稿，每段帶有時間戳記，例如：

```
0:23
So this fight, you really need to bait her into the
overhead slam, because that's the only window where
you can get a riposte safely
0:31
What you wanna do is...
```

### 2. 貼入 Claude，用呢個 prompt

<PromptBlock label="完整 prompt — 逐字稿變 1 頁打機筆記">
以下係一條 YouTube 攻略片嘅逐字稿：

遊戲：[遊戲名 + boss / 區域 / 任務名]
我嘅裝備 / 職業：[簡單描述]
我而家卡住嘅階段：[死過邊段，或者「全部」]

逐字稿:
[貼上成段]

請整成**1 頁打機筆記**，方便我用電話開住打機時即時參考。結構如下：

**🎯 戰前準備（3-5 點）**
- 推薦裝備 / 配置（攻略片入面提到嘅）
- 推薦增益狀態 / 消耗品
- 地圖 / 場地設定（如有）

**⚔️ 第一階段（同每個之後嘅階段）**
觸發條件: [HP %、預設劇情事件、或計時器]
每個攻擊模式：
- 名稱 / 視覺提示（敵人點郁或者燈光點變）
- 時機數字（無敵時間有幾耐 / 步數）
- 你應該做：閃避方向 / 擋格時機 / 補血空檔

**🚨 死亡陷阱（最易死嘅 3 樣嘢）**
攻略片入面創作者講過「新手永遠死喺度」嘅具體時刻，列出 + 1 句應對方法。

**💡 選擇性：速通 / 逃課打法（如片有提）**
有冇漏洞 / 捷徑，講明風險 + 適唔適合我嘅配置。

**⏱ 時間戳記索引**
最後一個小清單：邊個時間戳記講邊個階段，等我想核對一下可以快速跳返去段片。

語氣：好似你係副本隊長寫畀第一次打嘅隊員嘅簡報，唔好將成篇逐字稿倒晒出嚟。
</PromptBlock>

### 3. 印出嚟 / 儲存喺電話開住打

輸出通常 1-2 頁 A4。我嘅做法：

- **電腦打機**：另一個螢幕開住 Claude 視窗
- **手機 / iPad 打機**：將筆記分享落 Apple Notes 同 Notion，全螢幕喺另一部裝置打開
- **主機**：印張 A5 擺喺手掣隔籬

### 4. 更新你嘅筆記（重要）

打咗 5-10 場之後，你發覺筆記入面**有幾項資訊根本冇用**（例如片主推介法術流但你玩近戰）。追問：

> 「我已經打咗 10 次。第二階段我可以穩定過，但第三階段嘅 `XX 攻擊` 我永遠避唔到。我而家用近戰 + 輕盾，唔係攻略片個法師配置。請重寫第三階段部分去配合我嘅配置。」

呢個度身訂造通常令筆記由 90% 有用變成 100% 啱用。

## 變化例子

### 變化 1：速通路線提取

如果你想學速通（any%、100%、無傷等），速通片通常極長（1-3 個鐘全通關）：

<PromptBlock label="變化 1 — 速通路線 → 檢查點計劃">
以下係 [遊戲] 嘅 [類別，例：any% no major glitch] 世界紀錄速通片嘅逐字稿（[長度]）。

我嘅目標：[個人最佳紀錄 / 2 個鐘內 / 純粹學路線]
我嘅現時水平：[新手 / 打爆過一次 / 試過 1-2 次速通]

請整：

**路線概覽**
- 全程拆做 [N] 個段落（每個 5-15 分鐘）
- 每個段落嘅簡介（去邊 + 做乜）

**每個段落詳細資料**
每個段落：
- 開始 / 結束檢查點
- 必做動作（順序）
- 推薦技巧 / 漏洞（連難度評級）
- 常見死亡 / 重置觸發點
- 慳時間提示

**技巧詞彙表**
攻略片用嘅社群行話（zips, OOB, RBA 等），每個用 1 句解釋

**練習優先次序**
邊個段落你應該抽出來獨立練習，邊個係靠運氣 (RNG) 控制唔到嘅

**唔好做嘅嘢**
世界紀錄保持者用嘅極限技巧 (frame-perfect / TAS-only)，唔好試（會令你好氣餒）

語氣：速通教練，唔係維基百科。
</PromptBlock>

### 變化 2：收集品清單

玩 100% 收集嘅人，攻略片通常係逐個區域嘅收集指南：

<PromptBlock label="變化 2 — 全收集總清單">
以下係 [遊戲] [區域 / 章節 / DLC] 嘅收集品指南片逐字稿。

我而家進度：[第 X 章 / 已收集 Y%]
我而家去到嘅區域：[列出]

請整：

**按區域分類嘅總清單**
每個區域 / 房間：
- 收集品 1 — 描述 + 座標（如片有提）+ 獲取要求
- 收集品 2 — ...

**會錯過嘅清單**
如果我推進主線過咗某章節，會返唔到轉頭攞嘅收集品，用紅色標示。

**最佳化次序**
建議我下次玩點樣行，用最少回頭路攞最多嘢

**工具要求**
邊啲收集品要先解鎖某個能力 / 道具，列出依賴關係

**存檔警告**
有冇收集品需要特定對話選擇 / 第二次通關嘅分歧結局

語氣：全收集導遊。
</PromptBlock>

### 變化 3：裝備指南 → 我嘅專屬配置

最常見嘅 RPG / 類 RPG 攻略片就係「最強 XXX 配置」：

<PromptBlock label="變化 3 — 裝備指南調整做我嘅角色">
以下係 [遊戲] 嘅「最強 [職業 / 武器 / 專精] 配置」攻略片逐字稿。

我嘅角色現時狀態:
- 職業 / 等級: [填]
- 主武器: [填]
- 已解鎖嘅技能 / 被動技: [列出]
- 我玩嘅內容：[主線故事 / 終局副本 / PvP / 合作]

請：

**將配置調整畀我用**
- 我可以即刻做嘅嘢（用我而家有嘅）
- 我下一個里程碑應該去刷乜嘢（道具 / 等級 / 技能）
- 我而家用緊嘅嘢有冇需要重置 / 重新洗牌

**唔好用嘅嘢**
攻略片推薦但對我嚟講係陷阱嘅道具 / 被動技，講明原因

**長遠路線**
我個角色由而家去到攻略片個最終配置需要：
- 預估時數
- 必須做嘅主線活動
- 可以跳過嘅支線內容

**對比現有配置嘅傷害 / 生存力**
換裝之後預計提升幾多 %（有數字最好）

語氣：副本老手畀朋友嘅配置意見。
</PromptBlock>

## 一個思維框架

YouTube 攻略片嘅資訊密度其實好低——我哋 90% 時間係等創作者講開場白、炫耀自己嘅紀錄、賣廣告、講結語。**真正有得落手做嘅資訊通常只佔影片嘅 20-30%。**

Claude 嘅價值唔係取代影片創作者——你應該繼續訂閱同讚好你欣賞嘅頻道。Claude 嘅價值係 **將線性影片媒體變成隨機存取嘅參考資料**——你打 boss 嘅時候要嘅唔係故事，係千鈞一髮間嘅閃避時機。

最後提醒：**第一次睇條片仲係值得睇晒**。睇晒先至用逐字稿 → 變筆記。原因：創作者嘅語氣 / 節奏 / 性格係影片格式嘅情緒價值，筆記抽走唔到——但筆記嘅實用價值又係影片比唔上嘅。**兩者並存，唔係互相取代。**

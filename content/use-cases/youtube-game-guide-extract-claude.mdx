## 情境

你 stuck 喺 Elden Ring 嘅 Malenia。或者 Hollow Knight 嘅 Path of Pain。或者 Persona 5 嘅 Velvet Room boss。

你打開 YouTube，揾到一條「Malenia EASY KILL GUIDE — No Mimic Tear Required」。15 分鐘。

你開始睇——3 分鐘背景，2 分鐘 build setup，5 分鐘 phase 1 講解，5 分鐘 phase 2。

打 boss 中途，你只想知**「依家應該 dodge 邊個方向」**——但你要返去 video 由頭 scrub 揾返條 timing。死又再睇，再死又再睇，半個鐘只玩到 2 場 attempt。

問題：**Video 係 linear medium，但你需要嘅係 random-access 嘅 reference。**

呢篇用 Claude **10 分鐘**將 15 分鐘 video 變成**1 頁 cheat sheet**——按 phase 排好 timing、attack pattern、counter，print 出嚟或者 phone 開住，唔使再 scrub video。

## 跟住做

### 1. 攞 YouTube transcript

YouTube 大部分 video 都有 auto-generated transcript（包括非英文）。攞法：

**桌面 browser**：
1. 開條 video
2. 撳 `...` (more) → `Show transcript`
3. 整段 copy（包含 timestamp）

**iPhone / Android**：
- 用 Apple Notes 嘅 shortcut「Get YouTube Transcript」（一搜 shortcuts gallery 就揾到）
- 或者用 browser extension「YouTube Transcript & Summary」
- 或者去 `youtubetranscript.com`，paste URL

你會收到 ~500-1500 字嘅 transcript，每段帶 timestamp 例如：

```
0:23
So this fight, you really need to bait her into the
overhead slam, because that's the only window where
you can get a riposte safely
0:31
What you wanna do is...
```

### 2. Paste 入 Claude，用呢個 prompt

<PromptBlock label="完整 prompt — Video transcript → 1 頁 cheat sheet">
以下係一條 YouTube 攻略 video 嘅 transcript：

遊戲：[遊戲名 + boss / area / quest 名]
我嘅 build / class：[簡單描述]
我宜家 stuck 嘅 phase：[死過邊段，或者「全部」]

Transcript:
[paste 整段]

請整成**1 頁 cheat sheet**，phone 開住打可以即 reference。Structure 如下：

**🎯 Pre-fight setup（3-5 條 bullet）**
- 推薦 build / equipment（攻略片入面提到嘅）
- 推薦 buff / consumable
- Map / arena setup（如有）

**⚔️ Phase 1（同每個之後 phase）**
Trigger: [HP %, scripted event, 或 timer]
每個 attack pattern：
- 名 / 視覺 tell（敵人點動或燈光點變）
- Timing 數字（hit window in seconds / iframes / 步）
- 你應該做：dodge 方向 / parry timing / heal opening

**🚨 Death triggers（最易死嘅 3 樣嘢）**
攻略片入面 video creator 講過「new players always die here」嘅 specific moments，列出 + 1 句 counter。

**💡 Optional：speedrun / cheese tactic（如片有提）**
有冇 exploit / shortcut，講風險 + 適唔適合我 build。

**⏱ Timestamp index**
最後一個 mini-list：邊個 timestamp 講邊個 phase，等我想 verify 一下可以快速跳返去 video。

語氣：好似你係 raid leader 寫俾 first-time clear 隊員嘅 brief，唔好 dump 全部 transcript。
</PromptBlock>

### 3. Print / save phone 開住打

Output 通常 1-2 頁 A4。我嘅做法：

- **桌機玩 game**：另一個 monitor 開住 Claude chat
- **手機 / iPad 玩 game**：將 cheat sheet share 落 Apple Notes 同 Notion，full-screen 喺另一部 device
- **主機**：print A5 size 撕埋 controller 隔離

### 4. Update 你嘅 cheat sheet（重要）

打 5-10 attempt 之後，你發覺 cheat sheet 入面**有幾條根本冇用嘅 info**（例如 video 主推 spell build 但你 melee）。 follow up：

> 「我已經打咗 10 次。Phase 2 我可以 consistently 過，但 Phase 3 嘅 `XX attack` 我永遠 dodge 唔到。我宜家用 melee + light shield，唔係攻略片個 mage build。請 rewrite 個 Phase 3 section 配合我嘅 build。」

呢個 customize 通常令 cheat sheet 由 90% useful 變 100% useful。

## 變化

### 變化 1：Speedrun route extraction

如果你想學 speedrun（any%, 100%, 無傷等），speedrun video 通常極長（1-3 小時 full playthrough）：

<PromptBlock label="變化 1 — Speedrun route → checkpoint plan">
以下係 [遊戲] 嘅 [類別，例：any% no major glitch] world record speedrun video transcript（[長度]）。

我嘅 goal：[個人 PB 目標 / sub-2-hour / 純學 route]
我嘅 current 水平：[新手 / 完晒 normal / 試過 1-2 次 speedrun]

請整：

**Route overview**
- 全程拆 [N] 個 segment（每個 5-15 分鐘）
- 每個 segment 個 elevator pitch（去邊 + 做乜）

**每個 segment 詳細**
For each segment:
- Start / end checkpoint
- 必做 actions（順序）
- 推薦 tech / glitch（連 difficulty rating）
- Common death / reset trigger
- Time-save tips

**Tech 詞彙表**
攻略片用嘅 community jargon（zips, OOB, RBA 等），每個 1 句解

**Practice priority**
邊個 segment 你應該 isolate practice，邊個係 RNG 控唔到

**唔做嘅嘢**
WR runner 用嘅 frame-perfect / TAS-only tech，唔好試（會 frustrate）

語氣：speedrun coach，唔係 wiki dump。
</PromptBlock>

### 變化 2：Collectibles checklist

100% playthrough 嘅人，攻略片通常係 area-by-area collectible guide：

<PromptBlock label="變化 2 — Collectible 全收集 master list">
以下係 [遊戲] [area / chapter / DLC] 嘅 collectibles guide video transcript。

我宜家進度：[第 X chapter / 已收 Y%]
我而家可以 access 嘅 zone：[列出]

請整：

**Master checklist by area**
每個 area / room：
- Collectible 1 — 描述 + grid coord（如片有）+ access requirement
- Collectible 2 — ...

**Missable list**
如果我推進主線過咗某 chapter，會 lock 唔到嘅 collectibles，紅 flag highlight。

**Order optimisation**
建議我下一個 session 點 route，最少 backtrack 收最多嘢

**Tool requirement**
邊啲 collectible 要先解鎖某 ability / item，列出 dependency

**Save warning**
有冇 collectible 需要 specific dialogue choice / second playthrough 嘅 ending fork

語氣：completionist tour guide。
</PromptBlock>

### 變化 3：Build guide → 我嘅 specific build

最常見嘅 RPG/RPG-lite 攻略片就係「Best XXX Build For YYY」：

<PromptBlock label="變化 3 — Build guide adapt to my character">
以下係 [遊戲] 嘅「Best [class / weapon / spec] build」 video transcript。

我嘅角色 current state:
- Class / level: [填]
- 主武器 / weapon: [填]
- 已 unlock 嘅 skill / passive: [列出]
- 我玩嘅 content：[main story / endgame raid / PvP / co-op]

請：

**Adapt 個 build 為我**
- 我可以馬上做嘅嘢（用我宜家有嘅）
- 我下一個 milestone 應該 farm 乜（item / level / skill）
- 我宜家用嘅嘢有冇要 reset / re-roll

**唔好用嘅嘢**
攻略片推薦但對我嚟講 trap 嘅 item / passive，講原因

**Long-term path**
我 character 由宜家到攻略片個 final build 需要：
- 預估時數
- 必須做嘅 main path activity
- 可以 skip 嘅 side content

**對比現有 build 嘅 DPS / survivability**
換之後 expected 提升幾多 %（有 number 就好）

語氣：副本老手畀朋友嘅 build 意見。
</PromptBlock>

## 一個 mindset

YouTube 攻略片嘅 information density 其實好低——我哋 90% 時間係等 creator 講開場白、show off 自己嘅 PR、處理 sponsor、講 conclusion。**真正 actionable 嘅 info 通常剩返 video 嘅 20-30%。**

Claude 嘅 value 唔係取代 video creator——你應該繼續 sub 同 like 你 enjoy 嘅 channel。Claude 嘅 value 係 **將 linear video media 變成 random-access reference**——你打 boss 嘅時候要嘅唔係故事，係 split-second 嘅 dodge timing。

最後 reminder：**第一次睇 video 仲係值得睇晒**。睇晒先 transcript → cheat sheet。原因：creator 嘅 voice acting / pacing / personality 係 video format 嘅 emotional value，cheat sheet 抽走唔到——但 cheat sheet 嘅 functional value 又係 video 比唔上嘅。**兩個並存，唔係取代。**

GG，clear boss 之後返去 like + comment 條 video，creator 應得嘅。

## 情境

下星期一你飛台北 4 日。

你揾咗你阿媽 / 隔離鄰居 / 上門 sitter 嚟照顧主子。出發前一晚，你企喺廚房，揾 5 分鐘交代佢：

> 「主子朝早食 80g 乾糧，傍晚食濕糧⋯⋯不過如果佢唔肯食可以加啲 churu⋯⋯水要每日換新鮮，但啲碗抹乾淨唔好用洗潔精⋯⋯嗯，佢心臟藥每日傍晚 7 點⋯⋯吓，定係朝早 7 點？我問返獸醫先⋯⋯然後貓砂盆一日清兩次⋯⋯如果佢匿喺床底底唔好騷擾⋯⋯」

你講完，sitter 個樣：🥲

你哋雙方都知道，**5 分鐘口頭交代一定記唔到 50%**。

到你飛機升空，WhatsApp 開始響：

- 「主子邊度餵？」
- 「呢個係佢嗰隻 churu 嗎？」（影張錯嘅牌子）
- 「主子唔肯出嚟食點算？」
- 「主子有冇 contact lens？！」（朋友以為你話咩）

你喺台北永康街食小籠包，腦入面係主子嘅心臟藥到底有冇食。

呢篇用 Claude **15 分鐘**寫一份 sitter 即用嘅交接文件，sitter 跟住做唔使 message 你，你個 trip 真係 chill 到。

## 跟住做

### 1. 將你嘅 pet 作息記憶寫晒

打開 Claude chat，**唔好叫 Claude 出範本**。先將你個 pet 嘅實際作息寫晒（亂都得，後面 Claude 會整理）：

> 主子（8 歲男仔短毛貓，4kg，室內貓，已絕育，FIV 陰性）
>
> 食：朝 8 點 80g 乾糧（Royal Canin Indoor Adult），傍 7 點 1 罐濕糧（Hill's i/d wet）。佢揀食，如果唔肯食濕糧可以加 1 條 churu（INABA 雞肉味黃色嗰隻）。
>
> 水：3 個碗散開放——廚房、客廳、廁所角。每日全部換新鮮。唔好用洗潔精，用熱水沖。
>
> 藥：每日傍晚同濕糧一齊食。Vetmedin 1.25mg/粒半粒。藏喺 churu 入面成功率 90%。如果唔肯食藥要用 pill pocket（雪櫃有）。
>
> 貓砂：朝晚各掃一次。砂喺廚房櫃下層。每星期一一次完全換砂 + 洗盆。
>
> 個性：對人友善但內向。新人嚟頭 3 個鐘佢匿床底底正常。摸 OK 但唔好抱（會掙脫跌親）。最鐘意人坐梳化佢就過嚟坐大髀。觸發位：吸塵機、高聲、男人低沉聲（送貨員 / 維修工人）。
>
> 獸醫：[獸醫名]，[地址]，[電話]。緊急：SPCA 香港 24 小時 [電話]。
>
> 出問題即 call 我：[電話]。如 call 唔到，第二聯絡係我細妹 [電話]。

呢類隨手 dump 唔需要整齊，Claude 之後會結構化。

### 2. 用呢個 prompt 整交接文件

<PromptBlock label="完整 prompt — Sitter 交接文件">
我下星期一至四去 [目的地]，搵咗 [sitter — 親戚 / 朋友 / 專業 sitter] 嚟照顧我隻 pet。

📝 **Sitter 嘅背景**：
- 同主子熟唔熟：[完全唔識 / 見過 1-2 次 / 經常見]
- 養過 pet：[冇 / 養過貓 / 養過狗]
- 嚟我屋企頻率：[每日來 1 次 / 每日 2 次 / 留宿]
- 我可以聯絡佢嘅方式：[WhatsApp / 電話 / 都得]

🐾 **我 pet 嘅原始筆記**：
[貼你上面 dump 嘅筆記]

⏰ **我嘅 trip 日期**：
[填]

請整成 **1-2 頁 sitter 交接文件**，結構：

**📌 Page 1 — 每日作息（sitter 每日參考）**

🌅 **AM（建議 8-9 點）**
- [任務 1] — 量化（例：80g 乾糧）
- [任務 2]
- [檢查 / 觀察]

🌙 **PM（建議 7-8 點）**
- [任務 1 — 含藥藏入 churu 嘅逐步做法]
- [任務 2]
- [檢查]

🧹 **貓砂 / 清潔**
- 每日：[]
- 每週：[]

**📌 Page 2 — 參考卡**

🍽 **食物簡表**
- 乾糧：牌子 / 位置 / 每餐份量
- 濕糧：牌子 / 位置 / 每餐份量
- 零食 / 加料：churu 牌子 + 味道（影張相附上）
- ❌ Pet 食物敏感 / 唔食：[列出]

💊 **藥物時間表**
- 藥名 / 劑量 / 時間 / 藏入食物嘅方法
- 如果唔食：步驟 1（pill pocket） → 步驟 2（call 我）
- ⚠️ 補貨：[幾時要]

🚨 **緊急程序**
- 主子出現 [呼吸困難 / 持續嘔吐 / 不能站立 / 顯著行為改變] = 即 call：
  · 我嘅獸醫：[名 / 地址 / 電話]
  · 24 小時緊急：[SPCA / Cityu Vet 24h / 其他]
  · 我嘅電話：[電話]，第二聯絡：[親人名 / 電話]

🐾 **個性 + quirks**
- ✅ OK 做：[列 — 摸頭 / 坐大髀 / 玩逗貓棒]
- ❌ 唔好做：[列 — 抱起 / 高聲叫 / 用吸塵機 / 強拉佢出嚟]
- ⚠️ 觸發位：[列 — 男聲 / 雷聲 / 維修工人]
- 🏠 匿藏位：[床底 / 沙發後] = 正常，畀佢自己

🏠 **屋企後勤**
- 鎖匙：[邊度 + 點還]
- 電燈 / 冷氣設定：[溫度 / 時段]
- 補貨地點：[寵物店 / 超市 / Pet Pet Park 等]
- WiFi：[名 + 密碼]

📞 **每日通報計劃**
- 我希望 sitter 每日 [WhatsApp 1 張主子相 / 短訊確認餵咗] — 頻率 [一日 1 次 / 一日 2 次]
- 如果有任何不尋常嘢（食量、行為、屎尿）= 立即通報

語氣：友善主人體諒 sitter 嘅辛苦。結構清晰，唔好倒大段。
</PromptBlock>

### 3. Print + WhatsApp + 親身示範

交接文件出咗，做 3 樣部署：

1. **印 1 份 A4** 用磁石貼喺廚房雪櫃
2. **WhatsApp 個 PDF** 畀 sitter（方便佢隨時參考）
3. **走前一晚示範**：sitter 嚟過你屋企 30 分鐘，**跟住個文件一齊行一次**作息（你示範餵 / 藏藥 / 清貓砂 / 攞 churu）

呢個親身示範比口頭交代嘅記憶留存高 10 倍。

### 4. Trip 中最少通報

WhatsApp 設個每日「主子相 + ✓ 餵咗」嘅儀式。Sitter 唔需要長訊息，1 張相 + 1 個 ✓ emoji 就夠。

你食緊小籠包，腦袋唔再諗主子心臟藥。

## 變化

### 變化 1：寵物寄養（你送主子去 sitter 屋企）

唔係 sitter 嚟你屋企，而係你送主子去佢屋企（或者 pet hotel）：

<PromptBlock label="變化 1 — 寵物寄養送過去準備">
我送主子去 [pet hotel / 朋友屋企] 寄養 [幾日]。

[貼 pet 作息 — 同主篇一樣]

請整 **寄養專屬準備**：

**🧳 入袋清單**
- 必帶食物（計總份量 × 日數，加 20% buffer）
- 藥 + 詳細劑量指示卡
- 熟悉嘅毛毯 / 玩具（熟悉氣味助減壓）
- 寵物籠 + 運送計劃
- 獸醫紀錄 + 疫苗證明（pet hotel 通常要求）

**📋 送到時 5 分鐘交代**
你面對面同 sitter / pet hotel 員工講嘅重點（按重要性排）

**🏠 Pet 新環境適應**
- 首 24 小時預期行為（匿埋 / 唔食正常）
- Day 2-3 安頓預期
- ⚠️ 如果到 Day 3 仲完全唔食 = 即 call 獸醫 + 我

**📞 每日通報格式**
建議 sitter 每日寄畀你嘅資料：
- 食咗幾多
- 屎 / 尿正常？
- 行為（活躍 / 退縮）
- 1 張相

語氣：唔好過份小心，但保護主子嘅安全底線。
</PromptBlock>

### 變化 2：長期 sitter（超過 1 個月）

你出長 trip / 海外工作幾個月：

<PromptBlock label="變化 2 — 長期 sitter 入職">
我 [幾個月] 唔喺 HK，搵咗 [sitter 關係] 全程照顧主子。

[貼完整 pet 歷史 — 年齡 / 品種 / 醫療 / 性格 / 作息]

請整 **完整入職文件**：

**📚 Pet 歷史簿**
- 出生 / 領養歷史
- 醫療歷史（過去 3 年主要事件）
- 行為演變（佢點由 BB 貓變到宜家成貓）
- 同獸醫嘅關係 + 最近一次探訪結論

**🗓 每週 / 每月時間表**
- 每日作息（同 sitter 交代一樣）
- 每週：徹底換貓砂 / 量體重 / 影張紀錄相
- 每月：剪甲 / 刷牙 / 補貨庫存盤點
- 每季：獸醫健康檢查 / 洗牙（如有計劃）

**💰 財務安排**
- 每月預算（食物 $X / 砂 $Y / 雜費 $Z）
- 我轉帳嘅頻率
- 獸醫緊急基金門檻（超過 $XXX 我要批准）
- 收據點收（影相分享 / Google Drive 資料夾）

**🎯 決定權**
- Sitter 可以自己做嘅決定（常規獸醫探訪 / 同一系列食物轉牌子 / 清潔用品補貨）
- 要 call 我批准嘅（任何藥物改變 / 非緊急手術 / 轉寄養）
- 即做唔使等我回覆嘅（緊急救命 — 即刻送獸醫）

**🤝 建立信任**
- 第 1 個月：每日相 + 每週通話
- 第 2-3 個月：減到每週 3 次更新
- 之後：每週 check-in 除非有嘢

**🏃 退場計劃**
我返 HK 之前 1 星期：
- Sitter 點過渡主子返我作息
- 交接紀錄（藥變化、行為觀察、任何顯著嘢）

語氣：長期信任建立，唔係直升機式緊張，但主人話事嘅責任仲要保住。
</PromptBlock>

### 變化 3：多隻寵物同時交代

你有 2 隻貓 / 1 貓 + 1 狗 / 1 貓 + 1 倉鼠：

<PromptBlock label="變化 3 — 多寵物 sitter 交接">
我有 [N 隻 pet]，每隻：
- 主子（貓 8 歲） — [作息片段]
- Coco（狗 3 歲） — [作息片段]
- Sunny（倉鼠 1 歲） — [作息片段]

我 trip [幾日]，sitter [描述]。

請整多寵物交接：

**🐾 逐隻 pet 快速參考卡**（每隻獨立一頁，唔好撈埋）

**🔀 衝突 / 互動筆記**
- 貓同狗點分隔（餵食區 / 玩耍界線）
- 邊隻會嚇親邊隻（貓嚇親倉鼠 ok？狗妒忌貓拎咗注意力？）

**⏰ 主時間表（all pets 按時間順序）**
- 8am：[餵邊隻 / 邊隻 / 邊隻 / 順序]
- 12pm：...
- 7pm：...
- 10pm：...
（順序減少混亂，例：先貓後狗因為貓食得慢）

**🚨 多寵物緊急**
- 邊隻優先如果一齊出事
- 獸醫知唔知所有 pet（如果獸醫唔同，分開列）

**🧠 Sitter 精神負擔**
最重要 3 樣嘢 sitter 記得 — 嚟 sitter 一定唔會記晒所有細節，幫佢排優先。

語氣：體諒 — 照顧多 pet 比單一隻難好多。
</PromptBlock>

## 一個思維框架

養 pet 嘅主人通常會建立一種隱藏專業——你知道主子邊個聲音係肚餓 vs 沮喪，邊個匿埋係正常 vs 病。**呢類默會知識喺你腦入面毫不費力，但傳出去極難。**

Sitter 接手嘅 5 分鐘口頭交代，本質係**你 5 年養 pet 知識嘅有損壓縮**。Loss 越多，trip 中嘅焦慮越大。

Claude 嘅價值唔係取代你嘅 pet 專業，係**幫你將默會知識變成 sitter 睇得明嘅文件**。下次 sitter / 你阿媽 / Air-bnb 上 pet sitter 嚟，**重用同一份文件**就可以——你 5 年知識嘅邊際成本複利。

最後提醒：

- ✅ 交接文件唔等於拋低包袱。Sitter 仲係要每日通報，你仲係要緊急時 reachable。
- ⚠️ 第一次用新 sitter？建議**試住 24 小時**喺真 trip 之前 — 你喺 HK 就身邊可以救援。
- 🆘 Sitter 唔信？信自己直覺。主子嘅安全 > 你慳幾百蚊嘅 sitting fee。

Enjoy 你個 trip，主子安好等你返。

## 情境

小朋友返學撞到問題：

- 老師長期不公平對小朋友
- 學校政策唔合理（功課量、罰則、成績計算）
- 同學欺凌、家長未滿意處理
- 特殊學習需要嘅評估／支援唔足夠

你想寫信／email 同學校溝通，但：

- **太軟** —— 學校會敷衍處理，事情冇變
- **太硬** —— 學校會自我防衛、未來幾年小朋友返學壓力反彈喺自己屋企人身上
- **太激動** —— 顯得唔講道理，正當訴求變咗無理取鬧

呢度個甜蜜點係：**企得硬、但定位係合作**。 你嘅角色係夥伴而唔係對手。 呢篇用 Claude 寫到呢類信。

## 跟住做

### 1. 先收齊三樣嘢

- **事實經過**——時間、地點、發生咗咩。具體日期，唔係「最近」。
- **對小朋友嘅直接影響**——情緒、表現、健康。
- **你想要嘅結果**——具體要求，唔係「希望學校跟進」。

呢一步唔可以跳過。冇呢三樣，AI 出嘅信會係行貨投訴，學校好易回應「我哋會留意」就完場。

### 2. 用呢個 prompt

<PromptBlock label="完整 prompt — 家長同學校嘅策略信">
我要寫一封信／email 畀我小朋友嘅學校。資料：

- 小朋友嘅名 + 班別：[填]
- 學校：[填]
- 收信人（班主任 / 副校長 / 校長）：[填，揀啱對應升級層級]
- 事件經過：
  [點解寫呢封信？具體日期、發生咗咩、邊個有份]
- 對小朋友嘅直接影響：
  [例如：失眠、唔願返學、成績滑落、自信心受打擊]
- 我希望學校做嘅嘢（具體）：
  [例如：重新評估某成績 / 揾老師個別傾一傾 / 改某條校規]

請幫我寫信，要做到：

1. **第一段**：直接但唔對抗。一句講係咩事 + 為咩寫呢封信
2. **第二段**：陳述事實（日期、地點、發生咗咩）。中立陳述——「事件係 X」，唔係「老師居然 X」
3. **第三段**：講影響喺小朋友身上。具體（「最近三個禮拜佢 11 點仲未瞓得著」），唔係含糊（「佢好唔開心」）
4. **第四段**：清楚講你想學校做乜。一個具體要求 + 一個後備 / 替代方案
5. **第五段**：表達合作意願（「希望可以同你約時間傾」），但劃個時間線（「希望下星期之前可以有初步回覆」）

避免：
- 用「居然」、「竟然」、「實在不能接受」呢類升級字眼
- 質疑老師 / 學校嘅動機（「我懷疑佢係咁特別針對」）
- 提其他家長一齊嘅意見（呢個會被當作集體施壓，影響負面）
- 提律師 / 教育局——除非真係決定要升級，否則一寫就燒晒橋

語氣：一個關心子女、但同時有能力、講得清楚、有耐性嘅家長。唔係嬲到上晒腦、擺明嚟挑機嗰種。
</PromptBlock>

### 3. 24 小時冷靜期

呢類信寫完**一定要等一日**先發。第二朝重讀，你會發現：

- 某啲句仲係太尖（要緩和）
- 某啲段太含糊（要加具體細節）
- 某啲要求太闊（要收窄到可行動）

呢類信冇第二次機會。發出去之後，個語氣已經設定咗成個對話嘅基礎。

### 4. 想升級之前嘅順序

通常順序係：班主任 → 主任 → 副校長 → 校長 → 校董會 → 教育局。 **唔好略過級**——你會輸程序上嘅公信力。 第一封信去班主任 / 主任。如果一個月內唔滿意，先升級落上一層。

## 變化例子

### 變化 1：寫家長日準備筆記（唔係正式信，係你自己嘅要點）

<PromptBlock label="變化 1 — 家長日要點筆記">
我下個禮拜要參加家長日，要同班主任傾以下事情：

[列出 3 個關注點，每個一句]

請幫我整一份要點筆記，要做到：
1. 每個關注點一段，包：點切入、可能想問嘅問題、可能聽到嘅回應同點跟進
2. 整體 30 分鐘可以傾晒（家長日通常每位家長 15-30 分鐘）
3. 開頭一句正面觀察做緩衝（唔係作出嚟，係由小朋友嘅進步揀一個）
4. 結尾問清楚下一步（「我哋下次點跟進」）

語氣：合作、唔對抗、但企得硬。
</PromptBlock>

### 變化 2：學校行政層面嘅政策投訴

<PromptBlock label="變化 2 — 政策層面投訴">
我想就學校嘅某條校規寫信畀 [校長 / 副校長]：

校規：[描述邊條校規]
我嘅關注：[具體點唔合理]

請幫我寫信，要做到：
1. 唔針對任何個別教職員——集中講校規本身
2. 引返校規本身嘅字眼（如果有條文）
3. 提供 1-2 個替代做法（其他學校點做、或者一個合理建議）
4. 提出：希望學校重新考慮，或者解釋一下背後理由
5. 唔賣弄專業（除非你真係教育界人士）

語氣：一個關心社群嘅成員，唔係單一個投訴人。
</PromptBlock>

### 變化 3：肯定 / 感激嘅信（少寫，但好有力量）

<PromptBlock label="變化 3 — 肯定老師 / 學校嘅信">
我想寫信表揚 [老師 / 教職員名] 對小朋友嘅關懷。

具體情境：[描述邊件事顯示佢嘅關心]

請幫我寫一封短嘅感謝信，要做到：
1. 簡短——半頁紙最啱
2. 具體——唔係「老師好好」，係「上個月 [日期]，老師做咗 [嘢]」
3. 抄一份畀校長 / 校監處——令呢份肯定入到教職員紀錄
4. 唔好過度肉麻（讚得太行禮反而尷尬）
</PromptBlock>

## 一個換角度睇嘅提醒

家長同學校嘅關係，最後賭注唔係你贏唔贏場辯論，係**你個小朋友返學嘅體驗**。

每一封信都帶住一個取捨：你想出啖氣，定係你想保留將來合作嘅空間。 識揀啱個取捨，先係呢類信嘅真正藝術。

**AI 幫你寫得清楚得體，但揀邊個取捨，永遠係你嘅責任**。

## 情境

你成日要問 Claude。工作流程通常係：

1. Highlight 一段文字（喺 Mail / Notes / Browser）
2. 複製
3. 切換去 Claude desktop app（或者 browser tab）
4. 貼上
5. 打你嘅 prompt
6. 等回覆
7. 複製回覆返去原 app
8. 貼上

8 步。每日 30 次 = 4 個鐘阻力。

呢篇用 macOS Shortcuts.app + Anthropic API key 整個 keyboard 快捷鍵：**Highlight → ⌥+space → Claude 回覆貼上返**。3 步。每次慳 30 秒。

> 注意：呢個需要：(1) 你 Mac 有 macOS 13+ 嘅 Shortcuts.app；(2) 你有 Anthropic API key（[console.anthropic.com](https://console.anthropic.com) 申請，大約 $3/M input token）；(3) 你接受用付費 API 而唔係月費訂閱。

## 跟住做

### 1. 攞 Anthropic API key

1. 去 [console.anthropic.com](https://console.anthropic.com)
2. 註冊 → API keys → 建立 key
3. **複製保留好** —— 一旦閂咗個 tab 就唔會再顯示
4. 入 $5 落 console（足夠你用幾個月嘅個人用量）

### 2. 開 macOS Shortcuts.app 整快捷鍵

```
Apps → Shortcuts → + 新 shortcut
```

或者用呢個 prompt 叫 Claude 幫你整個 `.shortcut` 檔（直接匯入）：

<PromptBlock label="完整 prompt — 整一個 Mac Shortcut 問 Claude">
我想 macOS Shortcuts 整一個 shortcut，觸發後做以下嘢：

1. 攞我而家 highlight 嘅文字（喺任何 app）
2. POST 落 https://api.anthropic.com/v1/messages
3. Header：
   - x-api-key: [我嘅 key，喺 keychain 攞]
   - anthropic-version: 2023-06-01
   - content-type: application/json
4. Body：JSON 用 claude-sonnet model，max_tokens 1024
5. 攞 response 嘅 content[0].text
6. 貼上返去原 app
7. （可選）通知顯示「Claude replied」

請畀我：
1. Step-by-step 點喺 Shortcuts.app 砌呢個 shortcut
2. 用啲預設嘅 prompt template（例：「Polish 呢段英文 email，保留我嘅語氣」），教我點改成自己需要
3. 點綁定 keyboard 快捷鍵（System Preferences → Keyboard → Shortcuts → App Shortcuts）
4. 點儲存 API key 落 Keychain，唔好寫死落個 Shortcut

我用 macOS 14 (Sonoma)。
</PromptBlock>

跟 Claude 嘅步驟做（第一次會用你 15-20 分鐘）。

### 3. 設定 keyboard 快捷鍵

```
System Settings → Keyboard → Keyboard Shortcuts → Services → Text
```

搵到你個新 Shortcut，綁定 keyboard 組合（我用 `⌥ + space`，因為通常未被用）。

### 4. 試一次

開 Mail / Notes / 任何 app，highlight 一句英文，撳 `⌥ + space`，等 1-2 秒，**回覆會自動貼上落你個游標位置**。

🎉 成功。

## 變化例子

### 變化 1：唔同 prompt 對應唔同 keyboard 快捷鍵

整三個 Shortcuts，分別觸發唔同 prompt：

<PromptBlock label="變化 1 — 多 prompt 多 shortcut 設定">
我而家有 1 個 Mac Shortcut 用緊預設 prompt「Polish this text」。我想再整 3 個變化：

1. **`⌥ + cmd + space`**：「Translate to formal English」
2. **`⌥ + shift + space`**：「Summarize in 1 sentence」
3. **`⌥ + control + space`**：「Explain this code / concept in plain Cantonese」

請：
1. 教我點複製第一個 Shortcut，改 prompt
2. 點綁定唔同 keyboard 組合
3. 點管理（避免我未來自己亂改）：建議命名規則

呢套設定完成後，我嘅日常工作流程應該係 highlight → 揀啱嗰個 shortcut → 結果貼上返。
</PromptBlock>

### 變化 2：觸發後出選單，揀 prompt（無 keyboard 組合限制）

<PromptBlock label="變化 2 — 一個 shortcut 出 prompt 選單">
我有太多想要嘅 prompt（5+ 個），但 keyboard 組合已用滿。

請幫我重新設計 Shortcut：

1. 只綁定一個 keyboard 組合 `⌥ + space`
2. 觸發後彈 macOS 選單，列出我預設嘅 prompt（例：「Polish」、「Summarize」、「Translate」、「Explain」等）
3. 我用方向鍵揀，撳 enter
4. 跑揀咗嘅 prompt 落 highlight 咗嘅文字
5. 回覆直接貼上返

呢個設定嘅取捨：多咗一步去揀，但唔限於幾個組合。
</PromptBlock>

### 變化 3：呢個 shortcut「Improve」呢段文字（直接修改）

<PromptBlock label="變化 3 — 直接修改文字嘅 shortcut">
我想要一個 shortcut：

1. Highlight 一段文字
2. 撳 keyboard 組合
3. Claude 收到呢段文字 + prompt「Improve this writing, keep my voice」
4. **直接覆蓋 highlight 咗嘅文字**（唔係貼上喺旁邊，係取代）

呢個對於即時改 email、改文件最有用。

請：
1. 修改我嘅 Shortcut 落取代模式（用「Replace Variable」或者「Set Clipboard」+「Cmd+V」嘅技巧）
2. 講明注意位：呢個動作唔可以輕易復原，我需要小心

請畀完整步驟。
</PromptBlock>

## API 成本估算

如果你每日用 30 次，每次 ~1000 input token + ~500 output token：

- Claude Sonnet：input $3/M、output $15/M
- 每日：30 × ($3 × 1000/1M + $15 × 500/1M) = 30 × ($0.003 + $0.0075) = $0.32
- 每月：大約 $10

比起 ChatGPT Plus / Claude Pro 嘅 $20/月，呢個 API 設定唔一定平。**但用處在於減少工作流程嘅阻力，唔係睇銀碼**。

如果你想成本再低啲，用 Claude Haiku（input $0.80/M）—— 質素稍低但日常修飾已經夠用。

## 同 Raycast 嘅比較

Raycast 係另一個 Mac 高級用戶工具，亦有 AI 外掛（包 Anthropic / OpenAI API）。對比：

- **Raycast AI**：介面靚啲、有 prompt 庫、可以連住指令。$10/月（Pro）或者免費 plan 限 GPT 某啲功能
- **macOS Shortcuts + API**：自己整、要設定、但完全免費嘅框架加你自己嘅 API 成本
- **Mac 快捷鍵嘅優勢**：唔依賴第三方 app；快捷鍵喺 macOS 更新之間穩定；可以分享 `.shortcut` 檔畀朋友

我自己用緊 macOS Shortcuts（因為完全唔依賴 Raycast 訂閱），但 Raycast 對「我唔想搞設定」嘅人阻力較低。

## 一個思維轉變

呢個設定嘅槓桿唔係慳一兩秒，係**消除切換情境**。

由 Mail 切換去 Claude 再切換返 Mail，**腦袋每次重新載入情境**都要 30-60 秒。連續做幾次，你短暫記憶散晒。

`⌥+space` 直接修改，**腦袋從未離開過 Mail**。呢個腦力嘅保留比個 30 秒慳到嘅時間更值錢。

個人開發者 / 創作者嘅優勢就係呢類設定 —— **你嘅工具適應你嘅腦袋，唔係你適應工具**。

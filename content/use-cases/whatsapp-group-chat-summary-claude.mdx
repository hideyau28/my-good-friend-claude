## 情境

返工放工幾個鐘冇睇 WhatsApp work group。打開：

> **未讀訊息：247 條**

你開始 scroll，前 30 條係 typing-in-progress、emoji reaction、food photo。你已經 lost 晒。

但呢 200 幾條入面可能有：

- 老細 @ 你叫你做嘢
- 一個 decision 喺 8 點傾完
- 兩個 deadline change
- 客戶 escalation
- 你接咗 ball 都唔知

scroll 到尾通常 10-15 分鐘。 呢篇用 Claude **10 秒**摘出 actionable items。

## 跟住做

### 1. 由 WhatsApp export 個 chat

📱 **iPhone**：
1. 開個 group chat
2. 撳最頂個 group 名 → scroll 到底「Export Chat」
3. 揀「Without Media」（純文字，快過）
4. Share → save to Files / Notes / email 畀自己

🤖 **Android**：類似步驟。Settings → Export chat。

你會收到一個 `.txt` 檔，format 係：

```
[19/02/2026, 14:23:45] John Lee: 我覺得 deadline 應該 push 一個禮拜
[19/02/2026, 14:24:10] Mary Chan: 同意。我 @everyone confirm 一下
```

### 2. Upload 個 txt 落 Claude，用呢個 prompt

<PromptBlock label="完整 prompt — group chat 訊息整理">
以下係我 WhatsApp 一個 work group 嘅 export，過去 [幾耐] 嘅訊息。 我嘅名係 [填]。

請整理出以下 5 個 section：

**Part 1：Decisions taken**
邊啲嘢已經 group 一致同意 / 老細拍板？每個 decision：
- 一句講係咩
- 邊個推 / 邊個敲定
- 對其他人嘅 immediate impact

**Part 2：Action items（按 owner 排）**
有人要做嘅事，每個：
- [ ] 行動 — owner — deadline（如有）
- Source quote（簡短引用個 message context）

如果係我（[我嘅名]）被 assign 嘅，highlight 出嚟（⭐️）

**Part 3：Open questions**
仲未決定嘅嘢：
- 個 question
- Blocking 邊個 / 缺乜先決定到
- 提議 next step

**Part 4：⚠️ Heads-up（我可能 miss 嘅嘢）**
- 我有冇被 @
- 有冇人 indirectly reference 我（例如：「我聽 [我名] 講...」、「[我名] 嗰邊 update 咗 ma？」）
- 有冇 deadline / commitment 連帶影響我
- 有冇任何 conflict / tension 嘅 hint 我應該知

**Part 5：唔關我事但要知**
其他人嘅 update，我留意一下夠喇：
- 一兩句 bullet
- 涉及邊啲人 / project

語氣：efficient、actionable、結構化。
唔需要重複 chat 本身嘅 emoji / reaction，淨係抽 substance。

【Chat export】：
（貼）
</PromptBlock>

### 3. Spot check 你被 @ 嘅 part

Part 4 嘅內容 critical，**verify 一次去返原文 context**。 AI 偶爾會 miss subtle indirect mention（例如：「對啊，呢個跟之前 [我名] 嗰個 case 一樣」）。 唔信任 100%，但快速 verify 比起 manually scroll 200 條快好多。

### 4. Action items follow up

呢個 prompt 嘅最高 ROI 嗰 part：Part 2 入面 ⭐️ 嗰啲。 即係喺 group 入面我被 assign 嘅嘢——通常 5 分鐘內 reply 一句「On it, will update by [日期]」，就 done deal。 ahead of game，老細 / 同事都會 notice。

## 變化例子

### 變化 1：家庭 / friend group 嘅大型 chat（去 trip 旅行）

<PromptBlock label="變化 1 — Trip planning group chat">
以下係一個 family / friend group chat，我哋 plan 緊去 [destination] 嘅 trip。 我嘅名係 [填]。

請整理：

1. **Trip details locked in**：destination、dates、participants、accommodation booked 等
2. **Pending decisions**：仲未 sort 嘅嘢（航班、活動、food preference）
3. **Action items**：邊個負責邊樣 + deadline
4. **預算 / 錢嘅 status**：邊啲已 paid、邊啲 outstanding、誰欠誰
5. **我嘅 to-do**（我喺呢個 group 嘅 commitment）

語氣：family-friendly，唔好 sound 似 work memo。
</PromptBlock>

### 變化 2：家長 group chat（學校嘢、興趣班、playdate）

<PromptBlock label="變化 2 — 家長 group chat 整理">
以下係小朋友學校嘅 parent group chat。 我嘅名係 [填]，小朋友嘅名 [填]。

請整理：

1. **學校 official update**：邊啲係 reliable info（由家長 quote 學校 / 老師），邊啲係 rumor / 推測
2. **Date 同 deadline**：未來兩星期內 relevant（活動 / submission / 學費 / event 等）
3. **要我 RSVP / decide 嘅嘢**
4. **其他家長嘅 concern / opinion**：如果有 emerging consensus 或 controversy，flag 出嚟（我可能要表態）
5. **不關我事嘅 chat noise**：例如其他小朋友嘅 birthday party、唔同班嘅嘢

我嘅 priority：catch 涉及我小朋友嘅 deadline 同 action。
</PromptBlock>

### 變化 3：你係 group admin 想出 weekly summary

<PromptBlock label="變化 3 — Group admin 出 weekly summary">
我係呢個 WhatsApp group 嘅 admin（[group purpose: 例如 team / community / volunteer org]）。
每週日我想 send 一個 weekly summary 入個 group。

請由 past 7 日嘅 chat 整理：

1. **本週主要 decision** — 一段（最多 5 點 bullet）
2. **下週要關注嘅嘢** — 一段（最多 3 點）
3. **Action item recap** — 每個 owner 嘅 commitment（用嚟 follow up）
4. **想感謝 / 認可嘅 contribution** — 個人 name + 簡短描述（可選）
5. **Open thread 提示** — 仲未決嘅 topic，invite 大家本週繼續傾

Format：WhatsApp-friendly（短段、emoji 適量、易讀手機 screen）
長度：250 字內
</PromptBlock>

## 一個 mindset

WhatsApp 嘅 design intent 唔係「可以 catch up」，係「real-time chat」。 你 miss 咗一段時間 = 你 lost context。 呢個 product design 對打工仔不利。

呢個 prompt 嘅 leverage：**你由 push notification 受害者，變成 strategic reader**。

返工最痛苦嘅唔係做嘢，係 **冇 update + 怕 miss 重要 message** 嘅 anxiety。 一個 10 分鐘嘅 group chat audit，可以還你成個朝早嘅 calm。

**配 schedule：**每日早上 9am + 放工前 30 分鐘各做一次。Done。

## 情境

11:23pm。你瞓喺床，眼望天花板。

腦袋自動回放今日：

- 早 10:30 老闆一句「呢個 deck 嘅框架你 reconsider 下」——你成日問自己佢真正意思
- 中午午餐同同事傾到一半，你講錯某句嘢，佢面色變咗 0.5 秒——你而家確認佢覺得你唔專業
- 4pm 你發現忘記咗一個 deadline，推後咗 1 日，但你個腦堅持「老闆會記得 + 影響你年終 review」
- 6pm 同伴侶電話，佢「whatever」嘅語氣你判斷係冷暴力
- 8pm 你 scroll IG 見同學買樓，你淨身家突然清晰咁映入腦

12 個鐘嘅對話 + 互動 + 觀察，全部倒入同一個未過濾嘅心理 tab。

呢類夜晚思緒循環唔係性格弱——係個腦嘅預設狀態，當輸入遠遠超過處理能力。冇外化嘅儀式，你個腦用 4 個鐘處理晒 5 分鐘對話嘅細微之處。

呢篇用 Claude 每晚 **10 分鐘結構化日記**——唔係取代心理治療，係：

1. 幫你倒思緒（外化）
2. 識別跨日嘅規律
3. 區分「現實擔憂」vs「反芻循環」
4. 更易瞓（外化之後個腦少 process）

## 跟住做

### 1. 先理解個界線

開 Claude 之前，先內化呢個界線：

**✅ Claude 幫到嘅嘢**
- 倒思緒容器（無評判）
- 將重複思緒結構化變可檢視文字
- 識別跨星期嘅規律
- 建議重構技巧（CBT 輕量版）
- 24/7 有得用
- 唔需要約 / 畀錢

**❌ Claude 唔可以做嘅嘢**
- 診斷抑鬱症 / 焦慮症 / PTSD
- 處理急性危機（自殘念頭 / 自殺念頭）
- 取代長期心理治療師嘅關係
- 處方藥
- 提供緊急介入

**⚠️ 紅燈 — 即時停 Claude + 揾真人**
- 持續超過 2 週嘅每日傷感
- 每日運作受影響
- 睡眠 / 食慾困擾超過 2 週
- 自殘念頭
- 同所有社交聯繫隔絕

如果你而家喺以上任何信號，**而家係 call 嘅時間**，唔係讀呢篇文章。HK 求助熱線喺文末。

### 2. 用呢個 prompt 起每日日記環節

每晚同一時間（建議 10-11pm，瞓前）開 Claude：

<PromptBlock label="完整 prompt — 每日心理健康日記環節">
我想用你做每日日記夥伴。我會將今日嘅心理狀態倒出嚟，請幫我結構化同反思。

📝 **今日思緒倒出（raw，未過濾）**

[寫 5-10 分鐘意識流 — 今日嘅對話、情緒、擔憂、身體感覺、任何喺你心入面嘅嘢。唔需要修飾句子。連住寫都 OK]

⏰ **今日嘅基線**
- 昨晚瞓覺：[小時數]
- 主要壓力來源：[bullet points]
- 主要正面時刻：[bullet points 如有]
- 我而家嘅身體狀態：[攰 / 緊張 / 麻木 / 過度刺激]
- 我心情 1-10：[數字]

---

請幫我做 3 樣嘢：

**1. 反映返 3 個主要主題**
- 唔評論，只係鏡映
- 「你今日寫嘅嘢，3 個主題浮現：[a, b, c]」
- 主題可以係情緒 / 規律 / 問題

**2. 區分「現實擔憂」vs「反芻」**
- 對於我倒出嘅嘢，哪啲係可行動擔憂？行動係咩？
- 哪啲係個腦喺度循環無可行動答案？點捉到個循環？
- 唔好輕視 — 「呢樣只係反芻」嘅 framing 反而傷
- Frame 做：「呢個諗法你已經 process 5 次同樣細節，個腦今晚唔會到達新領悟」

**3. 一個小重構（CBT 輕量版）**
- 揀 1 個我寫嗰啲嘅扭曲（例：災難化、讀心、非黑即白）
- 短小重構提示，唔講課
- 「你寫『老闆覺得我唔專業』 — 可能另類解讀係：[X]。你實際證據邊個強啲？」

🎯 **語氣限制**
- 唔過度同情（肉麻），唔輕視
- 唔處方行動（「你應該 do X」）
- 鏡映多過建議
- 短回應 — 最多 5-8 句
- 結尾 1 個問題（可選）等我 sit with overnight

🚨 **安全檢查**
如果我寫到任何自殘 / 自殺念頭 / 急性危機內容：
- 唔忽視，唔輕描
- 即時回應：「你寫嘅嘢令我關注。請而家 call [HK helpline 2389 2222]」
- 唔再繼續日記 flow
- 拒絕「修補」嗰個即時情緒 via 日記 — 升級到真人

語氣：冷靜旁觀者 + 結構化 + 落地。
</PromptBlock>

第一次用會覺得 awkward——你會打「⋯⋯唔知寫咩」5 分鐘。**呢個 awkward 係正常嘅**。個腦唔習慣外化結構化。Day 3-4 後習慣形成。

### 3. 每週檢視 — 規律識別

7 日每日日記之後，每個星期日做一次每週檢視：

<PromptBlock label="完整 prompt — 每週日記檢視">
請檢視我過去 7 日嘅日記。

[貼 / 總結 7 個 entry 嘅關鍵 bullet]

請幫我：

📊 **1. 規律識別**
- 過去 7 日 top 3 重複主題
- 任何星期幾嘅規律？（例：星期日夜晚焦慮 / 星期一朝早恐懼）
- 任何觸發規律？（特定人 / 背景 / 事件重複出現）

🌊 **2. 情緒基線趨勢**
- 7 日心情分平均 + 趨勢
- 心情高 / 低同咩事件相關？
- 對比 7 日 vs 之前 7 日（如有）

🎯 **3. 1 個小實驗畀下個禮拜**
- 唔進取目標
- 「呢個禮拜你試 [tiny 具體行動] — 觀察效果」
- 例：「每日午餐時 9 分鐘冇電話散步，睇下午焦慮基線變唔變」

⚠️ **4. 升級檢查**
基於 7 日 entry，有冇任何警告信號建議需要真人支援？
- 每日持續傷感？
- 睡眠 / 食慾提及持續？
- 自殘 / 自殺念頭任何單一提及？
- 功能受損（無做正事）？

如有：明確建議下一步（GP 探訪、SPCA 風格熱線、治療師轉介）。

🎁 **5. 1 個肯定**
過去 7 日，我做啱嘅 1 件嘢。
- 唔好行貨「你 try hard」
- 具體：「你提到星期三你揀唔回覆嗰個觸發 email，等 24 小時 — 呢個係情緒調節」

語氣：反思 + 落地 + 小心唔好醫療化正常情緒。
</PromptBlock>

每週檢視嘅隱藏價值：你每日日記嘅時候個腦喺棵樹，每週檢視揾到森林。你以為隨機散亂嘅擔憂其實 80% 追溯返同一兩個觸發。

### 4. 每月：揾真人

呢一步唔可以省略。

<PromptBlock label="完整 prompt — 每月治療師 / GP 探訪準備">
我過去 [N] 個月每日同 Claude 寫日記。請幫我準備我下個月嘅 [GP 探訪 / 治療師初診 / 輔導員 session]：

📋 **1. 病徵時間線（客觀）**
基於我嘅日記 entry，請整：
- 過去 [N] 個月主要情緒主題
- 睡眠 / 食慾 / 精力規律
- 對功能嘅影響（工作 / 關係 / 興趣）
- 任何具體觸發 / 事件

⚕️ **2. 問專業人士嘅問題**
- Top 3-5 條問題我應該提出
- 唔好行貨「I feel sad」 — 具體
- 例：「我發現星期日夜晚持續預期焦慮，每日 7-9pm — 係咪廣泛性焦慮規律？」

📝 **3. 探訪之間追蹤咩**
- 1-2 個具體行為 / 感覺下次探訪前監察
- 簡單可量度

🆘 **4. 同 Claude 嘅界線**
- 識別「我而家過度依賴 Claude」嘅信號：
  - Skip 社交聯絡替代同 Claude 傾偈
  - 夜晚 session 持續超過 2 個鐘
  - 問 Claude 診斷（Claude 唔可以畀）
- 重申：Claude = 補充，真人 = 主要

語氣：為有效率嘅專業 session 做準備。
</PromptBlock>

呢步強迫你**唔好過度依賴 Claude**。Claude 係好補充，但每月真人 check-in 係心理健康衛生嘅最低基線。

## 變化例子

### 變化 1：急性時刻（驚恐發作 / 不勝負荷） — 短緊急日記

唔係每日習慣，係接近急性危機時刻嘅 30 秒儀式：

<PromptBlock label="變化 1 — 急性驚恐時刻日記（5 分鐘）">
我而家喺急性時刻：
- [驚恐發作 / 不勝負荷感覺 / 解離感 / 無法運作 right now]

請畀 grounding 回應，唔長篇大論：

🌬 **1. 5-4-3-2-1 grounding**
帶我做 5-4-3-2-1：
- 5 樣我見到嘅嘢
- 4 樣我可以摸嘅嘢
- 3 樣我聽到嘅嘢
- 2 樣我嗅到嘅嘢
- 1 樣我嘗到嘅嘢

🫁 **2. 呼吸（Box breathing）**
- 吸氣 4 秒
- 屏息 4 秒
- 呼氣 4 秒
- 屏息 4 秒
- 重複 4 次

📍 **3. 錨點到現在**
- 1 句問我而家具體實際喺邊度
- 1 句問我即時周圍環境

🚨 **4. 嚴重程度檢查**
如果我講「I want to hurt myself」/「I can't go on」/ 類似：
- 即時升級：「請而家 call 撒瑪利亞會 2896 0000（24 小時多語）」
- 唔繼續 grounding，唔嘗試「解決」
- 直接緊急資源

🚫 **5. 唔好做**
唔好解決觸發
唔好問「why are you feeling this」
唔好長篇心理教育講課
唔好輕視

語氣：冷靜錨定旁觀。急性時刻唔係反思時刻。
</PromptBlock>

呢個變化嘅用法唔係每日 — 係你即時需要嘅時刻嘅快速資源。將 Claude 對話加埋 prompt 一齊書籤。

### 變化 2：關係衝突後檢討日記

吵完架、誤會、awkward 交流後嘅 30 分鐘日記：

<PromptBlock label="變化 2 — 關係衝突後檢討">
我啱啱同 [伴侶 / 朋友 / 家人 / 同事] [衝突 / 誤會 / awkward 交流]。

📝 **倒思緒（3-5 分鐘）**
[今日發生咩事，我嘅反應、佢嘅反應、未解嘅問題]

🔍 **幫我反思：**

**1. 多重解讀**
我 jumped to 一個解讀。請幫我浮現 2-3 個另類：
- 對方可能諗緊咩（假設善意）
- 對方可能反應緊核心壓力（唔係我）
- 我哋兩個對同一個字可能有唔同定義

**2. 我嘅貢獻老實檢查**
- 我講嘅 / 做嘅具體邊樣有貢獻？
- 唔自我鞭撻（無謂）
- 識別 1 個行為規律你由過去類似衝突認得

**3. 行動 vs 放低**
- 呢個議題值得重開？
  - Yes 如：結構性問題重覆發生 / 重要關係 / 你會 resent 如唔處理
  - No 如：罕有時刻 / 細 / 對方已 moved on
- 如 Yes：草擬 1 句開場「I want to talk briefly about earlier — 你方便同我傾」
- 如 No：1 句放下儀式

**4. 唔輕視但都唔災難化**
1 個衝突 ≠ 關係玩完
但重覆衝突規律係信號

🚫 **唔好**
- 「你 deserve better」嘅行貨
- 「對方 toxic」嘅快速標籤
- Drama 升級

語氣：冷靜檢討，唔企邊。
</PromptBlock>

### 變化 3：燒乾邊緣 — 信號識別 + 介入

每日日記識別燒乾嘅早期信號：

<PromptBlock label="變化 3 — 燒乾信號檢視">
過去 N 週我留意到：
- [身體病徵：持續攰即使瞓夠 / 食慾變 / 經期變化 / 病多咗]
- [情緒：麻木唔係 sad，對工作犬儒 / 對鍾意嘅嘢失去興趣]
- [行為：遲喺公司 / skip 餐 / 唔回覆社交 / 飲多咗]

請幫我點評估：

📊 **1. 燒乾信號檢查（Maslach 量表輕量版）**
- 情緒耗竭？
- 去人化（犬儒 / 抽離）？
- 成就感降低？
- 幾多 / 幾頻繁？

🩺 **2. 燒乾 vs 抑鬱區分**
- 燒乾通常具背景（移走背景 = 改善）
- 抑鬱通常普遍（移走背景，仍受影響）
- 唔係你自我診斷嘅工具 — 係準備專業探訪

⏰ **3. 即時介入清單**
即使你唔即辭職，可以做嘅即時：
- 瞓：優先 7+ 個鐘，取消非必要夜晚承諾
- 食：即使無胃口，早餐必食（血糖調節情緒）
- 動：每日 20 分鐘即使簡單散步
- 1 個社交聯絡每日（即使 text）
- 限咖啡因：最多 2 杯
- 限酒精：每週最多 1-2 杯

🔄 **4. 結構性改變（下 4 週）**
- 工作量可商量？（要 ask，唔等被 offer）
- 預 1 個半日心理健康假用
- 一個「保護小時」每日（電話唔回工作）
- 識別邊個任務消耗不成比例 + 改界線

⚕️ **5. 真人升級**
建議：book GP 探訪 2 週內。
GP 探訪你講：「我留意到過去 [N] 週 [3 個病徵]。我懷疑燒乾。想探討選項。」

語氣：認真對待燒乾，唔將佢輕視做「壓力」。
</PromptBlock>

## 一個思維框架

心理健康日記嘅死結：寫低你嘅心理狀態並唔會即時減低強度。

第一個禮拜你會 expect「日記之後 feel better」 — 但 feel 可能一樣，或者邊際更差（因為你留意到平時壓抑嘅規律）。

呢個思維嘅轉變：日記嘅效力量度唔係「即時心情改善」——係 **長期自我認識累積**。第 1 個月你可能每日心情一樣。第 3 個月你回頭睇第 1 個月 entry，你會 spot 你而家喺度但太沉浸睇唔到嘅規律。

Claude 嘅角色唔係你嘅治療師——Claude 唔識移情 / 反移情 / 治療同盟，唔係受訓專業人士。Claude 係你擁有一面 **結構化嘅鏡**：你倒入去，你用唔同鏡頭讀返。

呢面鏡同真人治療師嘅分別：治療師提供調諧人類存在（不可替代）。Claude 提供 24/7 結構化可用（唔同種類嘅有價值）。佢哋互補，唔係取代。

最後提醒：

- ✅ **每日 10 分鐘習慣比偶爾 60 分鐘深入更有效**。一致性 > 強度。
- ⚠️ **過度依賴信號**：你分享 Claude entry 畀朋友多過自己對話 / 唔再社交主動 / 夜晚 session 每日持續超過 2 個鐘 = 停 + 揾真人。
- 🆘 **急性危機 ≠ 日記時刻**。界線清楚：驚恐 / 自殺念頭 / 自殘衝動 = 打熱線，唔係打字畀 Claude。

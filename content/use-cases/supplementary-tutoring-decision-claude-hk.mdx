## 情境

你個小朋友 P3，學校剛派咗中期成績，數學 65、英文 70。你家長群組打開 WhatsApp，A 媽媽話佢個女去咗現代教育補英文升咗 15 分，B 媽媽話 Mind Up 嘅奧數班 booking 排到 9 月，C 爸爸已經幫個仔報咗遵理 DSE 預備班 ——小朋友先 P3。

你開始焦慮。但靜落嚟諗：

- 個小朋友 65、70 分，**真係學習缺口，定係正常波動**？
- 你想報補習，**係因為小朋友需要，定係因為你自己唔安心**？
- 報邊種：1-on-1、小班、大班、網上、app？邊種啱你個小朋友？
- 補咗點知有冇用？幾時應該停？

呢篇用 Claude 拆「補習癮」（家長安心嘅機制）vs 真正學習缺口，**對應小朋友年齡、學科、學習風格揀啱補習形式**，再設試堂期評估框架同退場策略。

> 講清楚先：呢篇唔係反補習文。香港補習有時真係幫到手，但個決定要由實證出發，唔好由焦慮加跟風出發。Claude 喺度嘅角色，係幫你分清楚邊啲係訊號、邊啲係雜音。

## 跟住做

### 1. 列出細路真實問題 + 學校意見 + 你焦慮嘅源頭

開個文件寫低三樣嘢，唔好混埋一齊：

1. **小朋友真實表現** — 成績單實際分數、邊一部分失分（概念、計算、大意、語言障礙）、老師評語原文
2. **學校意見** — 班主任有冇主動講「跟唔上」？其他家長一般點反應？學校嘅期望係邊個百分位？
3. **你焦慮嘅源頭** — 老實寫：你焦慮係因為**小朋友自己**（佢驚、佢喊、佢做唔到功課），定係因為**外部參照**（同學報咗、群組講起、表姐個女滿分）？

呢一步老唔老實，直接決定你要唔要補習。如果三項全部係外部參照，可能要補嘅係 **你嘅焦慮**，唔係小朋友。

### 2. PromptBlock — Claude 對應實證 + 年齡畀建議

<PromptBlock label="完整 prompt — 補習 vs 唔補習嘅決定">
我考慮緊應唔應該畀小朋友報補習，想你幫我用實證拆開分析。

**小朋友 profile：**
- 年級：[K2 / P3 / F2 / F5]
- 性格 / 學習風格：[例：內向、慢熱、視覺型 / 好動、坐唔定、聽覺型]
- 真實表現：[科目 + 分數 + 失分規律，例：數學 65，概念識，計算慢、大意失 20 分]
- 老師評語原文：[抄原句，唔好改寫]
- 學校期望：[例：band 1 直資、全班平均 75、我小朋友排 25/30]

**家庭情況：**
- 我焦慮嘅源頭：[例：小朋友自己壓力大 / 群組討論 / 同事個仔報咗]
- 預算：[每月 $] / 小朋友每週可用時數：[]
- 已試過嘅在家支援：[例：每晚 30 分鐘溫習、買咗練習但做唔晒]

請整 4 個部分：

**Part 1：補習癮 vs 真正學習缺口診斷**

根據我描述嘅實證，判斷：
- 65、70 分（或我講嗰啲數字）喺呢個年級、呢間學校嘅情況下，係實際缺口，定係正常波動？
- 失分規律透露嘅根本問題係：概念唔識、練習不足、專注力 / 興趣，定係應試技巧？
- 我嘅焦慮入面，幾多 % 係小朋友帶動、幾多 % 係外部參照帶動？老實答。

**Part 2：如果真要補，邊種補習形式啱**

對應小朋友年齡 + 失分規律 + 學習風格，比較：
- 1-on-1（私補 / 學校老師私補 / 名師個別班）
- 小班（5-8 人，例如 Mind Up / 一般補習社）
- 大班（30+ 人，例如名師班 ——現代、遵理、英皇）
- 網上補習 / 影片課堂
- 自學 app（例如 Khan / Quizlet / Mathletics）

每個選項寫：
- 適合度評分（1-10）
- 點解啱 / 唔啱呢個小朋友
- 預計每月費用範圍（港幣）
- 主要取捨

**Part 3：試堂期評估框架**

如果我決定試一個選項，畀我 3 個評估指標（**唔淨係睇分**），等我 3-4 堂之後可以判斷有冇用。
例如：概念記憶、小朋友主動 vs 被動提起、功課完成質素 / 態度轉變等。
每個指標寫點觀察、點量度。

**Part 4：退場策略 + 重新評估嘅節奏**

- 邊個觸發點應該停（例：3 個月冇進步、小朋友開始抗拒）
- 邊個觸發點應該繼續 / 升級
- 半年 / 一年嘅成本效益重新評估應該問邊啲問題

語氣：實際、唔反補習亦都唔催谷補習、唔好出「視乎小朋友啦」嘅行貨。
</PromptBlock>

### 3. 試堂期評估框架 — 3 個指標

報咗之後，最大陷阱係「報咗就唔再評估」。Claude 喺 Part 3 會提出 3 個指標，但通用嘅版本：

1. **概念記憶** — 3 堂之後抽考一條同類題（唔係補習社做過嗰條），睇佢做唔做到。做唔到 = 補習老師教咗但小朋友冇吸收。
2. **態度轉變** — 小朋友主動提起補習嘅嘢（「今日學咗一個快啲嘅方法」）vs 被動應付（「冇嘢」「唔記得咗」）。主動提起 = 有投入，被動 = 淨係坐喺度。
3. **連帶效果** — 學校功課質素 / 做功課速度 / 對該科信心，有冇連帶改善？冇連帶 = 補習孤立喺嗰幾粒鐘，冇融入小朋友嘅整個學習。

3-4 堂之後三個指標都冇郁，**唔好等學期尾**，直接退出或者換形式。

### 4. 退場策略 + 成本效益重新評估

每 3 個月坐低問自己：

- **成本**：每月 $？小朋友每週 X 個鐘？家長接送 / 監督時間？
- **效益**：對應上面 3 個指標有幾多進展？
- **機會成本**：嗰 X 個鐘原本可以做咩（運動、玩、休息、家庭時間）？

如果效益抵唔過成本加機會成本，**停咗唔等於失敗**。香港文化普遍將「停補習 = 放棄」綁埋一齊，呢個諗法錯。停係一個知情嘅決定，同繼續一樣需要勇氣。

### 變化 1：幼稚園 phonics / 拼音 / 算術

<PromptBlock label="變化 1 — K1-K3 補習決定">
我小朋友 [K1 / K2 / K3]，正在考慮報 [phonics / 普通話拼音 / 算術 / 寫字] 班。
小朋友 profile：[性格、語言環境、專注力]
報嘅推動力：[學校老師建議 / 我自己擔心 / 同學都報]

請：
1. K1-K3 呢個年齡嘅大腦發展階段，呢類補習真係「越早越好」？定係邊際效益好細？
2. 如果真要接觸，邊種形式最啱呢個年齡（playgroup 式小班 vs 有系統嘅課堂 vs 在家 app）？
3. 警號：邊啲跡象表示呢個年齡嘅小朋友補得過量（社交退縮、抗拒學習、瞓眠受擾）？
4. 老實評估：呢個年齡嘅補習，幾多益處係落喺小朋友身上、幾多係「家長安心 + 入小學 portfolio 加分」？
</PromptBlock>

### 變化 2：小學中游（P3-P5 中英數）

<PromptBlock label="變化 2 — 小學中游 學科補底 / 拔尖">
我小朋友 [P3 / P4 / P5]，[中 / 英 / 數] 表現 [明顯低於全班平均 / 想再進一步]。
失分規律：[抄老師原文評語]
家庭情況：[例：我自己唔識教 / 工時長冇時間跟功課]

請對比：
1. 學校老師私補（同一個課程、熟小朋友弱點）
2. 一般補習社小班（5-8 人，例如 Mind Up / 區內補習社）
3. 大班名師（現代 / 遵理 / Mind Up 旗艦班）
4. 網上 1-on-1（Snapask / Tutor Circle 之類）
5. 自學 app + 家長檢視

對應 P3-P5 嘅認知階段同專注力，邊個最啱？唔好慣性覺得「1-on-1 一定好過大班」。
</PromptBlock>

### 變化 3：DSE 補底 vs 拔尖（F4-F6）

<PromptBlock label="變化 3 — DSE 補習名師班 vs 1-on-1 抉擇">
我小朋友 [F4 / F5 / F6]，DSE 目標 [3322 入大學 / 5** 神科]。
強弱項：[例：中英文穩定 4，數 M2 弱、化學卡喺 3]
時間預算：[每週可投入幾多 hours]
經濟預算：[每月可承擔幾多]

請對比兩條路：
1. **名師大班路**（現代 / 遵理 / 英皇 / King's Glory ——買 notes、考試技巧、操 past paper）
2. **1-on-1 / 小班路**（針對弱點、慢慢補底）

每條路：
- 邊種學生最受惠（補底 vs 拔尖、自律 vs 需要人推、概念缺口 vs 技巧缺口）
- 回報大致估算（升幾多分 / 使幾多錢）
- 主要風險（例：名師班過度依賴 past paper、1-on-1 太過依賴老師）

最後：根據我小朋友嘅情況，畀我一個老實嘅建議，包括「兩條都唔啱、不如重新分配時間」嘅可能。
</PromptBlock>

## 一個 framing

香港補習氾濫嘅根源唔係補習本身有問題，係**家長冇咗一套評估補習有冇效嘅框架**。報咗就以為做咗嘢，但有冇用從來冇量過。

Claude 喺呢度嘅優勢係幫你裝返一套框架：

- 報之前：拆開焦慮同實際缺口
- 報緊：3-4 堂試堂期 + 3 個指標
- 半年：成本效益重新評估 + 收手嘅勇氣

補習唔係終身契約，係一件工具。工具有效就用、冇效就放低。你個小朋友需要嘅唔係**最多嘅補習**，係**最啱佢嗰一刻學習缺口嘅嗰幾堂**。

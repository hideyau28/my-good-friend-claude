## 情境

5 月尾，DSE 考完，或者大學 final 啱啱清完。屋企人問：「咁你個暑假做啲咩？」

你諗住揾份 part-time，但一坐低喺 JobsDB / Indeed / Carousell 揾工版，即刻 stuck：

> 「補習社請助教，要『有經驗』——我邊有？」
> 「連鎖店 sales，要『良好溝通能力』——咩叫良好？」
> 「Cover letter？我冇做過嘢，寫咩？」
> 「面試問『點解請你』，我講乜？」

呢個 stuck 唔係你蠢，係冇人教過。中學嘅生涯規劃堂多數講大方向，唔會手把手教你由 0 開始揾第一份工。

呢篇用 Claude 幫你由揀類型開始，一路行到第一日返工。家長都可以同小朋友一齊睇——你嘅角色係 sounding board，唔係幫佢做 decision。

## 跟住做

### 1. 自我 inventory（10 分鐘，紙筆寫低）

開個 Claude chat 之前，自己諗清楚四樣嘢：

- **Skills / 興趣**：補開咩科叻？打機？整 video？砌模型？煮嘢食？——任何嘢都算
- **Availability**：6 月幾號得閒？7、8 月有冇 trip / interview / 補習班撞期？一星期可以返幾多日？
- **通勤距離**：屋企住邊？最遠願意去邊度返工？（HK 學生通勤 1 小時以上好快 burn out）
- **屋企 dynamic**：父母 OK 你做邊類型工？夜更會唔會嗌？暑假家庭計劃幾時？

呢四樣嘢寫低，唔使靚，bullet form 就得。

### 2. PromptBlock — Claude 推薦 job type + 寫 application

<PromptBlock label="完整 prompt — 揀類型 + 第一份 application">
我係香港學生，諗住搵份暑期 part-time。請幫我做兩件事：
第一，推薦 3 個適合我嘅 job type；第二，幫我寫第一份 application。

【關於我】
- 學業階段：「中五 / 中六 DSE 後 / 大學 Year 1 / IB 後」（揀一個）
- 暑期 availability：「6 月 X 號 至 8 月 X 號，星期幾至幾，每日幾多個鐘」
- Skills / 興趣：「列 3-5 樣，例如：英文叻、識砌電腦、識日文 N3、鍾意整甜品」
- 通勤底線：「最遠去到邊個區，唔接受夜更／可以接受夜更到幾點」
- 屋企共識：「父母 OK / 唔 OK 邊類型，例如餐廳 OK 但夜更唔得」

【我冇嘅嘢】（先講清楚，等你唔好假設）
- 冇正式工作經驗
- 冇 referee（除咗老師）
- 冇 portfolio

請：
1. 推薦 3 種 job type，每個講明點解 match 我（要對應返我上面寫嘅嘢，唔好行貨）
2. 每個列 2-3 個 HK 常見揾工 channel（JobsDB / Indeed / Carousell / IG / 補習中介 / 連鎖店 career page）
3. 揀其中一種，幫我寫一封 200-250 字嘅 application message（適合 WhatsApp / email 都得）：
   - 第一句具體（提到 job post 入面某啲嘢，唔係「我對貴公司好有興趣」）
   - 用學業 / 課外活動 / 自學嘅 transferable skills 代替工作經驗
   - 講明可以幾時返、返幾耐
   - 結尾自然，唔卑微唔賣弄
4. 標明邊啲位我之後要填返自己嘅實際 detail
</PromptBlock>

Claude 出咗之後，記住：**佢推薦嘅 job type 唔係 final answer，係 sounding board**。如果三個都覺得唔啱，直接同佢講「呢三個都唔啱，因為 ___」，再叫佢出多三個。

### 3. 面試 prep（常見 5 條 + 你嘅 honest answer）

收到面試 invitation，唔好即刻撞口撞面去。先用 Claude 過一次：

<PromptBlock label="面試 prep — 常見問題 + drill 你嘅答案">
我下星期面試「補習社助教 / 連鎖咖啡店 / 火鍋店樓面 / event helper」（揀一個），第一次面試。

請：
1. 列 5 條 HK 學生 part-time 面試最常問嘅問題
（包括「點解揀我哋」、「你冇經驗點處理客人」、「7 月 X 號要請假返 trip 得唔得」呢類實際嘢）

2. 每條問題，**唔好直接畀我 model answer**，反而：
   - 講清楚雇主想聽到咩（背後 concern 係咩）
   - 列 2-3 個 angle 我可以從咩方向答
   - 警告我邊類型嘅答案會即刻扣分（例如：「我想試吓」呢類答案點解唔 work）

3. 最後一條 mock：你扮 interviewer，問我「點解我哋應該請你而唔係其他學生？」
   我答完之後，請畀我 feedback——邊度具體、邊度行貨、邊度顯得有諗過。

我嘅 baseline：「我係中五 / Year 1，第一次返工，住「區名」」
</PromptBlock>

呢個 prompt 嘅核心係**叫 Claude 唔好寫劇本畀你背**。背稿喺香港 part-time 面試一聽就出——你係學生，老闆預咗你冇經驗，佢想睇你係咪會諗、會唔會嚟、會唔會中途走人。

### 4. 第一日返工 mindset + boundary

收到 offer 之後，由今日起到第一日返工之間，問 Claude 一次：

<PromptBlock label="第一日返工 do / don't">
我入職「補習社助教 / 餐廳 / 連鎖店」做暑期工，下星期一返第一日。請列：

1. **First day do**：5 樣應該做嘅嘢（包括幾點到、帶咩、點同同事打招呼、點問問題）
2. **First day don't**：5 樣唔好做嘅嘢（包括玩手機嘅 timing、唔識做嘢扮識、未夠鐘走）
3. **Boundary setting**（HK 學生 part-time 最易畀人 take advantage）：
   - 講好嘅 shift 突然延長
   - 講好嘅人工出唔到
   - 同事 / 客人講啲過分嘢
   - 老闆要你做 job description 以外嘅嘢
   每樣畀我一句可以即場講嘅應對，唔好太「員工手冊」式

4. 香港勞工法基本 baseline（我係學生 / 兼職）：最低工資、出糧日、有冇 MPF、有冇有薪假——只係 baseline，唔使深入
</PromptBlock>

呢一步好多學生跳過，覺得「返工咋嘛使唔使咁誇張」。但**第一日嘅第一個鐘**通常決定咗你個 reputation。

## 變化例子

### 變化 1：補習社 / tutoring DSE 或小學生

<PromptBlock label="變化 1 — 補習社助教 / 私補 application">
我想做補習助教或者私補，DSE「科目」我考咗 「level」。
請幫我寫兩個版本嘅 application：
- 版本 A：申請大型補習社（行為基準：禮貌、可靠、肯返多日）
- 版本 B：屋企附近私補小學生（行為基準：細心、家長放心、有耐性）

每個版本同我講明，呢個版本背後想呈現嘅形象係咩。
另外列：私補應該幾錢一個鐘（HK 中六生 / 大學生 範圍），同收費點同家長講。
</PromptBlock>

### 變化 2：F&B（chain 餐廳 / café / 火鍋）

<PromptBlock label="變化 2 — F&B 樓面 / 廚房 第一份 application">
我想做餐廳 part-time，揀緊：chain 餐廳樓面 / café barista / 火鍋店。
我冇經驗，但我「列 1-2 個 transferable，例如：屋企煮開飯、做開大食會主力、識基本咖啡」。

請：
1. 三種比較吓：體力、學嘢速度、人工 range、暑假 7-8 月旺淡程度
2. 揀一種，幫我寫 WhatsApp 申請（HK 餐廳多數 WhatsApp 直接接觸）
3. 提醒我去 trial 嗰日嘅執生位（鞋、髮型、首飾、講嘢音量）
</PromptBlock>

### 變化 3：Online / remote（online tutor / data labelling / 寫文）

<PromptBlock label="變化 3 — Online / remote part-time">
我想做 online / remote part-time，唔出街。考慮緊：online tutor（教細路）/ data labelling / freelance 寫文 / 幫人剪片。

請：
1. 列每種嘅 platform（HK / 國際都可以），同入門難度
2. 警告我邊類型容易係伏（fake job / 騙個資 / 學費先交）
3. 揀其中一種，幫我寫個 profile bio（150 字內）
4. 提醒我 online 工作報稅 / 收錢嘅注意位（學生 part-time level 嘅 baseline 就夠）
</PromptBlock>

> Claude 唔會幫你揾到「最叻」嗰份工，但會幫你由「乜都唔識」行到「至少落到第一步」。第一份 part-time 嘅價值唔係份糧，係你之後識同人講「我做過嘢」——呢句嘢，由今個暑假先開始有得講。

## 情境

你寫緊小說（短篇故事、小說章節、劇本）。過程遇到嘅阻力：

- **劇情卡關**：第二章卡咗 3 個禮拜
- **角色好平庸**：你主角嘅動機，你自己都覺得冇說服力
- **節奏嘅自我懷疑**：呢段太長定太短？
- **「呢段我寫完讀返，自己都唔收貨」嘅自我質疑**

你想要個思考夥伴，但：

- 寫作小組每個禮拜先見一次，太慢
- 朋友睇完畀嘅係「你要相信自己」呢類空泛意見
- 請編輯好貴、又要等

喺呢個情況，Claude 嘅用處好大。但要 **清楚劃好界線**：佢係陪練員，唔係代筆。

## 跟住做

### 1. 定義 Claude 嘅角色（關鍵設定）

第一條 prompt 嘅設定，會影響之後所有對話。如果設定得唔好，Claude 出嘅「草稿建議」會越嚟越多，最後**你會失去自己嘅寫作風格**。

<PromptBlock label="完整 prompt — 設定小說思考夥伴角色">
我寫緊一個 [短篇 / 長篇 / 劇本]：

- **類型**：[填]
- **目標長度**：[填]
- **階段**：[意念 / 大綱 / 初稿 / 修改 / 修飾]
- **視角**：[第一人稱 / 第三人稱貼身 / 第三人稱全知]
- **語氣**：[文學性 / 商業 / 黑暗 / 抒情 / 喜劇 / 等]
- **影響我寫作風格嘅作家**（我崇拜嘅）：[列 2-3 個]

請扮 **「靜靜傾偈嘅寫作夥伴」**，唔係編輯、唔係代筆。規則：

**你要做：**
1. **問問題去釐清意圖** —— 「你想讀者喺呢個章節完結有咩感覺？」
2. **反映返我做緊咩** —— 「我聽到你個角色 X 想 Y，但劇情入面嘅衝突似乎推佢去 Z」
3. **指出矛盾** —— 「第二章你確立咗 A，但第五章嘅鋪排假設咗唔係 A」
4. **建議方向，唔好寫死啲句子** —— 「呢度你可能想探索 X 嘅角度」

**你唔好做：**
1. **唔好幫我寫內文**——除非我特別講明「幫我草擬一句畀我比較」
2. **唔好用「好」/「壞」去評價**——用「呢個有冇達到你聲稱嘅意圖」呢種框架
3. **唔好引入重大劇情轉折**——除非我明確邀請
4. **唔好將我同著名作家比較**——「呢段似 [大師]」呢類評論冇幫助

我會：
- 分享我嘅大綱 / 草稿 / 具體段落
- 問具體問題（「呢個過渡順唔順？」）
- 偶爾發洩吓挫敗感

準備好嘅話，確認你明白你嘅角色，然後我會分享我而家卡住嘅地方。
</PromptBlock>

### 2. 三個常用模式

設定好之後，對話嘅典型模式：

**模式 A — 劇情 brainstorm**

<PromptBlock label="模式 A — 劇情 brainstorm">
我卡咗喺 [描述劇情點]。

我已經諗過：
- 選擇 1：[填]
- 選擇 2：[填]

但兩個都覺得唔滿意，原因：[你嘅直覺]

請：
1. 講清楚點解兩個選擇都唔好（用我之前講過嘅寫作意圖去判斷）
2. 點出 3 個我可能忽略咗嘅潛在張力
3. 提出 2-3 個我未諗過嘅新角度（唔好畀嗰啲廉價嘅「驚人轉折」）

唔好幫我寫內文。純粹 brainstorm 方向。
</PromptBlock>

**模式 B — 角色審問**

<PromptBlock label="模式 B — 角色審問">
我嘅角色 [名]：[簡單描述]

我覺得佢好平庸。請：
1. 問我 5-7 個關於呢個角色嘅具體問題（佢驚咩？佢每日下晝 5 點想做咩？佢有咩秘密？）
2. 我答完你之後，指出邊個答案觸發咗「呢個角色突然立體」嘅瞬間
3. 建議將邊個細節加入故事入面（具體放喺邊場戲）

唔好生成完整嘅角色傳記。你嘅角色係引導我諗深一層。
</PromptBlock>

**模式 C — 稿件點評（讀後反應）**

<PromptBlock label="模式 C — 稿件點評">
我分享一段 [章節 / 場景 / 1000 字]：

[貼上]

請用以下框架點評：

1. **讀者體驗**：第一句你會點樣繼續讀？你喺邊度停低？你會飛去邊度？
2. **意圖對齊**：呢段有冇達到我之前講過嘅章節意圖？
3. **技術問題**：時態一致性、視角出軌、代名詞唔清晰
4. **風格一致性**：邊度聽落似我之前分享過嘅樣本？邊度聽落唔似？
5. **建議**（按優先次序）：刪減 / 擴充 / 重組 / 唔好搞佢

⚠️ 唔好重寫我嘅內文——只係指出問題。
</PromptBlock>

### 3. 工作坊節奏

我嘅實際節奏：

- **第 1-2 個禮拜**：自己寫初稿。唔用 Claude（畀自己嘅寫作風格有慢慢確立嘅保護時間）
- **第 3 個禮拜**：第一次「模式 B 角色審問」環節
- **第 4-5 個禮拜**：寫稿，間中做 5-10 分鐘快閃「模式 A 劇情 brainstorm」
- **第 6 個禮拜**：第一次「模式 C 稿件點評」環節

唔好日日用。過分頻繁同 AI 對話 = 有失去你自己寫作風格嘅風險。

### 4. 紅線 — 你應該知

⚠️ **用 AI 去 brainstorm / 點評 = 可以**
⚠️ **用 AI 寫準備出版嘅內文 = 唔可以**

呢條界線喺出版界而家越來越嚴格：
- 大部分文學雜誌禁止「AI 共同創作」嘅投稿
- 大型圖書出版商出版時要求作者聲明
- 讀者越來越識得分辨 AI 生成文字嗰種「過分流暢」

你將佢當思考夥伴，你嘅文字 100% 係你自己嘅——呢條係老實嘅路。
直接用 AI 寫出嚟就發表——呢個係有問題嘅。你最終要對自己嘅作品保持誠實嘅態度。

## 變化例子

### 變化 1：奇幻 / 科幻嘅世界觀建構

<PromptBlock label="變化 1 — 世界觀陪練">
我建構緊 [奇幻 / 科幻] 世界。而家有：[簡述世界設定]

我卡喺世界觀建構：[具體問題 — 例：「呢個魔法系統嘅代價應該係咩」、「呢個社會嘅經濟點運作」]

請：
1. 點出 3 個我嘅設定可能會帶來、但我可能未諗過嘅影響
2. 提出 2-3 個「如果⋯⋯」嘅思想實驗去探索呢啲影響
3. 唔好畀「最終答案」我——保持開放嘅選擇

我自己會揀方向。
</PromptBlock>

### 變化 2：對白寫得生硬

<PromptBlock label="變化 2 — 對白點評">
我啲角色之間嘅對白，自己讀返覺得生硬 / 唔自然。

呢度有 [N] 段對白交流：

[貼上]

請：
1. 朗讀測試 — 邊段聽落唔似真人講嘢
2. 搵出規律（每個角色語氣一樣？資訊密度太平均？缺乏潛台詞？）
3. 建議我可以加入嘅唔同對白技巧（打斷、誤解、平行獨白）
4. 唔好逐句重寫——指出規律，等我自己修改
</PromptBlock>

### 變化 3：卡關 2 個禮拜，唔知點重新開始

<PromptBlock label="變化 3 — 寫作障礙診斷">
我寫小說卡關咗 [N 週]。唔係「冇時間」嘅卡——係坐低打字都打唔出。

請問我：
1. 我而家最近睇過咩好書
2. 我個故事最後寫得順嘅段落喺邊度
3. 我預期下一段點發展，但喺邊度遇到阻力
4. 我情感上對角色仲有冇投入

根據我嘅答案，搵出最有可能嘅原因（結構問題 / 角色問題 / 個人情緒 / 過度計劃），建議一個低門檻嘅重啟行動。
</PromptBlock>

## 一個真誠嘅備註

小說作家用 AI 嘅道德問題而家不斷爭論緊。我自己堅守嘅底線：

**AI 可以幫我諗。AI 唔應該替我寫。**

呢條線合理但模糊。你會同自己嘅良知交戰：「我分享個大綱，AI 出章節標題—— 算唔算共同創作？」

冇一個絕對嘅答案。每個作家要搵到自己嘅界線。但**至少願意去衡量呢條界線，代表你仍然擁有你嘅手藝**。停止反思呢條界線，AI 就會靜靜雞取代你。

呢個用法嘅最終價值：保持你嘅思考敏銳、保持你嘅手藝係自己嘅、保持寫小說嘅個人承擔。

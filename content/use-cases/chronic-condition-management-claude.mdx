> ⚠️ Claude 唔取代醫生。文中所有框架係自我追蹤工具，目的係準備你下次 GP 探訪帶有用數據。任何藥物 / 劑量 / 治療決定 = 只係醫生。

## 情境

你 GP 探訪 3 個月 1 次。每次 15 分鐘。

GP：「過去 3 個月血糖點？」
你：「⋯⋯OK 啦。上禮拜 fasting 6.2。」
GP：「平均點？」
你：「⋯⋯冇 record 嘅。Forgot。Lately seems OK。」
GP：「病徵點？」
你：「⋯⋯啱啱 4 月嗰陣有時攰。5 月 OK。」
GP 點頭，重新處方同樣藥物，安排下次 3 個月探訪。

呢個慢性病管理嘅隱形損失：
- **90 日你個腦記住 7 個數據點**（上週 + 重要 spike）
- **180 日你忘記 80% 趨勢**
- **GP 探訪基於你口頭報告 = 實際信號 10%**
- **藥物 / 劑量調整冇可能咁精準**

你藥物而家可能啱，但可能唔啱。3 個月後你 fasting 突然 7.5 → GP 升級劑量 50%。如果你有數據顯示實際漸進上升 + 2 個具體觸發事件 → GP 可能改 25% 而唔係，或者調整食飯時間。

呢篇用 Claude **每月慢性病紀錄**——專門設計：

1. 30 分鐘 / 月維護（可持續）
2. GP 可讀總結喺探訪時
3. 浮現趨勢 / 觸發 / 規律（你 missed）
4. 賦予你同 GP 同等嘅資訊

覆蓋 3 大常見 HK 慢性病：糖尿（Type 2）/ 高血壓 / 哮喘。框架通用 — 改畀甲狀腺 / IBS / 經痛 / 慢性痛 / 自身免疫都用得。

### 3. 每月檢視 + 趨勢識別

每月 1 次 30 分鐘檢視：

<PromptBlock label="完整 prompt — 每月慢性病檢視">
我完成 [N] 月每日紀錄。請每月檢視。

📊 **貼月度數據總結：**
[每日關鍵指標值、藥物遵從 Y/N、顯著事件]

📈 **需要嘅分析：**

**1. 指標趨勢（主要指標）**
- 月度平均 vs 目標
- 第 1 週 vs 第 4 週趨勢
- 變異（一致還是波動？）
- 最高 / 最低讀數 + 咩情況
- 同上月比較（如有）

**2. 藥物遵從度**
- 朝劑量漏食率
- 晚劑量漏食率
- 規律（例：旅行時永遠漏）？
- 調整建議？

**3. 觸發規律識別**

For 糖尿：
- 餐後 spike 相關性：[具體食物 / 時機 / 運動]
- 壓力水平 vs 血糖相關性
- 睡眠質素 vs 空腹血糖相關性

For 高血壓：
- 時段規律（朝高 / 晚高）
- 星期幾規律（工作日 vs 週末）
- 鈉攝取 vs 讀數
- 睡眠質素 vs 讀數
- 咖啡因時機

For 哮喘：
- 病徵觸發常見（過敏原 / 凍 / 運動）
- 急救吸入器用趨勢
- Peak flow 趨勢
- 環境因素（濕度 / 工作環境）

**4. 生活習慣因素相關性**
- 運動日 vs 指標：較好 / 較差 / 一樣？
- 壓力高日 vs 指標相關性
- 睡眠質素 vs 指標相關性

**5. 令人擔憂嘅趨勢（需要 GP 關注）**
- 任何指標持續超過門檻？
- 任何藥物副作用浮現？
- 任何新病徵？

📋 **輸出：畀 GP 探訪嘅月度總結**

將我嘅數據格式化成 1 頁總結我可以畀 GP 睇：
- 頂線指標趨勢
- 主要識別咗嘅觸發規律
- 畀 GP 嘅具體問題（3-5 條，排序）
- 藥物遵從老實報告
- 任何擔憂想明確討論

🚨 **升級檢查**
基於月度數據，我應該安排計劃外探訪？
- 任何讀數超過門檻連續 3+ 日：yes
- 之前未討論嘅新病徵：yes
- 任何藥物副作用令人擔憂：yes
- 否則：定期探訪 OK

語氣：數據導向、可行動、尊重臨床自主。
</PromptBlock>

每月檢視嘅隱藏價值：**識別你每日太沉浸睇唔到嘅觸發**。第 1 個月可能噪音，第 2-3 個月規律浮現。

### 4. 每季 GP 探訪準備

3 個月探訪前 1 個禮拜，整合 3 個月嘅每月檢視：

<PromptBlock label="完整 prompt — 每季 GP 探訪準備">
我下次 GP 探訪喺 [日期]。過去 3 個月每月紀錄。請整探訪準備。

📋 **Print 1 頁 handout（畀 GP）**

**頂部：**
- 我嘅病情 + 藥物方案
- 診斷距今多耐

**指標圖（過去 90 日）：**
- 主要指標趨勢（貼數據）
- 平均 / 範圍 / 變異
- 同上季比較
- vs 臨床目標

**識別咗嘅規律：**
- Top 3 觸發規律我哋留意到
- 生活習慣因素相關性
- 令人擔憂嘅趨勢（如有）

**畀 GP 嘅具體問題（3-5 條）：**
- 「[具體病徵] 持續 [Y] 週，同 [規律] 相關。調整治療？」
- 「藥物 X 副作用 [具體]，揀其他選項？」
- 「驗血應該頻率？」
- 「[生活習慣改變 A] 可行畀 [改善 Z]？」

**對醫生之前建議嘅 update：**
- 建議：[X]
- 我遵從：[%]
- 結果：[結果]

⏰ **探訪後勤**
- 15 分鐘探訪，頭 2 分鐘出示 1 頁總結
- 留 5 分鐘畀問題
- 唔好嘗試解釋一切 — 等 GP 檢視總結

🎯 **要帶咩**
- 1 頁總結 print
- 當前藥物樽（或清單）
- 保險咭
- 過往 lab 報告（最近 3 次探訪）
- 任何新病徵紀錄

💊 **藥物討論準備**
- 任何副作用要 flag
- 任何原因想減少 / 停 / 轉
- 補藥時間表 check

📅 **離開前安排下次探訪**
- 唔好「我之後 call」 — 出去前 book
- 早 book 確保下個 3 個月連續

🚨 **唔好**
- 隱瞞唔遵從（GP 如知道可以調整）
- 自己停藥唔講
- 自己診斷新病情
- 同朋友 / 家人治療比較

語氣：有準備嘅夥伴關係，唔係屈從。
</PromptBlock>

呢個準備嘅關鍵領悟：GP 探訪唔係你「report 完問題」，係合作 session。你帶數據準備好 = GP 有真信號可以做嘢 = 治療更精準。

## 變化例子

### 變化 1：多病情（共病）管理

如你同時管2-3 個病情：

<PromptBlock label="變化 1 — 多病情管理">
我同時 manage [糖尿 + 高血壓 + 高血脂 / 等] [N] 個病情。請整整合追蹤。

🎯 **多病情挑戰**
- 互動效應（例：糖尿加高血壓增加心血管風險）
- 藥物相互作用
- 唔同專科 / 唔同時間表
- 生活習慣因素重疊（飲食影響多個病情）
- 認知負擔高（追蹤 5+ 個每日指標）

📊 **整合紀錄設計**

```
每日（1.5 分鐘）：
- 空腹血糖朝
- 血壓朝 + 晚
- Peak flow（如都有哮喘）
- 膽固醇飲食遵從 Y/N
- 藥物朝 + 晚 Y/N（所有藥物）
- 任何病情顯著病徵
```

📅 **聯合探訪考慮**
- 1 個 GP 協調所有病情（偏好）
- 或專科級協調
- 帶全部數據 1 次探訪

💊 **藥物相互作用 check**
- 藥劑師年度檢視（免費服務）
- 全部藥物帶 1 個袋
- Check OTC + 補充劑相互作用

🍽 **飲食整合**
- 1 個飲食計劃處理多個病情
- 地中海 / DASH 兼容糖尿 + 心健康 + 高血壓
- 唔好試 3 個唔同飲食同時

🏃 **運動整合**
- 中等強度可持續所有病情
- 行路 + 輕阻力通常普遍安全
- 高強度受限（具體病情）

🧠 **心理健康重疊**
- 慢性病 + 抑鬱風險提升
- 追蹤心理健康指標同時
- 接觸治療師 / 輔導員做 network 一部分

🚨 **紅燈整合**
- 1 個病情病徵可能信號其他病情
- 例：糖尿 + 突然 BP spike = 心血管檢視

語氣：全面但唔壓倒。
</PromptBlock>

### 變化 2：女性月經 / 荷爾蒙相關追蹤

女性荷爾蒙週期影響慢性病：

<PromptBlock label="變化 2 — 女性週期意識慢性追蹤">
我女性，[病情 X] + 月經週期。請週期同步追蹤。

📐 **週期階段對常見病情嘅影響**

🩸 **月經期（Day 1-7）**
- 痛 / 抽搐打亂其他病情管理
- 心情波動
- 血糖波動（胰島素敏感性變化）
- 哮喘惡化（雌激素下跌）

🌱 **濾泡期（Day 7-14）**
- 精力高
- 血糖控制較易
- 哮喘普遍穩定

🥚 **排卵期（Day 14-16）**
- 輕微體溫上升
- 血糖控制穩定
- 部分哮喘週期中期惡化

🌖 **黃體期（Day 16-28）**
- 胰島素抵抗增加
- 血糖趨勢上
- PMS 疊加（心情、精力、食慾）
- 哮喘月經前惡化

📊 **整合追蹤**

```
標準紀錄之外，每日加：
- 週期日
- 估計階段
- 心情 / 精力
- 週期相關病徵
```

📈 **每月 + 週期檢視**
- 指標 vs 週期階段規律
- 具體階段對病情治療唔同？
- 月經前發作一致？

⚕️ **同 GP 討論**
- 如明確週期相關性：討論週期意識調整
- 例：糖尿黃體期胰島素劑量輕微改變
- 例：哮喘月經前預防吸入器

🚨 **週期紅燈**
- 唔規律週期（3+ 個月缺失）：GP 探訪（PCOS / 甲狀腺可能）
- 嚴重經痛（經痛致殘）：GP
- 週期中期出血：GP

語氣：尊重週期做數據維度。
</PromptBlock>

### 變化 3：兒科 / 細路病情（家長管理）

家長 manage 細路病情（兒童哮喘 / Type 1 糖尿 / 食物過敏）：

<PromptBlock label="變化 3 — 家長管理兒科病情">
我 manage [小朋友] [N 歲] 嘅 [病情]。我想追蹤 + GP 探訪準備。

🧒 **兒科追蹤特性**

📋 **家長紀錄（你填，唔係小朋友）**

```
每日（2-3 分鐘）：
- 病徵：[列]
- 藥物：朝 + 晚 + 按需 Y/N
- 今日遇到嘅觸發：[列]
- 睡眠質素 1-10
- 食時間表
- 運動 / 活動水平
- 學校 / 日託報告（任何事件）
- 任何不尋常心情 / 行為
```

🏥 **針對常見兒科**

**哮喘（兒童最常見慢性）**
- 觸發追蹤：寵物 / 塵 / 凍 / 運動 / 凍天氣 / 病
- Peak flow 如 6+ 歲 + 合作
- 急救吸入器用次數
- 急症室探訪 / 住院 Y/N

**濕疹（atopic dermatitis）**
- 發作位置 + 嚴重度
- 觸發食物 / 環境
- 保濕霜遵從 Y/N

## 一個思維框架

慢性病管理嘅死結：你係病情嘅每日專家，但 GP 係醫療決定權威。

呢個死結嘅源頭：GP 探訪 15 分鐘 / 3 個月 = 你「病情真實經驗」時間嘅 0.0017%。你 90 日 × 24 小時 = 2160 小時嘅真實經驗，GP 攞 15 分鐘。

呢個思維嘅轉變：你每月紀錄唔係「自我治療」 — 係 **將 0.0017% 信號可達性擴大畀 GP**。GP 而家用 2 頁數據工作而非 30 秒口頭報告。治療精準度顯著增加。

Claude 喺慢性病管理嘅角色：你獨自追蹤可持續少於 6 個月（認知負擔太重）。Claude 容器 + 結構化分析 = 6 個月 → 5 年可持續。你每日倒數據，Claude 每月反映規律，你每季同 GP 簡報。

最後提醒：

- ✅ **老實唔遵從 > 捏造遵從**。GP 根據你實際做咩調整，唔係你「應該」做咩。
- ⚠️ **藥物副作用 > 你以為係「自然老化過程」**。任何新病徵喺藥物開始後浮現 = flag GP。
- 🆘 **超過門檻嘅讀數**（例血糖隨機超過 15 mmol/L，BP 超過 180/110，peak flow 低於個人最佳 50%）：**唔好等下次定期探訪。即刻 call / 緊急護理**。

90 日後你 GP 探訪帶住數據出現。GP 第 1 次見你準備充分嘅病人 — 對治療軌跡改善長期顯著。

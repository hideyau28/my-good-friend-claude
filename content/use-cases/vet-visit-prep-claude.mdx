## 情境

你隻貓主子過去 3 日：

- 食量下降（平時食 80g，今日得 30g）
- 屎變硬，少咗
- 成日匿喺床下底唔肯出嚟

你 google「貓食量下降屎硬」——搜到 50 條 forum post：肝病？腎衰？毛球？壓力？食物中毒？牙痛？由「食壞嘢觀察一日」到「即刻送 ICU」全部都有。

你約咗下午 4:30 獸醫。**得 15 分鐘問診。** 你帶主子入去飛機籠，醫生問你：

> 「主子幾時開始唔對勁？」

你諗：「嗯⋯⋯應該禮拜一？定係禮拜日夜？」

> 「食量大概減咗幾多？」

「⋯⋯一半？三分二？我冇數過。」

> 「屎變硬係邊一日？」

「⋯⋯ um 唔記得。」

你揾錢、揾時間、揾交通帶主子去診所。**結果 15 分鐘入面有 8 分鐘係你喺度回想啲記憶**，醫生只能根據你一忽忽嘅描述開檢查。檢查結果幾時返到，就睇你個底有幾完整。

呢篇用 Claude **10 分鐘**將你雜亂無章嘅觀察整成一份畀獸醫睇得明嘅簡報——病徵時序、量化咗嘅變化、必問嘅 3-5 條問題、可能嘅鑑別診斷。

⚠️ Claude 唔係取代獸醫——但**唔好自己斷症**。Claude 嘅角色係整理你嘅觀察 + 教你問啱嘅問題，等獸醫嘅 15 分鐘最有效。

## 跟住做

### 1. 寫低過去 3-7 日最原始嘅觀察

打開 Claude Chat。**唔好就咁問「我貓有事點算」**——先一口氣打晒你啲原始筆記出嚟。語氣亂都無所謂：

> 主子 8 歲 公貓（已絕育）短毛貓 4kg。
> 
> 禮拜六晚我返到屋企（佢平時會跳落沙發迎接我），冇出嚟。我搵到佢喺床下底。摸佢冇反應，但冇明顯傷。
> 
> 禮拜日朝佢自己食咗少少乾糧，跟住瞓咗一日。
> 
> 禮拜一我返工出門，碗仲有大半。返到屋企水位低咗少少（平時飲乾），食碗大半樣。
> 
> 禮拜二（今日）朝 30g 都唔食晒，屎盆只有 1 粒小硬屎。
> 
> 平時：每日 80g 乾糧 + 1 罐濕糧。每日 2-3 砵屎水正常。

呢類時序筆記，就係 Claude 整獸醫簡報嘅原材料。

### 2. 用呢個 prompt

<PromptBlock label="完整 prompt — 整獸醫問診準備">
我主子（[簡單描述：種類 / 年齡 / 體重 / 公或乸 / 已絕育或未 / 室內或室外 / 有冇慢性病]）有以下症狀，下午 [時間] 約咗獸醫。

我嘅原始觀察：
[貼上你寫嘅時序筆記]

過去病史 / 疫苗：
- 上次睇獸醫：[幾時，原因]
- 慢性病：[填，例：FIV+ / 慢性腎病第 2 期 / 冇]
- 食緊嘅藥：[填]
- 最近一次打針：[幾時]

請整一份 **畀獸醫睇得明嘅簡報**（A4 1 頁內）：

**📋 1. 快速摘要（醫生睇 30 秒入腦嘅 1-2 句）**
今次主訴 + 嚴重度估計

**⏱ 2. 病徵時序（逐點寫，按日子排）**
每日寫:
- Day -X (yyyy-mm-dd 星期)
  · [症狀 1] — 可以嘅就量化（食量、屎量、體重、活動量等）
  · [症狀 2]
用紅色標示關鍵警號（例如：開始唔食 / 第一次屎變硬）

**🩺 3. 鑑別診斷（可能性清單 — 僅供參考）**
3-5 個可能成因，每個：
- 病名 + 1 句點解符合我主子嘅情況
- 接下來通常會做嘅檢查（驗血 / X-ray / 超聲波 / 驗尿）
**重要：呢個係幫你整理思路，唔係診斷。最終由醫生決定。**

**🎯 4. 5 條必問問題（按優先次序）**
你入到診室嘅頭 5 分鐘最值得問嘅嘢，例如：
1. 你嘅鑑別診斷頭三位係咩？
2. 今日要做邊啲檢查，先排除到最危險嗰啲？
3. 我帶主子返屋企應該留意邊啲徵狀？
4. 如果今晚情況惡化，邊間 24 小時急症你會推薦？
5. 預算 / 大概收費？

**💊 5. 我可能要主動問嘅嘢**
獸醫忙嘅時候未必主動講，要我自己問：
- 副作用 / 藥物相互作用
- 替代治療方案（例：口服 vs 打針）
- 幾耐覆診一次
- 保險索償需要嘅文件

**🚨 6. ⚠️ 警號 — 即送急症嘅徵狀**
今晚我返屋企應該留意嘅惡化徵狀（呼吸困難 / 持續嘔吐 / 企唔起身 / 牙肉變白 等）

語氣：好似一個做獸醫助護嘅朋友幫你準備，唔好說教，唔好堆砌行貨「帶寵物去獸醫」嗰啲廢話。
</PromptBlock>

### 3. 列印 / 存喺電話開住

出嚟通常一頁。我嘅做法：

- **iPhone**：分享落 Apple Notes → 入診所開大個螢幕，邊講邊對住睇
- **列印**：A4 1 張，獸醫好多時會欣賞你有準備好
- **email 畀自己**：睇完之後加返獸醫嘅回應入同一份檔，建立寵物病歷

### 4. 睇完 → 更新紀錄

睇完之後 2 分鐘打返獸醫講嘅嘢：

> 「獸醫確診 [病名]，開咗 [藥]，叫我 follow up [幾耐]。」

Claude 幫你更新紀錄，下次睇獸醫就有長期脈絡（對有慢性病嘅老貓老狗尤其重要）。

## 變化

### 變化 1：拍片 / 影屎樣本加描述

獸醫好多時要睇屎 / 嘔吐物 / 分泌物嘅樣。你可以**截圖 + 貼入 Claude 幫你描述**：

<PromptBlock label="變化 1 — 屎 / 嘔吐物 / 分泌物描述助手">
我主子嘅 [屎 / 嘔吐物 / 分泌物] 樣，影咗張相附咗喺度。

請幫我用醫療角度描述呢個樣，畀獸醫睇我嘅紀錄一眼睇得明：

**外觀**
- 顏色（用色碼或常見描述：焦油黑 / 咖啡渣色 / 粉紅 / 奶黃 等）
- 質地（液體 / 軟身成形 / 硬粒 / 帶黏液 等）
- 份量估計
- 有冇雜質（血 / 毛 / 未消化食物 / 寄生蟲 形狀）

**規律**
- 邊一日第一次出現
- 出現頻率
- 伴隨徵狀（嘔之前有冇乾嘔 / 屎前有冇用力）

**鑑別診斷指向**
呢個描述喺獸醫嘅鑑別診斷入面通常指向邊類問題（胃 / 下消化道 / 肝 / 泌尿 / 等）— 僅供參考

⚠️ 注意：我只係整理觀察，最終診斷一定要獸醫確認。
</PromptBlock>

### 變化 2：對比新舊獸醫 / 第二意見

如果你考慮緊換獸醫或者想攞 second opinion：

<PromptBlock label="變化 2 — 第二意見準備">
我主子喺 [現任獸醫] 度確診咗 [病名]。治療方案：
- [藥 / 治療 1]
- [藥 / 治療 2]
- 覆診: [幾耐 / 收費]

我考慮搵 [另一間獸醫] 攞第二意見。

請整：

**背景簡報（第二意見醫生需要嘅脈絡）**
1 段講主子嘅病史 + 現病史 + 現時治療方案

**5 條我應該問第二意見醫生嘅問題**
唔淨係要佢確認或推翻診斷，仲要問：
- 你嘅鑑別診斷同第一位醫生邊度唔同
- 有冇其他治療方案，各有咩取捨
- 檢查結果點解讀（如果第一位醫生有化驗報告）
- 預後嘅時間預期
- 如果你係主人，你會點處理

**準備帶嘅文件**
- 化驗報告
- X-ray / 超聲波
- 食緊嘅藥
- 保險表格（如有）

**禮節貼士**
要唔要同第一位醫生講你去攞第二意見，香港獸醫圈子細，唔好搞砸關係。
</PromptBlock>

### 變化 3：慢性病嘅長期追蹤

如果你主子有慢性病（CKD / 糖尿 / 甲狀腺亢進 / IBD 等）：

<PromptBlock label="變化 3 — 慢性病長期追蹤">
我主子確診咗 [慢性病]，現時第 [X] 期。

過去 6 個月化驗數值：
- 月份 1：[BUN / Creatinine / SDMA / T4 / 等，視乎病]
- 月份 2：[]
- 月份 3：[]
...

現時治療：
- [藥 + 劑量]
- 飲食：[處方糧名]
- 補充：[補充劑]

請整：

**趨勢分析**
- 邊個指標改善 / 穩定 / 惡化
- 同分期標準對比，我宜家惡化得幾快

**下次睇獸醫嘅議題**
- 邊個指標應該繼續追蹤
- 有冇劑量調整應該提出
- 有冇新徵狀我應該講

**生活調整**
- 飲食 / 運動 / 壓力管理嘅微調
- 屋企監測指標加 / 減（例如：每週體重 / 飲水量 / 屎尿次數）

**生活質素評估**
- 5 個生活質素指標評估，幫我決定下一步係積極治療 vs 紓緩照顧
- ⚠️ 如果生活質素低，獸醫可能會提出紓緩治療選項——我應該點喺心理上準備

語氣：有同理心嘅獸醫護士，唔好太機械化。
</PromptBlock>

## 一個心態

獸醫問診比 GP 短、收費高、次數少——但件事對你屋企嘅重要性一樣咁大。

唔好以為「醫生會問返我」就唔使準備。獸醫一日睇 30 隻動物，佢嘅腦力負荷已經爆晒。**你有準備好 = 佢可以專注喺臨床判斷，唔使浪費時間掘你啲記憶。**

最後提提你：

- ✅ Claude 嘅角色係整理觀察 + 準備問題 + 睇完之後整理紀錄
- ❌ Claude 唔係幫你斷症、唔係開藥、唔係推銷另類療法
- 🆘 如果主子出現急性症狀（呼吸困難 / 痙攣 / 企唔起身 / 持續嘔吐血），**直接送急症**，唔好用 Claude 拖時間

主子健康，家人安心。

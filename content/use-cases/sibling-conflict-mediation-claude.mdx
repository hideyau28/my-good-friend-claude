## 情境

5 月 16 日，星期六，9:47am。

過去 1 小時 你 brokered 3 場 fight:
- **8:12am**: 8 歲仔 Eric 玩緊 iPad, 11 歲女 Ariel walk by, 「不 small touch」 iPad screen. Eric scream. Ariel scream. 你 brokered 5 分鐘 walked away.
- **8:51am**: Cereal box 1 個剩, 兩個 同時 want. Negotiation collapse. Eric throw spoon. Ariel cry. 你 cut up 2 portion, brokered 5 分鐘 walked away.
- **9:30am**: Eric using shared family iPad, Ariel wants. Ariel: 「My turn!」 Eric: 「I just started!」 Ariel: 「Mom never enforce time!」 Eric: 「You played longest yesterday!」 你 brokered 8 分鐘 walked away.

你 9:47am sit down with coffee. Brain 嘅 mental thread:

- **「我 反 而 amplify conflict by mediating?」** — Theory: by 介入, you signal「conflict matters」, 鼓勵 future
- **「我 唔介入 = 細路 develop self-resolve skill?」** — Theory: parents stay out, 細路 learn
- **「我 介入 will dilute parental authority?」** — Theory: 每場 mediation is precedent
- **「我 唔介入 = Ariel will overpower Eric long-term?」** — Theory: age gap means weaker sibling gets steamrolled

呢個 sibling conflict paradox:

- **過度 介入**: 細路 唔 develop 自己 resolve skill + you exhausted
- **零 介入**: power imbalance creates resentment + injury risk
- **Inconsistent intervention**: 細路 read「parent random」 + 沒 trust
- **過 strict equal**: 「each take turn equally」 ignore developmental difference
- **過 lenient**: 大 仔 / 大 女 become bully

What 你 actually need: **systematic framework 區分 when 介入 + when stand back + how to intervene without making it worse**.

呢篇用 Claude **拆 conflict pattern + 提 出 specific intervention type + 家長 own emotional regulation**. Day 30 frequency 跌 30%, severity 跌 50%.

### 2. 用呢個 prompt 識別 「介入 vs leave」 framework

<PromptBlock label="完整 prompt — Intervention vs leave framework">
[paste 你 sibling dump]

請整 framework — 邊類 conflict need 介入, 邊類 leave them:

🚦 **Intervention level matrix**

🔴 **Level 1: Immediate intervention (always)**
- Physical hitting / kicking / biting
- Throwing object 危險
- Lock door trap sibling
- Real threat to safety
- Mental health attack:「I wish you were dead」/「you're a [slur]」
- 應 action: 即 stop, separate, 1-on-1 with each later

🟡 **Level 2: Pause and assess (sometimes)**
- Verbal escalation but no physical
- 1 細路 in tears
- 衝突 lasting >10 minutes already
- Same conflict repeating 3rd time today
- 應 action: 1 minute觀察, 介入 if 不 self-resolving

🟢 **Level 3: Stay out (most often)**
- 平 verbal disagreement
- Negotiation underway (even if loud)
- Frustration sound but no escalation
- Quick conflict + recover within 5 分鐘
- Game-like teasing (each smile despite「fight」)
- 應 action: walk away, 「我 in next room if 你 need」

🛑 **Level 4: Don't ever do**
- Force apologize / 強迫 hug
- 「You're older, you give in」
- Compare:「Why can't you be like ___」
- 1 cause 嘅 嘢 完全 remove punitive (vs neutral remove)
- Long lecture during heated moment
- Take side without hearing 兩 個
- 介入 公開 (public humiliation)

📋 **Decision tree**

聽到 fight noise:
1. **Within 10 sec: physical?** Yes → Intervene immediately. No → continue
2. **Verbal but tone increase?** Yes → 30 sec listen pause. No → walk away.
3. **30 sec listen — escalating?** Yes → 介入. No → 1 minute pause.
4. **1 minute later — resolved?** Yes → walk away (細路 self-resolved!). No → enter Level 2.

🎯 **Type of intervention by level**

**Level 1 (immediate)**
- Calm tone:「STOP.」
- Separate physically (your 兩 hand)
- 不 immediate punish — separate first
- Each goes own room for 5 分鐘 cool
- Re-engage when both regulated

**Level 2 (negotiated)**
- Sit between 兩 個
- 1 person speak 1 minute uninterrupted
- 另 1 person reflect:「You said [paraphrase]」
- Switch
- Question to 兩 個:「What's the actual problem here?」
- 細路 propose 2 solution
- Implement 1
- 不 force satisfaction outcome

**Level 3 (active non-intervention)**
- 不 hovering — actual walk away (kitchen / 另 room)
- 信 細路
- 細路 know 你 reachable but 不 watching

🔄 **Pattern interrupt 高 frequency conflict**

If conflict 3-5x daily:
- Identify hot zone (iPad / cereal / pre-dinner hunger)
- Adjust environment (cereal stock 2 boxes, iPad time slot defined)
- 預 預 預 — anticipated conflict often preventable
- Pre-conflict routine establish

🚨 **Red flag (deeper issue)**
- 1 細路 consistently aggressor + 1 consistently victim
- Conflict 嚴重 escalation severity 持續 增加
- 1 細路 withdraw, 不 fight back (concerning)
- Sleep disturb / refuse meal / anxiety
- 1 細路 self-harm thought

= 個 別 professional consult, 不 sibling 「they'll work it out」.

語氣：framework-based, not knee-jerk.
</PromptBlock>

Claude 出 framework. Print 出嚟. **頭 1 個禮拜 strict 跟** — 你 brain 默 react, 框 架 強迫 1 second pause assess.

### 3. Long-term: 建立 sibling relationship outside conflict

Mediation 解 short-term, 但 long-term sibling relationship 需要 separate cultivation:

<PromptBlock label="完整 prompt — Long-term sibling bonding">

請設計 long-term sibling positive interaction structure (over months):

📅 **每週 ritual**

🤝 **1-on-1 sibling time (parent-facilitated, optional initially)**
- 1 hr / week brother-sister activity
- 細路 自己 揀 activity (Lego together / 烘 bread / 玩 board game)
- Parent steps back after setup
- 不 force — encourage可能 success rate ↑ if low-pressure

🎮 **Collaborative game / project**
- 兩 個 against 1 challenge (vs each other)
- 「Build this Lego set together」
- 「Make this recipe together」
- Sibling 變 ally 短暂

🍳 **Helping each other**
- Older 教 younger 1 skill (Ariel teach Eric chess move)
- Younger help older 1 task (Eric carry stuff for Ariel)
- Cross-age contribution: 兩 個 see other's value

📅 **每月**

🏞 **Joint outing (without other distraction)**
- 1 outing 1 month brother-sister (you parent 1-on-1)
- 沒 phone, 沒 other family member
- Both feel 「special」

🎁 **Gift each other**
- 細路 buy / make 1 thing for sibling
- 你 facilitate budget / craft / shopping
- Small but 重要: love expressed concrete

📅 **每年**

🎂 **Sibling celebration**
- Each sibling birthday: other publicly recognize 「what I appreciate about [sibling]」
- 1 sentence each
- Build verbal positive trait habit

🧠 **Long-term mindset shifts**

🔁 **Stop comparison**
- 「Why can't you be like sibling」 = relationship poison
- Each celebrated for unique
- Praise 不 「same metric」

🔁 **Reframe sibling as ally**
- 「We're a family team」 framing
- 你 哋 sibling brand: 「Eric + Ariel」 unit
- 公 vs 私 dynamic 強 (vs 不 same family team)

🔁 **Permission to love different**
- 細路 兩 個 different personality
- 唔 expect equal closeness
- Some sibling close, some sibling like 「business」 — both OK long term

🌱 **The 80/20 reality**
- Sibling rivalry universal
- Adult sibling reconciliation common
- Childhood conflict 不 predict lifelong distance
- 但 childhood positive ritual increase long-term closeness

🎯 **Success metric (long view)**
- Adulthood: each call other 偶爾 (1+ time / month)
- Each show up for each other crisis
- Each prefer 1-2 hr together over not
- 唔 「best friend」 expectation — sibling 不 friend

⚠️ **Warning sign 不 normalize**
- Long-term aggression (1 child consistently 強壓 another)
- Abuse pattern (sexual / emotional / physical 真)
- Mental health impact on weaker sibling
- 1 細路 永遠 「withdrawn」 around the other

= seek professional family therapy

語氣：long-term cultivation, 不 quick fix.
</PromptBlock>

### 4. 你 own emotional regulation (parent side)

家長 own state 影響 conflict outcome 多 過 細路 dynamic:

<PromptBlock label="完整 prompt — Parent emotional regulation in sibling conflict">
[paste 你 sibling dump]

我 default state when fight starts: high arousal. 唔 calm. 請 design own regulation protocol:

🌡 **Pre-conflict state check**

Question 你 ask 自己 (3 second when hear fight start):
1. **我 own state right now**: tired / hungry / stressed / OK?
2. **我 last meal**: 1 hr ago / >3 hr ago (hangry parent = bad mediator)
3. **我 sleep**: >7 hr / less?
4. **我 anxiety baseline**: peaceful day / chaotic day?

If 1-2 indicate: 你 not optimal state.
- Quick: drink water, take 3 deep breaths, 1 min sit before respond
- Long: pre-need parent self-care (本 文 1 個 reminder)

🧘 **Immediate de-escalation (you, not 細路)**

When fight starts:
- 5 second pause
- 1 deep breath
- 唔 raise own voice (mirror effect)
- 不 jump to 「fix」 immediately
- 1 second internal question:「Level 1, 2, or 3?」 (從 framework)
- Then respond

📋 **Your own anti-default rules**

- 不 raise voice (細路 mirror)
- 不 sarcasm (細路 see)
- 不 ultimatum during heated moment (effective 0)
- 不 「we'll talk about this when dad comes home」 (delay 無用)
- 不 take side without hearing 兩 個
- 不 force apologize (insincere)
- 不 forbid emotions「don't cry」

💧 **Hydrate + 餓 management**
- 你 own water 大 cup nearby (always)
- 你 own snack stocked (avoid hangry-mediating)
- 細路 hangry trigger: pre-empt with snack 4pm-5pm

🛋 **「Mom in the corner」 strategy**
- Set 1 「mom's spot」 (你 own chair / corner)
- 「I am sitting here while 兩 個 work it out」
- 不 hover, but visible
- 細路 know 你 backup but 不 immediate intervene
- Most conflict 5 分鐘 self-resolve if 不 immediate parent fix

⏰ **Cool-down 你 own**
- Post-conflict 30 分鐘 你 own quiet time (if possible)
- 唔 dwell on what happened
- 不 share with partner 即 (let cool)
- Don't text friend immediately
- 1 cup tea / walk / read 10 分鐘

🤝 **Partner alignment**
- 1 weekend night both parents 1 hr review week's conflict
- 不 in front of 細路
- Discuss intervention philosophy
- Each support 對方 mediation approach (細路 detect inconsistency)
- 不 1 parent「good cop」 1 parent「bad cop」

😢 **When you lose it**
- Happens. 你 human.
- Within 24 hr, 個 別 apologize: 「I yelled earlier. That wasn't fair. I'm working on it.」
- 不 over-apologize / self-flagellate
- 細路 learn: parent makes mistake, owns it, models repair

🚨 **Parent burnout sign**
- 持續 dread fight time
- 想 cry frequently
- 心 baseline rage
- Self-medicate (alcohol / smoke / scroll)
- Resent 細路

= self-care escalation. Counsellor / therapist visit recommended. 細路 conflict 不 cause, but 反 映 parent state.

語氣：reminder 家長 own state matter most.
</PromptBlock>

## 一個 mindset

Sibling conflict 嘅 paradox: **你 介入 越多， 細路 嘅 conflict resolution skill 越弱**.

呢個 paradox 嘅 mechanism: 細路 brain 學 emotional regulation + negotiation through (controlled) friction. Parent 「solve」 every conflict = remove the learning opportunity. 細路 grow up dependent on external mediator.

呢個 mindset 嘅 reframe: 你 sibling conflict job 不係 「prevent all conflict」 — 係 **provide structure within which conflict 健康 to occur**. Level 1 你 intervene (safety). Level 3 你 walk away (skill development). Level 2 你 negotiate teach.

Claude 嘅 role 喺 sibling mediation: 你 own emotional default reactive. Claude 提供 framework distance — 你 dump scenario, Claude reflect Level X, suggest specific intervention. 30 個 framework iteration → 你 own brain internalize → 你 default response shift.

最後 reminder：

- ✅ **Sibling 鬥 嘴 不 normal — 係 universal**. Concern 嘅 不 frequency, but escalation pattern.
- ⚠️ **強迫 「sorry」 / 「hug it out」 = relationship poison**. 細路 learn perform reconciliation, not真 do.
- 🆘 **Bullying pattern (1 victim 1 aggressor 持續) = professional consult**. 不 「they'll work it out」.

明日早 7am 第 1 場 fight 5 分鐘 自然 form. 你 brain 默 react 「step in」. 5 second pause framework activate. Level check. Most likely Level 3 — walk away. 細路 5 分鐘 之後 self-resolve. 你 sip coffee. Day 30 frequency 跌 30%.

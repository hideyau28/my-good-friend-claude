## 情境

你睇咗單樓，業主 send 份租約 PDF 落你 WhatsApp，叫你「睇下 OK 就簽」。

呢份租約：

- **15-20 頁，律師體中文**（甚至英中混雜）
- **滿晒標準條文同行貨**——令你以為「咁多嘅，都係正常啦」
- **暗藏對你唔利嘅條款**——例如：押金條件、提前退租罰款、修葺責任、寵物／同住限制、續約加租機制

你簽之前要 30 分鐘讀清楚，但你冇法律底子，又冇兩個鐘。

呢篇教你用 Claude 做 30 分鐘嘅 **租約細節拆解**——揾出唔尋常嘅條款、預警可能蝕底嘅條文、列出你應該問業主嘅問題。

> **重要免責聲明**：呢個過程唔係法律意見。Claude 幫到你嘅，係**睇得明條約 + 識得問問題**，唔取代律師意見。涉及大金額、或者你好唔肯定，簽之前花 $1,500-2,000 揾律師睇半個鐘。

## 跟住做

### 1. 上傳份租約 PDF（如果係相機 scan，先 OCR）

Claude.ai 直接拖入對話框就得。如果係手機影嘅掃描 PDF（冇文字圖層），先用 Mac Preview 或者 Adobe Acrobat 做 OCR。

### 2. 用呢個 prompt

<PromptBlock label="完整 prompt — 香港租約 30 分鐘策略式拆解">
我即將簽呢份香港住宅租約，請幫我做以下分析。我嘅背景資料：

- 物業類型：[私人住宅 / 公屋租出 / 唐樓 / 服務式公寓]
- 月租：[填]
- 押金（按金 + 上期）：[填，通常 2+1 個月]
- 租期：[填，通常 2 年死約 + 1 年生約]
- 我嘅特殊情況：[例如：有寵物 / 想做裝修 / 同伴侶冇結婚但同住 / 想喺屋企做細生意]

請整理以下 4 個部分：

**Part 1：標準條文概覽**
列出呢份租約涵蓋嘅關鍵條文，每個一句講晒：租金、押金、租期、break clause、自動續約、水電煤、管理費、印花稅。逐個標明係咪市場標準。

**Part 2：⚠️ 紅旗條款（最重要嘅一段）**
逐個檢視以下類別，揾出對租客唔利嘅條款：

- **押金條件**：退還時間、扣留條件（係咪淨係「fair wear and tear」，定係「any damage」呢類過闊嘅寫法）
- **修葺責任**：邊啲屬租客、邊啲屬業主？特別留意「Tenant shall be responsible for all repairs」呢類一網打盡嘅寫法
- **提前退租 / break clause**：有冇？罰款幾多？合唔合理
- **轉租 / 同住人**：完全禁止、需要同意、定可以自由安排
- **寵物 / 裝修 / 改動**：明文點寫
- **續約機制**：點計加租？有冇封頂？跟市價？
- **違約終止**：點樣觸發？業主有冇單方面終止嘅權利
- **訪客 / 短租（Airbnb）**：通常禁止

每個紅旗都要畀：條款原文、指出點解對租客唔利、建議點傾。

**Part 3：欠缺嘅保障（租約冇寫嘅嘢）**
列出市場上合理、但呢份租約**冇涵蓋**嘅保障，例如：
- 業主入屋要事先通知
- 業主遲遲唔維修時，租客有咩追討途徑
- 租金支付方法嘅紀錄（防業主將來話冇收過）
- 不可抗力 / 物業變得唔適合居住

**Part 4：我應該問業主嘅問題（最多 5 條）**
根據以上分析，列出 5 條尖銳問題，我簽之前要傾或者要確認。

語氣：直接、實用、有前瞻性（提醒我將來可能撞到嘅問題）。
唔好喺每個部分都出行貨式免責聲明（信尾已經有 caveat）。
</PromptBlock>

### 3. 核對兩三個關鍵位

Claude 嘅 output 95% 啱，但呢類法律文件，**5% 嘅錯誤可以等於幾萬蚊**。至少核實：
- **押金條件嘅原文**——「fair wear and tear」同「any damage」差好遠
- **break clause 嘅準確字眼**——你要用退租條款嗰陣，邊一個字都好緊要
- **自動續約同續約租金**——業主通常喺呢度埋伏

### 4. 用 output 同業主傾（或者揾律師審）

如果紅旗得一兩個，你可以直接同業主傾（細節位業主通常肯讓）。
如果紅旗三個以上，**呢份租約唔尋常**，揾律師審過先簽會穩陣啲。

## 變化例子

### 變化 1：商業 / 鋪位 / 寫字樓租約

<PromptBlock label="變化 1 — 商業租約拆解">
同上面個 prompt 一樣，但呢份係商業租約（[鋪位 / 寫字樓 / 工業單位]）。
我嘅生意係：[填]

特別留意以下嘢：
- **准許用途條款**——有冇清楚寫明我嘅生意係准許嘅？（最常引起爭議）
- **雜費（outgoings）**——管理費、差餉、地租邊個畀？
- **還原責任**——退場時要還原成點？呢個可以幾萬蚊
- **裝修期同免租期**——點計？
- **交吉 / 提前退場**——商業租約嘅退租罰款通常好重
- **主租戶 / 商場租戶組合變動**——如果你係商場租戶，呢個對你影響好大

語氣：商業盡職審查。
</PromptBlock>

### 變化 2：你想退租，業主話要罰錢

<PromptBlock label="變化 2 — 退租爭議">
我而家已經簽咗租約 [幾耐]，因為 [原因] 想提早退租。
業主話我要畀 [幾多錢] 罰款。

請：
1. 檢視我份租約嘅 break / 提前終止條款（原文喺下面）
2. 確認業主索取嗰個金額係咪同條文一致
3. 揾出邊度有含糊位、我可以同佢傾
4. 列出我可以提出嘅替代方案（例如：揾人接手剩低嘅租期 / 畀租金直到業主搵到新租客 / 損失減輕嘅理據）
5. 草擬一封 email 畀業主，提出替代方案

【租約條文】：（貼相關條款）
</PromptBlock>

### 變化 3：你係業主，要起稿一份對你有利嘅租約

<PromptBlock label="變化 3 — 業主視角起草">
我有一個物業要租出，想草擬一份租約。
我嘅優先次序（由高到低）：
1. [例：保護物業免受損壞]
2. [例：租金收得穩定]
3. [例：保留將來重新發展物業嘅彈性]

請整一份標準住宅租約初稿，要做到：
1. 以香港標準租約（standard tenancy agreement）為基礎
2. 喺以下範疇加強業主保障（但唔好過分到變成無法執行）：
   - 押金條件
   - 修葺責任
   - 物業查察權
   - 退場還原
3. 同我講邊啲條款一般會被要求修改（即係租客有籌碼時會想改嗰啲）

Caveat：呢個 output 唔係法律意見，簽之前一定要律師審。
</PromptBlock>

## 一個心態

香港租約市場唔對等——**業主有現成行貨，租客好多時係第一次簽**。你睇唔出有伏，唔係你蠢，係資訊不對稱。

Claude 嘅優勢喺呢度最大：佢將業主嘅行貨條文，變成你睇得明嘅分析。30 分鐘嘅功夫，可能幫你避到幾萬蚊蝕底。

**但記住免責聲明**：呢個唔係法律意見。涉及大金額、或者你唔肯定，畀 $1,500 律師費永遠抵。

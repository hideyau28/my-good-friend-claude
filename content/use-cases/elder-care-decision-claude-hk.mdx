> ⚠️ Claude 唔取代社工 / 老人科醫生 / 律師。文中框架係幫你整理思緒同家庭對話 — 牽涉申請 / 法律 / 醫療決定，去社署「長者地區中心」或者父母嘅 case manager 度確認。

## 情境

朝早 6:45，醫院 call 你。

「[姓] 先生／太太，你媽咪／爸爸今朝喺廁所跌親，而家急症室。盤骨裂，要住院。」

你即刻請假落醫院。父母 82 歲，獨居（或者同另一位 80+ 歲嘅老伴住），平時行得食得，但呢半年開始「有啲忟忟憎憎」、「忘記咗食藥」、「行樓梯氣咳」。今次跌親係警號。

主診醫生第 5 日同你坐低：

> 「跌親本身好返，但骨頭未長返實之前要 6-8 週唔可以負重。出院之後，屋企人要諗下長期照顧安排 — 唔係淨係呢 8 週，係之後嘅 5-10 年。」

你諗起阿哥喺加拿大，阿妹喺深圳返工，你係香港唯一一個。你自己 40 幾歲，仔女仲讀書，全職返工。你而家面對 3 條可能路：

1. **請外傭 / 看護**喺父母屋企照顧
2. **送父母入安老院** / 護老院 / 護養院
3. **接父母落自己屋企住**，全家照顧

每條路都有成本、照顧質素、家庭關係、父母自尊嘅取捨。最痛苦嘅係 — **父母自己有意見**，而佢嘅意見可能同你嘅實際評估唔同。

呢一刻你 google 「香港安老院點揀」出嚟幾百個結果，全部好難消化。你個腦一片混亂，內疚 + 焦慮 + 疲勞混埋一齊。

呢篇用 Claude 幫你 **整理個決定，唔係幫你做決定**。Claude 嘅角色：

1. 列選項 + 各自取捨（成本 / 照顧程度 / 尊嚴）
2. 拆解香港政府資源（中央輪候冊、改善家居及社區照顧服務、樂悠咭等）
3. 同兄弟姊妹開家庭會議嘅講稿
4. 同父母敏感對話嘅 framing

最終決定 = 你 + 兄弟姊妹 + 父母 + 社工 + 醫生。Claude 幫你個腦清返。

### 1. 寫晒父母真實情況做基本資料

第一步唔係上網 search，係坐低 30 分鐘寫晒基本資料。冇基本資料你個決策矩陣會浮浮泛泛。

Claude 出反映。你會發現自己對某幾條問題「唔知道」 — 例如父母真實 ADL（你以為佢沖涼 OK，其實佢已經 3 個月冇沖過頭髮）。呢啲缺口就係你接住要核實嘅問題。

⏰ **時間**：30-45 分鐘。可以同住喺香港嘅兄弟姊妹一齊填，數據會更準。

### 2. 用決策矩陣列三條路嘅取捨

基本資料有咗，Claude 可以出個決策矩陣 — 唔係出答案，係出框架等你睇清楚每條路嘅成本。

<PromptBlock label="完整 prompt — 三條路決策矩陣">
基於我啱啱寫嘅基本資料（父母情況 + 屋企 + 家庭資源 + 父母意願），請幫我整一個 **3 條路決策矩陣**：

🅰 **請外傭 / 看護喺父母屋企**
🅱 **入安老院 / 護老院 / 護養院**
🅒 **接父母落子女屋企住**

每條路評估 8 個 dimension（1-5 分，1=極差，5=極好）：

1. **照顧程度配合** — 配唔配父母 ADL 同醫療需要
2. **每月成本**（香港 2026 typical range，要具體數）
3. **一次過 setup 成本**（裝修 / 按金 / 工人安置費）
4. **父母尊嚴 / 自主** — 父母仲喺熟悉環境？仲有自主感？

請列埋 **香港 2026 typical 成本 range**：

- 照顧程度上限：照顧到中度依賴，重度可能 unsafe

🅱 **院舍**（香港大類）
- **私營安老院**（自費）：[月費 range]

- **甲一級非資助宿位** vs **乙級** 照顧程度差別

🅒 **接落屋企**
- 屋企裝修（扶手 / 防滑 / 床 / 無障礙廁所）
- 可能仲要請 part-time 看護 / 鐘點
- 隱形成本：子女事業影響、配偶 burnout、細路空間壓縮

最後，請畀我 **3 個反思問題** 而唔係答案：
- 邊個 dimension 對我哋家庭最重要？
- 我哋而家係咪 react 緊急性、定係真係諗 5 年路？
- 父母自己會點 weigh 呢 8 個 dimension？

語氣：實際、唔造作、唔 push 任何一條路。
</PromptBlock>

Claude 會出個 table。**唔好即刻分享畀全家**。先自己沉澱一晚。

你會發現 — 矩陣唔會「告訴你揀邊條」。佢會浮現你冇諗過嘅取捨。例如：你以為「接落屋企」最孝順，矩陣提醒你父母可能會失去日日落公園做晨運嘅社交網絡，反而抑鬱。

### 3. 拎晒香港政府資源同輪候真實時間

呢一步好多夾心一代子女漏咗：**香港有相當多政府資源**，但要識揾。決策矩陣入面嘅成本會因為呢啲 subsidy 大幅變。

<PromptBlock label="完整 prompt — HK 長者政府資源同申請流程">
我父母 [年紀]，[medical condition]，住 [區]，[公屋 / 私樓 / 自置物業]。請幫我列晒 2026 年香港政府長者資源 + 申請流程 + 真實輪候時間。

請覆蓋以下類別：

🏥 **社署 — 長期照顧服務**

1. **中央輪候冊 — 資助長期護理服務**
   - 申請點開始（轉介？社署 hotline？老人科醫生？）
   - 評估流程（標準化評估機制 SCMC）
   - 「家居為本」vs「院舍為本」嘅評估結果
   - 點解資助院舍要輪 3-5 年 — 同時可以揀「改善買位計劃」加快

2. **改善家居及社區照顧服務（EHCCS）**
   - 上門照顧（沖涼、清潔、送飯、護士到診）
   - 適合：父母想留喺屋企，但需要 daily/weekly support
   - 申請：同上中央輪候冊掛鈎
   - 輪候：[typical range]

3. **綜合家居照顧服務（IHCS）— 普通個案 vs 體弱個案**
   - 收費低（個案類別 means-tested）
   - 包括陪診、護送、送飯、家居清潔

4. **日間護理中心 / 老人地區中心**
   - 父母日頭去中心，夜晚返屋企
   - 適合：父母獨居或老伴照顧者 burnout buffer
   - 收費 subsidised

💵 **現金 / 津貼**
- **長者生活津貼**（普通額 vs 高額）— means test
- **綜援**（如父母收入低於門檻）
- **公共福利金 — 高齡津貼（生果金）**
- **傷殘津貼**（如評估到 ADL 嚴重依賴）
- **樂悠咭** — 60 歲以上 $2 乘車

🏠 **環境改善**
- **長者社區照顧服務券**（CCSV）— 院舍 / 家居服務 voucher
- **長者院舍住宿照顧服務券**（RCSV）— 自選認可院舍
- **房屋署「長者住屋改善工程」** — 公屋戶免費裝扶手 / 防滑

⚖️ **法律 / 文件**
- **持久授權書（EPA）** — 父母仲清醒時做，授權子女處理財產
- **預設醫療指示（AMD）** — 父母自己決定 end-of-life care
- **平安紙 / 遺囑** — 唔係觸霉頭，係保護家庭和諧

🆘 **緊急 / 危機支援**
- 跌親後出院支援（醫管局「離院長者綜合支援計劃」）
- 護老者支援中心 / 護老者熱線

📞 **點開始（具體 action）**
請畀我一個 **「Day 1 / Week 1 / Month 1」行動清單**：
- Day 1：打邊個 hotline、揾邊個社工
- Week 1：去邊間長者地區中心登記、攞 SCMC 評估
- Month 1：父母嘅 EPA / AMD discussion 開始

⚠️ **誠實提醒**
- 香港資助宿位輪候真實時間（唔好 sugarcoat）
- 即使有 voucher，配額 / 質素 vary — 唔可以「申請完就 set」
- 任何申請都要父母同意 + 醫生 / 社工評估，唔可以子女單方面決定

語氣：實事求是，唔好過度樂觀，但同時令我知有資源。
</PromptBlock>

### 4. 家庭會議 + 同父母對話嘅講稿

最情緒上難嘅一步。揾完資源、列完矩陣，跟住要 **同兄弟姊妹開會** + **同父母開口**。Claude 可以幫你草擬講稿 — 唔係照讀，係浮現你想點 frame。

Context：

**Scenario 2 — 父母擔心拖累子女**（「唔好阻你哋」）

**Scenario 3 — 父母拒絕外人入屋**（文化 / 私隱抗拒）

- Trial：先試短時間 helper（每週 2 次 2 小時清潔），唔係 24 小時
- 揀同父母同方言 / 同年代 sensibility 嘅 helper
- 父母同 helper 之間嘅 social mismatch 係真，唔好 dismiss

**通用原則**
- 父母仲有能力嗰陣，**佢嘅一票加倍重**
- 唔好單方面決定，即使「客觀最好」
- 預期至少 3 次對話先有進展，第 1 次往往係浮現
- 自己 / sibling 喊 OK — 父母見到你哋難過，唔係 weakness 係 connection

語氣：温柔、實際、acknowledge 內疚同現實嘅 tension。我而家對呢個對話好驚。
</PromptBlock>

呢個講稿唔好照讀。沉澱 24 小時，然後改入你自己嘅聲音、你哋家庭嘅具體關係。打 video call 嗰陣記得做 facilitator，唔係 advocate。

## 變化

父母而家 ADL 全自理，認知正常，但今次跌親提醒你哋「窗口期」開始：

<PromptBlock label="變化 1 — 父母仍自主，提前準備">

請幫我整 **「輕力提前準備」清單**：

- 夜燈
- 床邊防跌軟墊

### 變化 2：失能 / 失智 — 照顧程度大幅升

父母重度依賴（不能自理 / 中後期認知障礙）：

<PromptBlock label="變化 2 — 重度依賴 / 失智父母">

🏥 **照顧程度 reality check**
- 重度依賴 / 失智 = **外傭一個人 unsafe**
  - 翻身、餵食、失禁、遊走、跌親風險都高
  - 傭工自己 burnout + 受傷風險
- 接落子女屋企 = **要全屋無障礙改造 + 至少 2 個 caregiver 輪更**
- 院舍方向 reasonable — 但要揀啱程度

🏥 **院舍程度分類**
- **安老院（Care and Attention Home）** — 中度依賴
- **護理安老院 / 護養院（Nursing Home）** — 重度依賴 + medical
- **醫院（長期病床）** — 末期 / palliative
- 失智父母：要揀「認知障礙症友善」設計嘅院舍（防遊走、reminiscence room）

🧠 **失智專屬**
- 黃昏症候群（Sundowning，黃昏混亂加劇）
- 遊走 — 屋企 unsafe
- 唔識自己食飯 / 失禁 — 子女落手照顧 = 子女 6 個月內 burnout
- 親人認唔出 — 子女情緒打擊最大

⚖️ **倫理 + 法律**
- 父母已失去心智能力 = 子女做決定要有 EPA 或者「精神健康條例」程序
- 院舍 admission 父母拒絕但子女覺得 necessary — 要社工 + 醫生介入
- 唔好內疚 — 重度照顧外判唔係不孝，係可持續

💔 **照顧者自身**
- 失智照顧者抑鬱率極高
- 暫託服務（Respite care）— 每月 1-2 週 break 唔係奢侈
- 護老者支援中心、家屬互助組

語氣：認可呢個階段嘅哀傷 — 父母「仲喺度但又唔再係佢」係模糊性失去。
</PromptBlock>

### 變化 3：海外 / 移民咗嘅子女 — 跨地照顧

## 一個心態

夾心一代嘅 quiet truth：你而家做嘅每一個決定，都喺內疚同現實之間嘅窄縫入面行。

「最好」嘅選項通常唔存在。你揀請外傭 — 父母 isolation；揀老人院 — 你成世記得佢喊「唔好留我喺度」；揀接落屋企 — 你婚姻、細路、自己事業全部要 absorb。每條路都有成本，**而呢個成本唔係你嘅錯**。

呢個心態嘅核心：**你唔係喺揀「最孝順」嘅路。你係喺揀「對全家最可持續嘅路」**。可持續 = 父母得到 adequate 照顧、子女唔 burnout、家庭關係唔崩、5 年後仲 sustainable。

亞洲文化會 frame 呢個做「孝」嘅 test。但「孝」嘅原本意思唔係自我犧牲 — 係 thoughtful care。Thoughtful = 攞晒資訊、open 同父母傾、協調兄弟姊妹、用社區資源、保住自己能力等之後 5-10 年仲行得到。

Claude 喺呢個過程嘅角色唔係情緒支援 — Claude 唔識替你承受內疚。Claude 嘅角色係**喺你個腦一片混亂嗰陣撐住個框架**：列選項、查資源、草擬講稿、反映你冇諗到嘅缺口。等你嘅情緒同直覺有空間出現，唔被事務淹沒。

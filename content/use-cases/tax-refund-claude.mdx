## 情境

12 月 20 日，你東京旅程最後一日。

成個旅程你買咗：

- Day 1：Don Quijote 化妝品 + 護膚 ¥18,500
- Day 2：UNIQLO + GU 衫 ¥12,000
- Day 3：Bic Camera 電子產品 ¥45,000
- Day 4：百貨公司 LV 護照套 ¥38,000
- Day 5：朝早秋葉原漫畫 + 玩具 ¥8,500

總共 ¥122,000，HK$ 約 6,200。

退稅 10% 大約 ¥12,200 = HK$620。**值得拎**。

你朝早 11am 到羽田機場，13:00 飛香港。

11:15am：check-in 之前過海關退稅櫃位。

你拎出張包好嘅 LV 護照套。海關員：

> 「Receipt please」

你紙袋裏面有 4-5 張收據混喺行李度。你揾出嚟。

> 「Bic Camera 電子收據 please」

你揾。

> 「Don Quijote 收據請出示」

你揾。

15 分鐘過去。

> 「Don Quijote 嘅免稅手續你已經喺店入面做咗 sales tax exemption——呢張收據有蓋印。你唔需要喺機場再處理。」

你：「等等⋯⋯即係 Don Quijote 化妝品 ¥18,500 嘅 8% 稅喺店已經減咗？」

> 「Yes」

OK。你呆咗一陣。

> 「LV 嘅收據 please」

你拎出 LV 收據同護照套。

> 「呢張已經喺百貨店蓋咗印——你畀嘅係含稅價，呢度退稅櫃位唔處理。」

你更加一頭霧水。「咁⋯⋯邊啲先可以退？」

> 「Bic Camera 嘅電子產品同百貨店櫃位外嘅購買。但⋯⋯收據呢？」

你揾咗 5 分鐘都揾唔到 Bic Camera 收據（係咪已經掉咗去酒店垃圾桶？）。

> 「冇收據就冇得退，sorry。」

你 12:30 就要登機。你企喺櫃位，先至醒起：

- Don Quijote ¥18,500 稅：已經喺店免咗（你以為要再退——其實已經慳咗）
- LV ¥38,000 稅：已經喺收據度打咗折
- Bic Camera ¥45,000 稅 = ¥4,500：因為冇收據，蝕咗
- UNIQLO / GU + 秋葉原：細件購買，可能未夠 ¥5,000 門檻

遊客退稅咁亂，根本原因係：

- **日本退稅系統有兩種流程**（店內即時 vs 機場後續——你以為一定要去機場處理）
- **收據管理冇一套有系統嘅計劃**（散落喺酒店、行李、袋入面）
- **物品類型資格唔同**（液體 / 消耗品 vs 一般商品分類）
- **門檻規則複雜**（每間店 ¥5,000+ 一般商品，又或者 ¥5,000+ 消耗品；消耗品要密封未開）

呢篇就教你用 Claude **出發前整追蹤器**、**旅程中每日記錄**、**機場最後索償次序**——唔再喺櫃位前面慌住卡死。

## 跟住做

### 1. 出發前搞清楚退稅基本

開 Claude，先了解你去嗰個地方嘅稅務系統：

<PromptBlock label="完整 prompt — 日本退稅系統大致理解">
我即將去日本玩 [N 日]，預算購物 [HK$ 金額]。我從未做過退稅。

請整 **日本退稅系統概覽** for HK 遊客：

🇯🇵 **日本消費稅**
- 稅率：10% 標準 / 8% 食物 + 飲品
- 持非日本護照停留少於 6 個月嘅遊客 = 合資格免稅

📋 **2 種流程**（重要區分）：

**Type A：店內即時免稅**（大部分情況）
- Don Quijote / Bic Camera / 大型連鎖店
- 你喺收銀出示護照
- 即時扣稅 = 你畀免稅價
- 收據 + 護照喺店蓋印
- **完成——唔需要再去機場處理**
- 你拎走嘅「消耗品」要密封、唔可以開（朱古力 / 化妝品 / 健康產品）

**Type B：機場退稅**（少數情況）
- 你喺店已經畀咗含稅價
- 收據留低
- 喺機場海關櫃位出示收據 + 物品
- 退稅以現金或信用卡 credit 形式畀返你

**大部分店家而家都用 Type A**，所以你整個旅程 99% 都唔會用到機場退稅流程。

📏 **門檻 + 限制**

✅ **一般商品** - 電子 / 衫 / 包 / 飾物
- 最低：同一日同一店 ¥5,000+
- 最高：冇限
- 可以開 / 即時用

✅ **消耗品** - 化妝品 / 食物 / 飲品 / 健康用品
- 最低：同一日同一店 ¥5,000+
- 最高：同一日同一店 ¥500,000
- **必須密封、唔可以開**，直到離開日本
- 海關可能抽查——唔可以喺日本用咗 / 食咗

❌ **唔可以溝埋計**：
- 同一張收據嘅一般商品 + 消耗品唔可以夾埋算門檻
- 例：¥3,000 衫 + ¥3,000 化妝品 = 兩邊都唔夠 ¥5,000 門檻、唔合資格

📝 **必要文件**
- 護照正本（唔係副本）
- 向收銀證明係你本人、護照同名
- 海關申報表（有啲店要填）——店會提供
- 收據：唔可以掉，亦唔可以摺（機場可能要驗）

🏷 **唔合資格免稅嘅項目**
- 喺店即場食 / 飲嘅飲品
- 旅行相關開支（酒店 / 交通 / 餐廳食飯）
- 二手 / 古董（部分情況）
- 已經寄返香港嘅（你必須親身帶走）

🏬 **常見連鎖店 + 佢哋嘅流程**：
- Don Quijote：Type A（店內，快）
- Bic Camera / Yamada Denki：Type A
- UNIQLO：Type A（唔係間間都做，只有特定分店處理免稅）
- 高端百貨（Mitsukoshi、Isetan、Takashimaya）：Type B（上 4F 客戶服務處理）
- Sezon / Daimaru：大部分係 Type B
- 細精品店 / 街市 / 自動售賣機：唔做免稅（你畀含稅價）

💰 **預期慳到幾多**
- 一般商品慳 10%（你畀標價嘅 90.9%）
- 消耗品慳 8%（你畀標價嘅 92.6%）
- 消費 HK$10,000 = 大約慳 HK$1,000
- 值得花啲時間

語氣：清晰區分兩個流程類型。
</PromptBlock>

搞清楚之後，你最重要嘅收穫係：**大部分情況喺店內已經處理咗——唔使去機場頂住壓力**。

- 計登機前需要幾多索償時間
- 「我有 ¥XXX 待退稅 → 機場處理要 YY 分鐘」

### 3. 機場最後索償次序（Type B 退稅項目）

最後一日喺機場嘅準備，避免登機前慌張：

<PromptBlock label="完整 prompt — 機場退稅索償最後次序">
我第 5 日飛香港，13:00 登機。我有 [N 張 Type B 退稅收據 = 待退 ¥XXX]。

請整個 **機場最後索償次序**：

⏰ **時機策略**

**到達機場**：預 11:00 到（13:00 登機，留 2 小時緩衝）
- 1 小時退稅處理
- 30 分鐘 check-in + 安檢
- 30 分鐘登機 + 行去登機閘口

**次序**：
1. **步驟 1：到達機場（11:00）**
   - 揾「Tax Refund」指示牌 / 海關櫃位
   - 通常喺國際出境 check-in 之後
   - 喺東京機場：羽田 T2 / T3、成田 T1 / T2 都有

2. **步驟 2：海關櫃位排隊（11:00 - 11:30）**
   - 避開繁忙時段（朝早 / 夜晚通常較快）
   - 大旺季（春節 / 黃金週）可能要排 1 個鐘以上

3. **步驟 3：文件檢查（11:30 - 11:45）**
   - 海關員會問你攞：
     - 護照
     - 登機證
     - 收據（按店分類）
     - 物品（部分要出示）
   - 海關員驗證之後喺收據蓋印
   - **關鍵**：物品必須未開（尤其消耗品）

4. **步驟 4：現金退稅或退返信用卡（11:45 - 12:00）**
   - 揀現金退稅（即收 ¥）或信用卡退稅（較慢，2-6 週）
   - 現金通常少少（每宗 $1-2 服務費）
   - 信用卡退返原本張卡

5. **步驟 5：Check-in + 安檢 + 登機（12:00 - 12:45）**

📋 **去機場前必要準備**

- [ ] 全部收據收齊 + 分類（Type B 優先）
- [ ] 消耗品密封好（手提或寄艙）
- [ ] 護照放喺手邊
- [ ] 登機證印好 / 喺電話
- [ ] 計數機 / notes app 預備好（核對退稅金額）

🎒 **收納策略**

消耗品（化妝品 / 朱古力 / 健康食品）要密封：
- 細件嘅放手提（海關多數要你出示物品）
- 放寄艙都得，只要妥善申報
- 最理想：30% 放手提（畀人睇樣本）+ 70% 放寄艙

一般商品（電子 / 袋 / 衫）：
- 唔一定要出示，但有時海關員會問
- 放手提或寄艙都得

🚨 **常見被拒原因**（你要避開）

- ❌ 收據唔齊 / 掉咗——無法補救
- ❌ 消耗品喺日本開咗 / 用咗
- ❌ 物品唔喺度（已經用咗）
- ❌ 冇帶護照（海關員無法核實）
- ❌ 轉機航班嘅登機證唔夠（必須出示直接離開日本嘅航班）
- ❌ 停留超過 6 個月 = 唔合資格遊客（唔係你嘅情況）

⏰ **如果排隊長、又趕住登機**

- 如果退稅金額少過 HK$200，而登機剩唔夠 30 分鐘，就唔好做喇
- 計一計：退 HK$200 vs 趕唔切上機要 HK$1,000+ 重新訂票
- 點睇都唔抵

🆘 **後備：拎唔到退稅點算**

- 部分退稅可以旅程後上 Global Blue 網站申索（歐洲情況）
- 日本情況：通常淨係機場做，旅程後申索好少見
- 部分店有「郵寄退稅」做法——喺店內問清楚

💰 **退稅預期**

- 現金退稅：即場攞 ¥
- 信用卡退稅：2-6 週後先出現
- 唔好以為成筆稅全數退返（手續費通常 ¥50-200）

📞 **如有問題**：
- 海關員唔肯蓋印 → 冷靜啲要求見主管
- 完全被拒退稅 → 要求書面寫低拒絕原因
- 收據出錯 → 如果有時間返原本店（24 小時內通常仲得）

語氣：有效率，亦提你排隊嘅不確定性。
</PromptBlock>

呢個次序嘅隱藏教訓：**退稅嘅預期價值，通常細過你慌住趕嘅成本**。退 ¥500 唔值得趕唔切上機。出發前準備充分 = 零壓力索償。

### 4. 用呢個 prompt 整買嘢策略（唔好白白蝕咗筆稅）

啲策略性決定：邊樣值得喺日本買、又可以免稅：

<PromptBlock label="完整 prompt — 稅策略性購物計劃">
我去日本玩 [N 日]，預算 HK$ [金額]。請整個策略性購物計劃：

🎯 **HK 遊客喺日本買咩值得？**

✅ **慳得最多（日本平 + 免稅）**
- 化妝品（SK-II / 資生堂 / 日本本土牌子）——比香港平 30-50% + 免稅 8%
- 電子產品（Bic Camera / Yodobashi）——部分型號香港冇得賣，又平
- 高級鋼筆 / 文具——質素好又平
- 特定零食 / 茶 / 工藝品——香港冇得買
- 牌子化妝品 + 護膚品大裝

🟡 **慳得有限**
- 一般時裝（UNIQLO / GU）——香港都有、同價
- 名牌（LV / Hermès / Gucci）——偶爾減價有著數，但通常香港都得
- Apple 產品——香港定價都好有競爭力
- 一般雜貨

❌ **唔值得為免稅咁麻煩**
- 樽裝水 / 飲品
- 每日食 / 餐廳（本身都唔免稅）
- 平嘅配件
- 單店少過 ¥5,000 嘅禮物（未夠門檻）

📊 **策略性消費原則**

要慳到最盡：
1. 集中喺少數大店買（夾夠 ¥5,000 門檻）
2. 揀有 Type A（店內即時免稅）嘅店——最易
3. 唔好將同類物品分散去多間店買
4. 預先研究大件物品定價（例：某部相機，香港 vs 日本）

🛒 **購物節奏建議**

**第 1-2 日：較大金額物品**
- Bic Camera / Yodobashi（電子）
- Don Quijote（化妝品 + 雜貨）
- 大型連鎖免稅有效率

**第 3-4 日：專門店 + 精品店**
- 百貨店 LV / 高端（4F 客戶服務免稅）
- 專門茶 / 朱古力 / 文具
- 細精品店唔做免稅，淨係買香港冇嘅嘢

**第 5 日（最後日）：手信，免稅唔做都罷**
- 機場 / 車站手信舖通常唔免稅
- 零食 / 畀公司同事嘅手信
- 最後一刻嘅細件購買

💰 **HK$15k 預算嘅分配例子**
- 慳得多嘅物品（化妝品 + 電子）：HK$8,000（慳大約 HK$640-800）
- 慳得中等（時裝 + 袋）：HK$5,000（慳大約 HK$400-500）
- 手信 + 細件：HK$2,000（無免稅）
- **稅總共慳到**：大約 HK$1,000-1,300

🎒 **後勤考慮**

- 行李重量上限影響大件購物
- 8kg 手提 + 23kg 寄艙 = 大約 30kg
- 大件電子產品要留意重量
- 易碎物品要照顧（相機 / 玻璃）

🚫 **退稅唔值得搞嘅情況**
- 單件少過 ¥5,000（未夠門檻）
- 你打算旅程中就用嘅物品（消耗品唔密封）
- 最後一日嘅衝動消費
- 重 / 易碎、唔值得加重行李嘅物品

✅ **買嘢時機要精明**
- 第 1-2 日買大件（有更多時間處理免稅 + 預埋海關檢查緩衝）
- 唔好喺第 5 日朝早衝動買（可能趕唔切免稅）
- 5pm cut-off：百貨店客戶服務免稅櫃位 6-7pm 就收

📋 **付錢前清單**（喺店、畀錢前）
- [ ] 呢間店做唔做免稅？（睇指示牌 + 問收銀）
- [ ] 夠唔夠門檻？（同店同日 ¥5,000+）
- [ ] 一般商品定消耗品分類清唔清楚？
- [ ] 帶咗護照、人喺現場

## 變化例子

### 變化 1：歐洲 VAT 退稅（Schengen + UK）

歐洲 VAT 同日本嘅系統完全唔同：

<PromptBlock label="變化 1 — 歐洲 VAT 退稅（Schengen）流程">
我去歐洲 [N 個國家] 玩 [N 日]。Schengen 區，可能去埋 UK。請整個 VAT 退稅框架：

🇪🇺 **歐洲 VAT（Value Added Tax）概覽**

- 稅率：各國唔同（德國 19%，法國 20%，意大利 22%，UK 20% 等）
- 持非歐盟護照嘅遊客 = 合資格
- 退稅通常係購買額嘅 8-15%（扣手續費後）

📋 **流程：全部喺機場退稅**（同日本主要喺店內唔同）

**步驟 1 — 喺店**：
- 出示護照（必須正本，唔係副本）
- 消費要 ≥ 該歐盟國家嘅最低門檻（通常 €50-175，視乎國家）
- 店填寫「Tax Free Form」（Global Blue / Planet / 部分店自家）
- 你拎走 form + 收據 + 商品

**步驟 2 — 出境機場（離開 Schengen / 歐盟最後一個歐盟國家）**：
- 海關櫃位喺退稅 form 蓋印
- 可能驗證物品（尤其高價嘅）
- 物品必須未用過（尤其名牌 / 電子）
- 之後將 form 寄畀經營者（Global Blue / Planet）

**步驟 3 — 退稅處理**：
- 信用卡退稅：2-6 週後出現喺卡
- 現金退稅：喺機場退稅櫃位（海關蓋印之後）
- 服務費：扣大約 3-5%

⚠️ **重要區分**

如果係多國旅程：
- 海關蓋印要喺 **離開 Schengen / 歐盟嘅最後一個歐盟國家** 做
- 例：意大利 → 法國 → 西班牙 → 飛香港：海關印喺西班牙（最後）
- 例：法國 → UK → 飛香港：去 UK 之前喺法國蓋印；UK 另計
- 唔可以喺中途轉機嗰度蓋印

🏬 **常見做免稅嘅店**：
- Hermès / LV / Chanel / 高級名牌
- 大型百貨店（Galeries Lafayette、KaDeWe）
- 部分高端餐廳——有啲餐廳 / 紅酒都可以免稅
- 大部分連鎖店都有得選擇加入

🚨 **常見被拒原因**：
- 未夠門檻（例如法國要 €175 以上）
- Form 填錯 / 漏咗簽名
- 物品睇得出用過（名牌袋有明顯磨損）
- 收據同 form 對唔上
- 護照過期 / 旅程期間遺失

⏰ **機場時機策略**

歐洲海關排隊通常比日本長：
- 高峰時段：黃金週 / 暑假 / 聖誕
- 預 2-3 小時機場緩衝
- 即日退現金嘅櫃位，會早過海關關門

💰 **退稅預期**：
- 法國買 €1,000（VAT 20%）：典型退 €166（€1000 / 1.2 × 0.2 = €166.67，扣 3-5% 費 = 淨 €130-150）
- 淨退：購買額嘅 13-15%

🏛 **個別國家嘅古怪之處**：
- **法國**：PABLO 機（比櫃位快，24 小時有）
- **德國**：先海關蓋印，再去獨立退稅櫃位
- **意大利**：排隊最長，要預多啲緩衝
- **UK**（脫歐後）：唔再屬 Schengen，免稅系統獨立，更嚴

📦 **高價購買（超過 €1,000）**

建議：
- 用信用卡（有保險保障）
- 購買時影相
- 留低盒 + 標籤 + 防塵袋
- 唔好寄返香港（必須親身帶）
- 空郵嘅成本之後會多過慳到嘅稅

🛍 **香港遊客最常買嘅免稅物品**：
- 名牌手袋（LV / Chanel / Hermès）
- 高端錶（Rolex / Patek）
- 設計師服裝（Burberry / Gucci）
- 紅酒 + 香檳（限個人用、手提）
- 香水 / 化妝品（留意液體手提限制）

語氣：詳細，提醒留意排隊同手續費。
</PromptBlock>

### 變化 2：一家大細一齊去（多人收據點管理 + 細件購買）

帶細路去玩，收據管理完全係另一回事：

<PromptBlock label="變化 2 — 家庭旅程退稅處理">
我帶 [我 + 伴侶 + N 個細路] 一齊去玩，每個家庭成員都會買嘢。請整個家庭模式嘅退稅策略：

🎯 **家庭旅程嘅稅務特性**

- 細路買嘅嘢：算唔算喺父母名下？視乎國家規則
- 多個人嘅購買累積埋
- 風險：收據散落喺幾個袋
- 大件行李加埋 BB 車嘅後勤

🇯🇵 **日本家庭規則**：
- 每個遊客有護照 = 每個都可以索免稅
- 細路自己有護照（香港 6 個月以上簽發嘅護照）——都計入家庭索償
- 細路嘅衫 / 玩具 / 糖可以用父母護照（細路同行申報）
- 用同一個父母嘅護照——唔好喺同一間店分太多單

📋 **加強版家庭收據記錄**

```
日期：___
邊個家庭成員用咗：[媽 / 爸 / 8 歲女自己 / 5 歲仔（媽名下）]
店：___
物品：___
金額：¥___
流程類型：A / B / NA
收據儲存：1 個集體信封
```

🎒 **集中管理收據**

**指定一個人做整個旅程嘅「稅務官」**：
- 每日完結時全部收據放入一個信封
- 唔好散落喺唔同人 / 唔同袋
- 檢查銀包 + 問細路攞返收據

📍 **細路購物嘅具體記錄**：
- 玩具：較易低過 ¥5,000 門檻 = 唔合資格
- 策略：同店同日合併幾件、夾夠 ¥5,000
- 例：5 歲仔玩具 ¥2,000 + 8 歲女 K-pop ¥3,500 = ¥5,500（媽護照）——合資格

🛒 **家庭模式買嘢**：
- 集中喺較少數嘅大店（Don Quijote / Bic Camera）夾夠門檻
- 專門玩具精品店（細店）——唔好嘥時間搞免稅，只接受含稅價
- Don Quijote：一站式家庭購物（Type A 即時免稅）

🎒 **行李分配**

- 每個父母分擔自己份額、平衡重量
- 細路嘅行李通常未夠重
- 可以嘅話將免稅物品分散喺行李度

🚨 **家庭特有風險**

- 細路拎住收據 → 揾唔到 / 掉咗
- 細路開咗消耗品（拆咗糖食）→ 嗰件就失去免稅資格
- 一家人周圍逛、多間店買細件 = 每間店都未夠門檻
- 父母攰到衝過免稅櫃位都唔索

✅ **家庭模式購物節奏**

- 第 1 日大型家庭購物（集中喺 Don Quijote 一個鐘）
- 第 2-3 日個別購物（各成員自己鍾意嘅嘢，細店）
- 第 4 日（最後）大金額（相機 / 袋 / 大件）
- 第 5 日機場留 2 小時緩衝（排隊 + 家庭後勤）

🛂 **海關檢查嘅家庭調整**

- 海關員可能問邊件物品屬邊個家庭成員
- 父母申報「呢件係買畀我細路」冇問題
- 海關員可能抽查細路玩具有冇開（少見但有可能）
- 每個家庭成員預多 5 分鐘喺海關

🆘 **遺失收據嘅應對**
- 細路掉咗一張收據 → 嗰單購買失去免稅
- 店無法補發（除非購買後少過 24 小時、而你又返到原本店）
- 計慳到嘅稅嘅時候，心理上預埋「掉咗 1-2 張收據」嘅緩衝

語氣：貼近家庭現實，明白父母管唔晒咁多。
</PromptBlock>

### 變化 3：高價單一物品（LV / 錶 / 珠寶）

買一件高價物品嘅具體策略：

<PromptBlock label="變化 3 — 高價單一物品免稅策略">
我計劃旅程期間買一件高價物品：[LV 袋 HK$30,000 / 錶 HK$50,000 / 珠寶 HK$80,000]。

請整個高價購買嘅稅務策略：

💎 **高價物品要考慮嘅嘢**

- 退稅慳得相當多（HK$50,000 × 8-15% = HK$4,000-7,500）
- 但風險同審查都更高
- 機場海關可能要驗
- 行李 / 手提嘅後勤較複雜
- 保險好重要（物品遺失 = 大損失）

🏬 **購買前研究**

對於名牌物品：
- 預先睇清楚香港同型號定價
- 預先比較唔同國家定價（LV 袋，巴黎 vs 香港）
- 比較二手市場（Vestiaire / 二手）
- 確認真偽
- 大型精品店預約 VIP（避免排隊）

📋 **購買時**

- 帶護照、放喺顯眼處（畀收銀確認）
- 用信用卡畀（唔好現金——有保險 + 有記錄）
- 核對收據姓名同護照一致
- 物品盒 + 防塵袋 + 認證咭一齊留低
- 喺店影物品 + 收據 + 你個樣（留低憑證）

🎒 **後勤：手提 vs 寄艙**

對於 HK$50,000+ 物品：
- 一律手提（寄艙行李遺失 = 災難）
- 機場海關接受申報手提
- 酒店退房前影狀況相（狀況證明）

⏰ **出境機場流程**

- 預 3-4 小時緩衝（高價物品要額外檢查）
- 海關員會：
  - 驗證物品喺度 + 未用過
  - 核對收據同退稅 form
  - 高價物品有時會問「點解買」
- 預埋抵港後再影物品狀況相（保險證據）

🛡 **保險保障**

對於 HK$30,000+ 物品：
- 確認旅遊保險有得保（細閱保單）
- 部分保單剔除高價單件
- 考慮另買物品保險
- 收據 + 相片對任何索償都必要

🚨 **抵港海關申報**

- 香港海關：個人自用物品通常冇問題
- 但商業數量 / 高價模式（3 件以上名牌）可能會問
- 香港唔收手信稅
- 被問就老實申報

💼 **轉售 / 商業用途考慮**

- 免稅好處明確係畀「個人用」
- 你即刻拎去二手轉售 = 有機會被指欺詐
- 個別國家規則，若發現轉售可能令退稅無效
- 唔好帶多件同型號

🆘 **途中遺失 / 損壞**

- 物品遺失：即刻報警 + 報保險索償
- 損壞：即刻影相 + 保留盒 + 通知店 + 報保險
- 收據至關重要——將電子副本 email 一份畀自己做後備

📞 **旅程後跟進**

- 檢查信用卡月結單（退稅金額有冇出現？）
- 旅程後先發現真偽問題 → 返原本店會好麻煩
- 將來再去嘅好處：成為店嘅會員，將來有優先待遇

💰 **值得計一計**

- HK$50,000 嘅錶喺日本買 vs 香港同型號 HK$48,000
- 免稅額外慳大約 HK$4,000
- 相對成個日本旅程 HK$15,000 嘅成本，總共慳到幾多？
- 通常係「好著數」，但唔足以成為「旅程嘅主要目的」

語氣：清醒分析高價物品，唔好衝動。

## 一個思維框架

遊客退稅嘅預期價值通常被高估。

以普通遊客嘅消費模式嚟睇：

- 5 日旅程平均消費：HK$3,000-5,000
- 退稅 8-12% = HK$240-600
- 機場處理時間：排隊 30-60 分鐘
- 趕飛機嘅壓力：一次因為退稅排隊而趕唔切上機 = HK$1,000+ 重新訂票 + 延誤一日
- **淨價值：每程 HK$200-500**

呢個價值唔算細，但唔值得你慌住趕、又或者打亂成個旅程嘅節奏。

換個角度睇：退稅唔係「攞盡額外收入」——而係 **一個有系統嘅儀式，無壓力咁收返一筆小息**。每日 5 分鐘收據記錄 + 搞清楚兩個流程類型 + 機場留 30 分鐘緩衝 = 零焦慮咁慳返大約 HK$500。

最後提醒：

- ✅ **大部分日本店係 Type A（店內即時）**。以為一定要去機場處理，係誤解。
- ⚠️ **收據管理比搞清楚系統更重要**。每日信封 + 拉鏈袋 = 九成成功。

## 情境

12 月 20 日，你最後一日東京 trip。

成 trip 你買咗：

- Day 1：Don Quijote 化妝品 + 護膚 ¥18,500
- Day 2：UNIQLO + GU 衫 ¥12,000
- Day 3：Bic Camera 電子產品 ¥45,000
- Day 4：百貨公司 LV 護照套 ¥38,000
- Day 5：朝早秋葉原漫畫 + 玩具 ¥8,500

總共 ¥122,000，HK$ 約 6,200。

退稅 10% 大約 ¥12,200 = HK$620。**值得拎**。

你 11am 朝早到羽田機場，13:00 飛 HK。

11:15am：check-in 前過海關退稅櫃位。

你拎出張包好嘅 LV 護照套。海關員：

> 「Receipt please」

你紙袋裏面有 4-5 張收據混喺行李度。你揾出嚟。

> 「Bic Camera 電子收據 please」

你揾。

> 「Don Quijote 收據請出示」

你揾。

15 分鐘過去。

> 「Don Quijote 嘅 tax-free 程序你已經喺 store 入面 done sales tax exemption — 呢張收據有 stamp。你唔需要喺機場再 process。」

你：「等等⋯⋯即係 Don Quijote 化妝品 ¥18,500 嘅 8% tax exemption already discounted at store？」

> 「Yes」

OK。你 confused 一陣。

> 「LV 嘅收據 please」

你拎出 LV 收據同護照套。

> 「呢個 case 已經 stamp by 百貨店 — 你已經 paid tax-included price，呢度 refund counter 唔處理。」

更 confused 你。「咁⋯⋯邊啲我可以 refund？」

> 「Bic Camera 嘅 electronics 同百貨店 outside-the-counter purchase。但⋯⋯where's the receipt？」

你揾 5 分鐘揾唔到 Bic Camera 收據（已掉去酒店垃圾桶？）。

> 「No receipt = no refund. Sorry.」

你 boarding 12:30。你企喺櫃位，意識到：

- Don Quijote ¥18,500 稅：已經喺店 exempt（你以為要再 refund — 其實已 saved）
- LV ¥38,000 稅：已經喺收據打折
- Bic Camera ¥45,000 稅 = ¥4,500：LOST 因為冇收據
- UNIQLO / GU + 秋葉原：細件購買，門檻可能低於 ¥5,000 最低

呢個遊客退稅混亂嘅根本原因：

- **日本退稅系統有 2 種流程**（店內即時 vs 機場後續 — 你以為要永遠機場處理）
- **收據管理冇有系統嘅計劃**（酒店收件箱 / 行李 / 袋散落）
- **物品類型資格唔同**（液體 / 消耗品 vs 一般商品分類）
- **門檻規則複雜**（每店 ¥5,000+ 一般商品 OR ¥5,000+ 消耗品；消耗品要密封未開）

呢篇用 Claude **trip 前整追蹤器**、**trip 中每日紀錄**、**機場最後索償次序** — 唔再喺櫃位 panic 卡住。

## 跟住做

### 1. Trip 前理解退稅基本

開 Claude，先了解你 trip 嗰個地方嘅稅務系統：

<PromptBlock label="完整 prompt — 日本退稅系統大致理解">
我即將去日本 [N 日] trip，預算購物 [HK$ 金額]。我從未做過退稅。

請整 **日本退稅系統概覽** for HK 遊客：

🇯🇵 **日本消費稅**
- 稅率：10% 標準 / 8% 食物 + 飲品
- 持非日本護照停留少於 6 個月嘅遊客 = 合資格免稅

📋 **2 種流程**（重要區分）：

**Type A：店內即時免稅**（大部分個案）
- Don Quijote / Bic Camera / 大型連鎖店
- 你喺收銀出示護照
- 即時扣稅 = 你 pay 免稅價
- 收據 + 護照喺店蓋章
- **完成 — 唔需要再機場 process**
- 你拎走嘅「消耗品」密封唔可以開（朱古力 / 化妝品 / 健康產品）

**Type B：機場退稅**（少數個案）
- 你喺店已 pay 含稅價
- 收據留低
- 機場海關櫃位出示收據 + 物品
- 退稅做現金 / 信用卡 credit

**大部分店家而家用 Type A**，所以你 trip 中 99% 唔用機場退稅流程。

📏 **門檻 + 限制**

✅ **一般商品** - 電子 / 衫 / 包 / 飾物
- 最低：同一日同一店 ¥5,000+
- 最高：冇限
- 可以開 / 即時用

✅ **消耗品** - 化妝品 / 食物 / 飲品 / 健康用品
- 最低：同一日同一店 ¥5,000+
- 最高：同一日同一店 ¥500,000
- **必須密封不開**直到離開日本
- 海關可能抽查 — 唔可以用 / 食喺日本

❌ **混合唔可以**：
- 同一張收據嘅一般商品 + 消耗品唔可以一齊算門檻
- 例：¥3,000 衫 + ¥3,000 化妝品 = 唔合資格任何一邊 ¥5,000 門檻

📝 **必要文件**
- 護照原本（NOT 副本）
- 你「同名護照」嘅事實對收銀
- 海關申報表（喺某店要填） — 店提供
- 收據：不可掉，亦不可摺（機場可能驗證）

🏷 **唔合資格免稅項目**
- 喺店即時食 / 飲嘅飲品
- 旅行相關開支（酒店 / 交通 / 餐廳餐）
- 二手 / 古董（部分個案）
- 已運去 HK 嘅（你必須親身帶走）

🏬 **常見連鎖店 + 佢哋嘅流程**：
- Don Quijote：Type A（店內，快）
- Bic Camera / Yamada Denki：Type A
- UNIQLO：Type A（NOT 全部店，只有特定分店處理免稅）
- 高端百貨（Mitsukoshi、Isetan、Takashimaya）：Type B（上 4F 客戶服務處理）
- Sezon / Daimaru：大部分 Type B
- 細 boutique / 街市 / 自動售賣機：唔處理免稅（你 pay 含稅價）

💰 **預期 saving**
- 10% on 一般商品（你 pay 90.9% of 標價）
- 8% on 消耗品（你 pay 92.6% of 標價）
- HK$10,000 spending = ~HK$1,000 saved
- 值得時間投資

語氣：清晰區分 2 個流程類型。
</PromptBlock>

Claude 清楚之後，你最重要嘅收穫：**大部分個案已店內處理 — 唔需要機場壓力**。

### 3. 機場最後索償次序（Type B 退稅項目）

最後一日機場準備，避免登機慌張：

<PromptBlock label="完整 prompt — 機場退稅索償最後次序">
我 Day 5 飛 HK，13:00 boarding。我有 [N 張 Type B 退稅收據 = 待 ¥XXX]。

請整 **機場最後索償次序**：

⏰ **時機策略**

**到達機場**：預 11:00（boarding 13:00，2 小時緩衝）
- 1 小時退稅 process
- 30 分鐘 check-in + 安檢
- 30 分鐘登機 + 行去 gate

**次序**：
1. **步驟 1：到達機場（11:00）**
   - 揾「Tax Refund」牌 / 海關櫃位
   - 通常喺國際出境 check-in 後
   - 喺東京機場：羽田 T2 / T3、成田 T1 / T2 都有

2. **步驟 2：海關櫃位排隊（11:00 - 11:30）**
   - 旺時避免（朝早 / 夜晚通常較快）
   - 大旺季節（春節 / 黃金週）排 1+ 小時可能

3. **步驟 3：文件檢查（11:30 - 11:45）**
   - 海關員問：
     - 護照
     - 登機證
     - 收據（按店分類）
     - 物品（必須出示部分）
   - 海關員 stamp 收據後驗證
   - **關鍵**：物品必須未開（特別消耗品）

4. **步驟 4：現金退稅或 credit（11:45 - 12:00）**
   - 揀現金退稅（¥ 即收）或信用卡退稅（較慢，2-6 週）
   - 現金通常稍少（每宗 $1-2 服務費）
   - 信用卡退原本卡

5. **步驟 5：Check-in + 安檢 + 登機（12:00 - 12:45）**

📋 **機場前必要準備**

- [ ] 全部收據收齊 + 分類（Type B 優先）
- [ ] 消耗品物品密封（手提或寄艙）
- [ ] 護照手邊
- [ ] 登機證 print / 電話
- [ ] 計數機 / notes app ready（驗證退稅金額）

🎒 **收納策略**

消耗品物品（化妝品 / 朱古力 / 健康食品）密封：
- 留手提如收細（海關多數要求出示物品）
- 喺 check-in 行李都 OK 如妥善申報
- 最好：30% 喺手提（樣本畀睇）+ 70% 喺寄艙

一般商品（電子 / 包 / 衫）：
- 唔強制出示，但有時海關員會問
- 兩邊手提 + 寄艙都 OK

🚨 **常見拒絕原因**（你要避免）

- ❌ 收據不齊 / 掉 — 不可恢復
- ❌ 消耗品開咗 / 用咗喺日本
- ❌ 物品不在（已用咗）
- ❌ 護照冇帶（海關員無法驗證）
- ❌ 轉機航班嘅登機證不足夠（必須出示直接出日本嘅航班）
- ❌ 超過 6 個月停留 = 不合資格遊客（不是你個案）

⏰ **如排隊長 + boarding 緊**

- Skip 退稅如金額少於 HK$200 + 少於 30 分鐘 boarding 時間
- 計數：HK$200 退稅 vs missed 機 HK$1000+ 重 book
- 嘅取捨唔值

🆘 **後備如拎唔到退稅**

- 部分退稅可以 post-trip 申索 via Global Blue 網站（歐洲個案）
- 日本個案：通常只機場，post-trip 罕有
- 部分店有「郵寄退稅」程式 — check 店內

💰 **退稅預期**

- 現金退稅：即攞 ¥ instant
- 信用卡退稅：appears 2-6 週後
- 唔好預期退稅全 saved ¥（手續費 ¥ 50-200 typical）

📞 **如有問題**：
- 海關員唔肯 stamp → 冷靜問主管
- 完全退稅拒絕 → 要求書面拒簽原因
- 收據錯誤 → 返原本店如時間（24 小時內通常 OK）

語氣：有效率 + 緩衝排隊不確定性。
</PromptBlock>

呢個次序嘅隱藏教訓：**退稅嘅預期價值通常細過你 panic-rushing 嘅成本**。¥500 退稅唔值 missed 機。Trip 準備充分 = 壓力歸零索償。

- 零食 / 手信畀公司
- 最後一刻細件購買

- 中等 saving（時裝 + 包）：HK$5,000（saved ~HK$400-500）
- 手信 + 細件：HK$2,000（無免稅）

## 變化例子

### 變化 1：歐洲 VAT 退稅（Schengen + UK）

歐洲 VAT 同日本完全唔同 system：

<PromptBlock label="變化 1 — 歐洲 VAT 退稅（Schengen）流程">
我去歐洲 [N 國 trip] [N 日]。Schengen + 可能 UK。請整 VAT 退稅框架：

🇪🇺 **歐洲 VAT（Value Added Tax）概覽**

- 稅率：各國唔同（Germany 19%，France 20%，Italy 22%，UK 20% 等）
- 持非歐盟護照嘅遊客 = 合資格
- 退稅通常購買 8-15%（手續費後）

📋 **流程：全部機場級退稅**（vs 日本主要店內）

**步驟 1 — 喺店**：
- 出示護照（必須原本，唔係副本）
- 入錢 ≥ EU 國家最低門檻（通常 €50-175 視國家）
- 店填寫「Tax Free Form」（Global Blue / Planet / 部分店自家）
- 你拎走 form + 收據 + 商品

**步驟 2 — 出境機場（離開 Schengen / 歐盟最後一個歐盟國家）**：
- 海關櫃位印 stamp 喺退稅 form
- 可能驗證物品（特別高價）
- 物品必須未用（特別 luxury / 電子）
- 之後寄 form 去經營者（Global Blue / Planet）

**步驟 3 — 退稅處理**：
- 信用卡退稅：2-6 週後出現喺卡
- 現金退稅：喺機場退稅櫃位（海關 stamp 後）
- 服務費：~3-5% 扣

⚠️ **重要區分**

對於多國 trip：
- 海關 stamp 喺 **離開 Schengen / EU 最後一個 EU 國家**
- 例：意大利 → 法國 → 西班牙 → 飛 HK：海關 stamp 喺西班牙（最後）
- 例：法國 → UK → 飛 HK：喺法國 stamp before UK；UK 另計
- 唔可以 stamp 喺中途轉機

🏬 **常見店免稅**：
- Hermès / LV / Chanel / 高級 luxury 牌子
- 大型百貨店（Galeries Lafayette、KaDeWe）
- 部分餐廳（高端） - 部分餐廳 / 紅酒可免稅
- 大部分連鎖店 opt-in

🚨 **常見拒絕原因**：
- 門檻未達（少於 €175 喺法國例）
- Form 填錯 / 簽名漏
- 物品見有用過（奢侈包用過 visible wear）
- 收據 + form 唔 match
- 護照過期 / trip 期間遺失

⏰ **機場時機策略**

歐洲海關排隊通常比日本長：
- 高峰時段：黃金週 / 暑假 / 聖誕
- 預 2-3 小時機場緩衝
- 同日退稅現金櫃位早過海關關門

💰 **退稅預期**：
- 法國買 €1,000（VAT 20%）：退典型 €166（€1000 / 1.2 × 0.2 = €166.67，減 3-5% 費 = €130-150 淨）
- 淨退：購買 13-15%

🏛 **特定國家奇癖**：
- **France**：PABLO 機（比櫃位快，24/7 有）
- **Germany**：海關 stamp 先，分開退稅櫃位
- **Italy**：最長排隊，預緩衝
- **UK**（脫歐後）：不再 Schengen，分開免稅系統，更嚴

📦 **高價購買（超過 €1,000）**

建議：
- 用信用卡（保險 cover）

- 留盒 + 標籤 + 防塵袋
- 唔好運去 HK（必須親身帶）

## 一個思維框架

遊客退稅嘅預期價值通常被高估。

- 5 日 trip 平均消費：HK$3,000-5,000
- 退稅 8-12% = HK$240-600
- 機場 process 時間 30-60 分鐘排隊

## 情境

你開咗 Tinder / Bumble / Hinge 三個月，swipe 過幾百個人，配對到嘅人傾兩三句就冷。你打開自己 profile 望多次：

> 「Love travelling, good food and sunset 🌅
> Family is everything ❤️
> Looking for someone genuine」

⚠️ 呢一段你會喺**九成 HK profile** 入面見到。可能加多隻狗 emoji、加多句「No drama please」、加多張西貢行山相。演算法對你冇偏見，但你嘅 profile 對演算法嚟講係**雜訊**——同其他幾百個 profile 撞晒，所以推畀你嘅人都係同一批慣性 swiper。

問題唔係你唔有趣，係你**將自己壓縮成最大公約數**。Profile 越籠統，引嚟嘅人越隨機，傾偈越 hea，sunk cost 越高。

呢篇用 Claude 重寫你 profile：唔係幫你扮 cool，係幫你講清楚你**真實**嘅身份訊號，拿揑「畀人睇到」同「篩走唔啱嘅人」之間嘅取捨，連第一條 opening message 都對應返。

## 跟住做

### 1. Brain dump：過去 30 日你真實做過嘅嘢

唔好諗 profile，唔好諗對住邊啲人。畀自己 15 分鐘喺 Apple Notes 列：

- **過去 30 日週末你真實做咗咩**（具體：「上禮拜六去咗深水埗食潮州打冷、之後去 Tai Kwun 睇個展、夜晚返屋企煲 Severance」，唔係「I love exploring HK」）
- **你最近一齊食飯、飲嘢嘅人**（同邊個傾偈會發吽哣、同邊個會傾到凌晨）
- **你電話 screen time 頭三位嘅 app**（誠實啲，唔好過濾）
- **你最近主動洗錢嘅 3 樣嘢**（一場 gig？一對波鞋？課程？）
- **你最近一次同人爭拗到面紅嘅話題**
- **你私底下嗰啲跟主流唱反調嘅睇法**（例：「我覺得西貢已經 over-rated」、「Sourdough 熱潮係 marketing」）
- **你完全唔接受嘅 dealbreaker**（具體：「對方覺得睇有字幕嘅戲好辛苦」）

呢個 brain dump 唔係寫畀人睇，係**畀 Claude 嘅原材料**。你寫得越誠實、越具體，個 output 越好。

### 2. 用呢個 prompt：寫 3 個 positioning draft

<PromptBlock label="完整 prompt — 由 brain dump 寫 3 個 dating profile draft">
我想重寫我嘅 dating app profile。我用緊 [揀 — Tinder / Bumble / Hinge / Coffee Meets Bagel]。

以下係我嘅原始 brain dump（過去 30 日真實做過、睇過、買過、爭拗過嘅嘢）：
[paste 你嘅 brain dump]

我嘅基本資料：
- 年齡：[填]
- 性別 + 性取向：[填]
- 而家喺 HK 邊區住：[填]
- 我搵緊：[揀 — 認真長期 / 開放 explore / 純結識朋友 / 未諗清楚]
- 我嘅 hard dealbreaker（最多 2 個）：[填]

請幫我寫 3 個唔同定位嘅 profile draft，每個 250 字以內（Bumble 限 300、Hinge 用 prompt 答題格式、Tinder 短啲），要做到：

**定位 A — 具體日常派**
- 由我 brain dump 入面揀 2-3 個最具體嘅細節
- 唔好出「I love food / travel / family」呢類人人都講得出嘅嘢
- 目標：對方睇完知道呢個禮拜六我大概會做咩

**定位 B — 觀點 / 品味派**
- 帶出我跟主流唱反調嘅睇法、或者強烈嘅品味（書、戲、音樂、食、城市觀察）
- 唔好攻擊性，但要有立場
- 目標：對方睇完知道我點睇世界，唔係淨係知道我做咩

**定位 C — 自覺幽默派**
- 用一個自嘲式 hook 開頭（唔好自貶到令人尷尬，係輕微 wink）
- 中段落返地，扣住一個真實細節
- 目標：對方睇完笑一下，但唔覺得我喺度扮棟篤笑

每個 draft 請額外標：
1. **「畀人睇到」vs「篩走人」評估**：呢個版本會吸引到 / 篩走邊類人？（具體講個取捨）
2. **太具體嘅風險**：邊一句可能令個 dating pool 太窄？
3. **太空泛嘅風險**：邊一句仲係空泛、可以再 sharp 啲？

**避免：**
- 「Looking for my person / partner in crime / plus one」
- 「No drama / No games / No hookups」（呢類負面框架反而會吸引到你想避開嘅人）
- 「Ask me anything」（懶嘅 hook）
- Emoji 多過 3 隻
- 列 5 樣 hobby 但每樣得一個字
- 「Fluent in sarcasm / sushi / movies」呢類 2017 年 template

語氣：HK 廣東話思維、英文夾雜 OK（Hinge / Bumble 以英文為主嘅場合），唔好成段中文硬譯到生晒。
</PromptBlock>

### 3. Photo selection：Claude 唔睇相，但可以教你揀

Claude 睇唔到你張相，但你可以用佢幫你審視成組相。將你 6 張相每張用一句話描述（「西貢行山自拍 + 太陽眼鏡」、「同朋友喺 LKF 飲嘢嘅合照」、「喺廚房整意粉嘅生活照」），交畀 Claude，問：

> 「呢 6 張相喺講緊邊 3 個故事？有冇相係多餘（同一個故事重複講）？我 brain dump 入面邊個身份訊號，欠咗一張對應嘅相？」

揀相邏輯：**6 張相應該涵蓋 3 個唔同面向**——一張清楚見樣（單獨、平視鏡頭）、一張活動 / 場景（show 你真實做嘅事）、一張有朋友佐證（同朋友影、但你係焦點）。第四到第六張係加分訊號——寵物、旅行、整嘢食、運動。

⚠️ HK 常見錯誤：6 張相全部都係合照（人哋 swipe 緊估邊個先係你）、或者全部西貢 / 日本 / 台北旅行相（畀人感覺你淨係識去嗰 3 個地方）。

### 4. 第一條 opening message：對應對方嘅 profile

Profile 改完，**第一條 message 都要對應**。唔好慣性丟一句「Hey :)」或者「How's your weekend?」。

<PromptBlock label="Opening message 模板">
我配對到一個人。對方 profile 寫嘅嘢（重點）：
[列 3 個對方 profile 入面嘅具體細節，例：「鍾意 vinyl」、「最近睇嘅書係 Pachinko」、「養咗一隻三色貓」]

我嘅 profile：
[paste 你嘅最終 profile]

我嘅目的：希望換到一條**有內容嘅回覆**，唔係「Haha thanks」。

請寫 3 條 opening message option，每條：
1. **第一句**——對應對方 profile 入面一個具體細節（唔係讚靚 / 讚 cool）
2. **第二句**——一條具體問題、或者一個輕輕拋出嘅觀點，邀請對方反應
3. **長度**：2-3 句，唔好寫成一大段
4. **避免**：「I noticed you...」開頭、「You seem...」加形容詞、過分讚美

3 個 option 要走唔同角度（一個好奇、一個輕輕挑機、一個分享返你自己一個類似嘅細節）。
</PromptBlock>

轉去 WhatsApp 嘅原則：傾過 5-8 條 message、雙方有來有往，你就可以提「不如出嚟飲杯嘢」。**唔好喺 dating app 入面傾足兩個禮拜**——對方嘅勢頭會散、你嗰份 swipe 紅利會用晒。

## 變化例子

### 變化 1：30+ 專業人士搵長期關係版

<PromptBlock label="變化 1 — 30+ 認真長期關係版">
我 30 至 40+，搵長期關係（可能會結婚 / 同居）。我嘅 brain dump：
[paste]

我嘅生活已經安頓落嚟（有事業、有日常 routine、喺 HK 有屋住或者租緊穩定嘅地方）。我搵嘅人都係 30+、有自己生活、唔係想搵人去救、又唔係想救人。

請寫 profile，要做到：
1. **唔扮後生** —— 唔用 Gen-Z 潮語、唔扮夜蒲、唔列一堆未做過嘅 bucket list
2. **顯出安頓但唔悶**：具體嘅日常（「Saturday morning 多數喺 Kennedy Town 嗰間 cafe 同朋友食 brunch」）反而比「I love adventure」更加吸引到成熟嘅人
3. **講清長期意向，但唔好嚇親人**：唔係寫「Looking for marriage / babies」（呢類擺到太前），而係用「Past the casual dating phase」呢類含蓄嘅訊號
4. **一個展露真我嘅訊號**：例如承認一個小缺點、或者一樣自己正在改善嘅嘢

避免：列職位 / 公司、講收入、列 Master degree（呢啲會吸引錯人）。
</PromptBlock>

### 變化 2：International / English-speaking, expat-friendly

<PromptBlock label="變化 2 — English-speaking expat-friendly 版本">
我嘅 dating pool 包括 expat、returnee、ABC、講國際語境嘅 HK 人。我用 English profile（Hinge / Bumble），但我嘅文化身份係 HK 人 / 喺 HK 長大。

我嘅 brain dump：
[paste]

請寫一個 English profile，要做到：
1. **扎根喺 HK，但唔係遊客視角**：唔好寫「I love dim sum and Star Ferry」（本地人唔會咁講、expat 覺得 try-hard）；要寫只有住喺度先講得出嘅細節（「The one bakery in Sai Ying Pun I keep going back to」）
2. **語言層次**：以英文為主，但自然咁帶一兩句粵語（例：「going for a 飲茶」、「post-work 落樓上 bar」）—— 帶出雙語身份，而唔係刻意表演自己係香港人
3. **拿揑對方對文化嘅熟悉程度**：你期望對方識 HK 識到咩程度？（新落地 vs 住咗 5 年以上 vs returnee）用一句含蓄帶出
4. **避免**：「Born and raised in HK but global citizen」（空泛）、列一大堆國家旅行清單（expat profile 過度老套）

語氣：踏實、好奇、唔好刻意扮國際人。
</PromptBlock>

### 變化 3：LGBTQ+ HK context

<PromptBlock label="變化 3 — LGBTQ+ HK context">
我嘅 dating app：[Tinder / Hinge / HER / Grindr / 其他]
我嘅 identity：[gay / lesbian / bi / pan / queer / trans / non-binary，或者你自己用得舒服嘅講法]
我嘅出櫃狀態：[完全出咗櫃 / 選擇性出櫃 / 低調]

我嘅 brain dump：
[paste]

HK LGBTQ+ 語境要特別留意嘅嘢：
- 圈子細、熟面孔多，profile 越空泛就越容易淹沒喺人海入面
- 部分人未完全出櫃（對屋企 / 同事），但又想搵到一段認真嘅關係
- Femme / masc / fluid 嘅形象訊號要清楚，但唔好被人簡化成一個 type 標籤

請寫 profile，要做到：
1. **身份訊號清楚但唔係標籤堆砌**：唔需要喺第一句塞 5 個 identity 字眼；用具體嘅文化 reference / 圈子細節自然帶出
2. **出櫃狀態點處理**：如果係選擇性出櫃，用含蓄嘅暗示（「Not on socials in that way」），唔好解釋得太白
3. **避開 LGBTQ+ profile 嘅慣性老套**：「Looking for something real in a sea of hookups」、「No more 1am `hey` messages」（呢類反應式框架反而會困住你，令成個語氣變負面）
4. **圈內 vs 主流**：你想吸引嘅主要係圈內人、定係開放跨圈子？（影響 reference 同 emoji 點揀）

語氣：自信、具體、有立場但包容。
</PromptBlock>

> 提你一句：dating app 嘅成功指標唔係 swipe right 率，而係**你配對到嘅人有冇質素**。Profile 寫得籠統，inbox 會塞爆但全部都係死胡同；profile 寫得真實又具體，inbox 數量少，但每個都接近你想要嘅人。
>
> Claude 嘅優勢唔係幫你扮另一個人，而係幫你**將本來淨係喺你腦入面嗰份自我認識，化成對方 5 秒就讀得明嘅訊號**。你嘅 brain dump 越誠實，個 profile 越似你；越似你，越篩到啱嘅人。
>
> 真誠又具體，永遠贏靚但籠統。

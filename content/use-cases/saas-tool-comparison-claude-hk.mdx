## 情境

你管緊一個 5 人 startup / 創意工作室。你訂閱緊：

- Notion ($10/月/人)
- Asana ($15/月/人)
- Linear ($10/月/人 — 工程師專用)
- Figma ($15/月/人)
- Slack ($8/月/人)
- 各人自己嘅訂閱（ChatGPT、Claude Pro、Canva⋯⋯）

5 人 × 6 個 SaaS = 每月燒 $300+。而且**功能重疊**（Notion 部分覆蓋到 Asana、Asana 部分覆蓋到 Linear）。

呢篇用 Claude 幫你做策略性對比 + 簡化工具組合。

## 跟住做

### 1. 列出你而家用緊 + 每個工具嘅實際用途

每個工具：
- 每月開支（每人收費 × 人數）
- 主要用途（你哋實際用嚟做乜，唔係功能清單）
- 邊個 team 用、邊個 team 唔用
- 接受度（80% 每日用？30%？）

冇呢個老實評估，AI 出嘅結果會變行貨「呢個工具強 X、嗰個強 Y」嘅推銷分析。

### 2. 用呢個 prompt

<PromptBlock label="完整 prompt — SaaS 工具組合審視">
我哋有 [N 人] 團隊，訂閱緊以下 SaaS：

[列出每個工具 + 收費 + 實際用途 + 採用率]

我哋嘅優先次序（排好）：

1. [例：團隊對齊 + 項目可見度]
2. [例：個別任務跟蹤]
3. [例：文件 / 知識庫]
4. [例：溝通]

請整 4 個部分：

**Part 1：功能重疊審視**
邊兩個工具我哋用途上重疊？具體：
- 我哋而家用 Notion 做 X，用 Asana 做 Y——其實 [工具] 一個就 cover 到兩樣
- 遷移成本：由 [工具 A] 走去 [工具 B] 邊樣難（範本 / 數據匯出 / 團隊重新培訓）

**Part 2：推薦簡化方案**
我哋理想嘅工具組合（揀 3-4 個工具最多，覆蓋上面優先次序）+
- 邊個工具保留
- 邊個工具刪走
- 邊個工具搬走（A → B）

**Part 3：成本影響**
我哋簡化之後每月嘅成本
年度節省
回本時間（包遷移時間成本）

**Part 4：風險 + 緩解**
- 採用風險：團隊成員抗拒轉換
- 功能缺口：刪走工具之後缺嘅具體功能點繞過
- 遷移時間表：建議分階段（唔好一日全部改）

語氣：有立場（畀推薦，唔好「it depends」嘅模稜兩可）。
</PromptBlock>

### 3. 同團隊傾 + 小規模試用

唔好單方面決定。將 AI 嘅建議帶落團隊週會：

- 提出成本考量
- 討論簡化方案
- 試用：揀一個小組試 2 個禮拜新工具組合
- 評估接受度 + 阻力

呢類工具整合通常需要 6-8 個禮拜完成。唔好急。

## 變化例子

### 變化 1：你係單人開發者 / freelancer

<PromptBlock label="變化 1 — 單人工具組合審視">
我係單人（[freelance / 開發者 / 創作者]）。
我嘅 SaaS 訂閱：[列出]

我嘅需要：
1. [例：客戶項目管理]
2. [例：個人任務清單]
3. [例：筆記 + 參考資料]

單人情況同團隊唔同——阻力低 / 決定快 / 採用率 100%。

請：
1. 推薦一個最簡工具組合（2-3 個工具最多）
2. 一個「all-in-one」嘅單一工具選擇（Notion / Obsidian / 純 .md file in iCloud / Capacities）
3. 為單人情境推薦邊種取向（最簡組合 vs all-in-one）
4. 每月成本對比
</PromptBlock>

### 變化 2：用 Claude 取代某個 SaaS

<PromptBlock label="變化 2 — Claude 直接取代 SaaS">
我考慮用 Claude Chat / Project 取代 [某個 SaaS]。

例如：
- Notion AI → Claude Chat 直接寫 / brainstorm
- Grammarly → Claude 修飾文字
- 寫社交媒體 caption 嘅工具 → Claude 一條 prompt

請：
1. 對比 Claude vs 該 SaaS 嘅具體用途
2. 取捨：Claude 嘅好處（深度）vs 該 SaaS 嘅好處（介面 / 工作流程）
3. 決定框架：邊類任務 Claude 真係 work，邊類仲係要保留個 SaaS
4. 推薦：刪走邊個 SaaS 最明顯有得贏，邊個邊際省錢唔值得 disruption
</PromptBlock>

## 一個思維轉變

每年做一次 SaaS 工具組合審視。唔需要徹底大改造——目標係**搵出重疊 + 取消已經冇用嘅訂閱**。

平均情況：HK 單人 / 細團隊每年慳 $1,500-3,000 嘅月度燒錢。呢個係「隱形加薪」——同樣生產力，少咗固定開支。

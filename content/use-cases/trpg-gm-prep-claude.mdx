## 情境

星期五晚 9 點，聽日下晝 2 點開團。

4 個玩家。第 5 場。佢哋上次喺一間酒館度遇到一個神秘貴族，諗住跟住接佢嘅任務。你之前準備好嘅劇情係：貴族派佢哋去鎮外山洞搵返件法器。

但你打開 group chat：

- 玩家 A：「我想查下嗰個貴族嘅底細」
- 玩家 B：「我去酒館問下其他酒客有冇聽過呢個人」
- 玩家 C：「我順便買隻馬」
- 玩家 D：「我問酒館老闆有冇關於山洞嘅傳聞」

⋯⋯你哋仲未行去你準備好嘅山洞。

呢個就係 TRPG GM 永遠要交嘅準備稅：**玩家會行去你冇準備嘅方向**。你可以：

- 硬拉佢哋返正軌（即係 railroad，破壞玩家自主權）
- 即場硬作（質素下降，前後容易出現矛盾）
- 預先準備 30 個應變方案（時間負擔太大）

呢篇教你用 Claude 喺**開團前 1 個鐘**，準備一啲模組化嘅積木——NPC、場景、劇情誘餌（hook）——等你即場可以隨時拼湊，唔使硬作。

## 跟住做

### 1. 餵世界觀背景畀 Claude

第一步永遠係：**Claude 唔知你個跑團世界係點。**唔好一開波就叫「畀個貴族 NPC 我」。先貼上你個世界觀設定（如果你冇，就寫個 200 字摘要）：

<PromptBlock label="完整 prompt — 跑團背景設定">
我做緊 GM 跑緊一個 [系統名，例：D&D 5e / Call of Cthulhu / Coriolis / 自家系統]，第 [N] 場，4 個玩家。

📚 **世界觀設定 (Setting)**：
- 世界：[簡單描述，2-3 句]
- 時代 / 科技水平：[例：中世紀奇幻 / 1920 年代宇宙恐怖 / 科幻探索]
- 我哋而家喺：[城市 / 地區 / 星球 / 太空站名 + 簡單描述]

🎭 **氣氛 (Tone)**：
- [嚴肅 / pulp 雜誌風 / 恐怖 / 政治角力 / 搞笑 / 混合]
- 玩家期望：[多戰鬥 / 多角色扮演 / 懸疑 / 探索 / 混合]

👥 **玩家角色 (PC)**：
- A 玩 [職業/定位/概念]，動機：[]
- B 玩 ...
- C 玩 ...
- D 玩 ...

🧠 **上次結局**：
- 場景發生喺：[]
- 玩家最後決定：[]
- 未解嘅伏線：[]
- 我預咗想推進去：[]

🚨 **可能會出事嘅應變情況**：
我預計玩家會做 A，但**可能會做 B / C / D**：
- B = []
- C = []

下面我會叫你幫我準備唔同嘅模組積木 — 請記住呢個設定 + 氣氛。回覆「收到」我就開始。

語氣：好似一個玩 TRPG 嘅 GM 朋友同你一齊度橋，唔係掟啲行貨奇幻範本畀我。
</PromptBlock>

⚠️ **貼士**：將呢個基本 prompt 存做 Claude Project 嘅系統指令，之後成個跑團活動都唔使重新貼一次。

### 2. 即場準備 5 個模組積木

收到 Claude「收到」之後，逐個要求：

**積木 1：3 個 NPC**

<PromptBlock label="積木 1 — 3 個模組化 NPC">
畀我 3 個喺呢個地區嘅 NPC。每個 NPC 我都可以**擺入任何場景**（酒館 / 街市 / 政治會議 / 意外現場），所以唔好綁死設定。

每個 NPC 要有：

- **名** + 種族 / 職業暗示
- **一句話點認得佢**（外表 / 說話小動作 — 例：「成日清喉嚨」/「隻戒指喺手指滑上滑落」）
- **表面欲望 (Want)** — 想要乜
- **內在需求 (Need)** — 真正缺乜（隱藏嘅，可能同表面欲望有衝突）
- **秘密** — 1 個如果玩家查深啲會搵到嘅嘢
- **劇情誘餌 (Plot hook)** — 點樣同玩家扯上關係（任務 / 煙幕 / 聯絡人）
- **演繹方式 (Voice)** — 我點樣扮演佢（口音 / 語速 / 用詞）

唔好出行貨嘅「笠住冷帽嘅高大神秘陌生人」。要有具體質感。
</PromptBlock>

**積木 2：1 個突發場景**

<PromptBlock label="積木 2 — 1 個突發場景（玩家走偏時用）">
畀我 1 個動態場景，當玩家偏離我嘅劇情時可以即場擺出嚟。

要求：
- **設定**：3-4 句，逐漸升溫嘅張力（唔係隨機遇敵，要有原因發生）
- **3 個介入點**：玩家可能會觸發嘅 3 種方式（被動觀察 / 主動介入 / 意外捲入）
- **2-3 個 NPC**（重用積木 1 或新角色，你自己決定）
- **目標 / 賭注**：場景解決咗有咩結果，唔解決有咩後果
- **轉折觸發點**：場景中段如果玩家做咗 X，局勢會突變成 Y
- **出口**：場景完咗之後帶出 3 個唔同嘅下一場景方向（唔好變掘頭路）
- **連返我主線劇情**：場景入面埋一條線索接駁主線
</PromptBlock>

**積木 3：1 場戰鬥遭遇**

<PromptBlock label="積木 3 — 1 場戰鬥遭遇（平衡 + 有趣）">
畀我 1 場戰鬥遭遇，[系統嘅平衡敵人預算 — 例：D&D 5e CR 5 對 4 人 5 級隊伍]。

要求：
- **環境地形**：3 個會影響戰術嘅明顯特徵（柱、水池、懸崖邊、燃燒物等）
- **敵人組合**：唔好 4 隻同數值嘅怪 — 要有角色多樣性（打手 / 控場 / 威脅核心 / 輔助）
- **戰術觸發點**：第 X 回合或 boss 血量跌到 50% 時，敵人會做 XX（改變戰術）
- **逃跑 / 投降選項**：boss 血量低時，玩家可以揀放過 / 殺死 / 盤問，每個分支有後果
- **戰利品 + 線索**：打完之後玩家可以搜到嘅 3 樣嘢，其中 1 樣連返去主線
</PromptBlock>

**積木 4：3 個傳聞 / 街坊八卦**

<PromptBlock label="積木 4 — 3 個玩家喺酒館 / 街市可能聽到嘅傳聞">
畀我 3 個玩家可以喺社交場合聽到嘅傳聞。

每個要有：
- **內容**（1-2 句）
- **真實性**：真 / 大部分真 / 煙幕 / 完全係大話
- **來源提示**：邊個階層 / 行業先會傳呢個傳聞
- **連接口**：呢個傳聞同邊條伏線連住（主線 / NPC 背景 / 支線任務種子）

要混合氣氛：1 個日常（食物加價）/ 1 個中等風險（某貴族失蹤）/ 1 個高風險（宇宙級 / 政治危機）。
</PromptBlock>

**積木 5：應付玩家出怪招嘅預備**

<PromptBlock label="積木 5 — 玩家典型出軌嘅應變">
我嘅玩家以前試過呢類出軌行為：
- 玩家 A：成日想引誘任何 NPC
- 玩家 B：成日想查底細 + 查記錄
- 玩家 C：成日要買騎呢嘢
- 玩家 D：成日想用魔法 / 駭客手段抄捷徑

請為每位準備一個**一句話回應**，等我可以即場攞出嚟用：

- A 嘅目標：NPC X 嘅反應（奉承 / 拒絕 / 接受但有附帶條件）
- B 嘅查底細：佢搵到 [3 條資料 — 2 條日常 1 條關劇情事]
- C 嘅騎呢道具：店主可能會有 [3 件唔尋常嘅道具]，每件 1 句描述 + 暗藏劇情誘餌
- D 嘅魔法捷徑：呢個地區嘅魔法有 [限制 / 代價 / 副作用]
</PromptBlock>

呢 5 舊積木通常 15-30 分鐘搞掂晒。你印出嚟，或者擺喺隔離部螢幕，玩緊嗰陣隨時參考。

### 3. 玩緊嗰陣即場用

玩家做預期之外嘅事？打開你份準備文件：

> 玩家：「我去查下個貴族底細」
> 你：（睇積木 5 - B 嘅回應）「⋯⋯經過半日四處打聽，你搵到佢叫 Lord Aldric Vance，3 年前由北部公爵領地嚟。佢老婆死於『不明病』，之後佢留咗一段時間先返首都。」（呢句包含 1 個劇情線索 = 不明病）

唔使硬作，流程都順暢。

## 變化例子

### 變化 1：玩完總結 → 下次準備嘅種子

玩完一齊食晚飯前 / 執嘢走人之前，趁記憶猶新：

<PromptBlock label="變化 1 — 玩完總結 + 下次準備種子">
我啱啱玩完第 [N] 場。發生咗：

[5-10 句講今次結局]

玩家最後決定：[]
未解伏線：
- A：[]
- B：[]
- C：[]

NPC 出場：[列出，註明 生/死/失蹤]

請：

**總結（畀玩家睇嘅）**
2-3 段敘事總結，我可以 send 落 group chat 等大家下次玩之前溫故知新。語氣係小說敘事，唔係 GM 筆記。

**GM 內部追蹤表**
- 未解伏線 + 劇情優先次序（高 / 中 / 低）
- NPC 狀態追蹤（邊個生存, 邊個嬲咗玩家, 邊個欠玩家主意）
- 玩家攞咗但未用嘅道具 / 線索，下次點樣提返起
- 玩家角色發展誘餌 — 邊個玩家嘅個人成長線應該下次推動

**下次準備清單**
我下次準備應該優先搞：
1. [最逼切嘅應變情況]
2. [下一個主要場景 / 遭遇]
3. [邊個玩家嘅角色發揮時刻]
</PromptBlock>

### 變化 2：玩家角色背景故事擴充

新玩家加入 / 開新角色：

<PromptBlock label="變化 2 — 玩家角色背景故事深化">
我嘅玩家寫咗以下角色背景故事：

[貼上]

我嘅世界觀設定 / 氣氛提示：[1-2 句]

請：

**抽出背景故事入面嘅劇情誘餌**
列出 3-5 個元素我可以喺跑團入面引用 / 呼應（NPC、地點、過去事件、未解疑團）

**性格一致性檢查**
有冇地方自相矛盾？有冇可能演變出角色缺點？

**3 個「呢個角色嘅未解問題」**
玩家自己未必意識到嘅背景故事空白，我可以慢慢揭曉

**第一場嘅個人誘餌**
1 個具體時刻 / NPC 遭遇，我可以擺喺第 1-2 場，等個角色即刻覺得自己「屬於呢個世界」

**長遠發展種子**
我可以埋下嘅 5-10 場個人發展線，呼應背景故事嘅未解決張力
</PromptBlock>

### 變化 3：玩家做意想不到行動嘅即場裁決

玩緊嗰陣，玩家做咗某啲機制上模稜兩可嘅事：

<PromptBlock label="變化 3 — 即場規則 / 公平裁決">
我玩 [系統]。玩家做緊：

[描述行動 — 例：用「法師之手」捉住 boss 把武器，再掟落熔岩]

我而家卡住喺：[邊度模稜兩可]

我嘅傾向係 [容許 / 限制 / 部分容許]，但驚 [副作用 — 例：開咗壞先例 / 破壞 boss 戰 / 打破行動經濟學]。

請：

**忠於系統嘅裁決**
按 [系統] 嘅字面規則 (RAW)，呢個行動點樣裁決

**公平嘅即興裁決**
如果字面規則唔清楚 / 太嚴，建議一個公平嘅「家規 (house ruling)」，符合系統精神

**先例風險**
如果我容許呢樣嘢，玩家下次會點樣濫用？我而家有冇辦法界定個裁決唔好開呢道門

**對目前進度嘅即時影響**
依個行動成功 / 失敗之後，場景點樣推進

語氣：熟讀規則嘅玩家 + 玩過好多年嘅 GM。
</PromptBlock>

## 一個思維框架

GM 嘅工作 80% 係 **準備**——剩低 20% 係 **即興發揮你冇準備嗰啲嘢**。

Claude 唔係取代你準備——好嘅 GM 永遠都係**對自己個跑團世界有感情投入**嘅人。Claude 嘅角色係**將準備時間由 3 個鐘變返 1 個鐘，留多 2 個鐘畀你同班朋友真正去玩**。

香港玩 TRPG 嘅人通常返工返到爆，加埋班朋友散落喺世界各地，一個月夾到一次時間已經好彩。每次準備質素提升 30%，成班人嘅享受程度會複利咁累積上去。

最後提一句：**唔好將 Claude 生成嘅 NPC / 場景一字不改咁照用**。睇佢嘅原始輸出，揀啲有火花嘅元素，**再加你自己嘅細節**（你哋班朋友之間嘅內部笑話、上次玩嘅回憶、香港文化共鳴例如「個貴族啲口音似官恩娜」）。AI 輸出 + 你嘅個人特色 = 你獨一無二嘅跑團世界。
